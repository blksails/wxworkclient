# 企业微信 Client 代码生成器

根据 `docs/apis/apis.json` 自动生成 Go client 代码，包括类型定义和 client 方法。

## 使用方法

### 1. 基本用法

```bash
# 生成所有 API
go run cmd/gencode/main.go

# 指定输入和输出路径
go run cmd/gencode/main.go \
  -input docs/apis/apis.json \
  -output generated \
  -package wxwork

# 只生成前 10 个 API（用于测试）
go run cmd/gencode/main.go -limit 10
```

### 2. 参数说明

- `-input`: APIs JSON 文件路径（默认：`docs/apis/apis.json`）
- `-output`: 生成代码输出目录（默认：`generated`）
- `-package`: 生成代码的包名（默认：`wxwork`）
- `-limit`: 限制生成的 API 数量，0 表示全部生成（默认：`0`）

### 3. 生成的文件

- `generated/types_generated.go`: 所有请求和响应类型定义
- `generated/client_generated.go`: Client 方法实现

## 示例

假设 `apis.json` 中有如下 API：

```json
{
  "id": "90194",
  "api_name": "user/create",
  "method": "POST",
  "api_url": "https://qyapi.weixin.qq.com/cgi-bin/user/create",
  "request_params": [
    {"name": "userid", "type": "string", "required": true},
    {"name": "name", "type": "string", "required": true}
  ],
  "response_params": [
    {"name": "errcode", "type": "int"},
    {"name": "errmsg", "type": "string"}
  ]
}
```

生成的代码：

```go
// types_generated.go
type UserCreateRequest struct {
    UserID string `json:"userid"`
    Name   string `json:"name"`
}

type UserCreateResponse struct {
    CommonResponse
}

// client_generated.go
func (c *Client) UserCreate(req *UserCreateRequest) (*UserCreateResponse, error) {
    var query url.Values
    if c.AccessToken != "" {
        query = url.Values{}
        query.Set("access_token", c.AccessToken)
    }
    return c.impls.UserCreate.Do("POST", "/cgi-bin/user/create", req, query)
}
```

## 特性

1. **自动类型推断**：根据字段名和类型描述推断 Go 类型
2. **命名规范**：自动转换为 Go 风格的命名（PascalCase）
3. **响应类型继承**：自动继承 `CommonResponse`
4. **注释生成**：保留原始 API 文档的描述信息
5. **可配置限制**：支持只生成部分 API 用于测试

## 注意事项

- 生成的文件头部包含 `// Code generated by gencode. DO NOT EDIT.`
- 只会处理包含有效 `api_name` 和 `method` 的 API
- 类型推断可能需要手动调整复杂类型
