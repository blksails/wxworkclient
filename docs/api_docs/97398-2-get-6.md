# 获取获客链接详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97398](https://developer.work.weixin.qq.com/document/path/97398)
- **文档 ID**: `97398`
- **API 名称**: `get`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/customer_acquisition/get`
- **分组信息**: 第 2 个接口，共 5 个

## 接口描述

企业可通过此接口根据获客链接id获取链接配置详情。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| link_id | string | 是 | 获客链接id。需要是当前应用创建 |

### 请求示例

```json
{
   "link_id":"LINK_ID_AAA"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| link.link_name | string | 获客链接的名称 |
| link.url | string | 获客链接实际的url |
| link.create_time | int32 | 创建时间 |
| link.skip_verify | bool | 是否无需验证，默认为true |
| link.mark_source | bool | 是否标记客户添加来源为该应用创建的获客链接; 仅对「营销获客」应用生效 |
| range.user_list | array | 该获客链接使用范围成员列表 |
| range.department_list | array | 该获客链接使用范围的部门列表 |
| priority_option.priority_type | int32 | 优先分配类型，1-全企业范围内优先分配给有好友关系的；2-指定范围内优先分配有好友关系的 |
| priority_option.priority_userid_list | array | priority_type为2时的指定成员列表 |

### 响应示例

```json
{
  "errcode": 0,
  "errmsg": "ok",
  "link":
	{
		"link_name":"LINK_NAME",
		"url":"https://work.weixin.qq.com/ca/xxxxxx",
		"create_time":1672502400,
		"skip_verify":true,
		"mark_source":true
	},
	"range":
	{
		"user_list":["rocky","sam"],
		"department_list":[1]
	},
	"priority_option":
	{
		"priority_type":2,
		"priority_userid_list":["tom","lisi"]
	}
}
```
