# 应用邮箱接收邮件事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97517](https://developer.work.weixin.qq.com/document/path/97517)
- **文档 ID**: `97517`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/mail/receive_email_callback`

## 接口描述

当应用邮箱收到邮件后，触发该事件。

## 请求信息

### 请求示例

```xml
<xml>
   <ToUserName><![CDATA[toUser]]></ToUserName>
   <FromUserName><![CDATA[sys]]></FromUserName>
   <CreateTime>1668831860</CreateTime>
   <MsgType><![CDATA[event]]></MsgType>
   <Event><![CDATA[app_email_change]]></Event>
   <ChangeType><![CDATA[receive_email]]></ChangeType>
   <Amount><![CDATA[2]]></Amount>
</xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| ToUserName | string | 企业微信CorpID |
| FromUserName | string | 此事件该值固定为sys，表示该消息由系统生成 |
| CreateTime | uint32 | 消息创建时间，unix时间戳 |
| MsgType | string | 消息类型，此时固定为：`event` |
| Event | string | 事件类型，此时固定为：`app_email_change` |
| ChangeType | string | 事件类型，此时固定为：`receive_email` |
| Amount | string | 应用邮箱当前的新邮件数 |
