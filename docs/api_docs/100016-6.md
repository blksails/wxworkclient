# 设置公钥

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100016](https://developer.work.weixin.qq.com/document/path/100016)
- **文档 ID**: `100016`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/set_public_key`

## 接口描述

授权完成后需要调用接口上传公钥(RSA-2048)。注意：设置公钥之后，消息才开始存档。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| public_key | string | 是 | 开发者为该企业生成的公钥，可以参考下方使用openssl命令生成 |
| public_key_ver | uint32 | 是 | 公钥对应的版本号，当重复调用该接口更换公钥时要求比旧公钥版本号大 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
