# 获取观看直播统计V2 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97132](https://developer.work.weixin.qq.com/document/path/97132)
- **文档 ID**: `97132`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/living/get_watch_stat_v2?access_token=ACCESS_TOKEN`

## 接口描述

通过该接口可以获取所有观看直播的人员统计

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| livingid |  | 是 | 直播的id |
| 自建应用 | 自建应用 | 否 | 配置到「上课直播 - 可调用接口的应用」和「家长可以使用的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 具有「直播」和「家校沟通」权限 |
| 第三方应用 | 第三方应用 | 否 | 具有「直播」和「家校沟通」权限 |

### 请求示例

```json
{"livingid":"livingid1","next_cursor":"NEXT_CURSOR"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| next_cursor |  | 上一次调用时返回的next_cursor，初次调用可以填"0" |
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| has_more |  | 是否结束。1：表示还有更多数据，需要继续拉取，0：表示已经拉取完所有数据。注意只能根据该字段判断是否已经拉完数据 |
| next_cursor |  | 当前数据最后一个cursor值，如果下次调用带上该值则从该cursor值往后拉，用于实现分页拉取 |
| stat_info |  | 统计信息列表 |
| stat_info.students |  | 观看直播的学生列表 |
| stat_info.students.student_userid |  | 学生的userid |
| stat_info.students.partyids |  | 学生所在的班级id列表 |
| stat_info.students.watch_time |  | 观看时长，单位为秒 |
| stat_info.students.enter_time |  | 首次进入直播时间 |
| stat_info.students.leave_time |  | 最后离开直播时间 |
| stat_info.students.is_comment |  | 是否评论，1表示评论，0表示没有评论 |
| stat_info.parents |  | 观看直播的家长列表 |
| stat_info.parents.parent_userid |  | 家长的userid |
| stat_info.parents.student_userid |  | 家长对应学生的userid |
| stat_info.parents.partyids |  | 家长对应学生所在的班级id列表 |
| stat_info.parents.watch_time |  | 观看时长，单位为秒 |
| stat_info.parents.enter_time |  | 首次进入直播时间 |
| stat_info.parents.leave_time |  | 最后离开直播时间 |
| stat_info.parents.is_comment |  | 是否评论，1表示评论，0表示没有评论 |
| stat_info.visitors |  | 观看直播的游客列表 |
| stat_info.visitors.nickname |  | 微信昵称 |
| stat_info.visitors.watch_time |  | 观看时长，单位为秒 |
| stat_info.visitors.enter_time |  | 首次进入直播时间 |
| stat_info.visitors.leave_time |  | 最后离开直播时间 |
| stat_info.visitors.is_comment |  | 是否评论，1表示评论，0表示没有评论 |
