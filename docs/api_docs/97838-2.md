# 收集表完成事件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97838](https://developer.work.weixin.qq.com/document/path/97838)
- **文档 ID**: `97838`

## 接口描述

当成员完成API创建的收集表后，触发该事件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 成员UserID |
| CreateTime |  | 否 | 消息创建时间，unix时间戳 |
| MsgType |  | 否 | 消息类型，此时固定为：event |
| Event |  | 否 | 事件类型，此时固定为：doc_change |
| ChangeType |  | 否 | 事件类型，此时固定为：form_complete |
| FormId |  | 否 | 收集表ID列表 |
