## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 此事件该值固定为sys，表示该消息由系统生成 |
| CreateTime |  | 否 | 消息创建时间，unix时间戳 |
| MsgType |  | 否 | 消息类型，此时固定为：event |
| Event |  | 否 | 事件类型，此时固定为：app_email_change |
| ChangeType |  | 否 | 事件类型，此时固定为：receive_email |
| Amount |  | 否 | 应用邮箱当前的新邮件数 |

### 请求示例

```xml
<xml><ToUserName><![CDATA[toUser]]></ToUserName><FromUserName><![CDATA[sys]]></FromUserName><CreateTime>1668831860</CreateTime><MsgType><![CDATA[event]]></MsgType><Event><![CDATA[app_email_change]]></Event><ChangeType><![CDATA[receive_email]]></ChangeType><Amount><![CDATA[2]]></Amount></xml>
```
