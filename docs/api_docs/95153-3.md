# 获取企业验证状态

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95153](https://developer.work.weixin.qq.com/document/path/95153)
- **文档 ID**: `95153`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/enterprise/get_auth_status`

## 接口描述

微信客服可接待的客户数，和企业是否已完成主体验证相关。企业未完成主体验证时，微信客服仅可累计接待100位客户。企业已验证时，微信客服可接待的客户数不受限制。开发者可获取状态后，在应用等地方提示企业去完成主体验证。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| auth_status | string | 企业验证状态，未验证/已验证 |
| customer_limit | int32 | 微信客服可接待的客户数 |

## 其他说明

### 概述

微信客服可接待的客户数，和企业是否已完成主体验证相关。企业未完成主体验证时，微信客服仅可累计接待100位客户。企业已验证时，微信客服可接待的客户数不受限制。开发者可获取状态后，在应用等地方提示企业去完成主体验证。
