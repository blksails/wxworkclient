# 删除会议报名信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98817](https://developer.work.weixin.qq.com/document/path/98817)
- **文档 ID**: `98817`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/enroll/delete?access_token=ACCESS_TOKEN`

## 接口描述

删除指定会议的报名信息，支持删除成员手动报名的信息和导入的报名信息。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议ID |
| enroll_id_list | object[] | 是 | 报名ID列表。详见EnrollID。 |
| enroll_id | string | 是 | 报名ID |

### 请求示例

```json
{"meetingid":"meetingid11234","enroll_id_list":[{"enroll_id":"123456"}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| total_count | int32 | 成功删除的报名信息数量 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","total_count":1}
```
