# wx.qy.startNFCReader

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92410](https://developer.work.weixin.qq.com/document/path/92410)
- **文档 ID**: `92410`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

打开 NFC 模块，仅支持安卓系统

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| Object object | object | 是 | 包含以下属性 |
| Object object.success | function | 否 | 接口调用成功的回调函数 |
| Object object.fail | function | 否 | 接口调用失败的回调函数 |
| Object object.complete | function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```text
wx.qy.startNFCReader({
  success: (res) => {
    wx.showModal({
      title: 'NFC-Start',
      content: JSON.stringify(res),
    })
  },
  fail: (res) => {
    console.log(JSON.stringify(res))
  }
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |

## 其他说明

### 错误说明

| 错误码 | 错误信息 | 说明 |
| --- | --- | --- |
| 0 | ok | 正常 |
| 13000 |  | 当前设备不支持NFC |
| 13001 |  | 当前设备支持NFC，但系统NFC开关未开启 |
