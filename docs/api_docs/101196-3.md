## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| ios |  | 否 | ≥5.0.2 |
| android |  | 否 | ≥5.0.2 |
| win |  | 否 | ≥5.0.2 |
| mac |  | 否 | 暂不支持 |

### 请求示例

```json
wx.qy.selectExternalChat({success:function(res){varchatIdList=res.chatIdList;}});
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| chatIdList | Array | 返回此次选择的客户群ID列表 |

## 其他说明

### wx.qy.selectExternalChat(Object object)

调用前提：必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态当前成员必须在应用的可见范围仅配置了客户联系功能的用户可调用仅支持「营销获客」分类的第三方应用可调用
「营销获客」类应用需要通过群活码接口为企业获客，在创建群活码时，需要传入群 Id本接口即提供一种合理且安全的方案，让应用可获得群 id

### 示例代码：

wx.qy.selectExternalChat({success:function(res){varchatIdList=res.chatIdList;}});
