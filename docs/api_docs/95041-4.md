# 微信小程序进入填写学生资料页面

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95041](https://developer.work.weixin.qq.com/document/path/95041)
- **文档 ID**: `95041`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

在微信环境里，当开发者检测到用户不在家校通讯录指定班级中，在微信小程序里可调用小程序接口wx.navigateToMiniProgram，唤起填写学生资料页面。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| appId | string | 是 | 固定填企业微信填写学生资料小程序appid: wx61866588582f5f71 |
| path | string | 是 | 跳转到小程序的路径，路径格式固定为：pages/index/index?strcorpid=${strcorpid}&strpartyid=${strpartyid} |
| strcorpid | string | 是 | 学校corpid |
| strpartyid | string | 否 | 班级的部门id，当传入班级部门id后，用户不能再修改校区和班级 |

### 请求示例

```text
wx.navigateToMiniProgram({
  appId: 'wx61866588582f5f71',
  path: 'pages/index/index?strcorpid=${strcorpid}&strpartyid=${strpartyid}',
  success(res) {
    // 打开成功
  }
})
```
