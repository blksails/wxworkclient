## 接口描述

为保障客户服务质量、提高内部协作效率和监管合规等原因，企业微信提供会话内容存档功能。企业可以统一设置存档的员工范围，并通过API获取开启存档员工的工作沟通内容，满足企业的外部监管合规和内部管理需求。
