# 配置可使用客户联系接口的应用

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/101215](https://developer.work.weixin.qq.com/document/path/101215)
- **文档 ID**: `101215`
- **API 名称**: `get_customer_contacts`
- **请求方法**: `GET`
- **接口地址**: `/cgi-bin/externalcontact/get_follow_user_list`
- **分组信息**: 第 2 个接口，共 9 个

## 接口描述

如需使用自建应用调用外部联系人相关的接口，需要在“可调用接口的应用”中进行配置。
