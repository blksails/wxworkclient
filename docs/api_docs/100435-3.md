## 其他说明

### ww.startBeaconDiscovery(params)

兼容性：企业微信
功能描述开始搜索附近的 iBeacon 设备。
ww.startBeaconDiscovery({uuids:['uuid']})
参数说明params: Object属性类型必填说明uuidsstring[]是iBeacon设备广播的 uuidssuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

开始搜索附近的 iBeacon 设备。

### params: Object

属性类型必填说明uuidsstring[]是iBeacon设备广播的 uuidssuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 示例代码

wx.startBeaconDiscovery({success(res){}})
