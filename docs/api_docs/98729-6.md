# tmp_external_userid的转换 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98729](https://developer.work.weixin.qq.com/document/path/98729)
- **文档 ID**: `98729`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/idconvert/convert_tmp_external_userid?access_token=ACCESS_TOKEN`

## 接口描述

将应用获取的外部用户临时idtmp_external_userid，转换为external_userid。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 1 | 1 | 否 | 会议 |
| 2 | 2 | 否 | 收集表 |
| 3 | 3 | 否 | 文档 |
| 1 | 1 | 否 | 客户 |
| 2 | 2 | 否 | 企业互联 |
| 3 | 3 | 否 | 上下游 |
| 4 | 4 | 否 | 互联企业（圈子） |
| access_token | string | 是 | 调用接口凭证 |
| business_type | uint32 | 是 | 业务类型。1-会议 2-收集表 3-智能表 |
| user_type | uint32 | 是 | 转换的目标用户类型。1-客户 2-企业互联 3-上下游 4-互联企业（圈子）详见上面关于user_type的说明 |
| tmp_external_userid_list | string[] | 是 | 外部用户临时id，最多不超过100个 |

### 请求示例

```json
{"business_type":1,"user_type":1,"tmp_external_userid_list":["ouXXX1","ouXXX2","ouXXX3"]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| results | obj[] | 转换成功的结果列表 |
| results[].tmp_external_userid | string | 输入的tmp_external_userid |
| results[].external_userid | string | 转换后的userid，user_type为1时返回 |
| results[].corpid | string | userid对应的corpid，user_type为2、3、4时返回 |
| results[].userid | string | 转换后的userid，user_type为2、3、4时返回 |
| invalid_tmp_external_userid_list | string[] | 无法转换的tmp_external_userid。可能非法或没有权限 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","results":[{"tmp_external_userid":"ouXXX1","external_userid":"EXTERNAL_USER_ID"},{"tmp_external_userid":"ouXXX2","corpid":"CORPID","userid":"USERID"}],"invalid_tmp_external_userid_list":["ouXXX3"]}
```
