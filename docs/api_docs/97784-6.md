# 获取日历详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97784](https://developer.work.weixin.qq.com/document/path/97784)
- **文档 ID**: `97784`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/oa/calendar/get`

## 接口描述

该接口用于获取应用在企业内创建的日历信息。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| cal_id_list[] | string | 是 | 日历ID列表，调用创建日历接口后获得。一次最多可获取1000条 |

### 请求示例

```json
{
	"cal_id_list": ["wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA"]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| calendar_list | object[] | 日历列表 |
| calendar_list[].cal_id | string | 日历ID |
| calendar_list[].admins | string[] | 日历的管理员userid列表 |
| calendar_list[].summary | string | 日历标题。1 ~ 128 字符 |
| calendar_list[].color | string | 日历颜色，RGB颜色编码16进制表示，例如:"#0000FF" 表示纯蓝色 |
| calendar_list[].description | string | 日历描述。0 ~ 512 字符 |
| calendar_list[].is_public | int32 | 是否公共日历。0-否；1-是 |
| calendar_list[].public_range | object | 公开范围。仅当是公共日历时有效 |
| calendar_list[].public_range.userids | string[] | 公开的成员列表范围 |
| calendar_list[].public_range.partyids | int32[] | 公开的部门列表范围 |
| calendar_list[].is_corp_calendar | int32 | 是否全员日历。0-否；1-是 |
| calendar_list[].shares | object[] | 日历通知范围成员列表。最多2000人 |
| calendar_list[].shares.userid | string | 日历通知范围成员的id |
| calendar_list[].shares.permission | int32 | 日历通知范围成员权限。 1：可查看  3：仅查看闲忙状态 |
