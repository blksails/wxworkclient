# 配置调试参数

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100082](https://developer.work.weixin.qq.com/document/path/100082)
- **文档 ID**: `100082`
- **API 名称**: `获取调试凭证`
- **请求方法**: `GET`
- **接口地址**: `https://wework.qpic.cn/wwpic3az/492237_1gNicNqvQNCcx4m_1721652005/0`
- **分组信息**: 第 1 个接口，共 5 个

## 接口描述

点击「调试凭证-获取」，弹窗二次确认后会通过服务商助手或企业微信团队将调试凭证推送消息给操作者。该凭证3天内有效，有效期内重复获取会获取到同样的凭证，并且不会刷新过期时间。

## 请求信息

### 请求示例

```text
GET https://wework.qpic.cn/wwpic3az/492237_1gNicNqvQNCcx4m_1721652005/0
```
