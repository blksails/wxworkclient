# 在企业微信内快速跳转到微信客服消息的界面

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94961](https://developer.work.weixin.qq.com/document/path/94961)
- **文档 ID**: `94961`
- **请求方法**: `wx.qy.navigateToKfChat`

## 接口描述

在企业微信内快速跳转到微信客服消息的界面

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| object.openKfId | String | 是 | 客服账号ID |
| object.externalUserId | String | 否 | 微信客户ID，若没指定，则跳转到客服账号的消息列表界面 |
| object.success | Function | 否 | 接口调用成功的回调函数 |
| object.fail | Function | 否 | 接口调用失败的回调函数 |
| object.complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```text
wx.qy.navigateToKfChat({
    openKfId: "wkAJ2GCAAAZSfhHCt7IFSvLKtMPxyAAA",
    externalUserId: "wmAJ2GCAAAZSfhHCt7IFSvLKtMPxyBBB",
    success: function(res) {
        // 回调
    }
});
```

## 其他说明

### 接口说明

在企业微信内快速跳转到微信客服消息的界面

### 调用前提

必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态。
当前企业已开启「微信客服」应用。
当前成员指定客服账号的坐席，且必须在应用的可见范围之内。

### 平台版本说明

| 平台 | 企业微信版本要求 |
| --- | --- |
| ios | ≥3.1.12 |
| android | ≥3.1.12 |
| win | ≥3.1.12 |
| mac | 暂不支持 |

### 应用权限说明

| 应用类型 | 可调用的应用 |
| --- | --- |
| 自建应用 | 所有 |
| 第三方应用 | 需具有“微信客服->获取基础信息”权限 |
