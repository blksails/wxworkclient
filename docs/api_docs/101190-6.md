# 编辑表格内容

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/101190](https://developer.work.weixin.qq.com/document/path/101190)
- **文档 ID**: `101190`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/spreadsheet/batch_update?access_token=ACCESS_TOKEN`

## 接口描述

该接口可以对一个在线表格批量执行多个更新操作。批量更新请求中的各个操作会逐个按顺序执行，直到全部执行完成则请求返回，或者其中一个操作报错则不再继续执行后续的操作。每一个更新操作在执行之前都会做请求校验（包括权限校验、参数校验等等），如果校验未通过则该更新操作会报错并返回，不再执行后续操作。单次批量更新请求的操作数量 <= 5。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 是 | 文档的docid |
| requests | object[] | 是 | 更新操作列表 |

### 请求示例

```json
{
	"docid": "DOCID",
	"requests": [
		{
			"add_sheet_request": {}
		},
		{
			"update_range_request": {}
		},
		{
			"delete_dimension_request": {}
		},
		{
			"delete_sheet_request": {}
		}
	]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| data.responses | object[] | 结果列表 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"data": {
		"responses": [
			{
				"add_sheet_response": {}
			},
			{
				"update_range_response": {}
			},
			{
				"delete_dimension_response": {}
			},
			{
				"delete_sheet_response": {}
			}
		]
	}
}
```
