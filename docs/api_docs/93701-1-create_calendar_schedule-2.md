# 接口创建日历和日程表现

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93701](https://developer.work.weixin.qq.com/document/path/93701)
- **文档 ID**: `93701`
- **API 名称**: `create_calendar_schedule`
- **请求方法**: `POST`
- **接口地址**: `https://wework.qpic.cn/wwpic/320159_q41SbCGtR2aJ8i6_1669818654/0`
- **分组信息**: 第 1 个接口，共 4 个

## 接口描述

接口创建的日历和日程，将在日历详情和日程详情标识出应用创建的信息。接口创建的日历，仅应用可添加日程，用户无法手动添加。通过接口创建、修改、删除日历或日程后，日程应用将自动向日历或日程相关成员推送通知。
