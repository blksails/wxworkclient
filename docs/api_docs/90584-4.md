# 设置通讯录同步完成 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90584](https://developer.work.weixin.qq.com/document/path/90584)
- **文档 ID**: `90584`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/sync/contact_sync_success?access_token=ACCESS_TOKEN`

## 接口描述

该API用于设置通讯录同步完成，解除通讯录锁定状态，同时使通讯录迁移access_token失效。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 查询注册状态接口返回的access_token（跟注册完成回调事件的AccessToken参数一致，请注意与provider_access_token的区别） |

## 响应信息

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
