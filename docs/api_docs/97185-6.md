# 获取订单详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97185](https://developer.work.weixin.qq.com/document/path/97185)
- **文档 ID**: `97185`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/license/get_order?provider_access_token=ACCESS_TOKEN`

## 接口描述

查询某个订单的详情，包括订单的状态、基础账号个数、互通账号个数、账号购买时长等。注意，该接口不返回订单中的账号激活码列表或者续期的账号成员列表。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 应用服务商的接口调用凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| order_id | string | 是 | 订单id |

### 请求示例

```json
{
	"order_id":"xxxxx"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| order | object | 订单详情 |
| order.order_id | string | 订单号 |
| order.order_type | int32 | 订单类型，1：购买账号，2：续期账号，5：应用版本付费迁移订单，6：历史合同迁移订单 |
| order.order_status | int32 | 订单状态，0：待支付，1：已支付，2：已取消（未支付，订单已关闭）3：未支付，订单已过期，4：申请退款中，5：退款成功，6：退款被拒绝，7：订单已失效 |
| order.corpid | string | 客户企业id，返回加密的corpid |
| order.price | int32 | 订单金额，单位分 |
| order.account_count | object | 订单的账号数详情 |
| order.account_count.base_count | int32 | 基础账号个数 |
| order.account_count.external_contact_count | int32 | 互通账号个数 |
| order.account_duration | object | 账号购买时长 |
| order.account_duration.months | int32 | 购买的月数，每个月按照31天计算 |
| order.account_duration.days | int32 | 购买的天数 |
| order.account_duration.new_expire_time | int32 | 新过期时间 |
| order.create_time | int32 | 创建时间 |
| order.pay_time | int32 | 支付时间 |
