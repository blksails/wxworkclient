## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| type | String | 是 | 设备发现方式，支持类型：qrcode（二维码识别）/bluetooth（蓝牙发现）/input（sn码输入）/choose（进入选择添加设备方式列表） |
| qrcode_url | String | 是 | 设备静态/动态二维码链接 |
| sn | String | 否 | 设备SN码 |
