# 获取企业成员头像

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92397](https://developer.work.weixin.qq.com/document/path/92397)
- **文档 ID**: `92397`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

获取企业成员头像

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| timeout | Number | 否 | 超时时间，单位 ms |
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```text
wx.qy.getAvatar ({
  success: function(res) {
    var avatar = res.avatar
  },
  fail: function(res) {
    console.log(res.fail_reason)
  }
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| avatar | String | 当前企业成员的头像url |
