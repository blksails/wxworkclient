# 上报异步任务结果

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99847](https://developer.work.weixin.qq.com/document/path/99847)
- **文档 ID**: `99847`
- **请求方法**: `SDK调用`
- **接口地址**: `program_async_job_call_back`

## 接口描述

可以通过这个接口上报异步任务的结果。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| errcode | int32 | 是 | 错误码，为0表示任务执行成功 |
| result | string | 否 | 任务结果。返回的结果必须符合模型或程序的输出协议。 |

### 请求示例

```json
{
	"errcode":0,
	"result":"aaaabbbccc"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok"
}
```
