# 授权组织架构权限通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97378](https://developer.work.weixin.qq.com/document/path/97378)
- **文档 ID**: `97378`

## 接口描述

当企业在管理端后台点击同意授权第三方应用的“组织架构”权限时，会触发该事件的回调。该事件会推送到第三方应用的指令回调URL上。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId |  | 否 | 第三方应用ID |
| AuthCorpId |  | 否 | 授权企业的CorpID |
| InfoType |  | 否 | 事件类型，此时固定为corp_arch_auth |
| TimeStamp |  | 否 | 时间戳 |

### 请求示例

```xml
<xml><SuiteId><![CDATA[ww4asffe99e54c0fxxxx]]></SuiteId><AuthCorpId><![CDATA[wxf8b4f85f3a79xxxx]]></AuthCorpId><InfoType><![CDATA[corp_arch_auth]]></InfoType><TimeStamp>1403610513</TimeStamp></xml>
```
