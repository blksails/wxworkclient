# 订单管理

- **接口地址**: `N/A`
- **分组信息**: 第 3 个接口，共 3 个

## 接口描述

企业成员可对订单进行开取发票和退款操作，管理员和普通成员有不同的订单管理入口和范围。

## 其他说明

### 订单管理入口区分

管理员：订单管理入口为企业微信移动端-工作台-企业管理-应用管理、企业微信移动端-工作台-应用市场-订单中心、企业管理后台-应用管理，管理员可管理企业的所有订单；
普通成员：订单管理入口为企业微信移动端-工作台-应用市场-订单中心，普通成员仅可管理自己购买的订单。
