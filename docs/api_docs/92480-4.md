# 发起无线投屏

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92480](https://developer.work.weixin.qq.com/document/path/92480)
- **文档 ID**: `92480`
- **请求方法**: `POST`
- **接口地址**: `https://wx.qy.startWecast`

## 接口描述

该接口调用需要配合硬件设备使用，硬件接入流程参考 无线投屏，目前仅支持第三方服务商接入。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| object.success | function | 否 | 接口调用成功的回调函数 |
| object.fail | function | 否 | 接口调用失败的回调函数 |
| object.complete | function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```text
wx.qy.startWecast({
  success: (res) => {
      //调用成功
  },
  fail: (res) => {
    console.log(JSON.stringify(res))
  }
})
```
