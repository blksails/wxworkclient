# 分享文档 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97744](https://developer.work.weixin.qq.com/document/path/97744)
- **文档 ID**: `97744`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/doc_share?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于获取文档、表格、智能表格及收集表的分享链接。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 否 | 文档id（docid、formid只能填其中一个） |

### 请求示例

```json
{"docid":"DOCID","formid":"FORMID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| share_url | string | 文档分享链接 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","share_url":"URL1"}
```
