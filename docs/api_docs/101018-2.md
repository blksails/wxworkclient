# 字段变更事件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/101018](https://developer.work.weixin.qq.com/document/path/101018)
- **文档 ID**: `101018`

## 接口描述

当成员在API创建的智能表格上修改了字段后，触发该事件。FieldId一次最多回调1000个，超过会分批回调
