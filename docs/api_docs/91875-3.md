# 异步任务回调通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91875](https://developer.work.weixin.qq.com/document/path/91875)
- **文档 ID**: `91875`
- **请求方法**: `POST`

## 接口描述

推送到服务商设置的接收事件回调url上。url配置路径为 应用管理-》开发接口-》通用开发参数请求方式：POST请求地址：https://127.0.0.1/service/receive?msg_signature=3a7b08bb8e6dbce3c9671d6fdb69d15066227608&timestamp=1403610513&nonce=380320359

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ServiceCorpId |  | 否 | 服务商corpid |
| InfoType |  | 否 | 事件类型，batch_job_result，通讯录导出完成回调 |
| TimeStamp |  | 否 | 回调事件的时间戳 |
| AuthCorpId |  | 否 | 企业corpid |
| JobId |  | 否 | 任务id,最大长度为64字符 |
| JobType |  | 否 | 任务类型，目前有contact_id_translate |

