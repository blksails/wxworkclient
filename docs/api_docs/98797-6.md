# 修改会议报名配置 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98797](https://developer.work.weixin.qq.com/document/path/98797)
- **文档 ID**: `98797`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/enroll/set_config?access_token=ACCESS_TOKEN`

## 接口描述

修改会议的报名配置和报名问题，且需要会议已开启报名。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议ID |
| approve_type | int32 | 否 | 审批类型：1：自动审批，默认自动审批。2：手动审批。 |
| is_collect_question | int32 | 否 | 是否收集问题：1：不收集，默认不收集问题。2：收集。 |
| question_list | object[] | 否 | 报名问题列表，非特殊问题按传入的顺序排序，特殊问题会优先放在最前面，仅开启收集问题时有效。详见Question。 |
| no_registration_needed_for_staff | bool | 否 | 本企业成员无需报名。true：本企业成员无需报名。false：默认配置，本企业成员及企业外成员需要报名。 |
| is_required | int32 | 是 | 是否必填：1：否2：是 |
| question_title | string | 否 | 问题标题，限制40个字符。说明：special_type 为特殊问题时，该字段无效。 |
| option_list | object[] | 否 | 问题选项列表，按传入的顺序排序，仅单选和多选时有效，最多8个选项，每个选项限40个汉字。详见Option。 |
| question_type | int32 | 否 | 问题类型：1：单选2：多选3：简答说明：special_type 为特殊问题时，该字段无效。 |
| special_type | int32 | 否 | 特殊问题类型：1：无2：手机号3：邮箱4：姓名5：公司名称说明：目前特殊问题均为简答题。 |
| content | string | 否 | 问题选项。每个选项限40个汉字。 |

### 请求示例

```json
{"meetingid":"MEETINGID1","approve_type":1,"is_collect_question":1,"no_registration_needed_for_staff":true,"question_list":[{"is_required":1,"question_type":1,"question_title":"acd","special_type":1,"option_list":[{"content":"asr"}]}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| question_count | int32 | 报名问题数量，不收集问题时该字段返回0。 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","question_count":12}
```
