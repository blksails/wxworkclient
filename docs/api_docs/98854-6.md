# 批量删除会议背景 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98854](https://developer.work.weixin.qq.com/document/path/98854)
- **文档 ID**: `98854`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/layout/batch_delete_background?access_token=ACCESS_TOKEN`

## 接口描述

根据背景 ID 删除多个会议背景。正在被会议应用的背景无法删除，请先设置成其他背景或恢复成会议的默认黑色背景后再行删除。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议 ID |
| background_id_list | string[] | 是 | 背景 ID 列表 |

### 请求示例

```json
{"meetingid":"MEETINGID","background_id_list":["BACKID1","BACKID2"]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
