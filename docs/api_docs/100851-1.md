# 监听页面返回事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100851](https://developer.work.weixin.qq.com/document/path/100851)
- **文档 ID**: `100851`

## 接口描述

监听页面返回事件。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| callback | Function | 回调函数，返回 false 则表示中断此次返回操作 |

## 其他说明

### 兼容性

企业微信 iOS、Android 2.2.0 开始支持，企业微信 PC、Mac 2.4.5 开始支持

### 使用限制

当页面左上角没有关闭按钮，不产生该事件
iOS 系统下使用手势返回时，不产生该事件
