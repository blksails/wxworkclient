# 邀请成员

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91127](https://developer.work.weixin.qq.com/document/path/91127)
- **文档 ID**: `91127`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/batch/invite`

## 接口描述

企业可通过接口批量邀请成员使用企业微信，邀请后将通过微信或邮件下发通知。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| user | array | 否 | 成员ID列表, 最多支持1000个。 |
| party | array | 否 | 部门ID列表，最多支持100个。 |
| tag | array | 否 | 标签ID列表，最多支持100个。 |

### 请求示例

```json
{
   "user" : ["UserID1", "UserID2", "UserID3"],
   "party" : [1, 2, 3],
   "tag" : [101, 102, 103]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| invaliduser | array | 非法成员列表 |
| invalidparty | array | 非法部门列表 |
| invalidtag | array | 非法标签列表 |

### 响应示例

```json
{
  "errcode" : 0,
  "errmsg" : "ok",
  "invaliduser" : ["UserID1", "UserID2"],
  "invalidparty" : [1, 2],
  "invalidtag": [101, 102]
}
```

## 其他说明

### 权限说明

须拥有指定成员、部门或标签的查看权限。第三方仅通讯录应用可调用。

### 更多说明

* user, party, tag三者不能同时为空；
* 如果部分接收人无权限或不存在，邀请仍然执行，但会返回无效的部分（即invaliduser或invalidparty或invalidtag）;
* 同一用户只须邀请一次，被邀请的用户如果未安装企业微信，在3天内每天会收到一次通知，最多持续3天。
* 因为邀请频率是异步检查的，所以调用接口返回成功，并不代表接收者一定能收到邀请消息（可能受上述频率限制无法接收）。
* 境外成员可用企业录入的邮箱登录企业微信。
