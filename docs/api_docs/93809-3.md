## 其他说明

### ww.startLiving(params)

兼容性：企业微信 3.1.0 开始支持
功能描述调起直播预约详情或进行中的直播间页面。
ww.startLiving({livingId:'LIVINGID'})
使用限制本接口必须使用应用身份进行注册应用需具有直播使用权限，见配置可使用直播的应用
参数说明params: Object属性类型必填说明livingIdstring是直播 IDsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明livingIdstring是直播 ID
错误信息errMsg说明startLiving:ok执行成功startLiving:fail no permission应用签名失败，或应用无直播权限startLiving:fail invalid living id不合法的直播IDstartLiving:fail not allow to cross corp不可跨企业使用直播IDstartLiving:fail not allow to cross app不可跨应用使用直播IDstartLiving:fail invalid parameter参数不合法startLiving:fail invalid department id不合法的班级IDstartLiving:fail exceed department id list size超过最大的班级个数

### 功能描述

调起直播预约详情或进行中的直播间页面。

### 使用限制

本接口必须使用应用身份进行注册应用需具有直播使用权限，见配置可使用直播的应用

### params: Object

属性类型必填说明livingIdstring是直播 IDsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明livingIdstring是直播 ID

### 错误信息

errMsg说明startLiving:ok执行成功startLiving:fail no permission应用签名失败，或应用无直播权限startLiving:fail invalid living id不合法的直播IDstartLiving:fail not allow to cross corp不可跨企业使用直播IDstartLiving:fail not allow to cross app不可跨应用使用直播IDstartLiving:fail invalid parameter参数不合法startLiving:fail invalid department id不合法的班级IDstartLiving:fail exceed department id list size超过最大的班级个数

### 示例代码

wx.invoke('startLiving',{"livingId":"LIVINGID",},function(res){if(res.err_msg=="startLiving:ok"){livingId=res.livingId;}});
