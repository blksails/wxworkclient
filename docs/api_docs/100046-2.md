## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| event_type |  | 否 | 事件类型，此时固定为：auth_knowledge_base |
| knowledge_base_id |  | 否 | 知识集ID |
| knowledge_base_name |  | 否 | 知识集名称 |
| timestamp |  | 否 | 授权时间时间戳 |
| event_type |  | 否 | 事件类型，此时固定为：unauth_knowledge_base |
| knowledge_base_id |  | 否 | 知识集ID |
| knowledge_base_name |  | 否 | 知识集名称 |
| timestamp |  | 否 | 取消授权知识集时间戳 |
| event_type |  | 否 | 事件类型，此时固定为：delete_knowledge_base |
| knowledge_base_id |  | 否 | 知识集ID |
| knowledge_base_name |  | 否 | 知识集名称 |
| timestamp |  | 否 | 删除知识集时间戳 |
| event_type |  | 否 | 事件类型，此时固定为：knowledge_base_learn_done |
| timestamp |  | 否 | 删除知识集时间戳 |
| knowledge_base_id |  | 否 | 知识集ID |
| knowledge_base_name |  | 否 | 知识集名称 |
| doc_id |  | 否 | 內容ID |
| learn_status |  | 否 | 学习状态。0-学习成功；1-学习失败 |

### 请求示例

#### 示例 1: 事件示例：

```json
{"event_type":"auth_knowledge_base","timestamp":1408091189,"auth_knowledge_base":{"knowledge_base_id":"xxxxxxx","knowledge_base_name":"zzzzz"}}
```

#### 示例 2: 取消授权知识集

```json
{"event_type":"unauth_knowledge_base","timestamp":1408091189,"unauth_knowledge_base":{"knowledge_base_id":"xxxxxxx","knowledge_base_name":"zzzzz"}}
```

#### 示例 3: 删除授权的知识集

```json
{"event_type":"delete_knowledge_base","timestamp":1408091189,"delete_knowledge_base":{"knowledge_base_id":"xxxxxxx","knowledge_base_name":"zzzzz"}}
```

#### 示例 4: 內容学习完成(每个內容学习完成都会回调一次)

```json
{"event_type":"knowledge_base_learn_done","timestamp":1408091189,"knowledge_base_learn_done":{"knowledge_base_id":"xxxxxxx","knowledge_base_name":"zzzzz","doc_id":1,"learn_status":0}}
```
