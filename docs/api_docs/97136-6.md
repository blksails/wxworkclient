# 添加网格 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97136](https://developer.work.weixin.qq.com/document/path/97136)
- **文档 ID**: `97136`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/report/grid/add?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/report/grid/add?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证。“获取access_token” |
| grid_name |  | 是 | 网格名称，不能超过30个字，同一个目标网格下，不能存在同名的同级子网格 |
| grid_parent_id |  | 是 | 父节点的id，网格结构层级最多支持10层 |
| grid_admin |  | 是 | 网格「负责人」userid列表，每个网格至少1个，最多20个负责人 |
| grid_member |  | 否 | 该节点的成员userid列表，不能超过100个，同一个成员最多能成功10个网格的管理员 |
| 自建应用 | 自建应用 | 否 | 配置到「居民联系 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 支持，需要勾选「居民联系权限」 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"grid_name":"grid_name","grid_parent_id":"grid_id","grid_admin":["zhangsan"],"grid_member":["lisi","invaliduser"]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| grid_id |  | 网格id |
| invalid_userids |  | 不合法的userid |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","grid_id":"grid_id","invalid_userids":["invaliduser"]}
```
