# 获取文档基础信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97743](https://developer.work.weixin.qq.com/document/path/97743)
- **文档 ID**: `97743`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/get_doc_base_info?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于获取指定文档、表格、智能表格及收集表的基础信息。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 是 | 文档docid |

### 请求示例

```json
{"docid":"DOCID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| docid | string | 文档docid |
| doc_name | string | 文档名字 |
| create_time | uint64 | 文档创建时间 |
| modify_time | uint64 | 文档最后修改时间 |
| doc_type | uint32 | 3: 文档 4: 表格   10:智能表格 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","doc_base_info":{"docid":"DOCID","doc_name":"DOC_NAME","create_time":1717071093,"modify_time":1717071093,"doc_type":3}}
```
