# 获取异步任务结果

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91882](https://developer.work.weixin.qq.com/document/path/91882)
- **文档 ID**: `91882`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/batch/getresult`

## 接口描述

获取异步任务结果

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 服务商provider_access_token，获取方法参见服务商的凭证 |
| jobid | string | 是 | 异步任务id，最大长度为64字节 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| status | int32 | 任务状态，整型，1表示任务开始，2表示任务进行中，3表示任务已完成 |
| type | string | 操作类型，字节串，目前有：contact_id_translate |
| result | object | 详细的处理结果。当任务完成后此字段有效 |
| result.contact_id_translate | object | 通讯录转换成功后的信息 |
| result.contact_id_translate.url | string | 通讯录转换成功后的url，需要用户通过oauth2授权登录或者单点登录在用户侧打开。该url24小时有效 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "status": 1,
    "type": "contact_id_translate",
    "result": {
        "contact_id_translate":{
            "url":"xxxx"
        }
    }
}
```

## 其他说明

### 权限说明

只能查询已经提交过的历史任务。
