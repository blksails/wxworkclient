## 接口描述

跳转到认领班级的界面

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |

### 请求示例

```json
wx.qy.claimClassAdmin({success:function(res){// 回调成功vardepartmentIds=res.departmentIds;//数组类型for(vari=0;i<departmentIds.length;i++){vardepartmentId=departmentIds[i];}},fail:function(res){// 失败处理console.log(res)}});
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| departmentIds | Array | 认领的班级部门id列表 |
| claimClassAdmin:ok |  | 执行成功 |
| claimClassAdmin:fail no permission |  | 必须先成功调用agentConfig |
| claimClassAdmin:fail user not in allow list |  | 当前成员不在应用可见范围 |
