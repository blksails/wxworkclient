## 接口描述

企业互联/上下游选人

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fromDepartmentId | number | 是 | 从指定的部门开始展示。-1 表示从自己所在部门开始，0 表示从最上层开始。移动端，当需要展开的部门不在应用可见范围内，则从应用的可见范围开始展开 |
| mode | string | 是 | 选择模式。合法值：single（单选）、multi（多选） |
| type | string[] | 是 | 选择限制类型，指定一个或多个。合法值：department（选择部门）、user（选择成员） |
| selectedDepartmentIds | string[] | 否 | 已选部门 ID 列表，用于多次选人时可重入 |
| selectedUserIds | string[] | 否 | 已选用户 ID 列表，用于多次选人时可重入。仅自建应用使用，第三方应用会忽略该字段 |
| selectedOpenUserIds | string[] | 否 | 已选用户 OpenUserID 列表，用于多次选人时可重入。仅第三方应用使用，自建应用会忽略该字段 |
| selectedChainDepartmentIds | Object[] | 否 | 已选上下游部门列表，用于多次选人时可重入。企业微信 3.1.20 |
| selectedChainUserIds | Object[] | 否 | 已选上下游用户列表，用于多次选人时可重入。企业微信 3.1.20 |
| selectedCorpGroupDepartmentIds | Object[] | 否 | 已选企业互联部门列表，用于多次选人时可重入 |
| selectedCorpGroupUserIds | Object[] | 否 | 已选企业互联用户列表 |
| success | Function | 否 | 成功回调 |
| fail | Function | 否 | 失败回调 |
| cancel | Function | 否 | 取消回调 |
| complete | Function | 否 | 完成回调 |

## 其他说明

### 使用限制

本接口必须使用应用身份进行注册。该接口仅可选择应用可见范围内的成员和部门。

### 使用说明

自建应用调用该接口时userid返回的是企业内部的userid，对于服务商该字段返回的是open_userid，同一个服务商，不同应用获取到企业内同一个成员的open_userid是相同的，最多64个字节。

### 错误信息

| errMsg | 说明 | | --- | --- | | selectCorpGroupContact:ok | 执行成功 | | selectCorpGroupContact:fail no permission | 应用身份鉴权失败 |
