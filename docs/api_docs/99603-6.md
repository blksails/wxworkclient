# 生成代支付key

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99603](https://developer.work.weixin.qq.com/document/path/99603)
- **文档 ID**: `99603`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/customer_acquisition/create_once_key`

## 接口描述

服务商可通过此接口生成用于对应链接代支付的key。key使用一次之后失效。生成key之后需要在14天内使用。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| link_id | string | 是 | 获客链接id |
| key_num | uint32 | 否 | 生成的代支付key数量，默认100，最大可填写1000 |

### 请求示例

```json
{
   "link_id":"xxxxxx",
   "key_num":1
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| keys | array | 生成的代支付key的列表 |
| keys[] | string | 代支付key |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "keys":[
			"xxxxxx"
    ]
}
```

## 其他说明

### 权限说明

获客助手组件可调用，需企业授权时为服务商代支付模式才可调用
