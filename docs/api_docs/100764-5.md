# 获取私密消息信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100764](https://developer.work.weixin.qq.com/document/path/100764)
- **文档 ID**: `100764`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

获取私密消息信息。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| params.shareTicket | string | 是 | getContext 接口返回的 shareTicket |
| params.success | Function | 否 | 成功回调 |
| params.fail | Function | 否 | 失败回调 |
| params.cancel | Function | 否 | 取消回调 |
| params.complete | Function | 否 | 完成回调 |

### 请求示例

```text
ww.getShareInfo({
  shareTicket: 'xxx'
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errMsg | string | 通用错误信息 |
| errCode | number | 通用错误码 |
| encryptedData | string | 转发信息的加密数据 |
| iv | string | 加密算法的初始向量 |

## 其他说明

### 使用限制

本接口必须使用应用身份进行注册
