# 外部联系人openid转换

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92292](https://developer.work.weixin.qq.com/document/path/92292)
- **文档 ID**: `92292`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/convert_to_openid?access_token=ACCESS_TOKEN`

## 接口描述

企业和服务商可通过此接口，将微信外部联系人的userid转为微信openid，用于调用支付相关接口。暂不支持企业微信外部联系人（ExternalUserid为wo开头）的userid转openid。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| external_userid | string | 是 | 外部联系人的userid，注意不是企业成员的账号 |

### 请求示例

```json
{
  "external_userid":"wmAAAAAAA"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| openid | string | 该企业的外部联系人openid |

### 响应示例

```json
{
 "errcode":0,
 "errmsg":"ok",
 "openid":"ooAAAAAAAAAAA"
}
```
