# 标签成员变更事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90641](https://developer.work.weixin.qq.com/document/path/90641)
- **文档 ID**: `90641`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`

## 接口描述

回调设置在授权应用可见范围内的标签的变更事件。由第三方应用调用接口触发的标签变更事件不回调给该应用本身。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 | 消息签名 |
| timestamp | string | 是 | 时间戳 |
| nonce | string | 是 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 第三方应用ID |
| AuthCorpId | string | 是 | 授权企业的CorpID |
| InfoType | string | 是 | 固定为change_contact |
| TimeStamp | string | 是 | 时间戳 |
| ChangeType | string | 是 | 固定为update_tag |
| TagId | string | 是 | 标签Id |
| AddUserItems | string | 是 | 标签中新增的成员userid列表，用逗号分隔 |
| DelUserItems | string | 是 | 标签中删除的成员userid列表，用逗号分隔 |
| AddPartyItems | string | 是 | 标签中新增的部门id列表，用逗号分隔 |
| DelPartyItems | string | 是 | 标签中删除的部门id列表，用逗号分隔。 |

### 请求示例

```xml
<xml>
	<SuiteId><![CDATA[ww4asffe99e54cxxxx]]></SuiteId>
	<AuthCorpId><![CDATA[wxf8b4f85f3a79xxxx]]></AuthCorpId>
	<InfoType><![CDATA[change_contact]]></InfoType>
	<TimeStamp>1403610513</TimeStamp>
	<ChangeType><![CDATA[update_tag]]></ChangeType>
	<TagId>1</TagId>
	<AddUserItems><![CDATA[zhangsan,lisi]]></AddUserItems>
	<DelUserItems><![CDATA[zhangsan1,lisi1]]></DelUserItems>
	<AddPartyItems><![CDATA[1,2]]></AddPartyItems>
	<DelPartyItems><![CDATA[3,4]]></DelPartyItems>
</xml>
```

| AddPartyItems | string | 标签中新增的部门id列表，用逗号分隔 |
