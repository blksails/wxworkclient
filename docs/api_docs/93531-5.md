# 获取配置的网格及网格负责人 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93531](https://developer.work.weixin.qq.com/document/path/93531)
- **文档 ID**: `93531`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/report/patrol/get_grid_info?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：GET（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/report/patrol/get_grid_info?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证。“获取access_token” |
| 自建应用 | 自建应用 | 否 | 配置到「巡查上报 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| grid_list.grid_id |  | 网格 ID |
| grid_list.grid_name |  | 网格名称 |
| grid_list.grid_admin |  | 网格管理员 userId 列表 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","grid_list":[{"grid_id":"grid_id","grid_name":1,"grid_admin":["zhangsan","lisi"]}]}
```
