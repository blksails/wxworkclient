## 接口描述

按照接口能力分为内外部会话、内外部会话及语音通话。服务商购买数据专区会话数据第三方接口时需指定企业，购买的接口仅可在指定的企业下使用。一个接口账号可用于调用一个员工的数据专区会话数据第三方接口，并使用展示组件展示对应的会话内容。
