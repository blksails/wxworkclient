## 接口描述

在第三方应用或代开发应用的权限详情页中，勾选开启“数据与智能专区权限”。

## 其他说明

### 开启应用权限

勾选后，设置需要分析的企业数据、使用的模型、使用的程序，以及是否使用会话展示组件。
