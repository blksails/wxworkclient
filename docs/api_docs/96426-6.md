# 概述

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96426](https://developer.work.weixin.qq.com/document/path/96426)

- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/kf/sync_msg?access_token=ACCESS_TOKEN`
