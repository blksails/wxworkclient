# wx.qy.selectCorpGroupContact

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94363](https://developer.work.weixin.qq.com/document/path/94363)
- **文档 ID**: `94363`
- **请求方法**: `N/A`
- **接口地址**: `wx.qy.selectCorpGroupContact`

## 接口描述

打开企业互联/局校互联/上下游通讯录选人功能，仅返回应用可见范围内的成员和部门。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| object.fromDepartmentId | Number | 是 | 表示打开的通讯录从指定的部门开始展示，-1表示打开的通讯录从自己所在部门开始展示, 0表示从最上层开始。移动端，当需要展开的部门不在应用可见范围内，则从应用的可见范围开始展开。 |
| object.mode | String | 是 | 选择模式，"single"表示单选，"multi"表示多选 |
| object.type | Array | 是 | 选择限制类型，指定"department"、"user"中的一个或者多个 |
| object.selectedDepartmentIds | Array | 否 | 已选部门ID列表。用于多次选人时可重入 |
| object.selectedUserIds | Array | 否 | 已选用户ID列表，仅自建应用使用，企业微信会忽略第三方应用填入的该字段。用于多次选人时可重入 |
| object.selectedOpenUserIds | Array | 否 | 已选用户ID列表，仅第三方应用使用，企业微信会忽略自建应用填入的该字段。用于多次选人时可重入 |
| object.selectedCorpGroupDepartmentIds | Array | 否 | 已选企业互联部门ID列表。用于多次选人时可重入 |
| object.selectedCorpGroupDepartmentIds.corpId | String | 是 | 已选企业互联的CorpId |
| object.selectedCorpGroupDepartmentIds.departmentId | String | 是 | 已选企业互联的部门ID |
| object.selectedCorpGroupUserIds | Array | 否 | 已选企业互联用户ID列表。用于多次选人时可重入 |
| object.selectedCorpGroupUserIds.corpId | String | 是 | 已选企业互联的CorpId |
| object.selectedCorpGroupUserIds.userId | String | 是 | 已选企业互联的用户userId，自建应用必填，企业微信会忽略第三方应用填入的该字段 |
| object.selectedCorpGroupUserIds.openUserId | String | 是 | 已选企业互联的用户openUserId，第三方应用必填，企业微信会忽略自建应用填入的该字段 |
| object.selectedChainDepartmentIds | Array | 否 | 已选上下游部门ID列表。用于多次选人时可重入 |
| object.selectedChainDepartmentIds.corpId | String | 是 | 已选上下游的CorpId |
| object.selectedChainDepartmentIds.departmentId | String | 是 | 已选上下游的部门ID |
| object.selectedChainUserIds | Array | 否 | 已选上下游用户ID列表。用于多次选人时可重入 |
| object.selectedChainUserIds.corpId | String | 是 | 已选上下游的CorpId |
| object.selectedChainUserIds.userId | String | 是 | 已选上下游的用户userId，自建应用必填，企业微信会忽略第三方应用填入的该字段 |
| object.selectedChainUserIds.openUserId | String | 是 | 已选上下游的用户openUserId，第三方应用必填，企业微信会忽略自建应用填入的该字段 |
| object.success | Function | 否 | 接口调用成功的回调函数 |
| object.fail | Function | 否 | 接口调用失败的回调函数 |
| object.complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```text
wx.qy.selectCorpGroupContact({
      fromDepartmentId: -1,
      mode: "single",
      type: ["department", "user"],
      selectedDepartmentIds: ["2","3"],
      selectedUserIds: ["lisi","lisi2"],
      selectedOpenUserIds: ["wabc3","wbcde"],
      selectedChainDepartmentIds: [
          {
              corpId:"ww3333",
              departmentId:"2"
          },
          {
              corpId:"ww4444",
              departmentId:"3"
          }
      ],
      selectedChainUserIds: [
          {
              corpId:"ww3333",
              userId:"userid123",
              openUserId:"wx1111"
          },
          {
              corpId:"ww4444",
              userId:"userid123",
              openUserId:"wx1111"
          }
      ],
      success: function(res) {
          // 回调函数内容
      }
});
```
