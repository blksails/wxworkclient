# 文件/文档标题展示组件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99348](https://developer.work.weixin.qq.com/document/path/99348)
- **文档 ID**: `99348`

## 接口描述

通过微盘选择文件或者文档选择文档返回的文件/文档url信息，可通过此展示组件，将对应的标题展示出来，以便用户快速识别所选择的文件/文档。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| type | string | 是 | 开放数据类型，此时固定为：fileName |
| fileName |  | 否 | 文件/文档标题名 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| openid | string | 数据ID，根据type取值而定 |

## 其他说明

### 权限说明

组件支持企业自建应用、代开发自建应用和第三方应用调用。

### 支持的元素属性

页面通过 open-data 元素上的属性控制需要渲染的内容：
属性类型必填说明typestring是开放数据类型，此时固定为：fileNameopenidstring是数据ID，根据type取值而定
type 的合法值
值说明fileName文件/文档标题名
type=fileName，此时 openid 对应 文件/文档url每 20ms 最多绑定 1000 个 open-data 元素，超出的部分将被忽略url错误，展示「链接错误」非本企业的文件/文档，且用户无浏览权限，展示「暂无权限查看该文档/文件」文件url，且应用无“微盘”权限，展示「暂无微盘应用权限」文档url，且应用无“文档”权限，展示「暂无文档应用权限」
