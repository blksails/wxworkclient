## 其他说明

### ww.setKeepScreenOn(params)

兼容性：企业微信 iOS、Android 4.0.20 开始支持
功能描述保持屏幕常亮。在企业微信内打开 H5 页面时，调用该接口让屏幕保持常亮。
ww.setKeepScreenOn({keepScreenOn:true,})
使用限制本接口必须使用应用身份进行注册成员必须在应用可见范围内
参数说明params: Object属性类型必填说明keepScreenOnboolean是是否保持常亮successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码
使用说明仅在当前页面生效，离开页面后设置失效。

### 功能描述

保持屏幕常亮。
在企业微信内打开 H5 页面时，调用该接口让屏幕保持常亮。

### 使用限制

本接口必须使用应用身份进行注册成员必须在应用可见范围内

### params: Object

属性类型必填说明keepScreenOnboolean是是否保持常亮successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 使用说明

仅在当前页面生效，离开页面后设置失效。

### 示例代码

wx.invoke('setKeepScreenOn',{"keepScreenOn":true,},function(res){});
