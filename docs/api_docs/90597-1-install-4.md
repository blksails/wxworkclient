# 从服务商网站发起

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90597](https://developer.work.weixin.qq.com/document/path/90597)
- **文档 ID**: `90597`
- **API 名称**: `install`
- **请求方法**: `GET`
- **接口地址**: `https://open.work.weixin.qq.com/3rdapp/install`
- **分组信息**: 第 1 个接口，共 2 个

## 接口描述

系统管理员在第三方服务商网站找到适用的应用后，可在服务商网站发起授权请求。第三方服务商需构造授权链接，引导用户进入授权页面完成授权过程，并取得临时授权码。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| suite_id | string | 是 | 套件ID |
| pre_auth_code | string | 是 | 预授权码 |
| redirect_uri | string | 是 | 授权完成回调URI |
| state | string | 是 | 用于第三方自行校验session，防止跨域攻击 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| auth_code | string | 临时授权码 |
| expires_in | int32 | 过期时间 |
| state | string | 参数值用于第三方自行校验session |
