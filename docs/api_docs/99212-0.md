# 购买接口许可支持关联代开发应用订单 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99212](https://developer.work.weixin.qq.com/document/path/99212)
- **文档 ID**: `99212`

## 接口描述

根据2023年3月14日《接口许可购买需关联应用订单》的通知，从2023年6月30日0:00起，合作伙伴购买接口许可需关联应用订单。企业微信已于5月中支持代开发应用下单，合作伙伴当前可为代开发应用客户进行应用下单，以免影响后续的接口许可购买。
