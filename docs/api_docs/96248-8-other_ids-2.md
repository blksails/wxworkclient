# 其他ID

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96248](https://developer.work.weixin.qq.com/document/path/96248)
- **文档 ID**: `96248`
- **API 名称**: `other_ids`
- **请求方法**: `GET`
- **分组信息**: 第 8 个接口，共 8 个

## 接口描述

除了corpid、userid、external_userid这三个最为基础的ID之外，还有其他区分企业主体与服务商主体的ID。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| customer_tag_id | string | 客户标签ID |
| wechat_customer_service_id | string | 微信客服ID |

## 其他说明

### 其他ID

除了corpid、userid、external_userid这三个最为基础的ID之外，其他区分企业主体与服务商主体的还有以下ID：客户标签ID、微信客服ID。
