# 创建家长

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92077](https://developer.work.weixin.qq.com/document/path/92077)
- **文档 ID**: `92077`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/user/create_parent`

## 接口描述

创建家长及其孩子的关系。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| parent_userid | string | 是 | 家长UserID。学校内必须唯一，可以与企业通讯录内成员UserID相同。不区分大小写，长度为1~64个字节。只能由数字、字母和“_-@.”四种字符组成，且第一个字符必须是数字或字母。 |
| mobile | string | 是 | 家长手机号 |
| to_invite | bool | 否 | 是否发起邀请，默认为true，仅验证的学校才能发起邀请。 |
| children | object[] | 是 | 家长的孩子列表，最多10 |
| children.student_userid | string | 是 | 学生的UserID |
| children.relation | string | 是 | 家长与学生的关系，最长32字节 |

### 请求示例

```json
{
	"parent_userid": "zhangsan_parent_userid",
    "mobile": "10000000000",
	"to_invite": false,
	"children":[
		{
			"student_userid": "zhangsan",
            "relation": "爸爸"
        },
        {
			"student_userid": "lisi",
            "relation": "伯父"
        }
     ]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "created"
}
```

## 其他说明

### 参数说明

access_token: 调用接口凭证。获取方法查看“获取access_token”
parent_userid: 家长UserID。学校内必须唯一，可以与企业通讯录内成员UserID相同。不区分大小写，长度为1~64个字节。只能由数字、字母和“_-@.”四种字符组成，且第一个字符必须是数字或字母。
mobile: 家长手机号
to_invite: 是否发起邀请，默认为true，仅验证的学校才能发起邀请。
children: 家长的孩子列表，最多10
children.student_userid: 学生的UserID
children.relation: 家长与学生的关系，最长32字节

### 权限说明

调用的应用需要满足如下的权限
自建应用: 配置到「家校沟通-读取和编辑家校通讯录的应用」中
代开发应用: 需拥有「家校沟通」可使用家校通讯录-家校通讯录编辑 权限
第三方应用: 需拥有「家校沟通」使用和编辑 权限
