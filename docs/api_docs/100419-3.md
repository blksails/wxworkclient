### ww.getBluetoothAdapterState([params])

兼容性：企业微信
功能描述获取本机蓝牙适配器状态。
ww.getBluetoothAdapterState()
参数说明params: Object属性类型必填说明successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码discoveringboolean是是否正在搜索设备availableboolean是蓝牙适配器是否可用

### 功能描述

获取本机蓝牙适配器状态。

### params: Object

属性类型必填说明successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码discoveringboolean是是否正在搜索设备availableboolean是蓝牙适配器是否可用

### 示例代码

wx.getBluetoothAdapterState({success:function(res){console.log(res)}})
