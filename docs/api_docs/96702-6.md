# 获取收款项目的商户单号 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96702](https://developer.work.weixin.qq.com/document/path/96702)
- **文档 ID**: `96702`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalpay/get_payment_info?access_token=ACCESS_TOKEN`

## 接口描述

获取对外收款项目中每笔收款的商户单号

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| payment_id |  | 是 | 收款项目单号。在发起对外收款中返回。 |
| 自建应用 | 自建应用 | 否 | 配置到「对外收款- 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 具有「对外收款」权限 |
| 第三方应用 | 第三方应用 | 否 | 具有「对外收款」权限 |

### 请求示例

```json
{"payment_id":"xxxx"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| out_trade_no_list |  | 收款单号列表。每笔支付对应一个收款单号 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","bill_list":[{"out_trade_no":"xxxx"},{"out_trade_no":"yyyy"}]}
```
