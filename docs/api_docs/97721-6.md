# 新增日程参与者 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97721](https://developer.work.weixin.qq.com/document/path/97721)
- **文档 ID**: `97721`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/oa/schedule/add_attendees?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于在日历中更新指定的日程参与者列表

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| schedule_id | string | 是 | 日程ID。创建日程时返回的ID |
| attendees | obj[] | 否 | 日程参与者列表。累计最多支持1000人 |

### 请求示例

```json
{"schedule_id":"17c7d2bd9f20d652840f72f59e796AAA","attendees":[{"userid":"userid2"}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| attendees.userid | string | 日程参与者ID不多于64字节 |
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
