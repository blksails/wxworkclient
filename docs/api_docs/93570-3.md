## 接口描述

获取当前客户群的群ID

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| 外部群聊工具栏 |  | 否 | 3.0.16 |
| 班级群的聊天工具栏 |  | 否 | 3.0.36 |
| 学生群的聊天工具栏 |  | 否 | 4.0.8 |

### 请求示例

```json
wx.qy.getCurExternalChat({success:function(res){varchatId=res.chatId//返回当前外部群的群聊ID}})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| chatId | String | 返回当前外部群的群聊ID |

## 其他说明

### wx.qy.getCurExternalChat(Object object)

调用前提：必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态当前成员必须在应用的可见范围。需要从特定入口进入页面才可调用，否则会报错：without context of external contact，可先通过调用wx.qy.getContext来判断进入小程序的入口，允许调用的入口说明参见“不同入口所需的权限说明”。从不同的入口进入的页面，应用需要满足相应的权限，否则会报“no permission”错误，所需的权限参见“不同入口所需的权限说明”。「营销获客」类应用仅可获取该应用带来的客户群
