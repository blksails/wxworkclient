# 删除客户端专用密码 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100184](https://developer.work.weixin.qq.com/document/path/100184)
- **文档 ID**: `100184`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/exmail/publicmail/delete_auth_code?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于删除公共邮箱的客户端专用密码

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | uint32 | 是 | 公共邮箱ID |
| auth_code_id | uint32 | 是 | 客户端专用密码ID |

### 请求示例

```json
{"id":1,"auth_code_id":1}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
