# 删除会议录制 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98206](https://developer.work.weixin.qq.com/document/path/98206)
- **文档 ID**: `98206`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/record/delete?access_token=ACCESS_TOKEN`

## 接口描述

删除会议的所有录制文件，该接口会删除会议录制 ID 里对应的所有云录制文件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meeting_record_id | string | 是 | 会议录制ID |
| meetingid | string | 是 | 会议ID |

### 请求示例

```json
{"meeting_record_id":"record","meetingid":"MEETINGID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
