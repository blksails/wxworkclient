# 应用异步调用专区程序

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99820](https://developer.work.weixin.qq.com/document/path/99820)
- **文档 ID**: `99820`
- **API 名称**: `53296`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/...`
- **分组信息**: 第 2 个接口，共 2 个

## 接口描述

开发者若要触发专区程序执行指定任务时，可通过应用异步调用专区程序发起程序调用。专区程序需要监听8080端口接收企微后台调用专区的请求。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| request_data | object | 是 | 请求数据，必须匹配当前能力设置的输入协议 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| response_data | object | 专区程序返回的结果，必须匹配当前能力设置的输出协议 |
