- **文档 ID**: `91529`

## 接口描述

监听NFC设备的消息回调。当用户离开小程序后，此接口无效。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| messageType | number | 否 | 消息类型 |
| 1 |  | 否 | 小程序在前台，当检测到 NFC 标签时回调 |
| 2 |  | 否 | 小程序退到后台，当用户不可见时，自动停止 NFC |

### 请求示例

```json
Page({onNFCReadMessage:function(e){console.log(JSON.stringify(e))if(1==e.messageType){wx.showModal({title:'NFC-Id',content:JSON.stringify(e),})}elseif(2==e.messageType){console.log('Auto-Stop')wx.showModal({title:'NFC-Auto-Stop',content:JSON.stringify(e),})}}})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| data.hex | string | 十六进制 id 序列 |
| data.reversed_hex | string | 字节反转十六进制 id 序列 |
| data.dec | number | 十进制 id 数值 |
| data.reversed_dec | number | 字节反转十进制 id 数值 |
