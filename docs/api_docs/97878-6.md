# 获取空间信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97878](https://developer.work.weixin.qq.com/document/path/97878)
- **文档 ID**: `97878`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/new_space_info?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于获取空间信息。包括：空间成员及权限及安全设置。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | Query参数，示例值: ACCESS_TOKEN |

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spaceid | string | 是 | 空间spaceid |

### 请求示例

```json
{"spaceid":"SPACEID"}
```
