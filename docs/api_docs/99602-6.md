# 获取代支付流水

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99602](https://developer.work.weixin.qq.com/document/path/99602)
- **文档 ID**: `99602`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/customer_acquisition/get_bill_list?suite_access_token=SUITE_ACCESS_TOKEN`

## 接口描述

服务商可通过此接口获取使用量代付流水。企业通过代付产生使用量后，次日可在获客助手中查看代付的订单记录，若需要与企业侧使用量统计周期一致，请按当天的0时0分01秒到第二天的0时0分0秒的时间戳查询。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| suite_access_token | string | 是 | 获客助手组件的应用凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| begin_time | int32 | 是 | 流水记录开始时间 |
| end_time | int32 | 是 | 流水记录结束时间 |
| auth_corpid | string | 是 | 授权企业corpid |
| cursor | string | 否 | 用于分页查询的游标，字符串类型，由上一次调用返回，首次调用可不填 |
| limit | int32 | 否 | 返回的最大记录数，默认值100，最大不超过1000 |

### 请求示例

```json
{
   "begin_time":1605171726,
   "end_time":1605172726,
   "auth_corpid": "xxxxxxxxx",
   "cursor":"CURSOR",
   "limit":10
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| next_cursor | string | 分页游标，在下次请求时填写以获取之后分页的记录，如果已经没有更多的数据则不返回该字段 |
| bill_list | object[] | 代支付流水记录列表 |
| bill_list[].timestamp | int32 | 消耗时间 |
| bill_list[].link_id | string | 获客链接id，获客链接已删除的情况不返回此字段 |
| bill_list[].state | string | 加好友时的state参数，为空则不返回 |
| bill_list[].price | int32 | 服务商代付金额，单位为分 |

### 响应示例

```json
{
   "errcode":0,
   "errmsg":"ok",
   "next_cursor":"CURSOR",
   "bill_list":[
      {
         "timestamp": 1605171726,
         "link_id": "xxxxxxx",
         "state": "REMARK",
		 "price":30000
      }
   ]
}
```
