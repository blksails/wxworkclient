# 关于ExternalUserId

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/101215](https://developer.work.weixin.qq.com/document/path/101215)
- **文档 ID**: `101215`
- **API 名称**: `external_user_id_info`
- **请求方法**: `GET`
- **接口地址**: `/cgi-bin/externalcontact/external_user_id_info`
- **分组信息**: 第 4 个接口，共 9 个

## 接口描述

ExternalUserId是企业微信外部联系人的标识，但需要开发者(尤其是第三方开发者)注意的是，对于同一个外部联系人，不同调用方（企业/第三方服务商）获取到的ExternalUserId是不同的。
