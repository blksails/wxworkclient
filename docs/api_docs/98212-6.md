# 获取录制转写段落信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98212](https://developer.work.weixin.qq.com/document/path/98212)
- **文档 ID**: `98212`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/record/transcript/get_paragraph_list?access_token=ACCESS_TOKEN`

## 接口描述

获取云录制转写的段落信息（段落总数、段落 ID）。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| record_file_id | string | 是 | 录制文件 ID。 |
| meetingid | string | 是 | 会议ID |

### 请求示例

```json
{"record_file_id":"record","meetingid":"MEETINGID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| audio_detect | int32 | 声纹识别状态0：未完成1：已完成说明：声纹识别是针对一台设备多人讲话场景，自动区分为多个发言人的能力。声纹识别与录制转写生成的过程独立。无需声纹识别或声纹识别已完成时，该值为1。 |
| paragraphs | object[] | 段落列表，详见Paragraph |
| pid | string | 段落 ID。 |
| start_time | int64 | 录制文件中的段落开始时间（毫秒）。 |
| end_time | int64 | 录制文件中的段落结束时间（毫秒）。 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","audio_detect":1,"paragraphs":[{"pid":"1111","start_time":111111111,"end_time":2222222222}]}
```
