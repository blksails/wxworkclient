### 请求示例

#### 示例 1: 功能描述

```json
ww.onGetWifiList(function(event){console.log(event)})
```

#### 示例 2: 示例代码

```json
wx.onGetWifiList(function(res){if(res.wifiList.length){for(vari=0;i<res.wifiList.length;i++){console.log("=====wifiList======")console.log("SSID:"+res.wifiList[i].SSID)console.log("BSSID:"+res.wifiList[i].BSSID)console.log("secure:"+res.wifiList[i].secure)console.log("secure:"+res.wifiList[i].signalStrength)console.log("====================")}}})
```

## 其他说明

### ww.onGetWifiList(callback)

兼容性：企业微信 Android 2.4.16 开始支持
功能描述监听 Wi-Fi 列表更新。
ww.onGetWifiList(function(event){console.log(event)})
参数说明callback: Function参数event: Object属性类型必填说明wifiListObject[]是Wi-Fi 列表数据属性类型必填说明SSIDstring是Wi-Fi 的 SSIDBSSIDstring是Wi-Fi 的 BSSIDsecureboolean是Wi-Fi 是否安全signalStrengthnumber是Wi-Fi 信号强度
