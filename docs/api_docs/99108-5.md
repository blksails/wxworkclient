# 获取应用权限详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99108](https://developer.work.weixin.qq.com/document/path/99108)
- **文档 ID**: `99108`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/agent/get_permissions`

## 接口描述

该API用于获取代开发应用或第三方应用用户授权的权限详情。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 代开发自建应用access_token，获取方法参见获取access_token 第三方应用access_token，获取方法参见获取企业凭证 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误描述 |
| app_permissions | string[] | 需要添加的权限列表，应用权限为字符串类型，应用权限对应的字符串描述参考附录。 |

### 响应示例

```json
{
	"errcode":0 ,
	"errmsg":"ok",
	"app_permissions":["xxx::xxx::xx"]
}
```

## 其他说明

### 权限说明

支持所有的代开发应用和第三方应用调用
