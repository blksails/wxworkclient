# 应用管理员变更通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95038](https://developer.work.weixin.qq.com/document/path/95038)
- **文档 ID**: `95038`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`

## 接口描述

当授权方在企业微信管理端修改了应用负责人，通过该接口通知服务商。服务商接收到变更通知之后，需自行调用获取应用的管理员列表进行授权内容变更比对。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 | 数据回调url |
| timestamp | string | 是 | 时间戳 |
| nonce | string | 是 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName | string | 是 | 企业微信CorpID |
| FromUserName | string | 是 | 固定为sys |
| CreateTime | int32 | 是 | 消息创建时间（整型） |
| MsgType | string | 是 | 消息类型，固定为event |
| Event | string | 是 | 事件类型，固定为change_app_admin |
| AgentID | int32 | 是 | 企业应用的id，整型。可在应用的设置页面查看 |

### 请求示例

```xml
<xml>
	<ToUserName><![CDATA[toUser]]></ToUserName>
	<FromUserName><![CDATA[sys]]></FromUserName>
	<CreateTime>1348831860</CreateTime>
	<MsgType><![CDATA[event]]></MsgType>
	<Event><![CDATA[change_app_admin]]></Event>
	<AgentID>1</AgentID>
</xml>
```
