# 下载直播回放

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93827](https://developer.work.weixin.qq.com/document/path/93827)
- **文档 ID**: `93827`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wxqy/download_living_replay`

## 接口描述

调起直播回放下载页面

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| livingId | string | 是 | 直播ID |
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```json
wx.qy.downloadLivingReplay({
	"livingId": "LIVINGID",
	fail: function(res) {
		console.log(res)
	},
});
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| res.errMsg | string | 错误信息，如果调用成功返回 ${apiName}:ok |

### 响应示例

```json
| res.errMsg | 说明 |
| --- | --- |
| qy.downloadLivingReplay:ok | 执行成功 |
| qy.downloadLivingReplay:fail no session or expired | 没有调用 wx.qy.login，或者 session_key 已过期 |
| qy.downloadLivingReplay:fail no permission | 应用需具有直播权限 |
| qy.downloadLivingReplay:fail user not in allow list | 当前成员不在应用可见范围 |
| qy.downloadLivingReplay:fail invalid living id | 不合法的直播ID |
| qy.downloadLivingReplay:fail not allow to cross corp | 不可跨企业使用直播ID |
| qy.downloadLivingReplay:fail not allow to cross app | 不可跨应用使用直播ID |
| qy.downloadLivingReplay:fail invalid parameter | 参数不合法 |
| qy.downloadLivingReplay:fail living has no replay | 不存在直播回放 |
| qy.downloadLivingReplay:fail replay is beging creating | 正在直播中，或回放正在生成中，稍后观看回放 |
| qy.downloadLivingReplay:fail create replay failed | 回放创建失败 |
| qy.downloadLivingReplay:fail invalid operator | 只允许直播的发起人下载直播回放 |
```

## 其他说明

### 权限说明

此接口将调起直播回放下载页面

* 此接口仅在企业微信3.1.0及以后版本支持，且目前仅PC端支持
* 必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态
* 当前成员必须在应用的可见范围
* 应用需具有直播使用权限。见[配置可使用直播的应用](#25967/%E9%85%8D%E7%BD%AE%E5%8F%AF%E4%BD%BF%E7%94%A8%E7%9B%B4%E6%92%AD%E7%9A%84%E5%BA%94%E7%94%A8)
* 只允许直播的发起人下载直播回放
