# 添加会议背景 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98851](https://developer.work.weixin.qq.com/document/path/98851)
- **文档 ID**: `98851`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/layout/add_background?access_token=ACCESS_TOKEN`

## 接口描述

对成功预定的会议添加会议背景，支持多个背景图片的添加。一场会议最多添加7个背景，且仅支持不超过10MB大小的 PNG 格式图片，分辨率最小为1920x1080。添加成功返回新增的会议背景信息。背景图片上传方式为异步上传，您可以通过订阅“素材上传结果”获取上传结果通知。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议 ID |
| image_list | object[] | 是 | 图片对象列表，详见Image |
| default_image_order | uint32 | 否 | 图片列表中会议需要使用的背景图片序号，从1开始计数。不填默认为1 |
| image_md5 | string | 是 | 背景图片 MD5（图片内容 MD5 的十六进制表示） |
| image_url | string | 是 | 背景图片 URL |

### 请求示例

```json
{"meetingid":"MEETINGID","image_list":[{"image_md5":"eax879asd3z","image_url":"https://image.qq.com/1jsaz.png"}],"default_image_order":1}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| selected_background_id | string | 会议应用的背景 ID |
| background_list | object[] | 背景对象列表，详见Background |
| background_id | string | 背景 ID |
| image_md5 | string | 背景图片 MD5（图片内容 MD5 的十六进制表示） |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","selected_background_id":"BACKID1","background_list":[{"background_id":"BACKID1","image_md5":"eax879asd3z"}]}
```
