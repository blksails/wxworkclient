# 订单管理接口

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100261](https://developer.work.weixin.qq.com/document/path/100261)
- **文档 ID**: `100261`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/advanced_api/get_order?provider_access_token=ACCESS_TOKEN`

## 接口描述

查询某个订单的详情。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 应用服务商的接口调用凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| order_id | string | 是 | 订单id |

### 请求示例

```json
{
	"order_id":"xxxxx"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| order | object | 参考Order结构说明 |
| order.advanced_api_type | int32 | 购买的高级接口类型，目前仅支持“会话内容数据接口”，type为1 |
| order.order_id | string | 订单号 |
| order.order_type | int32 | 订单类型：0-新购、1-增购、2-续期、3-升级 |
| order.order_status | int32 | 订单状态：0：待支付，1：已支付，2：已取消 3：未支付，订单已过期，4：申请退款中，5：退款成功，6：退款被拒绝 |
| order.custom_corpid | string | 客户企业id，返回加密的corpid |
| order.create_time | int32 | 订单创建时间 |
| order.buyer_userid | string | 下单用户id |
| order.paid_price | int32 | 应付金额 |
| order.chat_archive_api | object | 当extendapi_type为1时，返回该字段，参考ChatArchiveApi结构说明 |
| order.chat_archive_api.edition | int32 | 2-内外部会话，3-内外部会话及语音通话 |
| order.chat_archive_api.purchase_count | int32 | 购买人数 |
| order.chat_archive_api.purchase_duration_days | int32 | 使用时长，单位为天 |
| order.chat_archive_api.take_effect_time | int32 | 生效时间 |
| order.chat_archive_api.end_time | int32 | 到期时间 |
| order.chat_archive_api.original_price | int32 | 原价金额，目前仅升级订单返回 |
