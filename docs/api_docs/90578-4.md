# 创建推广二维码

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90578](https://developer.work.weixin.qq.com/document/path/90578)
- **文档 ID**: `90578`
- **请求方法**: `POST`
- **接口地址**: `https://open.work.weixin.qq.com/3rdservice/wework/register?register_code=REGISTER_CODE`

## 接口描述

根据不同的企业需求，可以将不同应用及设置生成一个推广二维码，用户扫码就能注册企业微信并默认安装预置的第三方应用。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| app_list[] | string | 是 | 预装的第三方应用和企业微信基础应用 |
| display_mode | string | 是 | 工作台的显示方式 |
| enable_address_book_migration | bool | 是 | 是否开启通讯录迁移接口 |
| enable_all_staff_group | bool | 是 | 是否开启全员群 |
| enable_auto_department_group | bool | 是 | 是否开启自动创建部门群 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |

## 其他说明

### 配置流程

企业微信对应用安装提供了用户自主授权安装的机制，但是当用户没有企业微信时，用户需要先注册企业微信再安装第三方应用，对于用户流程比较繁琐。基于这种情况，企业微信提供推广二维码功能，服务商只需要先配置好对应的推广二维码（即一系列应用及企业初始化设置），下载生成的二维码，用户扫码就能注册企业微信并默认安装预置的第三方应用。企业微信也提供了注册按钮，可以利用推广二维码生成的参数拼装出注册链接，用户点击该注册链接即可在注册完企业微信并默认安装预置的第三方应用。

### 使用推广二维码

推广二维码可以放在任何的网页和H5图文中，用户通过微信或企业微信都可以扫描二维码，没有企业微信的用户会被引导下载、安装、注册企业微信，注册成功后，预置的应用默认安装在工作台；有企业微信的用户会被引导到应用添加流程。

### 用户扫码注册后的流程

用户在注册页面提交资料完成企业微信的注册流程。企业微信端注册成功之后回调一个“注册完成事件”给服务商，同时判断通讯录迁移接口是否开启，若开启，则锁定用户通讯录。服务商侧接收到回调事件可以做自己的账户体系初始化等操作。

### Web页面放置推广二维码的交互建议

1. 注册链接的打开使用浏览器新标签页的方式，让用户在一个新的页面进行注册。 2. 然后在原页面弹层等待用户（类似PC端网银支付的流程）。 3. 当用户注册完成之后，企业微信端会回调通知服务商，用户已经注册完成，并且带上这个用户的标识和信息。 4. 服务商接收到注册完成通知之后进行页面的刷新即可较为流畅地进入到服务商自身的后台。 5. 用户也可以在新页面选择同时进入企业微信管理后台

### 注册按钮

除了提供二维码的注册方式之外，企业微信还提供注册按钮的推广方式。
