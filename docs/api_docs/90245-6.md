# 创建群聊会话 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90245](https://developer.work.weixin.qq.com/document/path/90245)
- **文档 ID**: `90245`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/appchat/create?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/appchat/create?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| name |  | 否 | 群聊名，最多50个utf8字符，超过将截断 |
| owner |  | 否 | 指定群主的id。如果不指定，系统会随机从userlist中选一人作为群主 |
| userlist |  | 是 | 群成员id列表。至少2人，至多2000人 |
| chatid |  | 否 | 群聊的唯一标志，不能与已有的群重复；字符串类型，最长32个字符。只允许字符0-9及字母a-zA-Z。如果不填，系统会随机生成群id |

### 请求示例

```json
{"name":"NAME","owner":"userid1","userlist":["userid1","userid2","userid3"],"chatid":"CHATID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| chatid |  | 群聊的唯一标志 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","chatid":"CHATID"}
```
