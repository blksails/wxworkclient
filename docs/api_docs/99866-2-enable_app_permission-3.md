# 如何接入

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99866](https://developer.work.weixin.qq.com/document/path/99866)
- **文档 ID**: `99866`
- **API 名称**: `enable_app_permission`
- **请求方法**: `PUT`
- **接口地址**: `https://yourdomain.com/enable_app_permission`
- **分组信息**: 第 2 个接口，共 5 个

## 接口描述

开启应用权限，勾选数据与智能专区权限。

## 请求信息

### 请求示例

```json
{
  "permission": "data_intelligence"
}
```

## 其他说明

### 注意

在第三方应用或代开发应用的权限详情页中勾选开启“数据与智能专区权限”。
