# 获取家校访问用户身份 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95791](https://developer.work.weixin.qq.com/document/path/95791)
- **文档 ID**: `95791`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/getuserinfo?access_token=ACCESS_TOKEN&code=CODE`

## 接口描述

该接口用于根据code获取家长或者学生信息

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| code |  | 是 | 通过成员授权获取到的code，最大为512字节。每次成员授权带上的code将不一样，code只能使用一次，5分钟未被使用自动过期。 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| DeviceId |  | 手机设备号(由企业微信在安装时随机生成，删除重装会改变，升级不受影响) |
| parent_userid |  | 家校通讯录里家长的userid |
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| DeviceId |  | 手机设备号(由企业微信在安装时随机生成，删除重装会改变，升级不受影响) |
| student_userid |  | 家校通讯录里学生的userid |

### 响应示例

#### 示例 1: 返回结果：

```json
{"errcode":0,"errmsg":"ok","DeviceId":"DEVICEID","parent_userid":"parent_userid"}
```

#### 示例 2: 返回结果：

```json
{"errcode":0,"errmsg":"ok","DeviceId":"DEVICEID","student_userid":"student_userid"}
```

#### 示例 3: 出错返回示例：

```json
{"errcode":40029,"errmsg":"invalid code"}
```
