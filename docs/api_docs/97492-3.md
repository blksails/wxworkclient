### ww.startMeeting(params)

兼容性：企业微信 3.1.0 开始支持
功能描述进入会议通过传入 meetingId 进入已有会议。
ww.startMeeting({meetingId:'MEETINGID'})
使用限制本接口必须使用应用身份进行注册应用必须具有会议使用权限，参考配置可使用会议的应用用户需要在应用可见范围内
参数说明params: Object属性类型必填说明meetingIdstring否会议 ID，可通过创建预约会议或通过调用ww.startMeeting()获取
为空的情况下创建快速会议successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明meetingIdstring是会议 ID
错误信息errMsg说明startMeeting:ok执行成功startMeeting:fail no permission应用签名校验失败，或成员不在应用的可见范围内，或应用未开启会议使用权限startMeeting:fail user not in allow list会议发起人必须在应用可见范围startMeeting:fail not allow to cross corp不可跨企业使用会议IDstartMeeting:fail invalid meeting id不合法的会议IDstartMeeting:fail not allow to cross app不可跨应用使用会议IDstartMeeting:fail invalid parameter参数不合法

### 功能描述

进入会议
通过传入 meetingId 进入已有会议。

### 使用限制

本接口必须使用应用身份进行注册应用必须具有会议使用权限，参考配置可使用会议的应用用户需要在应用可见范围内

### params: Object

属性类型必填说明meetingIdstring否会议 ID，可通过创建预约会议或通过调用ww.startMeeting()获取
为空的情况下创建快速会议successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明meetingIdstring是会议 ID

### 错误信息

errMsg说明startMeeting:ok执行成功startMeeting:fail no permission应用签名校验失败，或成员不在应用的可见范围内，或应用未开启会议使用权限startMeeting:fail user not in allow list会议发起人必须在应用可见范围startMeeting:fail not allow to cross corp不可跨企业使用会议IDstartMeeting:fail invalid meeting id不合法的会议IDstartMeeting:fail not allow to cross app不可跨应用使用会议IDstartMeeting:fail invalid parameter参数不合法
