# 编辑收集表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97824](https://developer.work.weixin.qq.com/document/path/97824)
- **文档 ID**: `97824`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/modify_form?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于编辑收集表。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| oper | uint32 | 是 | 操作类型。1：全量修改问题；2：全量修改设置 |
| formid | string | 是 | 收集表id |
| form_info.form_title | string | 否 | 收集表标题（操作1修改） |
| form_info.form_desc | string | 否 | 收集表描述（操作1修改） |
| form_info.form_header | string | 否 | 收集表表头背景图链接（操作1修改） |
| form_info.form_question.items | object[] | 是 | 问题数组 |
| form_info.form_question.items[].question_id | uint32 | 是 | 问题id，从1开始。如果是家校范围收集表，id从2开始。 |
| form_info.form_question.items[].title | string | 是 | 问题描述 |
| form_info.form_question.items[].pos | uint32 | 是 | 问题序号，从1开始。 |
| form_info.form_question.items[].status | uint32 | 是 | 问题状态。1：正常；2：被删除 |
| form_info.form_question.items[].reply_type | uint32 | 是 | 问题类型。1：文本；2：单选；3：多选；5：位置；9：图片；10：文件；11：日期；14：时间；15：下拉列表；16：体温；17：签名；18：部门；19：成员 22：时长 |
| form_info.form_question.items[].must_reply | bool | 是 | 是否必答 |
| form_info.form_question.items[].note | string | 否 | 问题备注 |
| form_info.form_question.items[].placeholder | string | 否 | 编辑提示 |
| form_info.form_question.items[].question_extend_setting | object | 否 | 问题的额外设置。不同问题类型有相应的设置，详见[question_extend_setting字段描述](#43942/question_extend_setting) |
| form_info.form_question.items[].option_item | object[] | 是 | 单选/多选/下拉列表题的选项列表 |
| form_info.form_question.items[].option_item[].key | uint32 | 是 | 选项key（1，2，3...） |
| form_info.form_question.items[].option_item[].value | string | 是 | 选项内容 |
| form_info.form_question.items[].option_item[].status | uint32 | 是 | 选项状态。1：正常；2：被删除 |
| form_info.form_setting.fill_out_auth | uint32 | 是 | 填写权限。0：所有人；1：企业内指定人/部门。若收集表当前为家校范围，则无法修改。 |
| form_info.form_setting.fill_in_range | object | 否 | 指定的可填写的人/部门 |
| form_info.form_setting.fill_in_range.userids | string[] | 否 | 企业成员userid列表 |
| form_info.form_setting.fill_in_range.departmentids | uint64[] | 否 | 部门id列表 |
| form_info.form_setting.setting_manager_range | object | 否 | 收集表管理员 |
| form_info.form_setting.timed_repeat_info | object | 否 | 定时重复设置项 |
| form_info.form_setting.timed_repeat_info.enable | bool | 否 | 是否开启定时重复 |
| form_info.form_setting.timed_repeat_info.remind_time | uint32 | 否 | 提醒时间 |
| form_info.form_setting.timed_repeat_info.repeat_type | uint32 | 否 | 重复类型。0：每周；1：每天；2：每月 |
| form_info.form_setting.timed_repeat_info.week_flag | uint32 | 否 | 每周几重复，只能repeat_type = 0 时填写。1：星期一；2：星期二；4：星期三；8：星期四；16：星期五；32：星期六；64：星期日 |
| form_info.form_setting.timed_repeat_info.skip_holiday | bool | 否 | 自动跳过节假日，只能repeat_type = 1 时填写。 |
| form_info.form_setting.timed_repeat_info.day_of_month | uint32 | 否 | 每月的第几天（1 - 31），只能repeat_type = 2时填写 |
| form_info.form_setting.timed_repeat_info.fork_finish_type | uint32 | 否 | 是否允许补填。0：允许；1：仅当天；2：最后五天内；3：一个月内；4：下一次生成前 |
| form_info.form_setting.allow_multi_fill | bool | 否 | 是否允许每人提交多份。默认false |
| form_info.form_setting.timed_finish | uint32 | 否 | 定时关闭。定时重复与定时结束互斥，若都填，优先定时重复 |
| form_info.form_setting.can_anonymous | bool | 否 | 是否支持匿名填写。默认false |
| form_info.form_setting.can_notify_submit | bool | 否 | 是否有回复时提醒。默认false |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok"
}
```
