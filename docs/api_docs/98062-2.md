# 创建文档 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98062](https://developer.work.weixin.qq.com/document/path/98062)
- **文档 ID**: `98062`

## 接口描述

此接口用于新建一个文档、表格、收集表、幻灯片或智能表格。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docType | Uint32 | 是 | 文档类型，3: 文档，4: 表格，5: 收集表，6: 幻灯片，10: 智能表格，其他类型暂不支持 |
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| ios |  | 否 |  |
| android |  | 否 |  |
| win |  | 否 |  |
| mac |  | 否 | >10.12 |
| 自建应用 | 自建应用 | 否 | 需具有“文档”权限 |
| 第三方应用 | 第三方应用 | 否 | 需具有“文档”权限 |
| 代开发应用 | 代开发应用 | 否 | 需具有“文档”权限 |

### 请求示例

```json
wx.qy.createDoc({docType:1,success:function(res){// 回调}});
```
