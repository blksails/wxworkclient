# 获取客户群详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96338](https://developer.work.weixin.qq.com/document/path/96338)
- **文档 ID**: `96338`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/groupchat/get`

## 接口描述

通过客户群ID，获取详情。包括群名、群成员列表、群成员入群时间、入群方式。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| chat_id | string | 是 | 客户群ID |
| need_name | int32 | 否 | 是否需要返回群成员的名字`group_chat.member_list.name`。0-不返回；1-返回。默认不返回 |

### 请求示例

```json
{
    "chat_id":"wrOgQhDgAAMYQiS5ol9G7gK9JVAAAA",
	"need_name" : 1
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| group_chat | object | 客户群详情 |
| group_chat.chat_id | string | 客户群ID |
| group_chat.name | string | 群名 |
| group_chat.owner | string | 群主ID |
| group_chat.create_time | uint32 | 群的创建时间 |
| group_chat.notice | string | 群公告 |
| group_chat.member_list | object[] | 群成员列表 |
| group_chat.member_list.userid | string | 群成员id |
| group_chat.member_list.type | int32 | 成员类型。 1 - 企业成员 2 - 外部联系人 |
| group_chat.member_list.unionid | string | 外部联系人在微信开放平台的唯一身份标识（微信unionid） |
| group_chat.member_list.join_time | uint32 | 入群时间 |
| group_chat.member_list.join_scene | int32 | 入群方式。 1 - 由群成员邀请入群（直接邀请入群） 2 - 由群成员邀请入群（通过邀请链接入群） 3 - 通过扫描群二维码入群 |
| group_chat.member_list.invitor | object | 邀请者。目前仅当是由**本企业内部成员**邀请入群时会返回该值 |
| group_chat.member_list.invitor.userid | string | 邀请者的userid |
| group_chat.member_list.group_nickname | string | 在群里的昵称 |
| group_chat.member_list.name | string | 名字。**仅当 need_name = 1 时返回** 如果是微信用户，则返回其在微信中设置的名字 如果是企业微信联系人，则返回其设置对外展示的别名或实名 |
| group_chat.admin_list | object[] | 群管理员列表 |
| group_chat.admin_list.userid | string | 群管理员userid |
| group_chat.member_version | string | 当前群成员版本号。可以配合客户群变更事件减少主动调用本接口的次数 |

## 其他说明

### 权限说明

企业需要使用配置到“可调用应用”列表中的自建应用secret所获取的accesstoken来调用。第三方应用需具有“企业客户权限->客户基础信息”权限。对于第三方/自建应用，群主必须在应用的可见范围。
