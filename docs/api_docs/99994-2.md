# 产生会话回调通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99994](https://developer.work.weixin.qq.com/document/path/99994)
- **文档 ID**: `99994`

## 接口描述

当企业下新产生消息时，企业微信专区后台会向具有企业会话内容权限的应用关联的程序推送相应的事件消息。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| event_type |  | 否 | 事件类型，此时固定为conversation_new_message |
| timestamp |  | 否 | 时间戳 |
| token |  | 否 | 10分钟内有效，用于调用获取会话记录，不超过128字节 |

### 请求示例

```json
{"event_type":"conversation_new_message","timestamp":1408091189,"conversation_new_message":{"token":"zzzzzzzzzzz"}}
```
