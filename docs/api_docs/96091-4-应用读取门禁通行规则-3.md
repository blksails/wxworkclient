# 获取硬件产生的数据

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96091](https://developer.work.weixin.qq.com/document/path/96091)
- **文档 ID**: `96091`
- **API 名称**: `应用读取门禁通行规则`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/accesscontrol/get_access_rules`
- **分组信息**: 第 4 个接口，共 7 个

## 接口描述

应用读取门禁通行规则接口用于获取企业下绑定的门禁设备的通行规则设置。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| access_rules | object[] | 通行规则列表 |
| access_rules[].rule_id | string | 规则ID |
| access_rules[].time_range | object | 通行时间范围 |
| access_rules[].location | string | 通行地点 |
