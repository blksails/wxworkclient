# 删除门禁通行规则

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96227](https://developer.work.weixin.qq.com/document/path/96227)
- **文档 ID**: `96227`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/devicedata/del_accesscontrol_rule`

## 接口描述

第三方应用向企业微信删除企业已经存在的门禁规则。注意：第三方应用须具有“设备接口信息权限-创建门禁规则”授权；企业须同时安装硬件设备和第三方应用；第三方应用只能删除自己创建的门禁规则

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 应用调用接口权限 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| rule_id | string | 是 | 企业微信返回的rule_id，门禁规则的标识id |

### 请求示例

```json
{
    "rule_id":"xxx"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码描述 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok"
}
```
