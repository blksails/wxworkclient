# 返回 ticket 的选人接口

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94516](https://developer.work.weixin.qq.com/document/path/94516)
- **文档 ID**: `94516`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

返回 ticket 的选人接口。用于第三方应用唤起选择企业通讯录成员，用户选择的范围区分成两部分回传给第三方应用。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fromDepartmentId | number | 是 | 从指定的部门开始展示。-1 表示从自己所在部门开始，0 表示从最上层开始 |
| mode | string | 是 | 选择模式。合法值：single（单选）、multi（多选） |
| selectedContextContact | boolean | 是 | 是否勾选当前环境的参与者，例如在群「+」号打开，默认勾选当前群成员 |
| selectedOpenUserIds | string | 否 | 已选用户 OpenID 列表，仅在 mode 为 multi 时支持。企业微信 3.1.12 |
| selectedTickets | string[] | 否 | 已选 Ticket 列表，仅在 mode 为 multi 时支持。企业微信 3.1.12 |
| success | Function | 否 | 成功回调 |
| fail | Function | 否 | 失败回调 |
| cancel | Function | 否 | 取消回调 |
| complete | Function | 否 | 完成回调 |

### 请求示例

```text
wx.invoke("selectPrivilegedContact", {
	"fromDepartmentId": -1,
	"mode": "multi",
	"selectedContextContact": 1,
	"selectedOpenUserIds": ["xxx", "yyy"],
	"selectedTickets": ["ticket1", "ticket2"]
}, function(res) {
	if (res.err_msg == "selectPrivilegedContact:ok") {
		var selectedUserList = res.result.userList;
		for (var i = 0; i < selectedUserList.length; i++) {
			var openUserId = selectedUserList[i].openUserId;
		}
		var selectedTicket = res.result.selectedTicket;
		var expiresIn = res.result.expiresIn;
		var selectedUserCount = res.result.selectedUserCount;
	}
});
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errMsg | string | 通用错误信息 |
| errCode | number | 通用错误码 |
| result | Object | 当调用成功时返回，对应用户选择的内容 |
| result.selectedTicket | string | 对应用户的选择集合凭证 |
| result.expiresIn | number | ticket 的有效时长，单位为秒，默认为 7 天 |
| result.selectedUserCount | string | 选择的用户数 |
| result.userList | Object[] | 选择的可见范围内的用户列表 |
| result.userList.openUserId | string | 成员 openUserId |
