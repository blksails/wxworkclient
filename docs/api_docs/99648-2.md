# 会议发起事件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99648](https://developer.work.weixin.qq.com/document/path/99648)
- **文档 ID**: `99648`

## 接口描述

当应用可见范围内的成员发起快速会议时或者作为首位参与者进入预约会议时，触发该事件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 会议发起者的UserID。如果是企业外部成员作为首位参与者进入预约会议，则该字段为此预约会议创建者的UserID |
| CreateTime |  | 否 | 消息创建时间，unix时间戳 |
| MsgType |  | 否 | 消息类型，此时固定为：event |
| Event |  | 否 | 事件类型，此时固定为：meeting_statistics |
| ChangeType |  | 否 | 类型，此时固定为：start_meeting |
| Status |  | 否 | 会议状态1:会议发起成功2:会议发起失败 |

### 请求示例

```xml
<xml><ToUserName><![CDATA[toUser]]></ToUserName><FromUserName><![CDATA[fromUser]]></FromUserName><CreateTime>1348831860</CreateTime><MsgType><![CDATA[event]]></MsgType><Event><![CDATA[meeting_statistics]]></Event><ChangeType><![CDATA[start_meeting]]></ChangeType<Status>1</Status></xml>
```
