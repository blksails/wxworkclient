# 获取配置了客户联系功能的成员列表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96310](https://developer.work.weixin.qq.com/document/path/96310)
- **文档 ID**: `96310`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/get_follow_user_list`

## 接口描述

企业和第三方服务商可通过此接口获取配置了客户联系功能的成员列表。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| follow_user | string[] | 配置了客户联系功能的成员userid列表 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "ok",
   "follow_user":[
    	"zhangsan",
		"lissi"
   ]
}
```

## 其他说明

### 权限说明

调用接口的应用需要满足如下的权限：

- 自建应用：配置到「客户联系 可调用接口的应用」中
- 代开发应用：具有「客户基础信息」权限
- 第三方应用：具有「客户基础信息」权限

应用仅可获取可见范围内的成员。
