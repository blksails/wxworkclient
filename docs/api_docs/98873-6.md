# 获取网络研讨会成员报名 ID - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98873](https://developer.work.weixin.qq.com/document/path/98873)
- **文档 ID**: `98873`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/webinar/enroll/query_by_tmp_openid?access_token=ACCESS_TOKEN`

## 接口描述

支持获取网络研讨会中已报名成员的报名 ID。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 网络研讨会ID。 |
| sorting_rules | uint32 | 否 | 查询报名 ID 的排序规则。当该账号存在多条报名记录（手机号导入、手动报名等）时，该接口返回的顺序。1：优先查询手机号导入报名，再查询成员手动报名，默认值。2：优先查询成员手动报名，再查手机号导入。 |
| tmp_openid_list | string[] | 是 | 当场会议的成员临时 ID（适用于所有成员）数组，单次最多支持500条。 |

### 请求示例

```json
{"meetingid":"MEETINGID","sorting_rules":1,"tmp_openid_list":["agaqwwgdkkddd"]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| enroll_id_list | object[] | 成员报名 ID 数组，仅返回已报名成员的报名 ID，若传入的成员无人报名，则无该字段。详见EnrollID。 |
| tmp_openid | string | 当场会议的成员临时ID，适用于所有成员。 |
| enroll_id | string | 报名ID。 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","enroll_id_list":[{"tmp_openid":"agaqwwgdkkddd","enroll_id":"mexxxxxxxx"}]}
```
