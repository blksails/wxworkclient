# 微信客服ID的转换

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97110](https://developer.work.weixin.qq.com/document/path/97110)
- **文档 ID**: `97110`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/idconvert/open_kfid?access_token=ACCESS_TOKEN`

## 接口描述

将企业主体下的微信客服ID转换成服务商主体下的微信客服ID。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证，第三方应用access_token或代开发应用access_token |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| open_kfid_list | array | 是 | 微信客服ID列表，最多不超过1000个 |

### 请求示例

```json
{
  "open_kfid_list":["KFID1","KFID2","KFID3","KFID4"]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| items | object[] | 微信客服ID转换结果 |
| items.open_kfid | string | 企业主体下的微信客服ID |
| items.new_open_kfid | string | 服务商主体下的微信客服ID，如果传入的open_kfid已经是服务商主体下的ID，则new_open_kfid与open_kfid相同。 |
| invalid_open_kfid_list | array | 无法转换的微信客服ID列表 |

### 响应示例

```json
{
 "errcode":0,
 "errmsg":"ok",
 "items":[
 	{
			"open_kfid":"KFID1",
			"new_open_kfid":"NEW_KFID1"
	},
	{
			"open_kfid":"KFID2",
			"new_open_kfid":"NEW_KFID2"
	},
	"invalid_open_kfid_list":["KFID3","KFID4"]
}
```
