## 接口描述

API创建的日历，仅允许管理员对该日历进行删除/修改信息/修改通知范围等操作。为了让企业实时获取用户的操作，当用户进行了某种操作后，企业微信会将相应的事件推送给日历所属应用配置的回调URL（自建应用见配置日历回调参数，第三方应用见[指令回调URL]()）。
