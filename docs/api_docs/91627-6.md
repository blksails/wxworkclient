# 发起语音电话 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91627](https://developer.work.weixin.qq.com/document/path/91627)
- **文档 ID**: `91627`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/pstncc/call?access_token=ACCESS_TOKEN`

## 接口描述

通过此接口发起语音电话，提醒员工查看应用推送的重要消息。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| callee_userid |  | 是 | 需要呼叫的列表 |
| 自建应用 | 自建应用 | 否 | 配置到「紧急通知 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"callee_userid":["james","paul"]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| states |  | 自动语音来电呼叫状态列表 |
| userid |  | 用户id |
| callid |  | 唯一标识一通呼叫的id |
| code |  | 呼叫结果状态：0成功发起呼叫，非0则失败 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","states":[{"code":0,"callid":"6-20190510201844181887818-4d0251082406000-out","userid":"james"},{"code":0,"callid":"6-20190510201844181887818-4d025109f806000-out","userid":"paul"}]}
```
