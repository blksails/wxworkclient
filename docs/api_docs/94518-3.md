## 其他说明

### ww.openExistedChatWithMsg(params)

兼容性：企业微信 3.1.8 开始支持
功能描述打开已有群聊并可选发送一条链接消息（link消息）。支持打开企业内部群、外部群、互联群。
ww.openExistedChatWithMsg({chatId:'chatId123',msg:{msgtype:'link',link:{title:'title1',desc:'desc1',url:'link1',imgUrl:'imgurl1'}}})
使用限制本接口必须使用应用身份进行注册
参数说明params: Object属性类型必填说明最低版本chatIdstring是打开的群聊 IDmsgObject否发送的链接消息。不传则表示仅进入群聊，不发消息企业微信 iOS、Android 3.1.8，企业微信 PC、Mac 4.0.2属性类型必填说明msgtype"link"是消息类型linkObject是属性类型必填说明titlestring否H5消息标题imgUrlstring否H5消息封面图片URLdescstring否H5消息摘要urlstring是H5消息页面urlsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码
错误信息errMsg说明openExistedChatWithMsg:ok执行成功openExistedChatWithMsg:fail_unsupported_msgtypemsgtype不合法openExistedChatWithMsg:fail_msg_link_missing_urlmsg.link.url未传入

### 功能描述

打开已有群聊并可选发送一条链接消息（link消息）。支持打开企业内部群、外部群、互联群。

### 使用限制

本接口必须使用应用身份进行注册

### params: Object

属性类型必填说明最低版本chatIdstring是打开的群聊 IDmsgObject否发送的链接消息。不传则表示仅进入群聊，不发消息企业微信 iOS、Android 3.1.8，企业微信 PC、Mac 4.0.2属性类型必填说明msgtype"link"是消息类型linkObject是属性类型必填说明titlestring否H5消息标题imgUrlstring否H5消息封面图片URLdescstring否H5消息摘要urlstring是H5消息页面urlsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 错误信息

errMsg说明openExistedChatWithMsg:ok执行成功openExistedChatWithMsg:fail_unsupported_msgtypemsgtype不合法openExistedChatWithMsg:fail_msg_link_missing_urlmsg.link.url未传入
