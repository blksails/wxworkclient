# 如何购买

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100008](https://developer.work.weixin.qq.com/document/path/100008)
- **文档 ID**: `100008`
- **API 名称**: `purchase_data`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/purchase_data?access_token=ACCESS_TOKEN`
- **分组信息**: 第 13 个接口，共 13 个

## 接口描述

企业授权服务商应用「数据与智能专区权限」30天内，服务商应用可免费使用企业会话数据进行分析和展示；免费使用时间到期后，服务商需购买企业会话内容数据接口。

## 其他说明

### 定价方案

按照会话内容数据的账号及时长收费，一个账号可用于分析和展示一个员工的会话数据；不同类型的会话内容数据定价不同。
