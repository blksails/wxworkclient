# 客户同意进行聊天内容存档事件回调 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92005](https://developer.work.weixin.qq.com/document/path/92005)
- **文档 ID**: `92005`

## 接口描述

配置了客户联系功能的成员添加外部联系人同意进行聊天内容存档时，回调该事件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 此事件该值固定为sys，表示该消息由系统生成 |
| CreateTime |  | 否 | 消息创建时间 （整型） |
| MsgType |  | 否 | 消息的类型，此时固定为event |
| Event |  | 否 | 事件的类型，此时固定为change_external_contact |
| ChangeType |  | 否 | 此时固定为msg_audit_approved |
| UserID |  | 否 | 企业服务人员的UserID |
| ExternalUserID |  | 否 | 外部联系人的userid，注意不是企业成员的账号 |
| WelcomeCode |  | 否 | 欢迎语code，可用于发送欢迎语 |
