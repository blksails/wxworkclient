# wx.qy.updateCorpGroupChat - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96972](https://developer.work.weixin.qq.com/document/path/96972)
- **文档 ID**: `96972`

## 接口描述

变更企业互联/上下游群成员接口

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| chatId | String | 是 | 指定的群会话id |
| userIdsToAdd | Array | 否 | 新增的企业成员列表，仅自建应用使用，第三方应用会忽略该字段 |
| openUserIdsToAdd | Array | 否 | 新增的企业成员列表，仅第三方应用使用，自建应用会忽略该字段 |
| corpGroupUserIdsToAdd | Array | 否 | 参与会话的企业互联/上下游成员列表 |
| updateCorpGroupChat:ok |  | 否 | 执行成功 |
| updateCorpGroupChat:fail no session or expired |  | 否 | 没有调用 wx.qy.login，或者 session_key 已过期 |
| updateCorpGroupChat:fail exceed user id list size |  | 否 | 超过人数上限 |
| updateCorpGroupChat:fail invalid parameter |  | 否 | 参数不合法 |
| updateCorpGroupChat:fail unsupported chat |  | 否 | 不支持群类型，目前仅支持企业互联/上下游群 |

### 请求示例

```json
wx.qy.updateCorpGroupChat({chatId:"CHATID",//通过企业微信创建群聊接口返回的chatIduserIdsToAdd:["lisi","lisi2"],//新增的企业成员列表，仅自建应用使用，第三方应用会忽略该字段openUserIdsToAdd:["wabc3","wbcde"],// 新增的企业成员列表，仅第三方应用使用，自建应用会忽略该字段corpGroupUserIdsToAdd:[// 新增的企业互联/上下游成员列表{corpId:"ww3333",// 企业CORPIDuserId:"userid123",// 成员ID，仅自建应用使用openUserId:"wx1111"// 成员OPEN_USERID，仅第三方应用使用},{corpId:"ww4444",// 企业CORPIDuserId:"userid123",// 成员ID，仅自建应用使用openUserId:"wx1111"// 成员OPEN_USERID，仅第三方应用使用}],success:(res)=>{console.log(JSON.stringify(res))},fail:(res)=>{console.log(JSON.stringify(res))}})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| corpGroupUserIdsToAdd.corpId | String | 企业互联/上下游企业corpId |
| corpGroupUserIdsToAdd.userId | String | 企业互联/上下游企业成员userId, 仅自建应用返回 |
| corpGroupUserIdsToAdd.openUserId | String | 企业互联/上下游企业成员openUserId，仅第三方应用使用返回 |
| success | function | 接口调用成功的回调函数 |
| fail | function | 接口调用失败的回调函数 |
