# 获取数据与智能专区授权信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100237](https://developer.work.weixin.qq.com/document/path/100237)
- **文档 ID**: `100237`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/get_corp_auth_info?access_token=ACCESS_TOKEN`

## 接口描述

第三方服务商应用可通过该接口获取企业数据与智能专区的授权详情。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | String | 是 | 调用接口凭证 |
| 自建应用 | 自建应用 | 否 | 不支持 |
| 代开发应用 | 代开发应用 | 否 | 需具备「数据与智能专区」权限 |
| 第三方应用 | 第三方应用 | 否 | 需具备「数据与智能专区」权限 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| auth_edition_list |  | 企业授权的会话版本。如果授权了多个会话版本，则返回授权的多个会话版本。 |
| auth_edition_list.edition |  | 企业授权的会话版本类型1-内部会话 2-内外部会话 3-内外部会话及语音通话 |
| auth_edition_list.auth_scope.userid_list |  | 企业授权存档的人员列表。注意，实际生效人员（参见获取授权存档的成员列表）取决于服务商购买的人数 |
| auth_edition_list.auth_scope.department_id_list |  | 企业授权存档的部门列表。注意，实际生效人员（参见获取授权存档的成员列表）取决于服务商购买的人数 |
| auth_edition_list.auth_scope.tag_id_list |  | 企业授权存档的标签列表。注意，实际生效人员（参见获取授权存档的成员列表）取决于服务商购买的人数 |
| auth_edition_list.status |  | 会话版本的当前状态1-试用中2-试用已到期3-付费使用中4-付费使用已到期5-免费使用中6-免费使用已到期7-付费待生效 |
| auth_edition_list.begin_time |  | 如果版本状态是试用中、试用已到期，即为试用的开始生效时间；付费使用中/付费使用已到期、免费使用中/免费使用已到期同理。如果版本状态是付费待生效，则为付费版本的预期开始生效时间。所有版本共用同一个试用期，如果是试用期已到期后授权且未购买的版本，则此字段不返回 |
| auth_edition_list.end_time |  | 如果版本状态是试用中、试用已到期，即为试用的结束生效时间；付费使用中/付费使用已到期、免费使用中/免费使用已到期同理。如果版本状态是付费待生效，则为付费版本的预期结束生效时间。所有版本共用同一个试用期，如果是试用期已到期后授权且未购买的版本，则此字段不返回 |
| auth_edition_list.msg_duration_days |  | 会话授权的时长，单位：天（一年按365天计） |
| auth_edition_list.auth_user_count |  | 企业授权存档的去重人数 |
