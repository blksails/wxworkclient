# 删除收集表事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98055](https://developer.work.weixin.qq.com/document/path/98055)
- **文档 ID**: `98055`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/callback/delete_form_event?access_token=ACCESS_TOKEN`

## 接口描述

当文档管理员删除API创建的收集表后，触发该事件。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName | string | 是 | 企业微信CorpID |
| FromUserName | string | 是 | 成员UserID |
| CreateTime | uint32 | 是 | 消息创建时间，unix时间戳 |
| MsgType | string | 是 | 消息类型，固定为：`event` |
| Event | string | 是 | 事件类型，固定为：`doc_change` |
| ChangeType | string | 是 | 事件类型，固定为：`delete_form` |
| FormId | string | 是 | 收集表ID列表 |

### 请求示例

```xml
<xml>
   <ToUserName><![CDATA[toUser]]></ToUserName>
   <FromUserName><![CDATA[fromUser]]></FromUserName>
   <CreateTime>1348831860</CreateTime>
   <MsgType><![CDATA[event]]></MsgType>
   <Event><![CDATA[doc_change]]></Event>
   <ChangeType><![CDATA[delete_form]]></ChangeType>
   <FormId><![CDATA[wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA]]></FormId>
   <FormId><![CDATA[wcjgewCwAAqeJcPI1d8Pwbjt7nttzBBB]]></FormId>
</xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
