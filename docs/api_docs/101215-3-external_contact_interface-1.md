# 使用客户联系相关接口

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/101215](https://developer.work.weixin.qq.com/document/path/101215)
- **文档 ID**: `101215`
- **API 名称**: `external_contact_interface`
- **请求方法**: `GET`
- **分组信息**: 第 3 个接口，共 9 个

## 接口描述

企业通过配置过权限的应用调用外部联系人选人相关接口，如：获取成员选择的外部联系人的userid以及通过该userid获取外部联系人详情。
