# 导出成员详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96294](https://developer.work.weixin.qq.com/document/path/96294)
- **文档 ID**: `96294`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/export/user`

## 接口描述

企业通讯录安全特别重要，企业微信将持续升级加固通讯录接口的安全机制。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| encoding_aeskey | string | 是 | Base64编码后的加密密钥。长度固定为43，从a-z, A-Z, 0-9共62个字符中选取，是AESKey的Base64编码。解码后即为32字节长的AESKey。加密方式采用AES-256-CBC方式，数据采用PKCS#7填充至32字节的倍数；IV初始向量大小为16字节，取AESKey前16字节。 |
| block_size | uint32 | 否 | 每块数据的人员数，支持范围[104,106]，默认值为106 |

### 请求示例

```json
{
	"encoding_aeskey": "IJUiXNpvGbODwKEBSEsAeOAPAhkqHqNCF6g19t9wfg2",
	"block_size": 1000000
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| jobid | string | 任务ID，可通过获取导出结果接口查询任务结果 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "ok",
   "jobid": "jobid_xxxxxxxxxxxxxxx"
}
```

## 其他说明

### 权限说明

仅会返回有权限的人员列表
