# 修改预约直播

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93720](https://developer.work.weixin.qq.com/document/path/93720)
- **文档 ID**: `93720`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/living/modify`

## 接口描述

修改预约直播。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| livingid | string | 是 | 直播id，仅允许修改预约状态下的直播id |
| theme | string | 否 | 直播的标题，最多支持60个字节 |
| living_start | uint32 | 否 | 直播开始时间的unix时间戳 |
| living_duration | uint32 | 否 | 直播持续时长 |
| type | uint32 | 否 | 直播的类型，0：通用直播，1：小班课，2：大班课，3：企业培训，4：活动直播。其中大班课和小班课仅k12学校和IT行业类型能够发起 |
| description | string | 否 | 直播的简介，最多支持300个字节 |
| remind_time | uint32 | 否 | 指定直播开始前多久提醒用户，相对于living_start前的秒数，默认为0 |

### 请求示例

```json
{
   "livingid": "XXXXXXXXX",
   "theme": "theme",
   "living_start": 1600100000,
   "living_duration": 3600,
   "description": "test description",
   "type": 1,
   "remind_time": 60
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "ok"
}
```
