## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| mch_id |  | 是 | 微信支付商户号,不超过32字节 |
| 自建应用 | 自建应用 | 否 | 配置到「对外收款- 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |
| mch_id |  | 是 | 微信支付商户号,不超过32字节 |
| allow_use_scope |  | 是 | 该商户号使用范围，支持按成员、部门或标签设置。 |
| 自建应用 | 自建应用 | 否 | 配置到「对外收款- 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

#### 示例 1: 请求示例：

```json
{"mch_id":"12334"}
```

#### 示例 2: 请求示例：

```json
{"mch_id":"12334","allow_use_scope":{"user":["zhangsan","lisi"],"partyid":[1],"tagid":[1,2,3]}}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| mch_id |  | 微信支付商户号,不超过32字节 |
| merchant_name |  | 微信支付商户号全称，不超过50字节 |
| bind_status |  | 商户号绑定状态。 1:申请中 2:已绑定 3:已撤销 |
| allow_use_scope |  | 该商户号使用范围，仅当商户号已绑定时才返回 |
| allow_use_scope.user |  | 使用范围内的企业用户id列表 |
| allow_use_scope.partyid |  | 使用范围内的部门id列表 |
| allow_use_scope.tagid |  | 使用范围内的标签id列表 |
| allow_use_scope.user |  | 使用范围内的企业用户id列表 |
| allow_use_scope.partyid |  | 使用范围内的部门id列表 |
| allow_use_scope.tagid |  | 使用范围内的标签id列表 |
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |

### 响应示例

通过该接口可以查询已绑定商户号信息以及商户号使用范围
请求方式：POST(HTTPS)请求地址：https://qyapi.weixin.qq.com/cgi-bin/externalpay/getmerchant?access_token=ACCESS_TOKEN
请求示例：
{"mch_id":"12334"}
参数说明：
参数必须说明mch_id是微信支付商户号,不超过32字节
权限说明：调用的应用需要满足如下的权限
应用类型权限要求自建应用配置到「对外收款- 可调用接口的应用」中代开发应用暂不支持第三方应用暂不支持
注：从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响查看详情

参数说明：
参数说明errcode返回码errmsg对返回码的文本描述内容mch_id微信支付商户号,不超过32字节merchant_name微信支付商户号全称，不超过50字节bind_status商户号绑定状态。 1:申请中 2:已绑定 3:已撤销allow_use_scope该商户号使用范围，仅当商户号已绑定时才返回allow_use_scope.user使用范围内的企业用户id列表allow_use_scope.partyid使用范围内的部门id列表allow_use_scope.tagid使用范围内的标签id列表

### 设置商户号使用范围

通过该接口可以设置已绑定商户号的使用范围
请求方式：POST(HTTPS)请求地址：https://qyapi.weixin.qq.com/cgi-bin/externalpay/set_mch_use_scope?access_token=ACCESS_TOKEN
请求示例：
{"mch_id":"12334","allow_use_scope":{"user":["zhangsan","lisi"],"partyid":[1],"tagid":[1,2,3]}}
参数说明：
参数必须说明mch_id是微信支付商户号,不超过32字节allow_use_scope是该商户号使用范围，支持按成员、部门或标签设置。allow_use_scope.user否使用范围内的企业用户id列表allow_use_scope.partyid否使用范围内的部门id列表allow_use_scope.tagid否使用范围内的标签id列表
权限说明：
应用类型权限要求自建应用配置到「对外收款- 可调用接口的应用」中代开发应用暂不支持第三方应用暂不支持
注：从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响查看详情
返回结果：
{"errcode":0,"errmsg":"ok"}
参数说明：
参数说明errcode返回码errmsg对返回码的文本描述内容
