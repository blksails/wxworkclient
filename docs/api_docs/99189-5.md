# wx.qy.wedriveSelectFile

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99189](https://developer.work.weixin.qq.com/document/path/99189)
- **文档 ID**: `99189`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

调用该接口，可唤起微盘选择器窗口，选择一个或多个微盘文件或文档。选中后返回对应文件的url。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| selectedFileNum | number | 是 | 选择文件的数量。1表示单选，大于1表示多选，上限为50 |
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```text
wx.qy.wedriveSelectFile({
	selectedFileNum: 1,
	success: function(res) {
		var selectedFileUrls = res.result.selectedFileUrls;  // 后续废弃
	    var selectedFileInfos = res.result.selectedFileInfos;
	},
	fail: function(res) {
		console.log(res)
	},
});
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| res.errMsg | string | 执行结果。"wedriveSelectFile:ok" 表示执行成功，否则表示报错信息 |
| res.result | Obj | 返回的数据 |
| res.result.selectedFileUrls | Array<string> | 选择的文件的 url 列表。后续废弃，请使用selectedFileInfos |
| res.result.selectedFileInfos | Array<object> | 选择的文件信息列表。企业微信版本 ≥4.1.8 才返回 |
| res.result.selectedFileInfos[].url | string | 选择的文件url |
| res.result.selectedFileInfos[].type | uint32 | 选择的文件类型。0: 其他，2: 文件，3: 文档，4: 表格，5: 收集表，6: 幻灯片，7: 思维导图，8: 流程图，10: 智能表格 |
