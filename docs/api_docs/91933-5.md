# 应用版本回调通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91933](https://developer.work.weixin.qq.com/document/path/91933)
- **文档 ID**: `91933`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`

## 接口描述

服务商收到付费版本变更通知后，需要通过获取企业授权信息接口拉取最新的版本信息。付费版本变更通知在多种情况下会收到。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 | 消息签名 |
| timestamp | string | 是 | 时间戳 |
| nonce | string | 是 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 第三方应用的SuiteId |
| PaidCorpId | string | 是 | 购买方corpid |
| InfoType | string | 是 | 固定为 change_editon |
| TimeStamp | string | 是 | 时间戳 |

### 请求示例

```xml
<xml>
    <SuiteId><![CDATA[ww4asffe99e54c0aaa]]></SuiteId>
    <PaidCorpId><![CDATA[wxf8b4f85f3a794aaa]]></PaidCorpId>
    <InfoType><![CDATA[change_editon]]></InfoType>
    <TimeStamp>1403610513</TimeStamp>
</xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
