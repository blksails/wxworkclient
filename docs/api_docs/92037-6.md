# 批量创建学生

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92037](https://developer.work.weixin.qq.com/document/path/92037)
- **文档 ID**: `92037`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/user/batch_create_student`

## 接口描述

批量创建学生接口，每次最多可创建100个学生。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| students | object[] | 是 | 学生列表，每次最多100个学生 |
| students[].student_userid | string | 是 | 学生UserID。学校内必须唯一。不区分大小写，长度为1~64个字节。只能由数字、字母和“_-@.”四种字符组成，且第一个字符必须是数字或字母。 |
| students[].mobile | string | 否 | 学生手机号 |
| students[].to_invite | bool | 否 | 是否发起邀请，默认为true |
| students[].name | string | 是 | 学生姓名，长度为1~32个字符 |
| students[].department | uint32[] | 是 | 学生所在的班级id列表,不超过20个 |

### 请求示例

```json
{
	"students":[
        {
		"student_userid": "zhangsan",
		"mobile": "18000000000",
		"to_invite": false,
		"name": "张三",
		"department": [1, 2]
		},
        {
		"student_userid": "lisi",
		"mobile": "18000000000",
		"name": "李四",
		"department": [3, 4]
		}
     ]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| result_list | object[] | 失败的学生列表 |
| result_list[].student_userid | string | 失败的学生userid |
| result_list[].errcode | int32 | 返回码 |
| result_list[].errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
	"errcode": 1,
	"errmsg": "invalid student_userid: zhangsan",
	"result_list": [
		{
			"student_userid": "zhangsan",
			"errcode": 1,
			"errmsg": "invalid student_userid: zhangsan"
		}
	]
}
```

## 其他说明

### 权限说明

调用的应用需要满足如下的权限

- 自建应用：配置到「家校沟通-读取和编辑家校通讯录的应用」中
- 代开发应用：需拥有「家校沟通」可使用家校通讯录-家校通讯录编辑 权限
- 第三方应用：需拥有「家校沟通」使用和编辑 权限

*注：* 从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 查看详情
