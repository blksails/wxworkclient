# 获取应用管理员列表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100073](https://developer.work.weixin.qq.com/document/path/100073)
- **文档 ID**: `100073`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/agent/get_admin_list`

## 接口描述

第三方服务商可以用此接口获取授权企业中某个第三方应用或者代开发应用的管理员列表(不包括外部管理员)，以便服务商在用户进入应用主页之后根据是否管理员身份做权限的区分。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 代开发自建应用access_token，获取方法参见获取access_token 第三方应用access_token，获取方法参见获取企业凭证 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码，0表示正常返回，可读取admin的管理员列表 |
| errmsg | string | 错误说明 |
| admin | object[] | 应用的管理员列表（不包括外部管理员）。成员授权模式下，不返回管理员列表 |
| admin[].userid | string | 管理员的userid |
| admin[].auth_type | int32 | 该管理员对应用的权限：0=发消息权限，1=管理权限 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "admin":[
        {"userid":"woxxxx","auth_type":1},
        {"userid":"woxxxx","auth_type":0}
    ]
}
```
