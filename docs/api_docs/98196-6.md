# 获取会议录制地址 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98196](https://developer.work.weixin.qq.com/document/path/98196)
- **文档 ID**: `98196`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/record/get_file_list?access_token=ACCESS_TOKEN`

## 接口描述

获取会议录制地址，可获取会议云录制的播放地址和下载地址。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meeting_record_id | string | 是 | 会议录制ID |
| meetingid | string | 是 | 会议ID |

### 请求示例

```json
{"meeting_record_id":"record","meetingid":"MEETINGID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| meeting_record_id | string | 会议录制 ID。 |
| meetingid | string | 会议 ID。 |
| meeting_code | string | 会议 code。 |
| title | string | 会议主题。 |
| record_files | object[] | 直播录制文件列表，详见RecordFile |
| record_file_id | string | 录制文件 ID。 |
| view_address | string | 播放地址。 |
| download_address | string | 下载地址，默认6小时过期 |
| download_address_file_type | string | 下载视频文件格式，例如：mp4。 |
| audio_address | string | 音频下载地址，默认6小时过期 |
| audio_address_file_type | string | 下载音频文件格式，例如：m4a。 |
| meeting_summary | object[] | 会议纪要文件列表，详见Summary |
| download_address | string | 下载文件地址，默认1小时过期 |
| file_type | string | 下载文件类型，例如：txt、pdf、docs。 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","meeting_record_id":"record","meetingid":"MEETINGID","meeting_code":"MEETINGOCDE","title":"meetingtitle","record_files":[{"record_file_id":"FILEID","view_address":"https://view.url/124","download_address":"https://download.url/234","download_address_file_type":"mp4","audio_address":"https://audio.download/342","audio_address_file_type":"m4a","meeting_summary":[{"download_address":"https://summary.url","file_type":"txt"}]}]}
```
