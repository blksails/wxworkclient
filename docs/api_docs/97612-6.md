# 停止发表企业朋友圈 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97612](https://developer.work.weixin.qq.com/document/path/97612)
- **文档 ID**: `97612`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/cancel_moment_task?access_token=ACCESS_TOKEN`

## 接口描述

企业和第三方应用可调用此接口，停止尚未发送的企业朋友圈发送任务。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| moment_id |  | 是 | 朋友圈id，可通过获取客户朋友圈企业发表的列表接口获取朋友圈企业发表的列表 |

### 请求示例

```json
{"moment_id":"momXXXXXXXXXX"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
