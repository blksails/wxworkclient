# 获取网格列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94481](https://developer.work.weixin.qq.com/document/path/94481)
- **文档 ID**: `94481`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/report/grid/list?access_token=ACCESS_TOKEN`

## 接口描述

可以查询目标网格的父网格及全部子网格ID列表

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。“获取access_token” |

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| grid_id | string | 否 | 网格id，不填则表示拉取根节点及其儿子节点 |
| 自建应用 | 自建应用 | 否 | 配置到「居民联系 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"grid_id":"grid_id"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| grid_list.grid_id | string | 网格id |
| grid_list.grid_name | int | 网格名称 |
| grid_list.grid_parent_id | string | 网格父节点的网格id |
| grid_list.grid_admin | array[string] | 该节点的管理员列表 |
| grid_list.grid_member | array[string] | 该节点的成员列表 |
| grid_list | array[object] | (从响应示例中推测) |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","grid_list":[{"grid_id":"grid_id","grid_name":1,"grid_parent_id":"grid_id","grid_admin":["zhangsan"],"grid_member":["lisi","11111"]}]}
```
