## 接口描述

获取系统信息

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | function | 否 | 接口调用成功的回调函数 |
| fail | function | 否 | 接口调用失败的回调函数 |
| complete | function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| brand | string | 否 | 设备品牌 |
| model | string | 否 | 设备型号 |
| pixelRatio | number | 否 | 设备像素比 |
| screenWidth | number | 否 | 屏幕宽度，单位px |
| screenHeight | number | 否 | 屏幕高度，单位px |
| windowWidth | number | 否 | 可使用窗口宽度，单位px |
| windowHeight | number | 否 | 可使用窗口高度，单位px |
| statusBarHeight | number | 否 | 状态栏的高度，单位px |
| language | string | 否 | 微信设置的语言 |
| version | string | 否 | 微信版本号 |
| system | string | 否 | 操作系统及版本 |
| platform | string | 否 | 客户端平台 |
| fontSizeSetting | number | 否 | 用户字体大小（单位px）。以微信客户端「我-设置-通用-字体大小」中的设置为准 |
| SDKVersion | string | 否 | 客户端基础库版本 |
| benchmarkLevel | number | 否 | 设备性能等级（仅Android小游戏）。取值为：-2 或 0（该设备无法运行小游戏），-1（性能未知），>=1（设备性能值，该值越高，设备性能越好，目前最高不到50） |
| environment | string | 否 | 小程序当前运行环境 |

### 请求示例

#### 示例 1: 示例代码

```json
wx.getSystemInfo({success(res){console.log(res.model)console.log(res.pixelRatio)console.log(res.windowWidth)console.log(res.windowHeight)console.log(res.language)console.log(res.version)console.log(res.platform)console.log(res.environment)}})
```

#### 示例 2: 示例代码

```json
try{constres=wx.getSystemInfoSync()console.log(res.model)console.log(res.pixelRatio)console.log(res.windowWidth)console.log(res.windowHeight)console.log(res.language)console.log(res.version)console.log(res.platform)console.log(res.environment)}catch(e){// Do something when catch error}
```

## 其他说明

### wx.getSystemInfo(Object object)

小程序可以在微信和企业微信中调用此接口，但是在企业微信中调用此接口时，会额外返回一个 environment 字段（微信中不返回），如此字段值为 wxwork，则表示当前小程序运行在企业微信环境中。注意：只有运行在企业微信环境中，才能调用wx.qy的接口。
获取系统信息
