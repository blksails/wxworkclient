# 获取企业凭证

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91201](https://developer.work.weixin.qq.com/document/path/91201)
- **文档 ID**: `91201`
- **API 名称**: `获取企业凭证`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/gettoken?corpid=ID&permanent_code=CODE`
- **分组信息**: 第 3 个接口，共 3 个

## 接口描述

用于操作授权企业相关接口，如通讯录管理，消息推送等。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| corpid | string | 是 | 企业CorpID |
| permanent_code | string | 是 | 永久授权码 |

### 请求示例

```text
POST https://qyapi.weixin.qq.com/cgi-bin/gettoken?corpid=ID&permanent_code=CODE
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
