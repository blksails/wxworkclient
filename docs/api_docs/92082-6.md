# 批量更新家长

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92082](https://developer.work.weixin.qq.com/document/path/92082)
- **文档 ID**: `92082`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/user/batch_update_parent`

## 接口描述

用于批量更新家长信息，包括家长UserID、手机号、孩子信息等。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| parents | object[] | 否 | 家长列表，每次最多100个 |
| parents[].parent_userid | string | 是 | 家长UserID。学校内必须唯一。不区分大小写，长度为1~64个字节。只能由数字、字母和“_-@.”四种字符组成，且第一个字符必须是数字或字母。 |
| parents[].new_parent_userid | string | 否 | 更新的家长UserID。不能与已经存在的家长userid相同。每个家长仅能更新一次。 |
| parents[].mobile | string | 否 | 家长手机号 |
| parents[].children | object[] | 否 | 家长的孩子列表，最多10个 |
| parents[].children[].student_userid | string | 是 | 学生的UserID |
| parents[].children[].relation | string | 是 | 家长与学生的关系，最长32字节 |

### 请求示例

```json
{
    "parents":[
        {
            "parent_userid": "zhangsan_baba",
            "new_parent_userid":"zhangsan_baba_new",
            "mobile": "10000000000",
            "children":[
                {
                    "student_userid": "zhangsan",
                    "relation": "爸爸"
                }
            ]
        },
        {
            "parent_userid": "lisi_mama",
            "mobile": "10000000001",
            "children":[
                {
                    "student_userid": "lisi",
                    "relation": "妈妈"
                }
            ]
        }
    ]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| result_list | object[] | 失败的学生列表 |
| result_list[].parent_userid | string | 失败的家长userid |
| result_list[].errcode | int32 | 返回码 |
| result_list[].errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
	"errcode": 1,
	"errmsg": "invalid parent_userid: lisi_parent_userid",
	"result_list": [
		{
			"parent_userid": "lisi_parent_userid",
			"errcode": 1,
			"errmsg": "invalid parent_userid: lisi_parent_userid"
		}
	]
}
```
