# 获取布局模板列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98844](https://developer.work.weixin.qq.com/document/path/98844)
- **文档 ID**: `98844`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/layout/list_template?access_token=ACCESS_TOKEN`

## 接口描述

获取企业下所有的布局模板列表。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| layout_template_list | object[] | 布局模板对象列表。详见LayoutTemplate |
| layout_template_id | string | 布局模板 ID。 |
| thumbnail_url | string | 缩略图 URL。 |
| picture_url | string | 布局图 URL（布局图中包含宫格 ID，该 ID 用于确定布局宫格和成员座次的映射关系，设置和修改会议布局时需要当做入参传递）。 |
| render_rule | string | 渲染规则（提供成员个人渲染布局图片的规则说明）。 |
