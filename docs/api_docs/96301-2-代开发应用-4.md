# 代开发应用

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96301](https://developer.work.weixin.qq.com/document/path/96301)
- **文档 ID**: `96301`
- **API 名称**: `代开发应用`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webhook/callback/update?access_token=ACCESS_TOKEN`
- **分组信息**: 第 2 个接口，共 3 个

## 接口描述

代开发应用会默认回调通讯录变更事件。事件回调到代开发应用回调URL上。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
