# 如何接入

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100008](https://developer.work.weixin.qq.com/document/path/100008)
- **文档 ID**: `100008`
- **API 名称**: `install_app_authorize`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/install_app_authorize?access_token=ACCESS_TOKEN`
- **分组信息**: 第 3 个接口，共 13 个

## 接口描述

企业安装服务商的第三方应用或代开发应用后，可授权服务商「数据与智能专区权限」。

## 其他说明

### 注意

应用安装时默认不授权「数据与智能专区权限」，企业需要在安装后进行二次授权。
