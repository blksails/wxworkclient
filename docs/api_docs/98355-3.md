# 成员离开等候室事件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98355](https://developer.work.weixin.qq.com/document/path/98355)
- **文档 ID**: `98355`

## 接口描述

当与会者进入启用等候室的会议时，主持人可允许与会者入会，也可将与会者移出等候室。对于API创建的会议，当主持人将与会者移出等候室或与会者主动离开等候室时，触发该事件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 操作者UserID。如果为系统触发事件，默认为sys；如果为成员操作事件，仅当是企业成员时有值。 |
| FromUserTmpOpenId |  | 否 | 操作者TmpOpenID。操作者的会中临时ID，仅成员操作事件有值。 |
| OperatedUser |  | 否 | 被操作者 |
