# 关于可见客户列表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96350](https://developer.work.weixin.qq.com/document/path/96350)
- **文档 ID**: `96350`
- **API 名称**: `获取客户朋友圈发表时选择的可见范围`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/get_visible_customer`
- **分组信息**: 第 2 个接口，共 3 个

## 接口描述

可见客户列表是指当前内容可在微信朋友圈中看到的客户列表，不代表最终可见的客户列表。比如某成员有5000位客户，发表内容时选择了3000位可见客户可见，则可见的客户列表为3000位客户。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| visible_customers | string[] | 可见客户列表 |
