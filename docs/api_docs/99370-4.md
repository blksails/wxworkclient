# 了解「微信客服组件」

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99370](https://developer.work.weixin.qq.com/document/path/99370)
- **文档 ID**: `99370`
- **请求方法**: `GET`
- **接口地址**: `https://open.work.weixin.qq.com/compApp/wwxxxxxxxx`

## 接口描述

申请接入微信客服组件后，企业可授权在你的场景中接入微信客服。授权后，该场景的用户可通过微信客服咨询企业，你可管理对应的客服咨询。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| state | string | 否 | 企业授权接入客服组件时的回调参数 |
| mode | string | 否 | 授权模式，1表示每次仅可选择一个客服账号进行授权 |

### 请求示例

```text
https://open.work.weixin.qq.com/compApp/wwxxxxxxxx?state=xxxxxx
https://open.work.weixin.qq.com/compApp/wwxxxxxxxx?mode=1
```

## 其他说明

### 更多说明

开发者可以在上文授权流程中所述的授权链接拼上指定参数，实现对应功能。a）拼接参数state，当企业授权接入客服组件时，授权成功通知或变更授权通知将回调该参数给开发者，开发者可依据该参数，明确知道企业是要在哪里接入客服。例: https://open.work.weixin.qq.com/compApp/wwxxxxxxxx?state=xxxxxx b）拼接上参数mode=1，当企业授权接入客服组件时，每次仅可选择一个客服账号进行授权 例: https://open.work.weixin.qq.com/compApp/wwxxxxxxxx?mode=1
