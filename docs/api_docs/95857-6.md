# 新建空间

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95857](https://developer.work.weixin.qq.com/document/path/95857)
- **文档 ID**: `95857`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/space_create`

## 接口描述

该接口用于在微盘内新建空间，创建者为应用本身。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| space_name | string | 是 | 空间标题 |
| auth_info | object[] | 否 | 空间其他成员信息 |
| auth_info[].type | uint32 | 否 | 成员类型 1:个人 2:部门 |
| auth_info[].userid | string | 否 | 成员userid,字符串 |
| auth_info[].departmentid | uint32 | 否 | 部门departmentid, 32位整型范围是[0, 2^32) |
| auth_info[].auth | uint32 | 否 | 成员权限 1:仅下载 4:可预览（仅专业版微盘企业可设置） 7:应用空间管理员(最多可指定3个，不支持设置部门) |
| space_sub_type | uint32 | 否 | 区分创建空间类型, 0:普通（目前只支持0） |

### 请求示例

```json
{
    "space_name": "SPACE_NAME",
    "auth_info": [{
        "type": 1,
        "userid": "USERID",
        "auth": 7
    }, {
        "type": 2,
        "departmentid": 1,
        "auth": 1
    }],
    "space_sub_type": 0
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| spaceid | string | 空间id |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "spaceid": "SPACEID"
}
```
