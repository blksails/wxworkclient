# 获取收件箱邮件列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97505](https://developer.work.weixin.qq.com/document/path/97505)
- **文档 ID**: `97505`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/exmail/app/get_mail_list?access_token=ACCESS_TOKEN`

## 接口描述

由于用户可向应用所绑定的邮箱回复邮件，故支持应用获取应用收件箱的邮件列表。该接口用于分页获取收件箱下，指定时间范围内的邮件id列表。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| begin_time | uint32 | 是 | 开始时间，unix时间戳 |
| end_time | uint32 | 是 | 结束时间，unix时间戳 |
| cursor | string | 否 | 上一次调用时返回的next_cursor，第一次拉取可以不填 |
| limit | uint32 | 否 | 期望请求的数据量，默认值为100，最大值为1000 |

### 请求示例

```json
{"begin_time":1668441600,"end_time":1668527999,"cursor":"CURSOR","limit":100}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| next_cursor | string | 应用邮箱账号中邮件未读数 |
| has_more | uint32 | 是否还有更多数据。0-没有 1-有 |
| mail_list | obj[] | 邮件列表 |
| mail_list[].mail_id | string | 邮件id |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","next_cursor":"NEXT_CURSOR","has_more":0,"mail_list":[{"mail_id":"MAIL_ID"}]}
```
