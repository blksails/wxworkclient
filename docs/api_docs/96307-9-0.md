# 接口调整说明 2018/12/21

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96307](https://developer.work.weixin.qq.com/document/path/96307)
- **文档 ID**: `96307`
- **分组信息**: 第 9 个接口，共 9 个

## 接口描述

企业微信管理端的“外部联系人”标签将替换为“客户联系”，原先该页面下的“员工对外信息展示”功能将移动至“我的企业”-“通讯录管理”下。同时，相关的JSAPI、小程序以及API接口也将有较大变更。相关变更将在2019年1月7日 00:00 生效，请各位企业开发者尽快做好调整。
