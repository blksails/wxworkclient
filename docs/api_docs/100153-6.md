# 更新家长 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100153](https://developer.work.weixin.qq.com/document/path/100153)
- **文档 ID**: `100153`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/user/update_parent?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/school/user/update_parent?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| parent_userid |  | 是 | 家长UserID。学校内必须唯一。不区分大小写，长度为1~64个字节。只能由数字、字母和“_-@.”四种字符组成，且第一个字符必须是数字或字母。 |
| new_parent_userid |  | 否 | 更新的家长UserID。不能与已经存在的家长UserID相同。每个家长仅能更新一次。 |
| mobile |  | 否 | 家长手机号 |
| children |  | 否 | 家长的孩子列表，该字段是全量更新，如果孩子列表为空则忽略该字段，最多10个 |
| 自建应用 | 自建应用 | 否 | 配置到「家校沟通-读取和编辑家校通讯录的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 需拥有「家校沟通」可使用家校通讯录-家校通讯录编辑权限 |
| 第三方应用 | 第三方应用 | 否 | 需拥有「家校沟通」使用和编辑权限 |

### 请求示例

```json
{"parent_userid":"zhangsan_parent_userid","new_parent_userid":"NEW_ID","mobile":"18000000000","children":[{"student_userid":"zhangsan","relation":"爸爸"},{"student_userid":"lisi","relation":"伯父"}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| children.student_userid |  | 学生的UserID |
| children.relation |  | 家长与学生的关系，最长32字节 |
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"updated"}
```
