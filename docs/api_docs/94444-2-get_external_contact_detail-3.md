# 开始开发

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94444](https://developer.work.weixin.qq.com/document/path/94444)
- **文档 ID**: `94444`
- **API 名称**: `get_external_contact_detail`
- **请求方法**: `POST`
- **接口地址**: `https://work.weixin.qq.com/api/doc#16685`
- **分组信息**: 第 2 个接口，共 4 个

## 接口描述

学校可以在相关的应用里调用外部联系人选人接口，获取成员选择的外部联系人的userid以及通过该userid获取外部联系人详情。

## 请求信息

### 请求示例

```text
学校可以在相关的应用里调用外部联系人选人接口，获取成员选择的外部联系人的userid以及通过该userid获取外部联系人详情。
```
