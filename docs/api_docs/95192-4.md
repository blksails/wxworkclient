# 进入应用客服会话

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95192](https://developer.work.weixin.qq.com/document/path/95192)
- **文档 ID**: `95192`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

第三方应用可以使用该接口提供按钮，让用户可快速打开应用客服的会话。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```text
wx.qy.openThirdAppServiceChat({
    success: function(res) {
        // 回调
    }
});
```

## 其他说明

### 平台版本说明

| 平台 | 企业微信版本要求 |
| --- | --- |
| ios | ≥3.1.18 |
| android | ≥3.1.18 |
| win | ≥4.1.6 |
| mac | ≥4.1.6 |

### 应用权限说明

| 应用类型 | 可调用的应用 |
| --- | --- |
| 自建应用 | 不支持 |
| 第三方应用 | 提前配置了客服 |
