# 产生会话回调通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99994](https://developer.work.weixin.qq.com/document/path/99994)
- **文档 ID**: `99994`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webhook/callback`

## 接口描述

当企业下新产生消息时，企业微信专区后台会向具有企业会话内容权限的应用关联的程序推送相应的事件消息。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| event_type | string | 是 | 事件类型，此时固定为conversation_new_message |
| timestamp | int32 | 是 | 时间戳 |
| conversation_new_message.token | string | 是 | 10分钟内有效，用于调用获取会话记录，不超过128字节 |

### 请求示例

```json
{
    "event_type": "conversation_new_message",
    "timestamp": 1408091189,
    "conversation_new_message": {
        "token": "zzzzzzzzzzz"
    }
}
```
