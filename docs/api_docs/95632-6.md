# 删除对接规则 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95632](https://developer.work.weixin.qq.com/document/path/95632)
- **文档 ID**: `95632`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/corpgroup/rule/delete_rule?access_token=ACCESS_TOKEN`

## 接口描述

上下游系统应用可通过该接口删除企业上下游规则

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| chain_id |  | 是 | 上下游id |
| rule_id |  | 是 | 上下游规则id |
| 自建应用 | 自建应用 | 否 | 配置到「上下游- 可调用接口的应用」中 |

### 请求示例

```json
{"chain_id":"Chxxxxxx","rule_id":1}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
