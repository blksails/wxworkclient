## 接口描述

唤起客户群列表，选择并返回客户群ID。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
|  | Function | 否 | 成功回调 |
|  | Function | 否 | 失败回调 |
|  | Function | 否 | 取消回调 |
|  | Function | 否 | 完成回调 |

### 请求示例

```text
ww.selectExternalChat()
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
|  | string | 通用错误信息 |
|  | number | 通用错误码 |
|  | string[] | 单次最多可选择 5 个 |

## 其他说明

### ww.selectExternalChat([params])

兼容性：企业微信 iOS、Android、PC 5.0.2 开始支持
功能描述唤起客户群列表，选择并返回客户群ID。

### 功能描述

唤起客户群列表，选择并返回客户群ID。

### 使用限制

本接口必须使用应用身份进行注册仅支持「营销获客」分类的第三方应用调用调用成员必须在应用的可见范围内

### params: Object

属性类型必填说明successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码chatIdListstring[]否单次最多可选择 5 个
