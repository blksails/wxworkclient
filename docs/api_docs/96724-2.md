## 接口描述

目录

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| strcorpid |  | 是 | 学校corpid |
| strpartyid |  | 否 | 班级的部门id，当传入班级部门id后，用户不能再修改校区和班级 |
| redirect_url |  | 是 | 重定向url，当用户填写完学生资料后，企业微信会重定向到该地址 |
| appId |  | 是 | 固定填企业微信直播小程序appid: wx61866588582f5f71 |
| path |  | 是 | 跳转到直播小程序的路径 |
| strcorpid |  | 是 | 学校corpid |
| strpartyid |  | 否 | 班级的部门id，当传入班级部门id后，用户不能再修改校区和班级 |

### 请求示例

```json
wx.navigateToMiniProgram({appId:'wx61866588582f5f71',path:'pages/index/index?strcorpid=${strcorpid}&strpartyid=${strpartyid}'success(res){// 打开成功}})
```

## 其他说明

### 微信H5页面唤起填写学生资料页面

开发者在h5页面中可以通过重定向到企业微信的组件地址让用户完成学生资料的填写，地址如下：https://work.weixin.qq.com/openSchoolFollow/${strcorpid}?strpartyid=${strpartyid}&redirect_url=${redirect_url}
请求参数
参数必须说明strcorpid是学校corpidstrpartyid否班级的部门id，当传入班级部门id后，用户不能再修改校区和班级redirect_url是重定向url，当用户填写完学生资料后，企业微信会重定向到该地址

### 微信小程序进入填写学生资料页面

开发者在微信小程序里可调用小程序接口wx.navigateToMiniProgram（点击查看更详细的调用方法），唤起填写学生资料页面。
请求参数
参数必须说明appId是固定填企业微信直播小程序appid: wx61866588582f5f71path是跳转到直播小程序的路径strcorpid是学校corpidstrpartyid否班级的部门id，当传入班级部门id后，用户不能再修改校区和班级
调用示例
wx.navigateToMiniProgram({appId:'wx61866588582f5f71',path:'pages/index/index?strcorpid=${strcorpid}&strpartyid=${strpartyid}'success(res){// 打开成功}})
