# 邮箱获取userid

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95892](https://developer.work.weixin.qq.com/document/path/95892)
- **文档 ID**: `95892`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/user/get_userid_by_email`

## 接口描述

通过邮箱获取其所对应的userid。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证，授权企业的token或上游获取的下游企业的token |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| email | string | 是 | 邮箱 |
| email_type | int32 | 否 | 邮箱类型：1-企业邮箱（默认）；2-个人邮箱 |

### 请求示例

```json
{
    "email":"12345@qq.com",
    "email_type":1
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| userid | string | 成员UserID。注意：已升级openid的代开发或第三方，获取的是密文userid |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"userid": "zhangsan"
}
```

## 其他说明

### 权限说明

应用须拥有指定成员的查看权限。

### 更多说明

请确保邮箱的正确性，若出错的次数超过企业人数上限的20%，会导致1天不可调用。
