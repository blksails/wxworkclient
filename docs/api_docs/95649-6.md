# 获取订单中的账号列表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95649](https://developer.work.weixin.qq.com/document/path/95649)
- **文档 ID**: `95649`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/license/list_order_account?provider_access_token=ACCESS_TOKEN`

## 接口描述

查询指定订单下的平台能力服务账号列表。若为购买账号的订单或者存量企业的版本付费迁移订单，则返回账号激活码列表；若为续期账号的订单，则返回续期账号的成员列表。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 应用服务商的接口调用凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| order_id | string | 是 | 订单号 |
| limit | uint32 | 否 | 返回的最大记录数，最大值1000，默认值500 |
| cursor | string | 否 | 用于分页查询的游标，由上一次调用返回，首次调用可不填 |

### 请求示例

```json
{
	"order_id" : "XXXXXXXX",
	"limit":1000,
	"cursor":"xxxx"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| next_cursor | string | 分页游标，再下次请求时填写以获取之后分页的记录 |
| has_more | int32 | 是否有更多。 0: 没有， 1: 有 |
| account_list | object[] | 账号列表 |
| account_list.active_code | string | 账号码，订单类型为购买账号时，返回该字段 |
| account_list.userid | string | 企业续期成员userid，订单类型为续期账号时，返回该字段。返回加密的userid |
| account_list.type | int32 | 账号类型：1:基础账号，2:互通账号 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"next_cursor": "xxxx",
	"has_more":1,
	"account_list":[
		{
			"active_code": "code1",
			"userid":"XXX",
			"type": 1
		},
		{
			"active_code": "code2",
			"userid":"XXX",
			"type": 2
		}
	]
}
```
