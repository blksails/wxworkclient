# 异步任务完成通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96305](https://developer.work.weixin.qq.com/document/path/96305)
- **文档 ID**: `96305`

## 接口描述

本事件是成员在使用异步任务接口时，用于接收任务执行完毕的结果通知。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 成员UserID |
| CreateTime |  | 否 | 消息创建时间（整型） |
| MsgType |  | 否 | 消息类型，此时固定为：event |
| Event |  | 否 | 事件类型：batch_job_result |
| JobId |  | 否 | 异步任务id，最大长度为64字符 |
| JobType |  | 否 | 操作类型，字符串，目前分别有：sync_user(增量更新成员)、 replace_user(全量覆盖成员）、invite_user(邀请成员关注）、replace_party(全量覆盖部门) |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| ErrCode |  | 返回码 |
| ErrMsg |  | 对返回码的文本描述内容 |
