## 其他说明

### 授权成功通知

企业授权微信客服组件时，企业微信后台会推送授权成功通知，会产生授权成功通知。除公共的字段信息外，会额外携带ExtraInfo.AuthOpenKfIdList参数，用于告知服务商用户当前授权是选择了哪些客服账号。
请求包体：
<xml><SuiteId><![CDATA[wxxxxxxx]]></SuiteId><AuthCode><![CDATA[AUTHCODE]]></AuthCode><InfoType><![CDATA[create_auth]]></InfoType><TimeStamp>1403610513</TimeStamp><State><![CDATA[STATE]]></State><ExtraInfo><AuthOpenKfIdList><OpenKfId><![CDATA[wk11111]]></OpenKfId><OpenKfId><![CDATA[wk22222]]></OpenKfId><OpenKfId><![CDATA[wk33333]]></OpenKfId></AuthOpenKfIdList></ExtraInfo></xml>
参数说明：
参数说明SuiteId等公共字段见授权成功通知ExtraInfo.AuthOpenKfIdList用户本次授权的客服账号列表

### 变更授权通知

企业修改授权的客服链接时，企业微信后台会推送变更授权通知。除公共的字段信息外，会额外携带ExtraInfo.AuthOpenKfIdList参数，用于告知服务商用户当前授权是选择了哪些客服账号。
请求包体：
<xml><SuiteId><![CDATA[wxxxxxxx]]></SuiteId><InfoType><![CDATA[change_auth]]></InfoType><TimeStamp>1403610513</TimeStamp><AuthCorpId><![CDATA[AUTHCORPID]]></AuthCorpId><State><![CDATA[STATE]]></State><ExtraInfo><AuthOpenKfIdList><OpenKfId><![CDATA[wk11111]]></OpenKfId><OpenKfId><![CDATA[wk22222]]></OpenKfId><OpenKfId><![CDATA[wk33333]]></OpenKfId></AuthOpenKfIdList></ExtraInfo></xml>
参数说明：
参数说明SuiteId等公共字段见变更授权通知ExtraInfo.AuthOpenKfIdList用户本次授权的客服账号列表

### 取消授权通知

当授权方（即授权企业）取消了对组件的授权，企业微信后台会推送取消授权通知。详见取消授权通知
