# 获取获客链接详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99484](https://developer.work.weixin.qq.com/document/path/99484)
- **文档 ID**: `99484`
- **API 名称**: `get`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/customer_acquisition/get?access_token=ACCESS_TOKEN`
- **分组信息**: 第 2 个接口，共 2 个

## 接口描述

服务商可通过此接口根据获客链接id获取链接配置详情。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| link_id | string | 是 | 获客链接id |

### 请求示例

```json
{
   "link_id":"LINK_ID_AAA"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| link.link_name | string | 获客链接的名称 |
| link.url | string | 获客链接实际的url |

### 响应示例

```json
{
  "errcode": 0,
  "errmsg": "ok",
  "link":
	{
		"link_name":"LINK_NAME",
		"url":"https://work.weixin.qq.com/ca/xxxxxx?comp_scene=xxxxxx",
	}
}
```
