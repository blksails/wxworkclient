## 其他说明

### 2025年2月10日

1.支持自定义线程池：支持使用自定义业务线程池执行业务逻辑，避免阻塞io线程。  2. 启动命令行参数优化：支持配置是否启用自定义业务线程池、io线程池大小、业务线程池大小、监控线程时间间隔以及调试模式以及参数。可在启动命令增加-h参数查看具体的启动参数说明 3.日志精简及优化：日志支持打印请求reqid，方便通过reqid定位问题 4.监控日志优化：支持启动命令指定监控日志时间间隔，以及支持打印当前繁忙的io线程数、业务线程数等 5.错误码优化：Invoke接口返回值细化，具体参考[specsdkinvoke接口返回值说明](#54166/specsdkinvoke%E6%8E%A5%E5%8F%A3%E8%BF%94%E5%9B%9E%E5%80%BC%E8%AF%B4%E6%98%8E) | Java
