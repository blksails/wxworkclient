## 其他说明

### ww.addDevice(params)

兼容性：企业微信 iOS、Android 4.0.18 开始支持
功能描述添加设备。
ww.addDevice({type:'qrcode',qrcode_url:'https://open.work.weixin.qq.com/connect?xxx',})
使用限制本接口必须使用应用身份进行注册发起用户需要有设备添加权限（超级管理员/设备管理员）
参数说明params: Object属性类型必填说明typestring是设备发现方式合法值说明qrcode识别传递的 qrcode_url如果是正确的链接，则直接进入到设备详情bluetooth进入蓝牙发现的列表页，进行搜索识别input进入输入 sn 流程若传入 sn，则直接进入设备详情choose进入选择添加设备方式列表qrcode_urlstring否设备静态 / 动态二维码链接当 type 为 qrcode 时必填snstring否设备 SN 码当 type 为 sn 时选填successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

添加设备。

### 使用限制

本接口必须使用应用身份进行注册发起用户需要有设备添加权限（超级管理员/设备管理员）

### params: Object

属性类型必填说明typestring是设备发现方式合法值说明qrcode识别传递的 qrcode_url如果是正确的链接，则直接进入到设备详情bluetooth进入蓝牙发现的列表页，进行搜索识别input进入输入 sn 流程若传入 sn，则直接进入设备详情choose进入选择添加设备方式列表qrcode_urlstring否设备静态 / 动态二维码链接当 type 为 qrcode 时必填snstring否设备 SN 码当 type 为 sn 时选填successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码
