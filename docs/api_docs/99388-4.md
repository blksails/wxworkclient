# 改单通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99388](https://developer.work.weixin.qq.com/document/path/99388)
- **文档 ID**: `99388`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`

## 接口描述

当服务商管理员修改订单价格之后，企业微信后台会推送改单通知到服务商的指令回调URL。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 | 消息体签名 |
| timestamp | string | 是 | 时间戳 |
| nonce | string | 是 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 第三方应用的SuiteId |
| PaidCorpId | string | 是 | 购买方corpid |
| InfoType | string | 是 | 固定为 change_order |
| TimeStamp | string | 是 | 时间戳 |
| OldOrderId | string | 是 | 原订单号 |
| NewOrderId | string | 是 | 改单后新的订单号，服务商可据此拉取购买信息。该OrderId由企业微信生成，不超过32个字符，每次修改价格都会产生新的订单号 |

### 请求示例

```xml
<xml>
    <SuiteId><![CDATA[ww4asffe99e54c0aaa]]></SuiteId>
    <PaidCorpId><![CDATA[wxf8b4f85f3a794aaa]]></PaidCorpId>
    <InfoType><![CDATA[change_order]]></InfoType>
    <TimeStamp>1403610513</TimeStamp>
    <OldOrderId><![CDATA[ORDERID]]></OldOrderId>
    <NewOrderId><![CDATA[ORDERID]]></NewOrderId>
</xml>
```
