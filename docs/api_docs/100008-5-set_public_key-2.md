# 如何接入

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100008](https://developer.work.weixin.qq.com/document/path/100008)
- **文档 ID**: `100008`
- **API 名称**: `set_public_key`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/set_public_key?access_token=ACCESS_TOKEN`
- **分组信息**: 第 5 个接口，共 13 个

## 接口描述

服务商需要调用设置公钥接口，为该企业设置加密消息的公钥。
