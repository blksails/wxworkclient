# 民生优惠条件查询

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97208](https://developer.work.weixin.qq.com/document/path/97208)
- **文档 ID**: `97208`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/license/support_policy_query?provider_access_token=ACCESS_TOKEN`

## 接口描述

查询企业是否满足民生行业政策的优惠条件。民生行业接口许可优惠政策于2023年3月31日到期，到期后不再支持查询。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 应用服务商的接口调用凭证，获取方法参见服务商的凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| corpid | string | 是 | 企业id，支持加密和非加密的corpid |

### 请求示例

```json
{
	"corpid" : "wwxxx"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| query_result | int32 | 查询结果。0表示不符合减免条件；1表示符合减免条件 |
| unsatisfied_reason | int32[] | 被查询企业不符合减免条件的原因对应的错误码列表 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"query_result": 1,
	"unsatisfied_reason":[701090, 701091]
}
```

## 其他说明

### 限制说明

查询的企业必须必须安装了服务商的第三方应用或者代开发应用。一个企业在30天内最多只能查询一次。

### 错误码说明

| 错误码 | 说明 |
| --- | --- |
| 701090 | 认证/验证状态不符合 |
| 701091 | 行业类型不符合 |
| 701092 | 统一社会信用代码不符合 |
| 701096 | 风控审核不通过 |
| 701110 | 学校不符合单校条件 |
| 701111 | 学校不符合局校条件 |
