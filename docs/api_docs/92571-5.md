# 获取配置了客户联系功能的成员列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92571](https://developer.work.weixin.qq.com/document/path/92571)
- **文档 ID**: `92571`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/get_follow_user_list?access_token=ACCESS_TOKEN`

## 接口描述

企业和第三方服务商可通过此接口获取配置了客户联系功能的成员列表。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| 自建应用 | 自建应用 | 否 | 配置到「客户联系 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 具有「客户基础信息」权限 |
| 第三方应用 | 第三方应用 | 否 | 具有「客户基础信息」权限 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| follow_user |  | 配置了客户联系功能的成员userid列表 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","follow_user":["zhangsan","lissi"]}
```
