### ww.getConnectedBluetoothDevices(params)

兼容性：企业微信
功能描述根据 uuid 获取处于已连接状态的设备。
ww.getConnectedBluetoothDevices({services:['FEE7']})
参数说明params: Object属性类型必填说明servicesstring[]是蓝牙设备主 service 的 uuid 列表successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码devicesObject[]是搜索到的设备列表属性类型必填说明namestring是蓝牙设备名称，某些设备可能没有deviceIdstring是用于区分设备的 ID

### 功能描述

根据 uuid 获取处于已连接状态的设备。

### params: Object

属性类型必填说明servicesstring[]是蓝牙设备主 service 的 uuid 列表successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码devicesObject[]是搜索到的设备列表属性类型必填说明namestring是蓝牙设备名称，某些设备可能没有deviceIdstring是用于区分设备的 ID

### 示例代码

wx.getConnectedBluetoothDevices({services:['FEE7'],success:function(res){console.log(res)}})
