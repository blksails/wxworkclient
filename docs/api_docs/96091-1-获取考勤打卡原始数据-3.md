# 获取硬件产生的数据

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96091](https://developer.work.weixin.qq.com/document/path/96091)
- **文档 ID**: `96091`
- **API 名称**: `获取考勤打卡原始数据`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/attendance/get_punch_data`
- **分组信息**: 第 1 个接口，共 7 个

## 接口描述

获取考勤打卡原始数据接口用于获取企业下绑定的考勤设备的打卡数据。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| punch_data | object[] | 打卡数据列表 |
| punch_data[].user_id | string | 用户ID |
| punch_data[].time | uint32 | 打卡时间戳 |
| punch_data[].location | string | 打卡地点 |
