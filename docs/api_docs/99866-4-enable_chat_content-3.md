- **文档 ID**: `99866`
- **API 名称**: `enable_chat_content`
- **请求方法**: `PUT`
- **接口地址**: `https://yourdomain.com/enable_chat_content`
- **分组信息**: 第 4 个接口，共 5 个

## 接口描述

企业开启会话内容以便服务商访问。

## 请求信息

### 请求示例

```json
{
  "enable": true
}
```

## 其他说明

### 注意

若授权时企业还未开启「企业会话内容」，授权页将展示开启指引。
