# 会话内容存储时间

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100099](https://developer.work.weixin.qq.com/document/path/100099)
- **文档 ID**: `100099`
- **API 名称**: `session_storage_time`
- **分组信息**: 第 3 个接口，共 5 个

## 接口描述

企业开启数据专区会话内容，并授权服务商应用数据与智能专区权限后，服务商应用可调用设置公钥接口设置会话数据的加密公钥，设置后企业微信将对授权范围内成员的会话内容加密存储，服务商可使用展示组件展示会话内容。企业可设置存储最近 90 天/180 天/1 年/2 年的会话内容，企业设置后，在数据专区会话数据第三方接口付费有效期内，存储企业自授权日起所设置存储时长的会话内容，服务商可使用展示组件展示对应的会话内容；过期的会话内容不存储，展示组件不展示。接口付费到期后，超过 30 天未续期，则之前付费期间存储的会话内容将不再存储，展示组件不展示。试用到期后，超过 30 天未付费，则之前试用期间存储的会话内容将不再存储，展示组件不展示。
