# 判断当前设备是否支持 NFC 能力

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92409](https://developer.work.weixin.qq.com/document/path/92409)
- **文档 ID**: `92409`
- **请求方法**: `Object object`
- **接口地址**: `wx.qy.getNFCReaderState`

## 接口描述

需要用户授权

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | function | 否 | 接口调用成功的回调函数 |
| fail | function | 否 | 接口调用失败的回调函数 |
| complete | function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```text
wx.qy.getNFCReaderState({
  success: (res) => {
    console.log(JSON.stringify(res))
    // todo what you want
  },
  fail: (res) => {
    wx.showModal({
      title: 'NFC-Not-Support',
      content: JSON.stringify(res),
    })
  }
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
