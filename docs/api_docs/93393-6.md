# 批量获取汇报记录单号 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93393](https://developer.work.weixin.qq.com/document/path/93393)
- **文档 ID**: `93393`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/oa/journal/get_record_list?access_token=ACCESS_TOKEN`

## 接口描述

企业可通过本接口获取企业一段时间内企业微信“汇报应用”汇报记录编号，支持按汇报表单ID、申请人、部门等条件筛选。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| starttime | uint32 | 是 | 开始时间 |
| endtime | uint32 | 是 | 结束时间,开始时间和结束时间间隔不能超过一个月 |
| cursor | uint32 | 是 | 游标首次请求传0，非首次请求携带上一次请求返回的next_cursor |
| limit | uint32 | 是 | 拉取条数 |
| filters | obj[] | 否 | 过滤条件 |
| 自建应用 | 自建应用 | 否 | 需要配置到「汇报 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"starttime":1606230000,"endtime":1606361304,"cursor":0,"limit":10,"filters":[{"key":"creator","value":"kele"},{"key":"department","value":"1"},{"key":"template_id","value":"3TmALk1ogfgKiQE3e3jRwnTUhMTh8vca1N8zUVNUx"}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| filters.key | string | -不多于256字节，creator指定汇报记录提单人；department指定提单人所在部门；template_id指定模板 |
| filters.value | string | -不多于256字节 |
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |
| journaluuid_list | string[] | 汇报记录id列表 |
| next_cursor | uint32 | 下一次拉取游标 |
| endflag | uint32 | 0代表还有数据，1代表已无数据 |
| 301065 |  | 无汇报应用数据拉取权限 |
| 301066 |  | 请求参数错误 |
| 301067 |  | 接口内部失败 |
