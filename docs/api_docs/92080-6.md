# 批量删除家长

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92080](https://developer.work.weixin.qq.com/document/path/92080)
- **文档 ID**: `92080`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/user/batch_delete_parent`

## 接口描述

批量删除家长

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| useridlist | array | 是 | 家长的userid列表，每次最多100个 |

### 请求示例

```json
{
	"useridlist": ["zhangsan", "lisi"]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| result_list | object[] | 批量操作结果 |
| result_list[].parent_userid | string | 家长userid |
| result_list[].errcode | int32 | 返回码 |
| result_list[].errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
	"errcode": 1111,
	"errmsg": "userid not found",
	"result_list": [
		{
			"parent_userid": "lisi",
			"errcode": 1111,
			"errmsg": "userid not found"
		}
	]
}
```

## 其他说明

### 权限说明

调用的应用需要满足如下的权限

- 自建应用：配置到「家校沟通-读取和编辑家校通讯录的应用」中
- 代开发应用：需拥有「家校沟通」可使用家校通讯录-家校通讯录编辑 权限
- 第三方应用：需拥有「家校沟通」使用和编辑 权限

*注：* 从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 [查看详情](https://developer.work.weixin.qq.com/document/51165)
