# 取消订单通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99353](https://developer.work.weixin.qq.com/document/path/99353)
- **文档 ID**: `99353`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`

## 接口描述

服务商或客户企业取消订单，会触发取消回调通知服务商

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 |  |
| timestamp | string | 是 |  |
| nonce | string | 是 |  |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 第三方应用的SuiteId |
| PaidCorpId | string | 是 | 购买方corpid |
| InfoType | string | 是 | cancel_order |
| TimeStamp | string | 是 | 时间戳 |
| OrderId | string | 是 | 订单号，付费订单的唯一标志。服务商可据此拉取购买信息。该OrderId由企业微信生成，不超过32个字符 |

### 请求示例

```xml
<xml>
    <SuiteId><![CDATA[ww4asffe99e54c0aaa]]></SuiteId>
    <PaidCorpId><![CDATA[wxf8b4f85f3a794aaa]]></PaidCorpId>
    <InfoType><![CDATA[cancel_order]]></InfoType>
    <TimeStamp>1403610513</TimeStamp>
    <OrderId><![CDATA[ORDERID]]></OrderId>
</xml>
```
