# 初始化企业微信Web登录组件，创建登录面板

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98622](https://developer.work.weixin.qq.com/document/path/98622)
- **文档 ID**: `98622`
- **请求方法**: `POST`
- **接口地址**: `ww.createWWLoginPanel(options)`

## 接口描述

初始化企业微信Web登录组件，创建登录面板。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| options.el | string / Element | 否 | 登录组件挂载元素，可以指定DOM元素或CSS选择器 |
| options.params | Object | 是 | 登录参数 |
| options.params.login_type | string | 否 | 登录类型，合法值：ServiceApp（第三方应用登录）、CorpApp（企业自建应用登录、服务商代开发应用登录） |
| options.params.appid | string | 是 | AppID，登录类型为企业自建应用/服务商代开发应用时填企业CorpID，第三方登录时填登录授权SuiteID |
| options.params.agentid | string | 否 | 企业自建应用/服务商代开发应用AgentID |
| options.params.redirect_uri | string | 是 | 登录成功重定向url，无需进行URLEncode |
| options.params.state | string | 否 | 登录state，用于保持请求和回调的状态，授权请求后原样带回给企业。该参数可用于防止CSRF攻击（跨站请求伪造攻击），建议带上该参数，可设置为简单的随机数加session进行校验 |
| options.params.redirect_type | string | 否 | 登录成功跳转类型，合法值：top（默认top window顶层页面跳转）、callback（通过onLoginSuccess回调用户授权code，开发者自行处理跳转）、self（登录组件跳转） |
| options.params.panel_size | string | 否 | 登录面板大小，合法值：middle（默认: 480x416px）、small（小尺寸: 320x380px） |
| options.params.lang | string | 否 | 语言类型，合法值：zh（中文）、en（英文） |
| options.onCheckWeComLogin | Function | 否 | 获取企业微信桌面端登录状态回调 |
| options.onLoginSuccess | Function | 否 | 企业微信登录成功回调，需配置redirect_type=callback |
| options.onLoginFail | Function | 否 | 企业微信登录错误回调，登录相关错误码 |
| options.onOpenInWecom | Function | 否 | 打开企业微信客户端成功回调 |

### 请求示例

```text
// 初始化登录组件
const wwLogin = ww.createWWLoginPanel({
  el: '#ww_login',
  params: {
    login_type: 'CorpApp',
    appid: 'wwbbb6a7b539f2xxxxx',
    agentid: '10000xx',
    redirect_uri: 'https://work.weixin.qq.com',
    state: 'loginState',
    redirect_type: 'callback',
  },
  onCheckWeComLogin({ isWeComLogin }) {
    console.log(isWeComLogin)
  },
  onLoginSuccess({ code }) {
    console.log({ code })
  },
  onLoginFail(err) {
    console.log(err)
  },
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| el | HTMLIFrameElement | 登录面板iframe元素 |
| unmount | Function | 卸载登录面板 |
