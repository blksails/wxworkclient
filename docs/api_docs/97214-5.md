# 考勤打卡原始数据回调

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97214](https://developer.work.weixin.qq.com/document/path/97214)
- **文档 ID**: `97214`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`

## 接口描述

当用户授权设备上报考勤打卡原始数据时，并且打卡用户在应用可见范围内时，企业微信回调该指令回调到应用配置的数据回调url中。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 | 消息签名 |
| timestamp | string | 是 | 时间戳 |
| nonce | string | 是 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName | string | 是 | 企业微信CorpID |
| FromUserName | string | 是 | 固定为sys |
| MsgType | string | 是 | 消息类型，固定为event |
| Event | string | 是 | 事件类型，固定为device_checkin_data_report |
| TimeStamp | string | 是 | 消息创建时间，UNIX时间戳 |
| DeviceSn | string | 是 | 设备SN |
| CheckInDataList | object | 是 | 考勤原始数据列表 |
| CheckInDataList.Item.OpenUserId | string | 是 | 考勤打卡成员openuserid |
| CheckInDataList.Item.UserType | int32 | 是 | 打卡人员类型，0-企业成员；1-访客；2-学生 |
| CheckInDataList.Item.CheckinTime | string | 是 | 考勤打卡时间戳 |

### 请求示例

```xml
<xml>
    <ToUserName><![CDATA[wpwLgYcwAAayDT6bUVnhZuKebVce4xxx]]></ToUserName>
    <FromUserName><![CDATA[sys]]></FromUserName>
    <MsgType><![CDATA[event]]></MsgType>
    <Event><![CDATA[device_checkin_data_report]]></Event>
    <TimeStamp>1673943021</TimeStamp>
    <CheckInDataList>
        <Item>
            <OpenUserId><![CDATA[wowLgYcwAAwh5EDz9jMMeGPGL5uhwxxx]]></OpenUserId>
            <UserType>0</UserType>
            <CheckinTime>1673943020</CheckinTime>
        </Item>
    </CheckInDataList>
    <DeviceSn><![CDATA[device_sn]]></DeviceSn>
</xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
