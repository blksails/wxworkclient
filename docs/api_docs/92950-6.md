# 修改自动升年级的配置

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92950](https://developer.work.weixin.qq.com/document/path/92950)
- **文档 ID**: `92950`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/set_upgrade_info`

## 接口描述

修改自动升年级的配置

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| upgrade_time | uint32 | 否 | 自动升年级的时间，该时间戳只有月和日有效，不传则默认为传0，代表的1月1号 |
| upgrade_switch | uint32 | 否 | 开启或关闭自动升年级。0：表示关闭，1：表示开启，不传默认关闭，传所有非1的值也视为关闭 |

### 请求示例

```json
{
   "upgrade_time": 1594090969,
   "upgrade_switch": 2
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| next_upgrade_time | uint32 | 下次升级的时间，只有年月日有效，如果该学校今年已经升过年级，那本参数会给明年对应日期的时间戳 |
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
   "next_upgrade_time": 1625587200,
   "errcode": 0,
   "errmsg": "ok"
}
```
