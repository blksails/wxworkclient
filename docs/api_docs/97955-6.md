# 重命名文件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97955](https://developer.work.weixin.qq.com/document/path/97955)
- **文档 ID**: `97955`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_rename`

## 接口描述

该接口用于对指定文件进行重命名。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fileid | string | 是 | 文件fileid |
| new_name | string | 是 | 重命名后的文件名 （注意：文件名最多填255个字符, 英文算1个, 汉字算2个） |

### 请求示例

```json
{
    "fileid": "FILEID",
    "new_name": "NEW_NAME"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| fileid | string | 文件fileid |
| file_name | string | 文件名字 |
| spaceid | string | 文件所在的空间spaceid |
| fatherid | string | 文件所在的目录fileid, 在根目录时为spaceid |
| file_size | uint64 | 文件大小 |
| ctime | uint64 | 文件创建时间 |
| mtime | uint64 | 文件最后修改时间 |
| file_type | uint32 | 文件类型, 1:文件夹 2:文件 3:文档(文档) 4:文档(表格) 5:文档(收集表) 6:文档(幻灯片) |
| file_status | uint32 | 文件状态, 1:正常 2:删除 |
| sha | string | 文件sha |
| md5 | string | 文件md5 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "file": {
        "fileid": "FILEID",
        "file_name": "FILE_NAME",
        "spaceid": "SPACEID",
        "fatherid": "FATHERID",
        "file_size": 1777777777,
        "ctime": 17777777777,
        "mtime": 177777777,
        "file_type": 2,
        "file_status": 1,
        "sha": "SHA",
        "md5": "MD5"
    }
}
```
