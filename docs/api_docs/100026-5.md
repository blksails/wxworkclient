# 会话名称搜索

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100026](https://developer.work.weixin.qq.com/document/path/100026)
- **文档 ID**: `100026`
- **请求方法**: `SDK调用`

## 接口描述

可调用该接口对企业的群会话名称进行搜索

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| query_word | string | 是 | 搜索的文本，要求大于等于2个UTF8字符 |
| limit | uint32 | 否 | 用于分页查询，每次请求返回的数据上限。最大为100，不填默认为50 |
| cursor | string | 否 | 用于分页查询的游标，字符串类型，由上一次调用返回，首次调用可不填 |

### 请求示例

```json
{
	"query_word": "VIP",
	"limit": 50,
	"cursor": "CURSOR"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| has_more | uint32 | 是否还有更多数据未获取，不填默认为0 0: 否 1: 是 |
| next_cursor | string | 下一次请求的cursor值 |
| chat_list | object[] | 搜索结果的群列表 |
| chat_list[].chatid | string | 群聊会话id |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"has_more": 1,
	"next_cursor": "CURSOR",
	"chat_list":[
		{
			"chatid": "CHATID1"
		},
		{
			"chatid": "CHATID2"
		}
	]
}
```

## 其他说明

### 权限说明

> 对于搜索会话名称，**仅支持有命名的群聊**，对于单聊和未命名的群聊不支持搜索
> 搜索会话名称时，基于当前最新的会话名称搜索

| 应用类型 | 权限要求 |
| --- | --- |
| 自建应用 | 需具备「数据与智能专区权限」 |
| 代开发应用 | 需具备「数据与智能专区权限」 |
| 第三方应用 | 需具备「数据与智能专区权限」 |
