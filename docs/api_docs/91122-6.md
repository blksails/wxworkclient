# 获取访问用户敏感信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91122](https://developer.work.weixin.qq.com/document/path/91122)
- **文档 ID**: `91122`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/auth/getuserdetail3rd?suite_access_token=SUITE_ACCESS_TOKEN`

## 接口描述

获取访问用户敏感信息

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| suite_access_token | string | 是 | 第三方应用的suite_access_token |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| user_ticket | string | 是 | 成员票据 |

### 请求示例

```json
{
   "user_ticket": "USER_TICKET"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| corpid | string | 用户所属企业的corpid |
| userid | string | 成员UserID |
| name | string | 成员姓名，此字段从2019年12月30日起，对新创建第三方应用不再返回真实name，使用userid代替name返回，2020年6月30日起，对所有历史第三方应用不再返回，第三方页面需要通过通讯录展示组件来展示名字 |
| gender | string | 性别。0表示未定义，1表示男性，2表示女性。仅在用户同意snsapi_privateinfo授权时返回真实值，否则返回0. |
| avatar | string | 头像url。仅在用户同意snsapi_privateinfo授权时返回真实头像，否则返回默认头像 |
| qr_code | string | 员工个人二维码（扫描可添加为外部联系人），仅在用户同意snsapi_privateinfo授权时返回 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "ok",
   "corpid":"wwxxxxxxyyyyy",
   "userid":"lisi",
   "name":"李四",
   "gender":"1",
   "avatar":"http://shp.qpic.cn/bizmp/xxxxxxxxxxx/0",
   "qr_code":"https://open.work.weixin.qq.com/wwopen/userQRCode?vcode=vcfc13b01dfs78e981c"
}
```
