# 获取硬件产生的数据

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96091](https://developer.work.weixin.qq.com/document/path/96091)
- **文档 ID**: `96091`
- **API 名称**: `应用修改门禁通行规则`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/accesscontrol/update_access_rule`
- **分组信息**: 第 6 个接口，共 7 个

## 接口描述

应用修改门禁通行规则接口用于修改企业下绑定的门禁设备的通行规则。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| rule_id | string | 是 | 规则ID |
| time_range | object | 是 | 通行时间范围 |
| location | string | 是 | 通行地点 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
