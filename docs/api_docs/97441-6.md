# 查询成员自定义id - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97441](https://developer.work.weixin.qq.com/document/path/97441)
- **文档 ID**: `97441`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/corpgroup/corp/get_chain_user_custom_id?access_token=ACCESS_TOKEN`

## 接口描述

上级企业自建应用/代开发应用通过本接口查询成员自定义 id

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证。上游企业应用access_token |
| chain_id |  | 是 | 上下游id |
| corpid |  | 是 | 已加入企业id |
| userid |  | 是 | 企业内的成员 |

### 请求示例

```json
{"chain_id":"Chxxxxxx","corpid":"xxxxx","userid":"xxxxx"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| user_custom_id |  | 成员自定义 id |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","user_custom_id":"1234"}
```
