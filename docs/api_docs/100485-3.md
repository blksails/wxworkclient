## 其他说明

### ww.getNetworkType([params])

兼容性：企业微信 iOS、Android，微信
功能描述获取网络状态。
ww.getNetworkType()
参数说明params: Object属性类型必填说明successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码networkTypestring是网络类型合法值说明wifiwifi2g2g3g3g4g4gnone无网络unknownAndroid下不常见的网络类型

### 功能描述

获取网络状态。

### params: Object

属性类型必填说明successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码networkTypestring是网络类型合法值说明wifiwifi2g2g3g3g4g4gnone无网络unknownAndroid下不常见的网络类型

### 示例代码

wx.getNetworkType({success:function(res){varnetworkType=res.networkType;// 返回网络类型2g，3g，4g，wifi}});
