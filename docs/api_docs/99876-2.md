# 成员申请的提交回调 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99876](https://developer.work.weixin.qq.com/document/path/99876)
- **文档 ID**: `99876`

## 接口描述

当企业开启通过接口设置高级功能的申请流程时，成员提交申请后，将会向「可调用接口的应用」设置的消息服务器发送成员申请事件回调。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信corpid |
| FromUserName |  | 否 | 申请人的userid |
| CreateTime |  | 否 | 操作时间 |
| MsgType |  | 否 | 消息的类型，此时固定为event |
| Event |  | 否 | 事件的类型，此时固定为submit_vip_account_approval |
| ApplyReason |  | 否 | 申请原因 |
| BusinessType |  | 否 | 申请的高级账号类型：1-邮件 2-文档 3-微盘 4-会议 |
| AgentID |  | 否 | 企业应用的id，整型。可在应用的设置页面查看 |
| ApplyId |  | 否 | 申请id，字符串 |

### 请求示例

```xml
<xml><ToUserName><![CDATA[toUser]]></ToUserName><FromUserName><![CDATA[fromUser]]></FromUserName><CreateTime>1715681559</CreateTime><MsgType><![CDATA[event]]></MsgType><Event><![CDATA[submit_vip_account_approval]]></Event><ApplyReason><![CDATA[我有很多客户资料,微盘空间不足]]></ApplyReason><BusinessType>1</BusinessType><ApplyId><![CDATA[ApplyId]]></ApplyId><AgentID>1</AgentID></xml>
```
