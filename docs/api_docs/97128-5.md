# 获取直播详情 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97128](https://developer.work.weixin.qq.com/document/path/97128)
- **文档 ID**: `97128`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/living/get_living_info?access_token=ACCESS_TOKEN&livingid=LIVINGID`

## 接口描述

请求方式：GET（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/school/living/get_living_info?access_token=ACCESS_TOKEN&livingid=LIVINGID

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| livingid |  | 是 | 直播ID |
| 自建应用 | 自建应用 | 否 | 配置到「上课直播 - 可调用接口的应用」和「家长可以使用的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 具有「直播」和「家校沟通」权限 |
| 第三方应用 | 第三方应用 | 否 | 具有「直播」和「家校沟通」权限 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| living_info |  | 直播信息 |
| living_info.theme |  | 直播主题 |
| living_info.living_start |  | 直播开始时间戳 |
| living_info.living_duration |  | 直播时长，单位为秒 |
| living_info.anchor_userid |  | 主播的userid |
| living_info.living_range |  | 直播范围，如果直播范围包含的是班级群，优先返回班级群对应班级的partyid，否则直接返回群名称 |
| living_info.living_range.partyids |  | 班级的partyid列表 |
| living_info.living_range.group_names |  | 群名称列表 |
| living_info.viewer_num |  | 观看直播总人数 |
| living_info.comment_num |  | 评论数 |
| living_info.open_replay |  | 是否开启回放，1表示开启，0表示关闭 |
| living_info.push_stream_url |  | 推流地址，仅直播类型为活动直播并且直播状态是待开播返回该字段 |
| living_info.online_count |  | 当前在线观看人数 |
| living_info.subscribe_count |  | 直播预约人数 |

### 响应示例

