# 报销发票接口及jsapi

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90419](https://developer.work.weixin.qq.com/document/path/90419)
- **文档 ID**: `90419`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/card/invoice/reimburse`

## 接口描述

用于在应用中选择微信卡包中的电子发票实现电子化报销，仅对认证的企业微信账号开放。

## 请求信息

### 请求示例

```json
{
  "invoice_list": [
    {
      "invoice_id": "123456",
      "title": "ABC Inc.",
      "amount": 100,
      "date": "2022-01-01"
    }
  ]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| invoice_list | object[] | 发票列表 |
| invoice_list[].invoice_id | string | 发票ID |
| invoice_list[].title | string | 发票抬头 |
| invoice_list[].amount | int32 | 发票金额 |
| invoice_list[].date | string | 开票日期 |

### 响应示例

```json
{
  "errcode": 0,
  "errmsg": "ok",
  "invoice_list": [
    {
      "invoice_id": "123456",
      "title": "ABC Inc.",
      "amount": 100,
      "date": "2022-01-01"
    }
  ]
}
```

## 其他说明

### 概述

最后更新：2017/09/07
报销发票接口及jsapi用于在应用中选择微信卡包中的电子发票实现电子化报销，该接口仅对认证的企业微信账号开放。具体操作步骤如下：
1. 用户在应用中发票报销，选择电子发票类型；
2. 应用调用JS-SDK“chooseInvoice接口”拉起微信卡包，展示发票列表
3. 用户选择电子发票提交报销；
4. 报销方调用API接口查询发票信息，核实后锁定电子发票
5. 报销方打款给用户后，调用接口核销电子发票

相关功能介绍请参考“电子发票平台”
