# 查询注册状态

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90582](https://developer.work.weixin.qq.com/document/path/90582)
- **文档 ID**: `90582`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/get_register_info?provider_access_token=PROVIDER_ACCESS_TOKEN`

## 接口描述

该API用于查询通过注册定制化新创建的企业注册状态，企业注册成功返回注册信息。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 应用提供商的provider_access_token，获取方法参见服务商的凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| register_code | string | 是 | 查询的注册码，register_code生成后的查询有效期为24小时。仅支持 注册完成回调事件或者获取注册码返回的register_code调用 |

### 请求示例

```json
{
    "register_code":"pIKi3wRPNWCGF-pyP-YU5KWjDDD"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码对应的描述 |
| corpid | string | 企业的corpid |
| contact_sync | object | 通讯录迁移的凭证信息。仅当注册推广包开启通讯录迁移接口时返回该参数 |
| contact_sync.access_token | string | 通讯录api接口调用凭证，有全部通讯录读写权限。（请注意与provider_access_token的区别） |
| contact_sync.expires_in | int32 | access_token凭证的有效时间（秒） |
| auth_user_info | object | 授权管理员的信息 |
| auth_user_info.userid | string | 授权管理员的userid |
| state | string | 用户自定义的状态值，参数值由接口 获取注册码 指定。若未指定，则无该字段 |
| template_id | string | 推广包ID |

### 响应示例

```json
{
    "errcode":0,
    "errmsg":"ok",
    "corpid":"wwddddccc7775555aaa",
    "contact_sync":
    {
        "access_token":"accesstoken000001",

    },
    "auth_user_info":
    {
        "userid":"zhangshan"
    },
	"state":"TestState123",
	"template_id":"tpltest"
}
```
