# 获取订单详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91909](https://developer.work.weixin.qq.com/document/path/91909)
- **文档 ID**: `91909`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/get_order?suite_access_token=SUITE_ACCESS_TOKEN`

## 接口描述

获取订单详情

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| orderid | string | 是 | 订单号 |

### 请求示例

```json
{
	"orderid" : "2018091822ks1sd3s"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| orderid | string | 订单号 |
| order_status | int32 | 订单状态。0-待支付，1-已支付，2-已取消， 3-支付过期， 4-申请退款中， 5-退款成功， 6-退款被拒绝 |
| order_type | int32 | 订单类型。0-新购应用，1-扩容应用人数，2-续期应用时间，3-变更版本 |
| paid_corpid | string | 客户企业的corpid |
| operator_id | string | 下单操作人员userid。部分情况没有该字段（如服务商代下单；服务商下的免支付订单等） |
| suiteid | string | 应用id |
| appid | int32 | 套件应用id（仅旧套件有该字段） |
| edition_id | string | 购买版本ID |
| edition_name | string | 购买版本名字 |
| price | int32 | 应付价格，单位分 |
| user_count | int32 | 购买的人数 |
| order_period | int32 | 购买的时长，单位为天 |
| order_time | int32 | 下单时间（UNIX时间戳） |
| paid_time | int32 | 付款时间（UNIX时间戳） |
| begin_time | int32 | 购买生效期的开始时间（UNIX时间戳） |
| end_time | int32 | 购买生效期的结束时间（UNIX时间戳） |
| order_from | int32 | 下单来源。0-企业下单；1-服务商代下单；2-代理商代下单; 3-服务商收银台下单 |
| operator_corpid | string | 下单方corpid |
| service_share_amount | int32 | 服务商分成金额，单位分 |
| platform_share_amount | int32 | 平台分成金额，单位分 |
| dealer_share_amount | int32 | 代理商分成金额，单位分 |
| dealer_corp_info.corpid | string | 代理商corpid |
| dealer_corp_info.corp_name | string | 代理商的企业简称 |
