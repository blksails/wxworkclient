# 设置授权配置

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90602](https://developer.work.weixin.qq.com/document/path/90602)
- **文档 ID**: `90602`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/set_session_info?suite_access_token=SUITE_ACCESS_TOKEN`

## 接口描述

该接口可对某次授权进行配置。可支持测试模式（应用未发布时）。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| suite_access_token | string | 是 | 第三方应用access_token |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pre_auth_code | string | 是 | 预授权码 |
| session_info | object | 是 | 本次授权过程中需要用到的会话信息 |
| session_info.appid | array | 否 | 允许进行授权的应用id，如1、2、3， 不填或者填空数组都表示允许授权套件内所有应用（仅旧的多应用套件可传此参数，新开发者可忽略） |
| session_info.auth_type | int32 | 否 | 授权类型：0 正式授权， 1 测试授权。 默认值为0。注意，请确保应用在正式发布后的授权类型为“正式授权” |

### 请求示例

```json
{
	"pre_auth_code":"xxxxx",
	"session_info":
	{
		"appid":[1,2,3],
		"auth_type":1
	}
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok"
}
```
