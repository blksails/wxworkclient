# 微信进入居民上报小程序 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97018](https://developer.work.weixin.qq.com/document/path/97018)
- **文档 ID**: `97018`

## 接口描述

政府单位可在自己的小程序配置「居民上报」小程序跳转入口，居民点击后进入「居民上报」小程序反馈问题。在微信小程序里可按以下参数调用小程序接口wx.navigateToMiniProgram，唤起「居民上报」小程序。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| appId |  | 是 | 固定填企业微信居民上报小程序appid: wx85c9a2814145dbeb |
| path |  | 是 | 跳转到小程序的路径，路径格式固定为：pages/feedback/feedback?strcorpid=${strcorpid}&userid=${userid} |
| strcorpid |  | 是 | 企业corpid |
| userid |  | 是 | 企业成员的userid |

### 请求示例

```json
wx.navigateToMiniProgram({appId:'wx85c9a2814145dbeb',path:'pages/feedback/feedback?strcorpid=${strcorpid}&userid=${userid}'success(res){// 打开成功}})
```
