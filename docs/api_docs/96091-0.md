## 其他说明

### 获取调用授权流程

以考勤门禁硬件为例，流程如下

### 授权变更回调

该回调触发时机为管理员同意授权，无论是首次安装还是后续变更，企业微信都会通过该事件告知服务商硬件数据授权有变化。详情可参考设备数据授权变更回调。

### 获取授权结果

服务商接收回调后可以调用获取授权结果。返回结果包含可被读取数据的硬件设备列表，此后应用可以使用相应的设备SN调用接口获取其产生的数据。详情可参考获取硬件数据授权结果

### 获取硬件产生的数据

在得到管理员授权后，应用可以依据授权的结果集，调用以下接口获取具体的数据。
获取考勤打卡原始数据应用获取温度检测记录应用获取门禁通行记录应用读取门禁通行规则应用写入门禁通行规则应用修改门禁通行规则应用删除门禁通行规则

### 接口调试工具

企业微信提供简易的接口模拟工具，方便厂商进行调试。开发者进入调试工具后，系统会颁发一个测试SN，开发者可以自行进行绑定，并使用该工具模拟产生数据。
注：1. 调试工具目前仅支持考勤门禁业务接口。2. 每个开发者名下单个企业最多可以申请1个测试SN，最多不超过10个企业。
