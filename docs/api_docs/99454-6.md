# 批量查询电子发票

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99454](https://developer.work.weixin.qq.com/document/path/99454)
- **文档 ID**: `99454`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/card/invoice/reimburse/getinvoiceinfobatch`

## 接口描述

报销方在获得用户选择的电子发票标识参数后，可以通过该接口批量查询电子发票的结构化信息。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| item_list[].card_id | string | 是 | 发票id |
| item_list[].encrypt_code | string | 是 | 加密code |

### 请求示例

```json
{
	"item_list": [
		{
		 "card_id":"CARDID1" ,
		 "encrypt_code":"ENCRYPTCODE1"
		},
		{
		 "card_id":"CARDID2" ,
		 "encrypt_code":"ENCRYPTCODE2"
		}
	]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| item_list | object[] | 发票列表 |
| item_list[].card_id | string | 发票id |
| item_list[].openid | string | 用户标识 |
| item_list[].type | string | 发票类型，如广东增值税普通发票 |
| item_list[].payee | string | 发票的收款方 |
| item_list[].detail | string | 发票详情 |
| item_list[].user_info | object | 发票的用户信息 |
| item_list[].user_info.fee | int32 | 发票加税合计金额，以分为单位 |
| item_list[].user_info.title | string | 发票的抬头 |
| item_list[].user_info.billing_time | int32 | 开票时间，为十位时间戳 |
| item_list[].user_info.billing_no | string | 发票代码 |
| item_list[].user_info.billing_code | string | 发票号码 |
| item_list[].user_info.tax | int32 | 税额,以分为单位 |
| item_list[].user_info.fee_without_tax | int32 | 不含税金额，以分为单位 |
| item_list[].user_info.detail | string | 发票详情，一般描述的是发票的使用说明 |
| item_list[].user_info.pdf_url | string | 这张发票对应的PDF_URL |
| item_list[].user_info.reimburse_status | string | 发报销状态INVOICE_REIMBURSE_INIT：发票初始状态，未锁定；INVOICE_REIMBURSE_LOCK：发票已锁定；INVOICE_REIMBURSE_CLOSURE：发票已核销 |
| item_list[].user_info.order_id | string | 订单ID |
| item_list[].user_info.check_code | string | 校验码 |
| item_list[].user_info.buyer_number | string | 购买方纳税人识别号 |
| item_list[].user_info.info | object[] | 商品信息结构 |
| item_list[].user_info.info[].name | string | 项目（商品）名称 |
| item_list[].user_info.info[].num | int32 | 项目数量 |
| item_list[].user_info.info[].unit | string | 项目单位 |
| item_list[].user_info.info[].fee | int32 | 单价，以分为单位 |
