# 参数详细说明

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98027](https://developer.work.weixin.qq.com/document/path/98027)
- **文档 ID**: `98027`
- **API 名称**: `Range`
- **分组信息**: 第 3 个接口，共 13 个

## 接口描述

表示从start_index开始的一段范围

## 请求信息

### 请求示例

```json
{
	"start_index": 10,
	"length": 5
}
```

## 其他说明

### 字段说明

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| start_index | uint32 | 起始位置，从0开始 |
| length | uint32 | 长度 |
