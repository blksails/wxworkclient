## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92374](https://developer.work.weixin.qq.com/document/path/92374)
- **文档 ID**: `92374`

## 接口描述

onUserOpStatistic(Object object)监听用户某些操作触发的事件，用于统计用途，目前仅支持转发事件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| op | String | 否 | 用户操作类型，目前仅有share，表示转发分享。 |
| path | String | 否 | 转发的小程序路径 |

### 请求示例

```json
page{onUserOpStatistic:function(e){if(e.op=='share'){varpath=e.path;}}}
```
