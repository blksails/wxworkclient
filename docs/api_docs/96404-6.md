# 添加客服账号 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96404](https://developer.work.weixin.qq.com/document/path/96404)
- **文档 ID**: `96404`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/kf/account/add?access_token=ACCESS_TOKEN`

## 接口描述

添加客服账号，并可设置客服名称和头像。目前一家企业最多可添加5000个客服账号。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| name | string | 是 | 客服名称不多于16个字符 |
| media_id | string | 是 | 客服头像临时素材。可以调用上传临时素材接口获取。不多于128个字节 |
| 自建应用 | 自建应用 | 否 | 配置到「微信客服- 可调用接口的应用」中，且在管理后台「通过API管理会话消息」-「企业内部开发」对应的自建应用的「可管理的客服账号」处，配置至少一个客服账号 |
| 第三方应用 | 第三方应用 | 否 | 具有“微信客服->管理账号、分配会话和收发消息”权限 |
| 代开发自建应用 | 代开发自建应用 | 否 | 具有“微信客服->管理账号、分配会话和收发消息”权限 |

### 请求示例

```json
{"name":"新建的客服账号","media_id":"294DpAog3YA5b9rTK4PjjfRfYLO0L5qpDHAJIzhhQ2jAEWjb9i661Q4lk8oFnPtmj"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |
| open_kfid | string | 新创建的客服账号ID |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","open_kfid":"wkAJ2GCAAAZSfhHCt7IFSvLKtMPxyJTw"}
```
