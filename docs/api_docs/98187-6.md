# 结束会议 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98187](https://developer.work.weixin.qq.com/document/path/98187)
- **文档 ID**: `98187`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/realcontrol/dismiss?access_token=ACCESS_TOKEN`

## 接口描述

结束一个进行中的会议。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议ID |
| force_dismiss | int32 | 否 | 是否强制结束会议，默认值为1：0：不强制结束会议，会议中有参会者，则无法强制结束会议1 ：强制结束会议，会议中有参会者，也会强制结束会议 |
| retrieve_code | int32 | 否 | 是否回收会议号，默认值为0：0：不回收会议号，可以重新入会1： 回收会议号，不可重新入会说明：周期性会议如果还有子会议，需设置为不回收会议号，否则会导致后续子会议无法正常进行。此字段对快速会议不生效，快速会议会强制收回会议号。 |

### 请求示例

```json
{"meetingid":"meetingid11234","force_dismiss":1,"retrieve_code":1}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
