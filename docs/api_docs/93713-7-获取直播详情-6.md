# API

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93713](https://developer.work.weixin.qq.com/document/path/93713)
- **文档 ID**: `93713`
- **API 名称**: `获取直播详情`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/live/get_detail`
- **分组信息**: 第 7 个接口，共 8 个

## 接口描述

获取直播详情接口用于获取直播的详细信息。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| meeting_id | string | 是 | 直播ID |

### 请求示例

```json
{
  "meeting_id": "123456"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| meeting_info | object | 直播信息 |

### 响应示例

```json
{
  "errcode": 0,
  "errmsg": "ok",
  "meeting_info": {
    "meeting_title": "Weekly Meeting",
    "start_time": 1608313200,
    "end_time": 1608316800,
    "anchor_userid": "zhangsan"
  }
}
```
