# 新建文件夹/文档

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97954](https://developer.work.weixin.qq.com/document/path/97954)
- **文档 ID**: `97954`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_create`

## 接口描述

该接口用于在微盘指定位置新建文件夹、文档。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spaceid | string | 是 | 空间spaceid |
| fatherid | string | 是 | 父目录fileid，在根目录时为空间spaceid |
| file_type | uint32 | 是 | 文件类型，1:文件夹 3:文档(文档) 4:文档(表格) |
| file_name | string | 是 | 文件名字（注意：文件名最多填255个字符，英文算1个，汉字算2个） |

### 请求示例

```json
{
    "spaceid": "SPACEID",
    "fatherid": "FATHERID",
    "file_type": FILE_TYPE,
    "file_name": "FILE_NAME"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| fileid | string | 新建文件的fileid |
| url | string | 文档的访问链接，仅在新建文档时返回 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "fileid": "FILEID",
    "url": "URL"
}
```

## 其他说明

### 参数说明

| 参数 | 类型 | 是否必须 | 说明 |
| --- | --- | --- | --- |
| spaceid | string | 是 | 空间spaceid |
| fatherid | string | 是 | 父目录fileid, 在根目录时为空间spaceid |
| file_type | uint32 | 是 | 文件类型, 1:文件夹 3:文档(文档) 4:文档(表格) |
| file_name | string | 是 | 文件名字（注意：文件名最多填255个字符, 英文算1个, 汉字算2个） |

### 权限说明

* 自建应用需配置到“可调用应用”列表中的应用secret所获取的accesstoken来调用（accesstoken如何获取？）
* 第三方应用需具有“微盘”权限
* 代开发自建应用需具有“微盘”权限
