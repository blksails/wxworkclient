# 管理会中设置 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98175](https://developer.work.weixin.qq.com/document/path/98175)
- **文档 ID**: `98175`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/realcontrol/set?access_token=ACCESS_TOKEN`

## 接口描述

管理进行中会议的设置项，例如：全体静音、是否允许参会者聊天设置、锁定会议、隐藏会议号和密码、是否开启等候室等。目前暂不支持 MRA 设备作为被操作者的情况

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议ID |
| mute_all | bool | 否 | 是否全体静音：true：全体静音false：关闭全体静音 |
| allow_unmute_self | bool | 否 | 是否允许成员自己取消静音，请求参数 mute_all 必传，且 mute_all = true 时设置才生效。true：允许false：不允许 |
| enable_enter_mute | int32 | 否 | 成员入会静音：0：关闭静音1：开启静音2：超过6人自动开启静音 |
| meeting_locked | bool | 否 | 是否锁定会议：true：锁定false：关闭锁定 |
| hide_meeting_code_password | bool | 否 | 隐藏会议号和密码：true：隐藏false：不隐藏 |
| allow_chat | int32 | 否 | 允许参会者聊天设置：0：允许参会者自由聊天1：仅允许参会者公开聊天2：仅允许私聊主持人 |
| allow_share_screen | bool | 否 | 是否允许参会者发起屏幕共享：true：允许false：不允许 |
| allow_external_user | bool | 否 | 是否仅企业成员可入会：true：仅企业成员可入会false：不限制 |
| play_ivr_on_join | bool | 否 | 成员入会是否播放提示音：true：成员入会播放提示音：false：不播放 |
| enable_waiting_room | bool | 否 | 是否开启等候室：true：开启false：关闭 |

### 请求示例

```json
{"meetingid":"MEETINGID","mute_all":true,"allow_unmute_self":true,"enable_enter_mute":1,"meeting_locked":true,"hide_meeting_code_password":true,"allow_chat":1,"allow_share_screen":true,"allow_external_user":true,"play_ivr_on_join":true,"enable_waiting_room":true}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
