# 获取多企业订单详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98893](https://developer.work.weixin.qq.com/document/path/98893)
- **文档 ID**: `98893`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/license/get_union_order?provider_access_token=ACCESS_TOKEN`

## 接口描述

查询某个多企业订单的详情，包括订单的状态、购买的企业、基础账号个数、互通账号个数、账号购买时长等。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 应用服务商的接口调用凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| order_id | string | 是 | 订单id |
| limit | uint32 | 否 | 返回的最大记录数，最大值1000，默认值500 |
| cursor | string | 否 | 用于分页查询的游标，填写上一次调用返回的 next_cursor，首次调用可不填 |

### 请求示例

```json
{
	"order_id": "ORDERID",
	"limit": 1000,
	"cursor": "CURSOR"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| order.order_id | string | 订单号 |
| order.order_type | int32 | 订单类型 8：多企业新购订单 |
| order.order_status | int32 | 订单状态 0：待支付 1：已支付 2：已取消（未支付，订单已关闭） 3：未支付，订单已过期 |
| order.price | int32 | 订单金额，单位分 |
| order.create_time | uint32 | 创建时间，Unix时间戳 |
| order.pay_time | uint32 | 支付时间，Unix时间戳 |
| has_more | int32 | 是否有更多。 0: 没有， 1: 有 |
| next_cursor | string | 分页游标，下次请求时填写到 cursor 以获取之后分页的记录 |
| buy_list | object[] | 多企业购买信息列表 |
| buy_list[].sub_order_id | string | 子订单id，可以调用获取订单中的账号列表接口以获取账号列表 |
| buy_list[].corpid | string | 客户企业id，返回加密的corpid |
| buy_list[].account_count | object | 订单的账号数详情 |
| buy_list[].account_count.base_count | int32 | 基础账号个数 |
| buy_list[].account_count.external_contact_count | int32 | 互通账号个数 |
| buy_list[].account_duration | object | 账号购买时长 |
| buy_list[].account_duration.months | int32 | 购买的月数，每个月按照31天计算 |
| buy_list[].account_duration.days | int32 | 购买的天数 |

```json
		"order_type": 1,

	},

		   "base_count": 100,
       "external_contact_count": 100
	     },

	   		"months": 2,

	    	 }
	}]
}
```
