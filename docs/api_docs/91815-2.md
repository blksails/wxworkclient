# 1.设置接收事件服务器

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91815](https://developer.work.weixin.qq.com/document/path/91815)
- **文档 ID**: `91815`

## 接口描述

订阅后，当企业内指定类型的企业微信“审批应用”单据流程发生变化时，会将审批单最新的流程状态回调给开发者。
