## 其他说明

### ww.openThirdAppServiceChat([params])

兼容性：企业微信 iOS、Android 3.1.18 开始支持，企业微信 PC、Mac 4.1.6 开始支持
功能描述进入应用客服会话。第三方应用可以使用该接口提供按钮，让用户快速打开应用客服的会话。。
ww.openThirdAppServiceChat()
使用限制本接口必须使用应用身份进行注册仅第三方应用可调用第三方应用需要提前配置客服当前用户需要有添加外部联系人权限
参数说明params: Object属性类型必填说明successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

进入应用客服会话。
第三方应用可以使用该接口提供按钮，让用户快速打开应用客服的会话。。

### 使用限制

本接口必须使用应用身份进行注册仅第三方应用可调用第三方应用需要提前配置客服当前用户需要有添加外部联系人权限

### params: Object

属性类型必填说明successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 示例代码

wx.invoke('openThirdAppServiceChat',{},function(res){if(res.err_msg=="openThirdAppServiceChat:ok"){}});
