# 下载文件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97915](https://developer.work.weixin.qq.com/document/path/97915)
- **文档 ID**: `97915`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_download?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于下载文件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fileid | string | 否 | 文件fileid（只支持下载普通文件，不支持下载文件夹或微文档） |

### 请求示例

```json
{"fileid":"FILEID","selected_ticket":"SELECTED_TICKET"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| selected_ticket | string | 微盘和文件选择器jsapi返回的selectedTicket。若填此参数，则不需要填fileid。 |
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| download_url | string | 下载请求url (有效期2个小时) |
| cookie_name | string | 下载请求带cookie的key |
| cookie_value | string | 下载请求带cookie的value |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","download_url":"DOWNLOAD_URL","cookie_name":"COOKIE_NAME","cookie_value":"COOKIE_VALUE"}
```
