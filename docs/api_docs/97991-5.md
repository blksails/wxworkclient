# 查询应用邮箱账号 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97991](https://developer.work.weixin.qq.com/document/path/97991)
- **文档 ID**: `97991`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/exmail/app/get_email_alias?access_token=ACCESS_TOKEN`

## 接口描述

应用调用此接口，可以查询自己的应用邮箱账号及别名邮箱

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| email | string | 当前发信账号的主邮箱地址，发邮件将会以此邮箱地址为发件人 |
| alias_list | string[] | 别名邮箱地址列表，能作为收件人收邮件 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","email":"youxiangceshi@ceshi.wecom.work","alias_list":["youxiang1@ceshi.wecom.work","youxiang2@ceshi.wecom.work"]}
```
