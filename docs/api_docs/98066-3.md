## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
|  | number | 是 | 文档类型
3: 文档，4: 表格, 5: 收集表, 6: PPT, 10: 智能表格，其他类型暂不支持 |
|  | Function | 否 | 成功回调 |
|  | Function | 否 | 失败回调 |
|  | Function | 否 | 取消回调 |
|  | Function | 否 | 完成回调 |

### 请求示例

#### 示例 1: 功能描述

```json
ww.createDoc({docType:3})
```

#### 示例 2: 示例代码

```json
wx.invoke('createDoc',{"docType":1},function(res){if(res.err_msg=="createDoc:ok"){vardocInfo=res.docInfo;vardocId=docInfo.docId;vardocName=docInfo.docName;}});
```

## 其他说明

### ww.createDoc(params)

兼容性：企业微信 4.1.0 开始支持
功能描述新建文档、表格或者收集表。
ww.createDoc({docType:3})
使用限制本接口必须使用应用身份进行注册签名应用必须具有文档使用权限当前用户必须在应用的可见范围之内在 Mac 端使用时，macOS 版本需 > 10.12
参数说明params: Object属性类型必填说明docTypenumber是文档类型
3: 文档，4: 表格, 5: 收集表, 6: PPT, 10: 智能表格，其他类型暂不支持successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码resultObject是属性类型必填说明selectedFileUrlsstring是选择的在线文档的URL列表
错误信息errMsg说明createDoc:ok执行成功createDoc:fail no permission应用签名校验失败，或成员不在应用的可见范围内，或应用未开启文档使用权限createDoc:fail doc app closed.基础应用“文档”如果未启用createDoc:fail form app closed.基础应用“收集表”如果没有启用

### 功能描述

新建文档、表格或者收集表。

### 使用限制

本接口必须使用应用身份进行注册签名应用必须具有文档使用权限当前用户必须在应用的可见范围之内在 Mac 端使用时，macOS 版本需 > 10.12

### params: Object

属性类型必填说明docTypenumber是文档类型
3: 文档，4: 表格, 5: 收集表, 6: PPT, 10: 智能表格，其他类型暂不支持successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码resultObject是属性类型必填说明selectedFileUrlsstring是选择的在线文档的URL列表

### 错误信息

errMsg说明createDoc:ok执行成功createDoc:fail no permission应用签名校验失败，或成员不在应用的可见范围内，或应用未开启文档使用权限createDoc:fail doc app closed.基础应用“文档”如果未启用createDoc:fail form app closed.基础应用“收集表”如果没有启用

### 示例代码

wx.invoke('createDoc',{"docType":1},function(res){if(res.err_msg=="createDoc:ok"){vardocInfo=res.docInfo;vardocId=docInfo.docId;vardocName=docInfo.docName;}});
