# 支付成功通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95804](https://developer.work.weixin.qq.com/document/path/95804)
- **文档 ID**: `95804`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/callback/pay_success_notify`

## 接口描述

当服务商购买接口调用许可账号并完成付款后，企业微信后台会推送“支付成功通知”到服务商的系统事件接收URL。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| ServiceCorpId | string | 服务商CorpID |
| InfoType | string | 事件类型，此时固定为license_pay_success |
| AuthCorpId | string | 客户企业CorpID |
| OrderId | string | 订单号。如果为多企业新购订单，该值为子订单号 |
| BuyerUserId | string | 服务商内下单用户UserID |
| TimeStamp | uint32 | 时间戳 |

### 响应示例

```xml
<xml>
	<ServiceCorpId><![CDATA[xxxx]]></ServiceCorpId>
	<InfoType><![CDATA[license_pay_success]]></InfoType>
	<AuthCorpId><![CDATA[yyyy]]></AuthCorpId>
	<OrderId><![CDATA[zzzz]]></OrderId>
	<BuyerUserId><![CDATA[USERID]]></BuyerUserId>
	<TimeStamp>1403610513</TimeStamp>
</xml>
```
