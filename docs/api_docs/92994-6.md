# 批量获取客户详情 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92994](https://developer.work.weixin.qq.com/document/path/92994)
- **文档 ID**: `92994`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/batch/get_by_user?access_token=ACCESS_TOKEN`

## 接口描述

企业/第三方可通过此接口获取指定成员添加的客户信息列表。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| userid_list |  | 是 | 企业成员的userid列表，字符串类型，最多支持100个 |
| cursor |  | 否 | 用于分页查询的游标，字符串类型，由上一次调用返回，首次调用可不填 |
| limit |  | 否 | 返回的最大记录数，整型，最大值100，默认值50，超过最大值时取最大值 |
| 自建应用 | 自建应用 | 否 | 配置到「客户联系 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 具有「客户基础信息」权限 |
| 第三方应用 | 第三方应用 | 否 | 具有「客户基础信息」权限营销获客类应用具有「建联客户信息」权限 |

### 请求示例

```json
{"userid_list":["zhangsan","lisi"],"cursor":"","limit":100}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| external_contact_list.external_contact |  | 客户的基本信息，可以参考获取客户详情 |
| external_contact_list.follow_info |  | 企业成员客户跟进信息，可以参考获取客户详情，但标签信息只会返回企业标签和规则组标签的tag_id，个人标签将不再返回 |
| next_cursor |  | 分页游标，再下次请求时填写以获取之后分页的记录，如果已经没有更多的数据则返回空 |
| fail_info.unlicensed_userid_list |  | 注：若请求中所有userid都无有效互通许可，接口直接报错701008。如果部分userid无有效互通许可，接口返回成功，该字段为无许可的userid列表 |
