# 上传图片 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90256](https://developer.work.weixin.qq.com/document/path/90256)
- **文档 ID**: `90256`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/media/uploadimg?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/media/uploadimg?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |

### 请求示例

```text
---------------------------acebdf13572468Content-Disposition:form-data;name="fieldNameHere";filename="20180103195745.png"Content-Type:image/pngContent-Length:220<@INCLUDE*C:\Users\abelzhu\Pictures\企业微信截图_20180103195745.png*@>---------------------------acebdf13572468--
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| url |  | 上传后得到的图片URL。永久有效 |

### 响应示例

```json
{"errcode":0,"errmsg":""，"url":"http://p.qpic.cn/pic_wework/3474110808/7a7c8471673ff0f178f63447935d35a5c1247a7f31d9c060/0"}
```

## 其他说明

### 上传的图片大小限制

图片文件大小应在 5B ~ 2MB 之间
