## 接口描述

可查看商户号的收入统计数据及收入明细数据，收入统计数据包括服务商商户号昨日、当日、本周、本月的到账金额，同时提供周维度的收入趋势图供查看，收入明细数据呈现了每笔订单的收入到账数据，可筛选及导出。
