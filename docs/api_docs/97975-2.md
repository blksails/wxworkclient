# 文件变更事件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97975](https://developer.work.weixin.qq.com/document/path/97975)
- **文档 ID**: `97975`

## 接口描述

当接口指定的管理员在企业微信客户端对应用上传的文件进行变更，触发该事件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 此事件该值固定为sys，表示该消息由系统生成 |
| CreateTime |  | 否 | 消息创建时间，unix时间戳 |
| MsgType |  | 否 | 消息类型，此时固定为：event |
| Event |  | 否 | 事件类型，此时固定为：wedrive_file_change |
| ChangeType |  | 否 | 变更类型。有以下值：create_file-创建文件 ，rename_file-重命名文件，update_file-更新文件内容，delete_file-删除文件，move_file-移动文件 |
| FileId |  | 否 | 文件fileid。可能有多个FileId节点，表示多个文件 |

### 请求示例

```xml
<xml><ToUserName><![CDATA[toUser]]></ToUserName><FromUserName><![CDATA[sys]]></FromUserName><CreateTime>1348831860</CreateTime><MsgType><![CDATA[event]]></MsgType><Event><![CDATA[wedrive_file_change]]></Event><ChangeType><![CDATA[xxx_file]]></ChangeType><FileId><![CDATA[spxxxxxxxxxxxxx]]></FileId></xml>
```
