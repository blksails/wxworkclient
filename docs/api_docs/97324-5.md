## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 商户号 | string[1,32] | 是 | 二级商户号，由企业微信生成并下发。 |
| 商户订单号 | string[6,32] | 是 | 商户系统内部订单号，只能是数字、大小 写字母_-*且在同一个商户号下唯一。 |

### 请求示例

```json
{"mchid":"xxx","out_trade_no":"xxx"}
```

## 响应信息

### 响应示例

{"errcode":0,"errmsg":"ok"}
