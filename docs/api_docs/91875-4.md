# 异步任务回调通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91875](https://developer.work.weixin.qq.com/document/path/91875)
- **文档 ID**: `91875`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/service/receive`

## 接口描述

推送到服务商设置的接收事件回调url上。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 | 消息签名 |
| timestamp | uint32 | 是 | 时间戳 |
| nonce | uint32 | 是 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ServiceCorpId | string | 是 | 服务商corpid |
| InfoType | string | 是 | 事件类型，batch_job_result，通讯录导出完成回调 |
| TimeStamp | uint32 | 是 | 回调事件的时间戳 |
| AuthCorpId | string | 是 | 企业corpid |
| JobId | string | 是 | 任务id,最大长度为64字符 |
| JobType | string | 是 | 任务类型，目前有contact_id_translate |

### 请求示例

```xml
<xml>
	<ServiceCorpId><![CDATA[wx233kasfa2jkfAAA]]></ServiceCorpId>
	<InfoType><![CDATA[batch_job_result]]></InfoType>
	<TimeStamp>1502682173</TimeStamp>
	<AuthCorpId><![CDATA[wxf1b4f49f3a794AAA]]></AuthCorpId>
	<BatchJob>
	   <JobId><![CDATA[S0MrnndvRG5fadSlLwiBqiDD52FZk4]]></JobId>
	   <JobType><![CDATA[contact_id_translate]]></JobType>
	</BatchJob>
</xml>
```
