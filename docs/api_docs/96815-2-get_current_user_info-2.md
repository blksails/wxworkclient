# 获取当前使用者信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96815](https://developer.work.weixin.qq.com/document/path/96815)
- **文档 ID**: `96815`
- **API 名称**: `get_current_user_info`
- **请求方法**: `GET`
- **分组信息**: 第 2 个接口，共 6 个

## 接口描述

获取当前使用者属于哪个企业的信息。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| user_corpid | string | 当前使用者所属的企业ID |
