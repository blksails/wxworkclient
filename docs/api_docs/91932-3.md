# 退款通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91932](https://developer.work.weixin.qq.com/document/path/91932)
- **文档 ID**: `91932`
- **请求方法**: `POST`

## 接口描述

当某个客户发起有效的退款，经服务商在管理端同意后（或过期自动完成退款），会触发回调事件给服务商，服务商收到退款事件后，需自行通过相关接口重新获取这个客户最新的购买状态

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId |  | 否 | 第三方应用的SuiteId |
| PaidCorpId |  | 否 | 购买方corpid |
| InfoType |  | 否 | 固定为refund |
| TimeStamp |  | 否 | 时间戳 |
| OrderId |  | 否 | 订单号，付费订单的唯一标志。服务商可据此拉取购买信息。该OrderId由企业微信生成，不超过32个字符 |
