## 其他说明

### ww.updateMomentsSetting(params)

兼容性：企业微信 iOS、Android 3.1.12 开始支持
功能描述设置朋友圈封面与签名。
ww.updateMomentsSetting({signature:'个性签名',imgUrl:'https://work.weixin.qq.com/'})
使用限制本接口必须使用应用身份进行注册应用需有客户联系功能权限当前成员必须配置了客户联系功能当前成员必须在客户朋友圈使用范围当前成员必须具备外部沟通管理成员使用权限
参数说明params: Object属性类型必填说明signaturestring否个性签名空字符串表示清空个性签名使用限制最多支持 30 字imgUrlstring否朋友圈封面封面图建议为正方形，尺寸建议为 1125*1125 像素successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码
使用说明同时设置了签名跟封面url，客户端更新顺序为先更新签名，再更新封面图url（封面图若不符合要求会让用户重新调整）。

### 功能描述

设置朋友圈封面与签名。

### 使用限制

本接口必须使用应用身份进行注册应用需有客户联系功能权限当前成员必须配置了客户联系功能当前成员必须在客户朋友圈使用范围当前成员必须具备外部沟通管理成员使用权限

### params: Object

属性类型必填说明signaturestring否个性签名空字符串表示清空个性签名使用限制最多支持 30 字imgUrlstring否朋友圈封面封面图建议为正方形，尺寸建议为 1125*1125 像素successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 使用说明

同时设置了签名跟封面url，客户端更新顺序为先更新签名，再更新封面图url（封面图若不符合要求会让用户重新调整）。

### 示例代码

wx.invoke("updateMomentsSetting",{signature:"个性签名",// 个性签名imgUrl:"http://www.xxx/"// 封面url},function(res){if(res.err_msg=="updateMomentsSetting:ok"){}});
