# 获取学生付款结果

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94553](https://developer.work.weixin.qq.com/document/path/94553)
- **文档 ID**: `94553`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/get_payment_result`

## 接口描述

获取学生付款结果

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| payment_id | string | 是 | 收款项目id，由jssdk的发起班级收款接口或者小程序的发起班级收款接口返回 |

### 请求示例

```json
{
   "payment_id":"xxxx"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| project_name | string | 项目名称 |
| amount | int32 | 金额 |
| payment_result | object[] | 学生付款信息列表 |
| payment_result[].student_userid | string | 学生账号 |
| payment_result[].trade_state | int32 | 付款状态。0表示未付款；1表示已付款 |
| payment_result[].trade_no | string | 订单号 |
| payment_result[].payer_parent_userid | string | 付款家长账号 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"project_name": "学费",
	"amount": 998,
	"payment_result": [{
		"student_userid": "xxxx",
		"trade_state": 1,
		"trade_no": "xxxxx",
		"payer_parent_userid": "zhangshan"
	}]
}
```
