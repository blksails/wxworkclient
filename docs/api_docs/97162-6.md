# 获取代开发应用模板凭证

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97162](https://developer.work.weixin.qq.com/document/path/97162)
- **文档 ID**: `97162`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/get_suite_token`

## 接口描述

该API用于获取代开发应用模板凭证（suite_access_token），该接口实际上与获取第三方应用凭证是同一个接口。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| suite_id | string | 是 | 第三方应用id或者代开发应用模板id。第三方应用以ww或wx开头应用id（对应于旧的以tj开头的套件id）；代开发应用以dk开头 |
| suite_secret | string | 是 | 第三方应用secret 或者代开发应用模板secret |
| suite_ticket | string | 是 | 企业微信后台推送的ticket |

### 请求示例

```json
{
	"suite_id":"wwddddccc7775555aaa" ,
	"suite_secret": "ldAE_H9anCRN21GKXVfdAAAAAAAAAAAAAAAAAA",
	"suite_ticket": "Cfp0_givEagXcYJIztF6sfbdmIZCmpaR8ZBsvJEFFNBrWmnD5-CGYJ3_NhYexMyw"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| suite_access_token | string | 第三方或者代开发应用模板access_token，最长为512字节 |
| expires_in | uint32 | 有效期（秒） |

### 响应示例

```json
{
	"errcode":0,
	"errmsg":"ok",
	"suite_access_token":"61W3mEpU66027wgNZ_MhGHNQDHnFATkDa9-2llMBjUwxRSNPbVsMmyD-yq8wZETSoE5NQgecigDrSHkPtIYA",
	"expires_in":7200
}
```

## 其他说明

### 注意

因历史原因，该接口在调用失败时才返回errcode。没返回errcode视为调用成功
