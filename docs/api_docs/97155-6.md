# 获取个人居民上报数据统计 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97155](https://developer.work.weixin.qq.com/document/path/97155)
- **文档 ID**: `97155`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/report/resident/get_user_status?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/report/resident/get_user_status?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证。“获取access_token” |
| userid |  | 是 | 成员的userId |
| 自建应用 | 自建应用 | 否 | 配置到「居民上报 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"userid":"zhangsan"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| processing |  | 办理中 |
| added_today |  | 今日上报 |
| solved_today |  | 今日办结 |
| pending |  | 待受理 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","processing":1,"added_today":1,"solved_today":1,"pending":1}
```
