# 应用授权的token

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90593](https://developer.work.weixin.qq.com/document/path/90593)
- **文档 ID**: `90593`
- **API 名称**: `获取第三方应用凭证`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/get_provider_token`
- **分组信息**: 第 2 个接口，共 3 个

## 接口描述

获取第三方应用凭证，用于访问应用授权的接口。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| suite_id | string | 是 | 第三方应用ID |
| suite_secret | string | 是 | 第三方应用密钥 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| provider_access_token | string | 第三方应用凭证，最长为512字节 |
| expires_in | uint32 | 凭证有效期（秒） |
