# 注意事项

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93701](https://developer.work.weixin.qq.com/document/path/93701)
- **文档 ID**: `93701`
- **API 名称**: `notes`
- **请求方法**: `GET`
- **分组信息**: 第 3 个接口，共 4 个

## 接口描述

一些关于日程接口的注意事项：企业微信需要更新到4.0.20及以上版本，日历/日程不可跨应用操作，日历/日程通知范围内的成员需在应用的可见范围内，每个企业可创建的公共日历数量上限为3万个，每个应用可创建的日历本数量上限为1万个，每个企业用户被共享的日历数上限为100个，每个应用每天创建的日程数量上限为2万个。
