# 获取企业成员个人二维码

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92398](https://developer.work.weixin.qq.com/document/path/92398)
- **文档 ID**: `92398`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

获取企业成员个人二维码

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| timeout | Number | 否 | 超时时间，单位 ms |
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```text
wx.qy.getQrCode ({
  success: function(res) {
    var qrCode = res.qrCode
  }
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| qrCode | String | 当前企业成员的个人二维码url |

## 其他说明

### 调用前提

1、必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态
2、要求用户在应用可见范围内，且每次调用都需要用户同意确认
