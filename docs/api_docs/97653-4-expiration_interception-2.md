# 应用到期拦截

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97653](https://developer.work.weixin.qq.com/document/path/97653)
- **文档 ID**: `97653`
- **API 名称**: `expiration_interception`
- **请求方法**: `N/A`
- **接口地址**: `N/A`
- **分组信息**: 第 4 个接口，共 4 个

## 接口描述

应用到期拦截相关信息，包括试用到期和服务到期。

## 其他说明

### 试用到期

企业在应用市场中首次添加第三方应用时，可选试用版本无需付费。应用试用到期后，企业需购买付费版本方可继续使用。

### 服务到期

企业购买付费版本后，将按照企业所购买的时长生效服务。应用服务到期后，企业需续费后方可继续使用。
