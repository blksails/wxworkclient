## 其他说明

### ww.queryCurrHWOpenTalk([params])

兼容性：企业微信 2.5.0 开始支持
功能描述查询当前是否在视频会议。
ww.queryCurrHWOpenTalk()
参数说明params: Object属性类型必填说明successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码inTalkTypestring是当前通话的类型合法值说明None当前不在任何通话中HWOpenTalk视频会议中VoIPvoip通话中SystemCall系统通话中ticketstring是如果当前通话类型为视频会议，会将当前会议码票据（enterHWOpenTalk 中携带的 ticket）回传

### 功能描述

查询当前是否在视频会议。

### params: Object

属性类型必填说明successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码inTalkTypestring是当前通话的类型合法值说明None当前不在任何通话中HWOpenTalk视频会议中VoIPvoip通话中SystemCall系统通话中ticketstring是如果当前通话类型为视频会议，会将当前会议码票据（enterHWOpenTalk 中携带的 ticket）回传
