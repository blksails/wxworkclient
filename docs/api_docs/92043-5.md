# 获取部门学生详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92043](https://developer.work.weixin.qq.com/document/path/92043)
- **文档 ID**: `92043`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/user/list`

## 接口描述

获取部门学生详情，包括学生信息、家长信息等。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| department_id | string | 是 | 获取的部门id |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| students | object[] | 学生列表 |
| students[].student_userid | string | 学生UserID |
| students[].name | string | 学生名字 |
| students[].department | uint32[] | 学生所在的班级id列表，不超过20个 |
| students[].parents | object[] | 学生家长列表 |
| students[].parents[].parent_userid | string | 家长的userid |
| students[].parents[].relation | string | 学生与家长的关系 |
| students[].parents[].mobile | string | 家长手机号，第三方不可获取 |
| students[].parents[].is_subscribe | int32 | 家长是否关注了“学校通知”，0-未关注，1-已关注 |
| students[].parents[].external_userid | string | 家长的external_userid，仅当家长已关注才返回 |
