# 调起扫一扫

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90525](https://developer.work.weixin.qq.com/document/path/90525)
- **文档 ID**: `90525`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

调起企业微信扫一扫。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| params.needResult | boolean | 否 | 扫描结果是否由企业微信处理 为 false 时直接返回扫描结果 |
| params.scanType | string[] | 否 | 扫码类型 |
| params.success | Function | 否 | 成功回调 |
| params.fail | Function | 否 | 失败回调 |
| params.cancel | Function | 否 | 取消回调 |
| params.complete | Function | 否 | 完成回调 |

### 请求示例

```text
ww.scanQRCode({
  needResult: true,
  scanType: ['qrCode']
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errMsg | string | 通用错误信息 |
| errCode | number | 通用错误码 |
| resultStr | string | 扫码结果，needResult为true时返回 |
