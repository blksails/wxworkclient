# 企业主体下的external_userid

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96248](https://developer.work.weixin.qq.com/document/path/96248)
- **文档 ID**: `96248`
- **API 名称**: `external_userid_enterprise`
- **请求方法**: `GET`
- **分组信息**: 第 4 个接口，共 8 个

## 接口描述

external_userid是企业的外部联系人ID，由企业微信系统生成，企业无法在管理端页面中查看，只能通过 API 获取到。
