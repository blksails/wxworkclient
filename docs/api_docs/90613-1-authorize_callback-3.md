# 应用授权

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90613](https://developer.work.weixin.qq.com/document/path/90613)
- **文档 ID**: `90613`
- **API 名称**: `authorize_callback`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`
- **分组信息**: 第 1 个接口，共 2 个

## 接口描述

企业微信服务器向应用的“指令回调URL”推送授权、通讯录变更、ticket变化等事件消息。消息结构体使用创建应用时的EncodingAESKey进行加密。
