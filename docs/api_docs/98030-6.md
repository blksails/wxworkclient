# 获取表格数据

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98030](https://developer.work.weixin.qq.com/document/path/98030)
- **文档 ID**: `98030`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/spreadsheet/get_sheet_range_data?access_token=ACCESS_TOKEN`

## 接口描述

本接口用于获取指定范围内的在线表格信息，单次查询的范围大小需满足以下限制：1. 查询范围行数 <=1000 2. 查询范围列数 <=200 3. 范围内的总单元格数量 <=10000

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 是 | 在线表格唯一标识 |
| sheet_id | string | 是 | 工作表ID，工作表的唯一标识 |
| range | string | 是 | 查询的范围，格式遵循 A1表示法 |

### 请求示例

```json
{
	"docid": "DOCID",
	"sheet_id": "AABBCC",
	"range": "A1:B2"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| data.result | object | 表格数据 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"data": {
		"result": {

		}
	}
}
```
