# 关于客户群ID

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/101215](https://developer.work.weixin.qq.com/document/path/101215)
- **文档 ID**: `101215`
- **API 名称**: `customer_group_id_info`
- **请求方法**: `GET`
- **接口地址**: `/cgi-bin/externalcontact/customer_group_id_info`
- **分组信息**: 第 5 个接口，共 9 个

## 接口描述

企业微信为每个客户群生成唯一的ID（chat_id），同样需要注意的是，对于同一个群，不同调用方（企业/第三方服务商）获取到的chat_id是不同的。
