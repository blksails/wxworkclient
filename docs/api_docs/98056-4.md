# 修改收集表设置事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98056](https://developer.work.weixin.qq.com/document/path/98056)
- **文档 ID**: `98056`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/callback/modify_form_settings?access_token=ACCESS_TOKEN`

## 接口描述

当文档管理员在API创建的文档上修改收集表设置时，包括管理员权限变更事件（增删改）、收集范围变更事件（增删改）、其他收集表设置调整，触发该事件。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName | string | 是 | 企业微信CorpID |
| FromUserName | string | 是 | 成员UserID |
| CreateTime | uint32 | 是 | 消息创建时间，unix时间戳 |
| MsgType | string | 是 | 消息类型，固定为：`event` |
| Event | string | 是 | 事件类型，固定为：`doc_change` |
| ChangeType | string | 是 | 事件类型，固定为：`form_settings_change` |
| FormId | string[] | 是 | 收集表ID列表 |
