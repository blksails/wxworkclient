# 获取下级/下游企业的access_token

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96873](https://developer.work.weixin.qq.com/document/path/96873)
- **文档 ID**: `96873`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/corpgroup/corp/gettoken`

## 接口描述

获取应用可见范围内下级/下游企业的access_token，该access_token可用于调用下级/下游企业通讯录的只读接口。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证，上级/上游企业应用access_token |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| corpid | string | 是 | 已授权的下级/下游企业corpid |
| business_type | int32 | 否 | 填0则为企业互联/局校互联，填1则表示上下游企业，默认0 |
| agentid | uint32 | 是 | 已授权的下级/下游企业应用ID |

### 请求示例

```json
{
	"corpid": "wwabc",
	"business_type":1,
	"agentid": 1111
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 出错返回码，为0表示成功，非0表示调用失败 |
| errmsg | string | 返回码提示语 |
| access_token | string | 获取到的下级/下游企业调用凭证，最长为512字节 |
| expires_in | uint32 | 凭证的有效时间（秒） |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "ok",
   "access_token": "accesstoken000001",
   "expires_in": 7200
}
```
