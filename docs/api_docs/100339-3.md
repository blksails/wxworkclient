#### 示例 1: 功能描述

```json
ww.getShareInfo({shareTicket:'xxx'})
```

#### 示例 2: 示例代码


## 其他说明

### ww.getShareInfo(params)

兼容性：企业微信 3.1.8 开始支持
功能描述获取私密消息信息。
ww.getShareInfo({shareTicket:'xxx'})
使用限制本接口必须使用应用身份进行注册
参数说明params: Object属性类型必填说明shareTicketstring是getContext 接口返回的 shareTicketsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码encryptedDatastring是转发信息的加密数据ivstring是加密算法的初始向量

### 功能描述

获取私密消息信息。

### 使用限制

本接口必须使用应用身份进行注册

### params: Object

属性类型必填说明shareTicketstring是getContext 接口返回的 shareTicketsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码encryptedDatastring是转发信息的加密数据ivstring是加密算法的初始向量

### 示例代码

wx.invoke("getShareInfo",{shareTicket:"xxxx",},function(res){if(res.err_msg=="getShareInfo:ok"){encryptedData=res.encryptedData;iv=res.iv;}});
