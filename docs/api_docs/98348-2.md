# 成员入会事件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98348](https://developer.work.weixin.qq.com/document/path/98348)
- **文档 ID**: `98348`

## 接口描述

每一个与会者加入API创建的会议时，触发该事件。与会者可以是普通与会者，也可以是主持人。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 操作者UserID。如果为系统触发事件，默认为sys；如果为成员操作事件，仅当是企业成员时有值。 |
| FromUserTmpOpenId |  | 否 | 操作者TmpOpenID。操作者的会中临时ID，仅成员操作事件有值。 |
| CreateTime |  | 否 | 消息创建时间，unix时间戳 |
| MsgType |  | 否 | 消息类型，此时固定为：event |
| Event |  | 否 | 事件类型，此时固定为：meeting_change |
| ChangeType |  | 否 | 事件类型，此时固定为：join_meeting |
| MeetingId |  | 否 | 会议ID |
