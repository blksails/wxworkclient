- **分组信息**: 第 1 个接口，共 2 个

## 接口描述

使用 docker-export 将运行环境导出 tar 包，用于数据与智能专区的镜像导入。请特别注意不能用 docker-save 命令生成 tar 包，否则可能无法执行启动命令。
