- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100858](https://developer.work.weixin.qq.com/document/path/100858)
- **文档 ID**: `100858`

## 接口描述

加入视频会议。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
|  | string | 是 | 会议码，从对应的视频会议硬件上获取 |
|  | string | 是 | 会议码票据，调用方可以根据会议码生成对应的会议码票据此票据会在调用ww.queryCurrHWOpenTalk时返回，便于换回对应的会议码调用方需要记录票据到会议码的映射关系 |
|  | Function | 否 | 成功回调 |
|  | Function | 否 | 失败回调 |
|  | Function | 否 | 取消回调 |
|  | Function | 否 | 完成回调 |

### 请求示例

#### 示例 1: 功能描述

```json
ww.enterHWOpenTalk({code:code,ticket:ticket})
```

#### 示例 2: 示例代码

```json
wx.invoke('enterHWOpenTalk',{'code':"",'ticket':"",},function(res){if(res.err_msg=="enterHWOpenTalk:ok"){// 发起加入会议请求成功}else{// 发起加入会议请求失败}});
```

|  | string | 通用错误信息 |
|  | number | 通用错误码 |

## 其他说明

### ww.enterHWOpenTalk(params)

兼容性：企业微信 2.5.0 开始支持
功能描述加入视频会议。
ww.enterHWOpenTalk({code:code,ticket:ticket})
使用限制只能加入同企业硬件创建的视频会议。
参数说明params: Object属性类型必填说明codestring是会议码，从对应的视频会议硬件上获取ticketstring是会议码票据，调用方可以根据会议码生成对应的会议码票据此票据会在调用ww.queryCurrHWOpenTalk时返回，便于换回对应的会议码调用方需要记录票据到会议码的映射关系successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

加入视频会议。

### 使用限制

只能加入同企业硬件创建的视频会议。

### params: Object

属性类型必填说明codestring是会议码，从对应的视频会议硬件上获取ticketstring是会议码票据，调用方可以根据会议码生成对应的会议码票据此票据会在调用ww.queryCurrHWOpenTalk时返回，便于换回对应的会议码调用方需要记录票据到会议码的映射关系successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

wx.invoke('enterHWOpenTalk',{'code':"",'ticket':"",},function(res){if(res.err_msg=="enterHWOpenTalk:ok"){// 发起加入会议请求成功}else{// 发起加入会议请求失败}});
