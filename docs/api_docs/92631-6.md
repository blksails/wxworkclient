# 获取审批模板详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92631](https://developer.work.weixin.qq.com/document/path/92631)
- **文档 ID**: `92631`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/oa/gettemplatedetail?access_token=ACCESS_TOKEN`

## 接口描述

服务商可通过第三方应用Secret调用本接口，获取企业微信“审批应用”内，由服务商添加的模板的详情。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。须使用第三方应用的secret获取，获取方式参考：文档-获取access_token |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| template_id | string | 是 | 模板的唯一标识id。可在“获取审批单据详情”、“审批状态变化回调通知”中获得，也可在使用“复制/更新模板到企业”接口回调中获得。注：此id为企业内模板的实例id，非服务商后台对应模板的id |

### 请求示例

```json
{
   "template_id" : "ZLqk8pcsAoXZ1eYa6vpAgfX28MPdYU3ayMaSPHaaa"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| template_names | object[] | 模板名称，若配置了多语言则会包含中英文的模板名称，默认为zh_CN中文 |
| template_content | object | 模板控件信息 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "template_names": [...],
    "template_content": {...}
}
```

## 其他说明

### 附录

包含Data控件、Selector控件、Contact控件、Table、Attendance控件、Vacation控件、Tips控件的config说明

### 错误说明

包含错误码301025和301026的说明
