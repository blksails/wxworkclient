# 标签成员变更事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96304](https://developer.work.weixin.qq.com/document/path/96304)
- **文档 ID**: `96304`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

该事件会回调给通讯录同步助手以及代开发自建应用

## 请求信息

### 请求示例

```xml
<xml>
	<ToUserName><![CDATA[toUser]]></ToUserName>
	<FromUserName><![CDATA[sys]]></FromUserName>
	<CreateTime>1403610513</CreateTime>
	<MsgType><![CDATA[event]]></MsgType>
	<Event><![CDATA[change_contact]]></Event>
	<ChangeType><![CDATA[update_tag]]></ChangeType>
	<TagId>1</TagId>
	<AddUserItems><![CDATA[zhangsan,lisi]]></AddUserItems>
	<DelUserItems><![CDATA[zhangsan1,lisi1]]></DelUserItems>
	<AddPartyItems><![CDATA[1,2]]></AddPartyItems>
	<DelPartyItems><![CDATA[3,4]]></DelPartyItems>
</xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| ToUserName | string | 企业微信CorpID |
| FromUserName | string | 此事件该值固定为sys，表示该消息由系统生成 |
| CreateTime | int32 | 消息创建时间 （整型） |
| MsgType | string | 消息的类型，此时固定为event |
| Event | string | 事件的类型，此时固定为change_contact |
| ChangeType | string | 固定为update_tag |
| TagId | string | 标签Id |
| AddUserItems | string | 标签中新增的成员userid列表，用逗号分隔 |
| DelUserItems | string | 标签中删除的成员userid列表，用逗号分隔 |
| AddPartyItems | string | 标签中新增的部门id列表，用逗号分隔 |
| DelPartyItems | string | 标签中删除的部门id列表，用逗号分隔 |
