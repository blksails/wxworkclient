# 获取标签成员

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90349](https://developer.work.weixin.qq.com/document/path/90349)
- **文档 ID**: `90349`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/tag/get`

## 接口描述

获取标签成员列表，返回应用可见范围内的成员信息。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| tagid | string | 是 | 标签ID |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| tagname | string | 标签名 |
| userlist | object[] | 标签中包含的成员列表 |
| userlist[].userid | string | 成员账号 |
| userlist[].name | string | 成员名称，代开发自建应用需要管理员授权才返回该字段；此字段从2019年12月30日起，对新创建第三方应用不再返回，2020年6月30日起，对所有历史第三方应用不再返回，后续第三方仅通讯录应用可获取，未返回名称的情况需要通过通讯录展示组件来展示名字 |
| partylist | int32[] | 标签中包含的部门id列表 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "ok",
   "tagname": "乒乓球协会",
   "userlist": [
         {
             "userid": "zhangsan",
             "name": "李四"
         }
     ],
   "partylist": [2]
}
```

## 其他说明

### 权限说明

无限制，但返回列表仅包含应用可见范围的成员；第三方可获取自己创建的标签及应用可见范围内的标签详情
