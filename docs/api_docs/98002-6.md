# 获取公共邮箱详情 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98002](https://developer.work.weixin.qq.com/document/path/98002)
- **文档 ID**: `98002`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/exmail/publicmail/get?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于获取公共邮箱详细信息，包含公共邮箱名称、权限信息。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id_list | uint32 | 是 | 公共邮箱ID列表 |

### 请求示例

```json
{"id_list":[1,2]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| id | uint32 | 公共邮箱ID |
| email | string | 公共邮箱地址 |
| name | string | 公共邮箱名称 |
| userid_list | string | 有权限使用公共邮箱的成员UserID列表 |
| department_list | uint32 | 有权限使用公共邮箱的部门列表 |
| tag_list | uint32 | 有权限使用公共邮箱的标签列表 |
| alias_list | string | 公共邮箱的别名列表 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","list":[{"id":1,"email":"apitest3@gzdev.com","name":"测试","userid_list":{"list":["member","member2"]},"department_list":{"list":[1,2]},"tag_list":{"list":[1,2]},"alias_list":{"list":["lisi@gzdev.com","wangwu@gzdev.com"]}}]}
```
