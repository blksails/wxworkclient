# 获取文件信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97920](https://developer.work.weixin.qq.com/document/path/97920)
- **文档 ID**: `97920`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_info?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于获取指定文件的信息。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fileid | string | 是 | 文件fileid |

### 请求示例

```json
{"fileid":"FILEID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| fileid | string | 文件fileid |
| file_name | string | 文件名字 |
| spaceid | string | 文件所在的空间spaceid |
| fatherid | string | 文件所在的目录fileid, 在根目录时为fileid |
| file_size | uint64 | 文件大小 |
| ctime | uint64 | 文件创建时间 |
| mtime | uint64 | 文件最后修改时间 |
| file_type | uint32 | 1: 文件夹 2:文件 3: 文档(文档) 4: 文档(表格) 5:文档(收集表) 6:文档(幻灯片) |
| file_status | uint32 | 文件状态, 1:正常 2:删除 |
| sha | string | 文件sha。可用于确认是否跟与上传的文件一致，或避免重复上传相同的文件 |
| md5 | string | 文件md5。可用于确认是否跟与上传的文件一致，或避免重复上传相同的文件 |
| url | string | 仅微文档类型返回访问链接 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","file_info":{"fileid":"FILEID","file_name":"FILE_NAME","spaceid":"SPACEID","fatherid":"FATHERID","file_size":FILE_SIZE,"ctime":CTIME,"mtime":MTIME,"file_type":FILE_TYPE,"file_status":FILE_STATUS,"sha":"SHA","md5":"MD5","url":"URL"}}
```
