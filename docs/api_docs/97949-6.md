# 获取邀请链接

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97949](https://developer.work.weixin.qq.com/document/path/97949)
- **文档 ID**: `97949`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/space_share`

## 接口描述

该接口用于获取空间邀请分享链接。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spaceid | string | 是 | 空间spaceid |

### 请求示例

```json
{
    "spaceid": "SPACEID"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| space_share_url | string | 邀请链接 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "space_share_url": "SPACE_SHARE_URL"
}
```
