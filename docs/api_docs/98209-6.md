# 获取录制文件访问统计 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98209](https://developer.work.weixin.qq.com/document/path/98209)
- **文档 ID**: `98209`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/record/get_statistics?access_token=ACCESS_TOKEN`

## 接口描述

获取会议录制 ID 对应的访问数据，按照天维度返回。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meeting_record_id | string | 是 | 会议录制ID |
| start_time | int32 | 否 | 查询起始时间戳，UNIX 时间戳（单位秒）。说明：默认展示最近31天的数据，时间区间不允许超过31天。 |
| end_time | int32 | 否 | 查询结束时间戳，UNIX 时间戳（单位秒）。说明：默认展示最近31天的数据，时间区间不允许超过31天。 |

### 请求示例

```json
{"meeting_record_id":"record","meetingid":"MEETINGID","start_time":16666666,"end_time":1788888888}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| summaries | object | 统计结果列表，详见Summary |
| date | string | 统计时间，格式：yyyy-MM-dd。 |
| view_count | int32 | 观看次数（当天数据），默认0。 |
| download_count | int32 | 下载次数（当天数据），默认0。 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","summaries":[{"date":"2033-01-01","view_count":12,"download_count":12}]}
```
