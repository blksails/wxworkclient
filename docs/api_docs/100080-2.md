## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 自建应用 | 自建应用 | 否 | 无 |
| 代开发应用 | 代开发应用 | 否 | 无 |
| 第三方应用 | 第三方应用 | 否 | 无 |
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 此事件该值固定为sys，表示该消息由系统生成 |
| CreateTime |  | 否 | 消息创建时间 （整型） |
| MsgType |  | 否 | 消息的类型，此时固定为event |
| Event |  | 否 | 事件的类型，此时固定为security |
| ChangeType |  | 否 | 此时固定为change_domain_ip |

### 请求示例


## 其他说明

### 企业微信域名IP变更事件

企业微信的域名或IP发生变更时，回调此事件给企业。
权限要求：
应用类型权限要求接收事件开关自建应用配置到「我的企业 - 设置 - 域名IP - 可调用API的应用」中无代开发应用暂不支持无第三方应用暂不支持无
事件示例：
<xml><ToUserName><![CDATA[toUser]]></ToUserName><FromUserName><![CDATA[sys]]></FromUserName><CreateTime>1403610513</CreateTime><MsgType><![CDATA[event]]></MsgType><Event><![CDATA[security]]></Event><ChangeType><![CDATA[change_domain_ip]]></ChangeType></xml>
参数说明：
参数说明ToUserName企业微信CorpIDFromUserName此事件该值固定为sys，表示该消息由系统生成CreateTime消息创建时间 （整型）MsgType消息的类型，此时固定为eventEvent事件的类型，此时固定为securityChangeType此时固定为change_domain_ip
