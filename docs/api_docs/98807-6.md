# 审批会议报名信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98807](https://developer.work.weixin.qq.com/document/path/98807)
- **文档 ID**: `98807`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/enroll/approve?access_token=ACCESS_TOKEN`

## 接口描述

批量审批会议的报名信息。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议ID |
| action | int32 | 是 | 审批动作：1：取消批准2：拒绝3：批准说明：取消批准后状态将变成待审批。 |
| enroll_id_list | string[] | 是 | 报名ID列表 |

### 请求示例

```json
{"meetingid":"meetingid11234","action":1,"enroll_id_list":["mexxxxxxx"]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| handled_count | int32 | 成功处理的数量。 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","handled_count":10}
```
