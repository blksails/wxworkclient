# 开启调试模式

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100082](https://developer.work.weixin.qq.com/document/path/100082)
- **文档 ID**: `100082`
- **API 名称**: `应用开启调试模式`
- **请求方法**: `GET`
- **接口地址**: `https://wework.qpic.cn/wwpic3az/403081_7Npz6oeoQu-dIZm_1721124095/0`
- **分组信息**: 第 5 个接口，共 6 个

## 接口描述

调用文档中的API接口应用开启调试模式以及SDK接口SDK开启调试模式打开调试模式。调用SDK接口时需要使用「1.1 获取调试凭证」中获得的调试凭证。这两个接口均调用成功后，调试模式才会生效。
