## 其他说明

### ww.getConnectedWifi([params])

兼容性：企业微信 iOS、Android 2.4.16 开始支持
功能描述获取已连接中的 Wi-Fi 信息。
ww.getConnectedWifi()
参数说明params: Object属性类型必填说明successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码wifiObject是Wi-Fi 信息属性类型必填说明SSIDstring是Wi-Fi 的 SSIDBSSIDstring是Wi-Fi 的 BSSIDsecureboolean是Wi-Fi 是否安全signalStrengthnumber是Wi-Fi 信号强度

### 功能描述

获取已连接中的 Wi-Fi 信息。

### params: Object

属性类型必填说明successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码wifiObject是Wi-Fi 信息属性类型必填说明SSIDstring是Wi-Fi 的 SSIDBSSIDstring是Wi-Fi 的 BSSIDsecureboolean是Wi-Fi 是否安全signalStrengthnumber是Wi-Fi 信号强度

### 示例代码

wx.getConnectedWifi({success:function(res){console.log(res.wifi)}})
