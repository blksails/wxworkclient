## 其他说明

### ww.uploadImage(params)

功能描述上传图片。
ww.uploadImage({localId:localId,isShowProgressTips:true})
参数说明params: Object属性类型默认值必填说明localIdstring是需要上传的图片的本地 ID可通过 chooseImage 接口获得isShowProgressTipsbooleantrue否是否显示进度提示successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码serverIdstring是返回图片的服务器端 ID
使用说明上传的图片有效期 3 天，可用素材管理接口下载图片到自己的服务器，此处获得的 serverId 即 media_id。

### 功能描述

上传图片。

### params: Object

属性类型默认值必填说明localIdstring是需要上传的图片的本地 ID可通过 chooseImage 接口获得isShowProgressTipsbooleantrue否是否显示进度提示successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码serverIdstring是返回图片的服务器端 ID

### 使用说明

上传的图片有效期 3 天，可用素材管理接口下载图片到自己的服务器，此处获得的 serverId 即 media_id。

### 示例代码
