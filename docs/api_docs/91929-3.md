# 下单成功通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91929](https://developer.work.weixin.qq.com/document/path/91929)
- **文档 ID**: `91929`
- **请求方法**: `POST`

## 接口描述

当企业在应用市场购买付费应用完成下单后，或者服务商在管理端为企业代下单后，企业微信后台会推送下单成功通知到服务商的指令回调URL。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId |  | 否 | 第三方应用的SuiteId |
| PaidCorpId |  | 否 | 购买方corpid |
| InfoType |  | 否 | 固定为 open_order |
| TimeStamp |  | 否 | 时间戳 |
| OrderId |  | 否 | 订单号，付费订单的唯一标志。服务商可据此拉取购买信息。该OrderId由企业微信生成，不超过32个字符 |
| OperatorId |  | 否 | 下单操作者userid，当服务商或代理商代下单时，该字段为空 |
