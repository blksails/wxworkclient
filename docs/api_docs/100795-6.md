# 获取企业授权信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100795](https://developer.work.weixin.qq.com/document/path/100795)
- **文档 ID**: `100795`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/v2/get_auth_info?suite_access_token=SUITE_ACCESS_TOKEN`

## 接口描述

通过永久授权码换取企业微信的授权信息。永久code的获取，是通过临时授权码使用get_permanent_code接口获取到的permanent_code。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| auth_corpid | string | 是 | 授权方corpid |
| permanent_code | string | 是 | 永久授权码，通过get_permanent_code获取 |

### 请求示例

```json
{
	"auth_corpid": "auth_corpid_value",
	"permanent_code": "code_value"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| dealer_corp_info.corpid | string | 代理服务商企业微信id |
| dealer_corp_info.corp_name | string | 代理服务商企业微信名称 |
| auth_corp_info.corpid | string | 授权方企业微信id |
| auth_corp_info.corp_name | string | 授权方企业名称 |
| auth_corp_info.corp_type | string | 授权方企业类型，认证号：verified, 注册号：unverified |
| auth_corp_info.corp_square_logo_url | string | 授权方企业方形头像 |
| auth_corp_info.corp_user_max | uint32 | 授权方企业用户规模 |
| auth_corp_info.corp_full_name | string | 授权方企业的主体名称(仅认证或验证过的企业有)，即企业全称。企业微信将逐步回收该字段，后续实际返回内容为企业名称，即auth_corp_info.corp_name。 |
| auth_corp_info.subject_type | uint32 | 企业类型，1. 企业; 2. 政府以及事业单位; 3. 其他组织, 4.团队号 |
| auth_corp_info.verified_end_time | uint32 | 认证到期时间 |
| auth_corp_info.corp_scale | string | 企业规模。当企业未设置该属性时，值为空。成员授权下，即auth_info.agent.auth_mode为1时值为空 |
| auth_corp_info.corp_industry | string | 企业所属行业。当企业未设置该属性时，值为空。成员授权下，即auth_info.agent.auth_mode为1时值为空 |
| auth_corp_info.corp_sub_industry | string | 企业所属子行业。当企业未设置该属性时，值为空。成员授权下，即auth_info.agent.auth_mode为1时值为空 |
| auth_corp_info.corp_ex_name | object | 企业其他认证的名称，仅认证企业才有 |
| auth_corp_info.corp_ex_name.name_list | string | 企业其他认证的企业简称列表（不包括corp_name），仅认证企业才有 |
| auth_info.agent | object[] | 授权的应用信息 |
| auth_info.agent[].agentid | int32 | 授权方应用id |
| auth_info.agent[].name | string | 授权方应用名字 |
| auth_info.agent[].square_logo_url | string | 授权方应用方形头像 |
| auth_info.agent[].round_logo_url | string | 授权方应用圆形头像 |
| auth_info.agent[].appid | int32 | 旧的多应用套件中的对应应用id，新开发者请忽略 |
| auth_info.agent[].auth_mode | int32 | 授权模式，0为管理员授权；1为成员授权 |
| auth_info.agent[].is_customized_app | bool | 是否为代开发自建应用 |
| auth_info.agent[].privilege | object | 应用对应的权限 |
| auth_info.agent[].privilege.level | int32 | 权限等级。1:通讯录基本信息只读 2:通讯录全部信息只读（已废弃） 3:通讯录全部信息读写 4:单个基本信息只读 5:通讯录全部信息只写（已废弃） 0:为代开发应用，该值无意义，固定为0，代开发应用权限可使用获取应用权限接口获取 |
| auth_info.agent[].privilege.allow_party | int32[] | 应用可见范围（部门） |
| auth_info.agent[].privilege.allow_tag | int32[] | 应用可见范围（标签） |
| auth_info.agent[].privilege.allow_user | string[] | 应用可见范围（成员） |
| auth_info.agent[].privilege.extra_party | int32[] | 额外通讯录（部门） |
| auth_info.agent[].privilege.extra_user | string[] | 额外通讯录（成员） |
| auth_info.agent[].privilege.extra_tag | int32[] | 额外通讯录（标签） |
| auth_info.agent[].shared_from.corpid | string | 共享了应用的企业corpid，仅当企业互联或上下游共享应用触发的安装时才返回 |
| auth_info.agent[].shared_from.share_type | int32 | 共享了途径，0表示企业互联，1表示上下游 |
