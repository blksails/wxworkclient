# 获取收款订单列表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98053](https://developer.work.weixin.qq.com/document/path/98053)
- **文档 ID**: `98053`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/paytool/get_order_list`

## 接口描述

服务商可以使用该接口查询指定时间段内创建的各种业务的收款订单列表

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 服务商调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| business_type | uint32 | 否 | 业务类型 取值范围：1 - 普通第三方应用 2 - 代开发应用 3 - 行业解决方案 |
| start_time | uint32 | 否 | 起始时间 |
| end_time | uint32 | 否 | 结束时间 |
| cursor | string | 否 | 用于分页查询的游标，字符串类型，由上一次调用返回，首次调用不填 |
| limit | uint32 | 是 | 分页，预期请求的数据量，取值范围 1 ~ 2000 |
| nonce_str | string | 是 | 随机字符串，长度要求在32字节以内，用于保证签名不可预测及防重放攻击。 需保证15分钟内不能重复，推荐随机字符串生成算法 |
| ts | uint64 | 是 | unix时间戳（中国时区）,精确到秒。 注意业务系统的机器时间与腾讯的时间相差不能超过15分钟 |
| sig | string | 是 | 数字签名。见签名算法。签名所需密钥获取路径： 工作台->企业微信服务商助手->工具->收银台->收银台API调用密钥 |

### 请求示例

```json
{
    "business_type":1,
    "start_time":1670643437,
    "end_time":1671507437,
    "cursor":"tJzlB9tdqfh-g7i_J-ehOz_TWcd7dSKa39_AqCIeMFw",
    "limit":0,
    "nonce_str":"129031823",
    "ts":1548302135,
    "sig":"mPOwVW/vQ74xN+b+Yu1KMa9RrmhKJaJjAtXHTof+EpU="
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |
| next_cursor | string | 分页游标，下次请求时填写以获取之后分页的记录 |
| has_more | uint32 | 是否还有更多数据 |
| pay_order_list | obj[] | 订单列表 |
| pay_order_list[].order_id | string | 订单号 |
| pay_order_list[].create_time | uint64 | 订单创建时间 |
| pay_order_list[].custom_corpid | string | 客户企业的corpid |
| pay_order_list[].buy_content | string | 购买内容 |
| pay_order_list[].origin_price | uint64 | 原价金额 |
| pay_order_list[].paid_price | uint64 | 实付金额。免支付订单实付金额返回0 |
| pay_order_list[].order_status | uint32 | 订单状态。取值范围为： 1 - 待支付 2 - 已支付 3 - 订单取消 4 - 支付过期 5 - 退款申请中 6 - 已退款 7 - 交易完成 8 - 待企业确认 9 - 已部分退款 |
| pay_order_list[].order_from | uint32 | 订单来源。取值范围为： 1 - 客户下单 2 - 服务商创建 |
| pay_order_list[].creator | string | 订单创建人 |
| pay_order_list[].pay_type | uint32 | 支付方式 0-客户支付；1-服务商代支付；2-免支付 |
