# wx.qy.getSystemInfo - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95046](https://developer.work.weixin.qq.com/document/path/95046)

## 接口描述

获取企业微信系统信息

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | function | 否 | 接口调用成功的回调函数 |
| fail | function | 否 | 接口调用失败的回调函数 |
| complete | function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| version | string | 否 | 企业微信微信版本号 |

### 请求示例

```json
wx.qy.getSystemInfo({success(res){console.log(res.version)}})
```

## 其他说明

### wx.qy.getSystemInfo(Object object)

小程序可以企业微信中调用此接口，获得企业微信终端的版本号
获取企业微信系统信息
