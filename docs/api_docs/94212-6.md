# 获取成员假期余额

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94212](https://developer.work.weixin.qq.com/document/path/94212)
- **文档 ID**: `94212`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/oa/vacation/getuservacationquota`

## 接口描述

通过本接口可获取应用可见范围内各个员工的假期余额数据。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。必须使用第三方应用accesstoken获取，获取方式参考：文档-获取access_token |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| userid | string | 是 | 需要获取假期余额的成员的userid |

### 请求示例

```json
{
    "userid": "ZhangSan"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码，详情见错误码说明 |
| errmsg | string | 错误码对应的错误信息提示 |
| lists | object[] | 假期列表 |
| lists.id | int32 | 假期id |
| lists.assignduration | uint32 | 发放时长，单位为秒 |
| lists.usedduration | uint32 | 使用时长，单位为秒 |
| lists.leftduration | uint32 | 剩余时长，单位为秒 |
| lists.vacationname | string | 假期名称 |
| lists.real_assignduration | uint32 | 假期的实际发放时长，通常在设置了按照实际工作时间发放假期后进行计算 |

## 其他说明

### 权限说明

自建应用：配置到「审批 - 可调用接口的应用」中
代开发应用：具有「审批」权限
第三方应用：具有「审批」权限
1.第三方应用可以获取第三方应用添加的模板详情。
2.自建应用的Secret可获取企业自建模板的模板详情。
3.接口调用频率限制为600次/分钟。
注：从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 查看详情

### 错误说明

301063 - 参数错误
