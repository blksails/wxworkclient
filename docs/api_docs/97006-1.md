# wx.qy.updateMomentsSetting - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97006](https://developer.work.weixin.qq.com/document/path/97006)
- **文档 ID**: `97006`

## 接口描述

具有客户联系权限的企业成员，可通过该接口设置个人的朋友圈封面和签名。当前暂仅支持在企业微信内调用。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| signature | string | 否 | 个性签名， 可不填（imgUrl跟signature必须二填1，否则报错"missing arguments"）。若不填该字段，表示不更新个性签名。若填了该字段且为空字符串，表示清空个性签名；最多支持输入30个中文或字母，若超过报错"invalid signature size" |
| imgUrl | string | 否 | 朋友圈封面，可不填（imgUrl跟signature必须二填1，否则报错"missing arguments"）。封面图片建议为正方形，尺寸建议为1125*1125像素（图片若非正方形，会唤起用户调整封面图片界面）；图片大小建议不超过10M；若填了该字段，该字段不能为空，若为空或者图片地址无效，报错"invalid imgUrl" |
