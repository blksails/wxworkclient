- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90216](https://developer.work.weixin.qq.com/document/path/90216)

## 接口描述

请求方式：GET（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/tag/list?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|

| taglist |  | 标签列表 |
| tagid |  | 标签id |
