# 获取下级企业加入的上下游 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97442](https://developer.work.weixin.qq.com/document/path/97442)
- **文档 ID**: `97442`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/corpgroup/get_corp_shared_chain_list?access_token=ACCESS_TOKEN`

## 接口描述

上级企业自建应用/代开发应用通过本接口查询下级企业所在上下游

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证。上游企业应用access_token |
| corpid |  | 否 | 已加入企业id |

### 请求示例

```json
{"corpid":"xxxxx"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| chains |  | 上下游列表 |
| chains.chain_id |  | 上下游id |
| chains.chain_name |  | 上下游名称 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","chains":[{"chain_id":"xxx","chain_name":"xxx"}]}
```
