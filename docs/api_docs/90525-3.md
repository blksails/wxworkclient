## 其他说明

### ww.scanQRCode([params])

功能描述调起企业微信扫一扫。
ww.scanQRCode({needResult:true,scanType:['qrCode']})
参数说明params: Object属性类型默认值必填说明needResultbooleanfalse否扫描结果是否由企业微信处理为 false 时直接返回扫描结果scanTypestring[]否扫码类型合法值说明qrCode扫描二维码barCode扫描条形码successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码resultStrstring否扫码结果，needResult为true时返回

### 功能描述

调起企业微信扫一扫。

### params: Object

属性类型默认值必填说明needResultbooleanfalse否扫描结果是否由企业微信处理为 false 时直接返回扫描结果scanTypestring[]否扫码类型合法值说明qrCode扫描二维码barCode扫描条形码successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码resultStrstring否扫码结果，needResult为true时返回
