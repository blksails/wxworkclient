# 小程序发送通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92415](https://developer.work.weixin.qq.com/document/path/92415)
- **文档 ID**: `92415`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/notify/send?access_token=ACCESS_TOKEN`

## 接口描述

小程序关联的应用可以发送小程序通知消息。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| touser | string | 是 | 接收消息的用户，成员ID列表，最多支持1000个。 |
| agentid | uint32 | 是 | 企业应用的id，整型。可在应用的设置页面查看。 |
| msgtype | string | 是 | 消息类型，此时固定为：miniprogram_notice。 |
| miniprogram_notice | object | 是 | 小程序通知消息内容。 |
| enable_id_trans | bool | 否 | 表示是否开启id转译，0表示否，1表示是，默认0。 |
| enable_duplicate_check | bool | 否 | 表示是否开启重复消息检查，0表示否，1表示是，默认0。 |
| duplicate_check_interval | uint32 | 否 | 表示是否重复消息检查的时间间隔，默认1800s，最大不超过4小时。 |

### 请求示例

```json
{
  "touser": "UserID1|UserID2",
  "agentid": 1000005,
  "msgtype": "miniprogram_notice",
  "miniprogram_notice": {
    "appid": "wx123",
    "page": "index",
    "title": "会议室预订成功通知",
    "description": "4月27日 16:16-17:00",
    "emphasis_first_item": true
  },
  "enable_id_trans": 0,
  "enable_duplicate_check": 0,
  "duplicate_check_interval": 1800
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码。 |
| errmsg | string | 错误信息。 |

### 响应示例

```json
{
  "errcode": 0,
  "errmsg": "ok"
}
```
