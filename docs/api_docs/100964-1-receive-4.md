# 授权成功通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100964](https://developer.work.weixin.qq.com/document/path/100964)
- **文档 ID**: `100964`
- **API 名称**: `receive`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`
- **分组信息**: 第 1 个接口，共 3 个

## 接口描述

企业微信后台会推送授权成功通知。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 否 | 消息签名 |
| timestamp | string | 否 | 时间戳 |
| nonce | string | 否 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 第三方应用的SuiteId或者代开发应用模板id |
| AuthCode | string | 是 | 临时授权码，最长为512字节。用于获取企业永久授权码。10分钟内有效 |
| InfoType | string | 是 | create_auth |
| TimeStamp | string | 是 | 时间戳 |
| State | string | 是 | 构造授权链接指定的state参数 |
| ExtraInfo | string | 否 | 额外信息，部分场景会携带，在其他文档中分别描述 |

### 请求示例

```xml
<xml>
	<SuiteId><![CDATA[ww4asffe9xxx4c0f4c]]></SuiteId>
	<AuthCode><![CDATA[AUTHCODE]]></AuthCode>
	<InfoType><![CDATA[create_auth]]></InfoType>
	<TimeStamp>1403610513</TimeStamp>
	<State><![CDATA[123]]></State>
	<ExtraInfo></ExtraInfo>
</xml>
```
