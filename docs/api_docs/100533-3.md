## 其他说明

### ww.stopVoice(params)

功能描述停止播放。
ww.stopVoice({localId:localId})
参数说明params: Object属性类型必填说明localIdstring是需要停止的音频的本地 ID通过 stopRecord 接口获得successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

停止播放。

### params: Object

属性类型必填说明localIdstring是需要停止的音频的本地 ID通过 stopRecord 接口获得successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

wx.stopVoice({localId:''// 需要停止的音频的本地ID，由stopRecord接口获得});
