## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| contact_profile |  | 否 | 从联系人详情进入 |
| single_chat_tools |  | 否 | 从单聊会话的工具栏进入 |
| group_chat_tools |  | 否 | 从群聊会话的工具栏进入 |
| chat_attachment |  | 否 | 从会话的聊天附件栏进入 |
| single_kf_tools |  | 否 | 从微信客服的工具栏进入 |
| chain_single_chat_tools |  | 否 | 从上下游单聊会话的工具栏进入 |
| chain_group_chat_tools |  | 否 | 从上下游群聊会话的工具栏进入 |
| internal_group_chat_board |  | 否 | 从内部群群看板进入 |
| normal |  | 否 | 除以上场景之外进入，例如工作台，聊天会话等 |

### 请求示例

```json
wx.qy.getContext({success:function(res){varentry=res.entry,//返回进入小程序的入口类型varshareTicket=res.shareTicket;}})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| entry | String | 返回进入小程序的入口类型，目前有normal、contact_profile、single_chat_tools、group_chat_tools、chat_attachment等，详见以下“entry说明” |
| shareTicket | String | 从企业微信3.1.8及以上版本支持，仅从私密消息卡片进入时返回该字段 |

## 其他说明

### wx.qy.getContext(Object object)

调用前提：1. 此接口仅在企业微信3.0.24及以后版本支持，微信端不支持（微信开发者工具也不支持）2. 必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态3. 当前成员必须在应用的可见范围4. 从企业微信3.1.6版本开始，所有应用皆可调用，包括自建应用与第三方应用
调用该接口可以获得用户是从哪个入口打开页面，从而决定是否可以调用工具栏相关的接口
