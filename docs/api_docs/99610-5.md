# 获取组件授权信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99610](https://developer.work.weixin.qq.com/document/path/99610)
- **文档 ID**: `99610`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/customer_acquisition/get_comp_auth_info`

## 接口描述

服务商可通过此接口获取企业的获客助手组件授权信息。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| pay_mode | int32 | 使用扣除方式。0为扣企业的，1为扣服务商的 |
| price | int32 | 当扣款方式为服务商代付(pay_mode=1)时，代付单价，单位为分 |
| link_auth_mode | int32 | 链接授权方式。0为企业选择链接进行授权，1为关联应用创建的链接 |
| auth_apps | object[] | 当1为关联应用创建链接时，返回对应的关联应用信息列表 |
| auth_apps[].agentid | int32 | 关联应用的agentid |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "pay_mode": 1,
    "price": 300,
    "link_auth_mode": 1,
    "auth_apps": [
        {
            "agentid": 1000030
        },
        {
            "agentid": 1000031
        }
    ]
}
```

## 其他说明

获客助手组件可调用
