# 删除客服账号 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96405](https://developer.work.weixin.qq.com/document/path/96405)
- **文档 ID**: `96405`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/kf/account/del?access_token=ACCESS_TOKEN`

## 接口描述

删除已有的客服账号

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| open_kfid | string | 是 | 客服账号ID。不多于64字节 |
| 自建应用 | 自建应用 | 否 | 配置到「微信客服- 可调用接口的应用」中 |
| 第三方应用 | 第三方应用 | 否 | 具有“微信客服->管理账号、分配会话和收发消息”权限 |
| 代开发自建应用 | 代开发自建应用 | 否 | 具有“微信客服->管理账号、分配会话和收发消息”权限 |

### 请求示例

```json
{"open_kfid":"wkAJ2GCAAAZSfhHCt7IFSvLKtMPxyJTw"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
