# 上传文档图片 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99933](https://developer.work.weixin.qq.com/document/path/99933)
- **文档 ID**: `99933`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/image_upload?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于上传图片

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 是 | 文档ID，通过新建文档接口创建后获得 |
| base64_content | string | 是 | base64之后的图片内容 |
| size | string | 否 | 图片的大小 |

### 请求示例

```json
{"docid":"docid","base64_content":"Content"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| url | string | 图片的url |
| height | string | 图片的高 |
| width | string | 图片的宽 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","url":"URL","height":234,"width":234,"size":81259}
```
