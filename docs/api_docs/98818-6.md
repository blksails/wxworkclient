# 创建用户专属参会链接 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98818](https://developer.work.weixin.qq.com/document/path/98818)
- **文档 ID**: `98818`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/create_customer_short_url?access_token=ACCESS_TOKEN`

## 接口描述

使用该接口，可为会议生成多个专属入会链接，不同链接以customer_data进行区分。通过用户入会、用户进入等候室等事件，或通过获取等候室成员列表的 API 可查询到该参数。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议ID |
| customer_data | string | 是 | 用户专属字段，长度不超过256 字节。 |

### 请求示例

```json
{"meetingid":"meetingid11234","customer_data":"CUSTOMERDATA"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| meeting_short_url_customer_data | object[] | 用户专属参会链接对象。详见CustomerData。 |
| customer_data | string | 用户专属字段 |
| meeting_short_url | string | 用户专属参会链接 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","meeting_short_url_customer_data":{"customer_data":"CUSTOMERDATA","meeting_short_url":"URL"}}
```
