# 创建代开发模版

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97201](https://developer.work.weixin.qq.com/document/path/97201)
- **文档 ID**: `97201`
- **API 名称**: `create_template`
- **请求方法**: `POST`
- **接口地址**: `https://wework.qpic.cn/wwpic/762533_k5FAM8I1STG3uPt_1626354201/0`
- **分组信息**: 第 1 个接口，共 3 个

## 接口描述

创建代开发应用模版，用于服务商向企业提供自建应用代开发的载体。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 代开发应用模版logo | string | 是 | 必须为正方形，像素为750*750，1M以内的jpg、png图片，不可使用企业微信logo或进行二次修改 |
| 代开发应用模版名称 | string | 是 | 2-10个字，建议包含品牌名，不可使用特殊字符及特殊词汇 |
| 代开发应用模版介绍 | string | 是 | 4-20个字，简单清晰描述应用功能和使用场景 |
| 分类 | string | 是 | 根据应用的功能及场景进行选择 |
| 目标行业 | string | 是 | 根据应用目标用户的行业进行选择 |
| 代开发模版回调URL | string | 是 | 填写带https://头的有效URL链接，用于接收企业微信验证URL的请求 |

### 请求示例

```text
POST https://wework.qpic.cn/wwpic/762533_k5FAM8I1STG3uPt_1626354201/0

Body:
{
  "代开发应用模版logo": "logo.jpg",
  "代开发应用模版名称": "CRM",
  "代开发应用模版介绍": "用于客户关系管理",
  "分类": "工具",
  "目标行业": "IT",
  "代开发模版回调URL": "https://callback.url"
}
```

2. 配置代开发模版信息填写标准...
