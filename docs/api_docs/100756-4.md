# 获取「转发」按钮点击状态并自定义分享内容

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100756](https://developer.work.weixin.qq.com/document/path/100756)
- **文档 ID**: `100756`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

获取「转发」按钮点击状态并自定义分享内容。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| params.title | string | 否 | 分享标题 |
| params.desc | string | 否 | 分享描述 |
| params.link | string | 否 | 分享链接  在微信上分享时，该链接的域名必须与企业某个应用的可信域名一致 |
| params.imgUrl | string | 否 | 分享图标 |
| params.enableIdTrans | number | 否 | 企业微信 4.0.20 是否开启id转译, 1 为开启，0 为关闭，默认为0 仅供第三方应用使用 必须使用应用身份进行注册 |
| params.success | Function | 否 | 成功回调 |
| params.fail | Function | 否 | 失败回调 |
| params.cancel | Function | 否 | 取消回调 |
| params.complete | Function | 否 | 完成回调 |

### 请求示例

```text
ww.onMenuShareAppMessage({
    title: '企业微信',
    desc: '让每个企业都有自己的微信',
    link: 'https://work.weixin.qq.com/',
    imgUrl: 'https://res.mail.qq.com/node/ww/wwmng/style/images/index_share_logo$13c64306.png',
    success() {
        // 用户确认分享后回调
    },
    cancel() {
        // 用户取消分享后回调
    }
})
```

## 其他说明

### 使用限制

仅激活成员数超过 200 人且已经认证的企业才可在微信上调用

### 使用说明

微信客户端即将废弃该接口。
