# 获取群聊会话 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/101028](https://developer.work.weixin.qq.com/document/path/101028)
- **文档 ID**: `101028`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartsheet/groupchat/get?access_token=ACCESS_TOKEN`

## 接口描述

该接口可以查询通过智能表格自动化创建的群聊信息。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| docid |  | 是 | 智能表格ID。需为当前应用创建的智能表格 |
| chat_id |  | 是 | 群聊ID，需为智能表自动规则创建群聊ID |

### 请求示例

```json
{"docid":"DOCID","chat_id":"CHATID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| name |  | 群聊名称 |
| owner |  | 群主 |
| user_list |  | 群成员列表，仅返回当前企业成员列表 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","name":"NAME","owner":"userid2","user_list":["userid1","userid2","userid3"]}
```
