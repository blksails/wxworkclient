## 接口描述

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| result-id | string | 是 | 异步任务结果 ID |
| secret-key | string | 是 | 异步任务结果解密 key |
