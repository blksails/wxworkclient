# 新建文件夹/文档 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97882](https://developer.work.weixin.qq.com/document/path/97882)
- **文档 ID**: `97882`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_create?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于在微盘指定位置新建文件夹、文档（更多文档接口能力可见文档API接口说明）。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spaceid | string | 是 | 空间spaceid |
| fatherid | string | 是 | 父目录fileid, 在根目录时为空间spaceid |
| file_type | uint32 | 是 | 文件类型, 1:文件夹 3:文档(文档) 4:文档(表格) |
| file_name | string | 是 | 文件名字（注意：文件名最多填255个字符, 英文算1个, 汉字算2个） |

### 请求示例

```json
{"spaceid":"SPACEID","fatherid":"FATHERID","file_type":FILE_TYPE,"file_name":"FILE_NAME"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| fileid | string | 新建文件的fileid |
| url | string | 文档的访问链接，仅在新建文档时返回 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","fileid":"FILEID","url":"URL"}
```
