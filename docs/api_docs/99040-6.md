# 更新会议嘉宾列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99040](https://developer.work.weixin.qq.com/document/path/99040)
- **文档 ID**: `99040`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/set_guests?access_token=ACCESS_TOKEN`

## 接口描述

通过会议ID更新会议嘉宾列表。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议ID。 |
| guests | object[] | 是 | 会议嘉宾列表，详见Guest。 |
| area | string | 是 | 国家/地区代码（例如：中国传86，不是+86，也不是0086）。 |
| phone_number | string | 是 | 手机号。 |
| guest_name | string | 否 | 嘉宾名称。最大长度限制16个字符 |

### 请求示例

```json
{"meetingid":"ID","guests":[{"area":"86","phone_number":"xxxxxxxxx","guest_name":"xxxx"}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
