# 修改自动升年级的配置 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92949](https://developer.work.weixin.qq.com/document/path/92949)
- **文档 ID**: `92949`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/set_upgrade_info?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/school/set_upgrade_info?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| upgrade_time |  | 否 | 自动升年级的时间，该时间戳只有月和日有效，不传则默认为传0，代表的1月1号 |
| upgrade_switch |  | 否 | 开启或关闭自动升年级。0：表示关闭，1：表示开启，不传默认关闭，传所有非1的值也视为关闭 |
| 自建应用 | 自建应用 | 否 | 配置到「家校沟通- 读取和编辑家校通讯录的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 需拥有「家校沟通」可使用家校通讯录-家校通讯录编辑权限 |
| 第三方应用 | 第三方应用 | 否 | 需拥有「家校沟通」使用和编辑权限 |

### 请求示例

```json
{"upgrade_time":1594090969,"upgrade_switch":2}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| next_upgrade_time |  | 下次升级的时间，只有年月日有效，如果该学校今年已经升过年级，那本参数会给明年对应日期的时间戳 |
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"next_upgrade_time":1625587200,"errcode":0,"errmsg":"ok"}
```
