## 接口描述

唤起微盘选择器，选择微盘中的文件

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
|  | number | 是 | 选择文件的数量1 表示单选，大于 1 表示多选，上限为 50 |
|  | Function | 否 | 成功回调 |
|  | Function | 否 | 失败回调 |
|  | Function | 否 | 取消回调 |
|  | Function | 否 | 完成回调 |

### 请求示例

#### 示例 1: 功能描述

```json
ww.wedriveSelectFile({selectedFileNum:1,})
```

#### 示例 2: 示例代码

```json

```

### 功能描述

唤起微盘选择器，选择微盘中的文件
在微盘中选择一个或多个具有可分享权限的微盘文件或在线文档，返回选中文件的 url。

### 使用限制

本接口必须使用应用身份进行注册所使用的应用必须具有微盘和文档使用权限当前成员必须在应用的可见范围之内若用户对某文件不具备「分享」权限（微盘自定义权限取消勾选“分享”权限），则无法选择该文件。在 Mac 端使用时，macOS 版本需 > 10.12

### params: Object

属性类型必填说明selectedFileNumnumber是选择文件的数量1 表示单选，大于 1 表示多选，上限为 50successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码resultObject是属性类型必填说明最低版本selectedFileUrlsstring[]是选择的文件的 URL 列表selectedFileInfosObject[]是选择的文件信息列表企业微信 4.1.8属性类型必填说明urlstring是选择的文件urltypenumber是选择的文件类型。0: 其他，2: 文件，3: 文档，4: 表格，5: 收集表，6: 幻灯片，7: 思维导图，8: 流程图，10: 智能表格
