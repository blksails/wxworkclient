## 接口描述

数据与智能专区可存储企业的数据，当前支持企业会话内容数据和知识集数据，后续将根据企业的需要支持更多的数据。企业可将企业在专区的数据授权给服务商的应用，用以满足企业的数据分析需要。

## 其他说明

### 企业会话内容

包括企业成员内部、外部单聊和群聊内容以及语音通话内容，使用前企业需要先在「企业管理后台 - 安全与管理 - 管理工具 - 数据与智能专区 - 企业会话内容」启用。

### 企业知识集

为企业的知识集内容，企业可在「企业管理后台 - 安全与管理 - 管理工具 - 数据与智能专区 - 知识集」进行配置.
