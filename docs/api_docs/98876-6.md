# 获取网络研讨会报名信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98876](https://developer.work.weixin.qq.com/document/path/98876)
- **文档 ID**: `98876`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/webinar/enroll/list?access_token=ACCESS_TOKEN`

## 接口描述

获取已报名观众数量和报名观众答题详情。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 网络研讨会 ID |
| cursor | string | 否 | 分页查询用，将上一个请求返回的next_cursor字段传入。第一次查询时可不传值 |
| limit | uint32 | 否 | 分页大小，最大50条。默认值为50 |

### 请求示例

```json
{"meetingid":"MEETINGID1","status":1,"cursor":"CURSOR","limit":20}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| status | uint32 | 审批状态筛选字段，默认返回全部。审批状态：0：全部。1：待审批。2：已拒绝。3：已批准。 |
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| has_more | bool | 是否还有待拉取的成员列表 |
| next_cursor | string | 分页用，下一次拉取列表将该字段填入cursor字段 |
| enroll_list | object[] | 当前页的报名列表。详见Enroll。 |
| enroll_id | string | 报名 ID |
| enroll_time | string | 报名时间（utc +8，非时间戳） |
| enroll_source_type | uint32 | 报名来源：1：成员手动报名2：批量导入报名 |
| nick_name | string | 昵称。若通过手机号导入报名且未设置昵称，则该字段显示手机号 |
| status | uint32 | 报名状态：1：待审批2：已拒绝3：已批准 |
| userid | string | 为企业内成员时返回userid |
| tmp_openid | string | 当场会议的成员临时 ID |
| enroll_code | string | 报名码，用于电话入会的凭证 |
| answer_list | object[] | 答题对象列表，详见Answer。 |
| answer_content | string[] | 回答内容：单选/简答只有一个元素，多选会有多个 |
| is_required | uint32 | 是否必填：1：否2：是 |
| question_num | uint32 | 问题编号，1,2,3...等形式 |
| question_title | string | 问题标题 |
| question_type | uint32 | 问题类型：1：单选2：多选3：简答 |
| special_type | uint32 | 特殊问题类型：1：无2：手机号3：邮箱4：姓名5：公司名称说明：目前特殊问题均为简答题 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","has_more":false,"next_cursor":"CURSOR","enroll_list":[{"enroll_id":1386442,"enroll_time":"2023-03-23 10:00:00","enroll_source_type":2,"nick_name":"ccc","status":3,"userid":"USERID","tmp_openid":"MSOPENID","enroll_code":"1111111","answer_list":[{"answer_content":["A"],"is_required":1,"question_num":1,"question_title":"testqus","question_type":1,"special_type":1}]}]}
```
