# 获取硬件授权结果

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96097](https://developer.work.weixin.qq.com/document/path/96097)
- **文档 ID**: `96097`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/devicedata/get_auth_info`

## 接口描述

此接口用于第三方应用获取企业授权的硬件设备列表，后续第三方应用可获取这批设备产生的数据，例如考勤记录、门禁通行记录、测温记录等。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。见获取access_token |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| agentid | uint32 | 否 | 应用id，仅套件应用需要填写该字段 |

### 请求示例

```json
{
	"agentid":1
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码描述 |
| device_list | object | 授权读取数据的设备列表 |
| device_list.item.SN | string | 设备SN |
| device_list.item.remark_name | string | 企业设置的备注名称 |
| device_list.item.default_name | string | 设备出厂时的名称 |
| device_list.item.model_name | string | 设备出厂时的型号 |
| device_list.item.device_type | int32 | 设备类型：1-考勤/门禁设备 |
| device_list.item.device_ability | int32 | 设备能力： 1-具备考勤能力； 2-具备门禁能力； 4-具备测温能力； 设备能力字段可叠加，例如：返回5表示设备具备考勤和测温能力 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"device_list": {
		"item":[
			{
				"device_sn":"SN",
				"remark_name":"设备备注名",
				"default_name":"设备出厂名",
				"model_name":"PRODUCT_MODEL",
				"device_type":1,
				"device_ability":1
			}
		]
	}
}
```
