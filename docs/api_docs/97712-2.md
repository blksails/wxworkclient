## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 自建应用 | 自建应用 | 否 | 微信客服消息和事件 |
| 代开发应用 | 代开发应用 | 否 | 无 |
| 第三方应用 | 第三方应用 | 否 | 无 |
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 此事件该值固定为sys，表示该消息由系统生成 |
| CreateTime |  | 否 | 消息创建时间，unix时间戳 |
| MsgType |  | 否 | 消息类型，此时固定为：event |
| Event |  | 否 | 事件类型，此时固定为：kf_account_auth_change |
| AuthAddOpenKfId |  | 否 | 新增授权的客服账号列表，多个AuthAddOpenKfId节点表示多个新增账号 |
| AuthDelOpenKfId |  | 否 | 取消授权的客服账号列表，多个AuthDelOpenKfId节点表示多个取消账号 |

### 请求示例

```xml
<xml><ToUserName><![CDATA[toUser]]></ToUserName><FromUserName><![CDATA[sys]]></FromUserName><CreateTime>1348831860</CreateTime><MsgType><![CDATA[event]]></MsgType><Event><![CDATA[kf_account_auth_change]]></Event><AuthAddOpenKfId><![CDATA[wkxxxx1]]></AuthAddOpenKfId><AuthDelOpenKfId><![CDATA[wkxxxx2]]></AuthDelOpenKfId></xml>
```

## 其他说明

### 客服账号授权变更事件

当应用授权的客服账号发生变化（新增/取消客服账号的授权），触发该回调。
示例
<xml><ToUserName><![CDATA[toUser]]></ToUserName><FromUserName><![CDATA[sys]]></FromUserName><CreateTime>1348831860</CreateTime><MsgType><![CDATA[event]]></MsgType><Event><![CDATA[kf_account_auth_change]]></Event><AuthAddOpenKfId><![CDATA[wkxxxx1]]></AuthAddOpenKfId><AuthDelOpenKfId><![CDATA[wkxxxx2]]></AuthDelOpenKfId></xml>
说明
参数说明ToUserName企业微信CorpIDFromUserName此事件该值固定为sys，表示该消息由系统生成CreateTime消息创建时间，unix时间戳MsgType消息类型，此时固定为：eventEvent事件类型，此时固定为：kf_account_auth_changeAuthAddOpenKfId新增授权的客服账号列表，多个AuthAddOpenKfId节点表示多个新增账号AuthDelOpenKfId取消授权的客服账号列表，多个AuthDelOpenKfId节点表示多个取消账号
