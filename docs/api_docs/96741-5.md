# 获取部门家长详情 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96741](https://developer.work.weixin.qq.com/document/path/96741)
- **文档 ID**: `96741`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/user/list_parent?access_token=ACCESS_TOKEN&department_id=DEPARTMENT_ID`

## 接口描述

请求方式：GET（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/school/user/list_parent?access_token=ACCESS_TOKEN&department_id=DEPARTMENT_ID

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| department_id |  | 是 | 获取的部门id |
| 自建应用 | 自建应用 | 否 | 配置到「家校沟通- 读取和编辑家校通讯录的应用」或「家校沟通- 读取家校通讯录的应用」或“家长可使用应用”列表中 |
| 代开发应用 | 代开发应用 | 否 | 具有「家校沟通」权限 |
| 第三方应用 | 第三方应用 | 否 | 具有「家校沟通」使用权限 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| parents.parent_userid |  | 家长的userid |
| parents.mobile |  | 家长手机号，第三方不可获取;代开发应用需要管理员授权该权限才返回 |
| parents.is_subscribe |  | 家长是否关注了“学校通知”，0-未关注，1-已关注 |
| parents.children |  | 家长孩子列表 |
| parents.children.student_userid |  | 学生的userid |
| parents.children.relation |  | 家长与学生关系 |
| parents.children.name |  | 学生姓名 |
| parents.external_userid |  | 家长的external_userid,仅当家长已关注才返回。对同一个服务商来说，同一个家长微信在不同学校下返回的家长的external_userid是一样的 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","parents":[{"parent_userid":"zhangsan_parent","mobile":"18900000000","is_subscribe":1,"external_userid":"xxx","children":[{"student_userid":"zhangsan","relation":"爸爸","name":"张三"}]},{"parent_userid":"lisi_parent","mobile":"18900000001","is_subscribe":0,"children":[{"student_userid":"lisi","relation":"妈妈","name":"李四"}]}]}
```
