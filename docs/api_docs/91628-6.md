# 获取接听状态 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91628](https://developer.work.weixin.qq.com/document/path/91628)
- **文档 ID**: `91628`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/pstncc/getstates?access_token=ACCESS_TOKEN`

## 接口描述

通过此接口，了解员工是否已接听语音电话。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证。 |
| callee_userid |  | 是 | 用户id |
| callid |  | 是 | 发起自动语音来电callid |
| 自建应用 | 自建应用 | 否 | 配置到「紧急通知 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"callee_userid":"james","callid":"6-20190510201844181887818-4d0251082406000-out"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| istalked |  | 0.表示未接听，1.表示接听 |
| calltime |  | 呼叫发起时间戳 |
| talktime |  | 通话时长单位（s） |
| reason |  | 呼叫结果状态：0正常结束 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","istalked":1,"calltime":1557306531,"talktime":2,"reason":0}
```
