# 获取成员多次收消息详情 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100133](https://developer.work.weixin.qq.com/document/path/100133)
- **文档 ID**: `100133`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/customer_acquisition/get_chat_info?access_token=ACCESS_TOKEN`

## 接口描述

目录

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 自建应用 | 自建应用 | 否 | 配置到「客户联系 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 具有「查询使用量、创建和编辑获客链接、获取客户列表」权限 |
| 第三方应用 | 第三方应用 | 否 | 具有「获客助手」权限 |
| access_token |  | 是 | 调用接口凭证 |
| chat_key |  | 是 | 成员多次收消息事件中回调的会话信息凭据ChatKey，回调后30分钟内有效 |

### 请求示例

```json
{"chat_key":"CHAT_KEY"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| userid |  | 成员的userid |
| external_userid |  | 客户id |
| chat_info.recv_msg_cnt |  | 成员收到的此客户的消息次数 |
| chat_info.link_id |  | 成员添加客户的获客链接id |
| chat_info.state |  | 成员添加客户的state |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","userid":"zhangsan","external_userid":"woAJ2GCAAAXtWyujaWJHDDGi0mACAAA","chat_info":{"recv_msg_cnt":3,"link_id":"caXXXXXXX","state":"STATE"}}
```

## 其他说明

### 获取成员多次收消息详情

企业和服务商可通过此接口获取成员多次收消息情况，如次数、客户id等信息。
权限说明:调用接口的应用需要满足如下的权限：
应用类型权限要求自建应用配置到「客户联系 可调用接口的应用」中代开发应用具有「查询使用量、创建和编辑获客链接、获取客户列表」权限第三方应用具有「获客助手」权限
请求方式:POST(HTTP)
请求地址:https://qyapi.weixin.qq.com/cgi-bin/externalcontact/customer_acquisition/get_chat_info?access_token=ACCESS_TOKEN
请求示例
{"chat_key":"CHAT_KEY"}
参数说明:
参数必须说明access_token是调用接口凭证chat_key是成员多次收消息事件中回调的会话信息凭据ChatKey，回调后30分钟内有效
返回结果:
{"errcode":0,"errmsg":"ok","userid":"zhangsan","external_userid":"woAJ2GCAAAXtWyujaWJHDDGi0mACAAA","chat_info":{"recv_msg_cnt":3,"link_id":"caXXXXXXX","state":"STATE"}}
参数说明:
参数说明errcode返回码errmsg对返回码的文本描述内容userid成员的useridexternal_userid客户idchat_info.recv_msg_cnt成员收到的此客户的消息次数chat_info.link_id成员添加客户的获客链接idchat_info.state成员添加客户的state
