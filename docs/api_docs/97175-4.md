# 重置永久授权码通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97175](https://developer.work.weixin.qq.com/document/path/97175)
- **文档 ID**: `97175`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`

## 接口描述

在服务商管理端的代开发应用详情页，点击“重新获取secret”会触发该事件的回调，服务商收到回调事件后，可使用AuthCode通过代开发授权应用secret的获取接口获取代开发应用最新的secret（即permanent_code字段）。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 |  |
| timestamp | string | 是 |  |
| nonce | string | 是 |  |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 代开发自建应用的SuiteId |
| AuthCode | string | 是 | 临时授权码，最长为512字节。用于获取企业永久授权码。10分钟内有效 |
| InfoType | string | 是 | reset_permanent_code |
| TimeStamp | string | 是 | 时间戳 |

### 请求示例

```xml
<xml>
	<SuiteId><![CDATA[dk4asffe9xxx4c0f4c]]></SuiteId>
	<AuthCode><![CDATA[AUTHCODE]]></AuthCode>
	<InfoType><![CDATA[reset_permanent_code]]></InfoType>
	<TimeStamp>1403610513</TimeStamp>
</xml>
```
