# 获取「客户数据统计」企业汇总数据

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95492](https://developer.work.weixin.qq.com/document/path/95492)
- **文档 ID**: `95492`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/kf/get_corp_statistic`

## 接口描述

通过此接口，可以获取咨询会话数、咨询客户数等企业汇总统计数据

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| open_kfid | string | 是 | 客服账号ID |
| start_time | uint32 | 是 | 起始日期的时间戳，填这一天的0时0分0秒。取值范围：昨天至前180天 |
| end_time | uint32 | 是 | 结束日期的时间戳，填这一天的0时0分0秒。取值范围：昨天至前180天 |

### 请求示例

```json
{
    "open_kfid": "OPEN_KFID",
    "start_time": 1645545600,
    "end_time": 1645632000
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |
| statistic_list | object[] | 统计数据列表 |
| statistic_list[].stat_time | uint32 | 数据统计日期，为当日0点的时间戳 |
| statistic_list[].statistic | object | 一天的统计数据 |
| statistic_list[].statistic.session_cnt | uint64 | 咨询会话数 |
| statistic_list[].statistic.customer_cnt | uint64 | 咨询客户数 |
| statistic_list[].statistic.customer_msg_cnt | uint64 | 咨询消息总数 |
| statistic_list[].statistic.upgrade_service_customer_cnt | uint64 | 升级服务客户数 |
| statistic_list[].statistic.ai_session_reply_cnt | uint64 | 智能回复会话数 |
| statistic_list[].statistic.ai_transfer_rate | float | 转人工率 |
| statistic_list[].statistic.ai_knowledge_hit_rate | float | 知识命中率 |
| statistic_list[].statistic.msg_rejected_customer_cnt | uint64 | 被拒收消息的客户数 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "statistic_list" : [
            {
                "stat_time" : 1645545600,
                "statistic" : {
                    "session_cnt" : 2,
                    "customer_cnt" : 1,
                    "customer_msg_cnt" : 6,
                    "upgrade_service_customer_cnt" : 0,
                    "ai_session_reply_cnt" : 1,
                    "ai_transfer_rate" : 1,
                    "ai_knowledge_hit_rate" : 0,
                    "msg_rejected_customer_cnt" : 1
                }
            }
        ]
}
```
