# 创建订单

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97634](https://developer.work.weixin.qq.com/document/path/97634)
- **文档 ID**: `97634`
- **API 名称**: `create_order`
- **请求方法**: `POST`
- **接口地址**: `https://wework.qpic.cn/wwpic/803527_rH-6cZNoTMyLBNX_1669809366/0`
- **分组信息**: 第 1 个接口，共 3 个

## 接口描述

前往 “服务商助手 - 工具 - 收银台 “ 创建订单，创建成功后可获取订单支付二维码/链接。

## 请求信息

### 请求示例

```json
N/A
```

## 其他说明

### 选择应用

支持选择已经上线发布的第三方应用、代开发模板和行业方案，选择后即可完成创建订单。

### 更多配置

为满足更为个性化的订单创建需求，服务商可以开启“更多配置”，开启后可配置详细的订单参数，包括订单类型、购买人数、购买时长、额外优惠和指定企业等。

### 创建完成

订单创建完成后，服务商可获取订单的支付链接或二维码，转发给客户企业的管理员和成员打开后，即可购买应用并支付订单。
