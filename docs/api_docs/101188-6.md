# 获取文档数据

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/101188](https://developer.work.weixin.qq.com/document/path/101188)
- **文档 ID**: `101188`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/document/get?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于获取文档数据

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 是 | 文档的docid |

### 请求示例

```json
{
	"docid": "DOCID"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| version | uint32 | 文档版本 |
| document | object(Node) | 文档内容根节点 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"version": 10,
	"document": {
		...
	}
}
```
