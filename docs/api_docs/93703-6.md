# 创建日程

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93703](https://developer.work.weixin.qq.com/document/path/93703)
- **文档 ID**: `93703`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/oa/schedule/add`

## 接口描述

该接口用于在日历中创建一个日程。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| schedule.admins | string | 否 | 日程的管理员userid列表，管理员必须在共享成员的列表中。最多指定3人 |
| schedule.attendees | object[] | 否 | 日程参与者列表。最多支持1000人 |
| schedule.attendees.userid | string | 是 | 日程参与者ID 不多于64字节 |
| schedule.summary | string | 否 | 日程标题。0 ~ 128 字符。不填会默认显示为“新建事件” |
| schedule.description | string | 否 | 日程描述 不多于1000个字符 |
| schedule.reminders | object | 否 | 提醒相关信息 |
| schedule.reminders.is_remind | int32 | 否 | 是否需要提醒。0-否；1-是 |
| schedule.reminders.is_repeat | int32 | 否 | 是否重复日程。0-否；1-是 |
| schedule.reminders.remind_before_event_secs | uint32 | 否 | 日程开始（start_time）前多少秒提醒，当is_remind为1时有效。 |
| schedule.reminders.remind_time_diffs | int32[] | 否 | 提醒时间与日程开始时间（start_time）的差值，当is_remind为1时有效，可以指定多个提醒时间 |
| schedule.reminders.repeat_type | uint32 | 否 | 重复类型，当is_repeat为1时有效 |
| schedule.reminders.repeat_until | uint32 | 否 | 重复结束时刻，Unix时间戳，当is_repeat为1时有效。不填或填0表示一直重复 |
| schedule.reminders.is_custom_repeat | uint32 | 否 | 是否自定义重复。0-否；1-是。当is_repeat为1时有效 |
| schedule.reminders.repeat_interval | uint32 | 否 | 重复间隔 仅当指定为自定义重复时有效 |
| schedule.reminders.repeat_day_of_week | uint32[] | 否 | 每周周几重复 仅当指定为自定义重复且重复类型为每周时有效 |
| schedule.reminders.repeat_day_of_month | uint32[] | 否 | 每月哪几天重复 仅当指定为自定义重复且重复类型为每月时有效 |
| schedule.reminders.timezone | uint32 | 否 | 时区。UTC偏移量表示(即偏离零时区的小时数)，东区为正数，西区为负数。 |
| schedule.location | string | 否 | 日程地址 不多于128个字符 |
| schedule.start_time | uint32 | 是 | 日程开始时间，Unix时间戳 |
| schedule.end_time | uint32 | 是 | 日程结束时间，Unix时间戳 |
| schedule.cal_id | string | 否 | 日程所属日历ID。该日历必须是access_token所对应应用所创建的日历。 |
| schedule.is_whole_day | uint32 | 否 | 是否设置为全天日程。0-否；1-是 |
| agentid | uint32 | 否 | 授权方安装的应用agentid。仅旧的第三方多应用套件需要填此参数 |

			"userid": "userid2"

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |
| schedule_id | string | 日程ID |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg" : "ok",
	"schedule_id":"17c7d2bd9f20d652840f72f59e796AAA"
}
```
