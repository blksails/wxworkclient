# 修改日程事件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97773](https://developer.work.weixin.qq.com/document/path/97773)
- **文档 ID**: `97773`

## 接口描述

应用通过API创建日程后，当日程管理员修改了日程时，触发该事件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 成员UserID |
| CreateTime |  | 否 | 消息创建时间，unix时间戳 |
| MsgType |  | 否 | 消息类型，此时固定为：event |
| Event |  | 否 | 事件类型，此时固定为：modify_schedule |
| CalId |  | 否 | 日历ID |
| ScheduleId |  | 否 | 日程ID |
