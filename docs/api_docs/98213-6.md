# 获取录制转写搜索结果 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98213](https://developer.work.weixin.qq.com/document/path/98213)
- **文档 ID**: `98213`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/record/transcript/search?access_token=ACCESS_TOKEN`

## 接口描述

根据指定内容搜索录制转写

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| record_file_id | string | 是 | 会议录制文件ID |
| meetingid | string | 是 | 会议ID |
| text | string | 是 | 搜索的文本 |

### 请求示例

```json
{"record_file_id":"record","meetingid":"MEETINGID","text":"test"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| hits | object[] | 搜索结果列表，详见Result。 |
| timelines | object[] | 搜索结果时间戳对象列表，详见TimeLine。可用于时间轴上的预览。 |
| pid | string | 段落 ID。 |
| sid | string | 句子 ID。 |
| offset | int32 | text 相对词的偏移。 |
| length | int32 | 匹配长度。 |
| pid | string | 搜索结果所在的段落 ID。 |
| sid | string | 搜索结果所在的句子 ID。 |
| start_time | int64 | 录制文件中的词开始时间（毫秒）。 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","hits":[{"pid":"11","sid":"2","offset":10,"length":10}],"timelines":[{"pid":"11","sid":"2","start_time":11111}]}
```
