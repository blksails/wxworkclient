# 设置企业的许可自动激活状态

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97199](https://developer.work.weixin.qq.com/document/path/97199)
- **文档 ID**: `97199`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/license/set_auto_active_status`

## 接口描述

服务商可以调用该接口设置授权企业的许可自动激活状态。设置为自动激活后，对应授权企业的员工使用服务商应用时，接口许可表现为自动激活。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 应用服务商的接口调用凭证，获取方法参见服务商的凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| corpid | string | 是 | 企业corpid，要求服务商为企业购买过接口许可，购买指支付完成，购买并退款成功包括在内 |
| auto_active_status | int32 | 是 | 许可自动激活状态。0：关闭，1：打开 |

### 请求示例

```json
{
	"corpid":"CORPID1",
	"auto_active_status":1
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok"
}
```
