## 接口描述

判断企业微信专有接口、回调、参数等是否在当前版本可用。注意wx.canIUse判断的是微信小程序通用接口（即wx.[接口名]）是否在当前版本可用。

## 请求信息

### 请求示例

```text
wx.qy.canIUse('login.success.code')wx.qy.canIUse('getNFCReaderState')wx.qy.canIUse("getAvatar.success.avatar")
```

## 其他说明

### boolean wx.qy.canIUse(string schema)

判断企业微信专有接口、回调、参数等是否在当前版本可用。注意wx.canIUse判断的是微信小程序通用接口（即wx.[接口名]）是否在当前版本可用。

### 返回值

boolean
当前版本是否可用
