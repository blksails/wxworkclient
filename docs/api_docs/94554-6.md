# 获取订单详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94554](https://developer.work.weixin.qq.com/document/path/94554)
- **文档 ID**: `94554`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/get_trade`

## 接口描述

获取订单详情

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| payment_id | string | 是 | 收款项目id，由发起班级收款接口返回 |
| trade_no | string | 是 | 订单号，由获取学生付款结果返回 |

### 请求示例

```json
{
   "payment_id":"xxxx",
   "trade_no":"xxxx"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| transaction_id | string | 微信交易单号 |
| pay_time | int32 | 交易时间 |

### 响应示例

```json
{
	"errcode":0,
	"errmsg":"ok",
	"transaction_id":"xxxxx",
	"pay_time":12345
}
```

## 其他说明

### 权限说明

只允许获取由应用本身创建的收款项目的订单详情
