# 获取空间信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97950](https://developer.work.weixin.qq.com/document/path/97950)
- **文档 ID**: `97950`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/new_space_info`

## 接口描述

该接口用于获取空间信息。包括：空间成员及权限及安全设置。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spaceid | string | 是 | 空间spaceid |

### 请求示例

```json
{
    "spaceid": "SPACEID"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| space_info | object | 空间信息 |
| space_info.spaceid | string | 空间id |
| space_info.space_name | string | 空间名 |
| space_info.auth_list | object | 权限列表 |
| space_info.auth_list.auth_info | object[] | 权限信息列表 |
| space_info.auth_list.auth_info[].type | int32 | 类型 |
| space_info.auth_list.auth_info[].userid | string | 用户ID |
| space_info.auth_list.auth_info[].auth | int32 | 权限 |
| space_info.auth_list.quit_userid | string[] | 退出用户ID列表 |
| space_info.space_sub_type | int32 | 空间子类型 |
| space_info.secure_setting | object | 安全设置 |
| space_info.secure_setting.enable_watermark | bool | 启用水印 |
| space_info.secure_setting.add_member_only_admin | bool | 仅管理员添加成员 |
| space_info.secure_setting.enable_share_url | bool | 启用分享链接 |
| space_info.secure_setting.share_url_no_approve | bool | 分享链接无需审批 |
| space_info.secure_setting.share_url_no_approve_default_auth | int32 | 分享链接无需审批默认权限 |
| space_info.secure_setting.enable_share_external | bool | 启用外部分享 |
| space_info.secure_setting.enable_share_external_admin | bool | 仅管理员开启外部分享 |
| space_info.secure_setting.enable_space_add_external_member | bool | 启用空间添加外部成员 |
| space_info.secure_setting.enable_space_add_external_member_admin | bool | 仅管理员开启空间添加外部成员 |
| space_info.secure_setting.enable_confidential_mode | bool | 启用保密模式 |
| space_info.secure_setting.default_file_scope | int32 | 默认文件范围 |
| space_info.secure_setting.create_file_only_admin | bool | 仅管理员创建文件 |
