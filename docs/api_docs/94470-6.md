# 获取学生付款结果 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94470](https://developer.work.weixin.qq.com/document/path/94470)
- **文档 ID**: `94470`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/get_payment_result?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/school/get_payment_result?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| payment_id |  | 是 | 收款项目id，由jssdk的发起班级收款接口或者小程序的发起班级收款接口返回 |

### 请求示例

```json
{"payment_id":"xxxx"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| payment_result |  | 学生付款信息列表，详情见PayInfo说明 |
| student_userid |  | 学生账号 |
| trade_state |  | 付款状态。0表示未付款；1表示已付款 |
| trade_no |  | 订单号 |
| payer_parent_userid |  | 付款家长账号 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","project_name":"学费","amount":998,"payment_result":[{"student_userid":"xxxx","trade_state":1,"trade_no":"xxxxx","payer_parent_userid":"zhangshan"}]}
```
