ww.hideMenuItems({menuList:['menuItem:setFont']})
参数说明params: Object属性类型必填说明menuListstring[]是要隐藏的菜单项successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码
