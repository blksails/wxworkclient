# 发起对外收款

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95948](https://developer.work.weixin.qq.com/document/path/95948)
- **文档 ID**: `95948`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

该接口可以用于商户对客户发起收款请求。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| paymentType | Int | 否 | 收款方式。 0：在聊天中收款 1：收款码收款，不填默认为在聊天中收款。 |
| totalFee | Int | 否 | 收款金额，单位为分，允许范围1～5,000,000分。可为空。若为空或者非法，客户端会忽略该字段 |
| description | String | 否 | 收款说明，“在聊天中收款” 不超过32个字，“收款码收款”不超过16个字。若为空或者超出最大长度，唤起收款页面时，客户端会忽略该字段，由用户填写 |

### 请求示例

```text
wx.qy.createExternalPayment({
	paymentType: 0,
    totalFee: 300,
    description: '可乐一罐',
    success: function(res) {
            var paymentId  = res.paymentId;
    }
});
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errMsg | String | 返回结果信息 |
| paymentId | String | 收款项目id，可使用该id“获取收款项目的商户单号” |

### 响应示例

```json
{
     "errMsg": "createExternalPayment:ok",
     "paymentId": "xxxxx"
}
```

## 其他说明

### 调用前提

1. 在企业微信4.0.12及以后版本支持；
2. 必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态；
3. 应用需要具有对外收款权限；
4. 发起的用户必顡在应用可见范围，且实名；
5. 允许第三方应用、代开发应用和自建应用调用。
