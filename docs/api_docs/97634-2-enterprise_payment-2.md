# 企业支付

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97634](https://developer.work.weixin.qq.com/document/path/97634)
- **文档 ID**: `97634`
- **API 名称**: `enterprise_payment`
- **请求方法**: `POST`
- **接口地址**: `N/A`
- **分组信息**: 第 2 个接口，共 3 个

## 接口描述

客户企业的成员和管理员均可通过二维码/链接进行下单支付。根据服务商所配置的订单信息的差异，客户企业的支付流程表现不同。

## 其他说明

### 企业支付流程

根据服务商所配置的订单信息的差异，客户企业的支付流程表现如下：
1) 没有开启更多配置
2) 开启更多配置，但未指定企业
3) 开启更多配置，且指定了企业
4) 企业可通过微信支付/网银进行订单支付。
