# 获取下级企业付费版本信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93404](https://developer.work.weixin.qq.com/document/path/93404)
- **文档 ID**: `93404`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/xxx/get_subcorp_pay_info`

## 接口描述

获取下级企业付费版本信息的接口

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| pay_info | object | 付费信息 |
| pay_info.corpid | string | 企业ID |
| pay_info.pay_type | int32 | 付费类型 |
