# 获取订单列表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91910](https://developer.work.weixin.qq.com/document/path/91910)
- **文档 ID**: `91910`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/get_order_list?suite_access_token=SUITE_ACCESS_TOKEN`

## 接口描述

获取订单列表

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| start_time | int32 | 是 | 起始时间，UNIX时间戳 |
| end_time | int32 | 是 | 终止时间，UNIX时间戳 |
| test_mode | int32 | 否 | 指定拉取正式或测试授权的订单。默认值为0，其中 0-正式授权，1-测试授权。 |

### 请求示例

```json
{
	"start_time" : 1542091572,
	"end_time" : 1542177972,
	"test_mode" : 1
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| order_list | object[] | 订单列表 |
| order_list[].orderid | string | 订单号 |
| order_list[].order_status | int32 | 订单状态。0-待支付，1-已支付，2-已取消， 3-支付过期， 4-申请退款中， 5-退款成功， 6-退款被拒绝 |
| order_list[].order_type | int32 | 订单类型。0-新购应用，1-扩容应用人数，2-续期应用时间，3-变更版本 |
| order_list[].paid_corpid | string | 客户企业的corpid |
| order_list[].operator_id | string | 下单操作人员userid。如果是服务商代下单，没有该字段。 |
| order_list[].suiteid | string | 应用id |
| order_list[].appid | int32 | 套件应用id（仅旧套件有该字段） |
| order_list[].edition_id | string | 购买版本ID |
| order_list[].edition_name | string | 购买版本名字 |
| order_list[].price | int32 | 应付价格，单位分 |
| order_list[].user_count | int32 | 购买的人数 |
| order_list[].order_period | int32 | 购买的时长，单位天 |
| order_list[].order_time | int32 | 下单时间（UNIX时间戳） |
| order_list[].paid_time | int32 | 付款时间（UNIX时间戳） |
| order_list[].begin_time | int32 | 购买生效期的开始时间（UNIX时间戳） |
| order_list[].end_time | int32 | 购买生效期的结束时间（UNIX时间戳） |
| order_list[].order_from | int32 | 下单来源。0-企业下单；1-服务商代下单；2-代理商代下单; 3-服务商收银台下单 |
| order_list[].operator_corpid | string | 下单方corpid |
| order_list[].service_share_amount | int32 | 服务商分成金额，单位分 |
| order_list[].platform_share_amount | int32 | 平台分成金额，单位分 |
| order_list[].dealer_share_amount | int32 | 代理商分成金额，单位分 |
| order_list[].dealer_corp_info | object | 代理商信息（仅当有渠道商报备后才会有此字段） |
| order_list[].dealer_corp_info.corpid | string | 代理商corpid |
| order_list[].dealer_corp_info.corp_name | string | 代理商的企业简称 |
