# 获取下级/下游企业小程序session - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93355](https://developer.work.weixin.qq.com/document/path/93355)
- **文档 ID**: `93355`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/miniprogram/transfer_session?access_token=ACCESS_TOKEN`

## 接口描述

上级/上游企业通过该接口转换为下级/下游企业的小程序session

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。下级/下游企业的 access_token |
| userid | string | 是 | 通过code2Session接口获取到的加密的userid不多于64字节 |
| session_key | string | 是 | 通过code2Session接口获取到的属于上级/上游企业的会话密钥-不多于64字节 |

### 请求示例

```json
{"userid":"wmAoNVCwAAUrSqEqz7oQpEIEMVWDrPeg","session_key":"n8cnNEoyW1pxSRz6/Lwjwg=="}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| userid | string | 下级/下游企业用户的ID |
| session_key | string | 属于下级/下游企业的会话密钥 |
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |

### 响应示例

```json
{"userid":"abcdef","session_key":"DGAuy2KVaGcnsUrXk8ERgw==","errcode":0,"errmsg":"ok"}
```
