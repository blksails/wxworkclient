# 更改用户功能属性 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98008](https://developer.work.weixin.qq.com/document/path/98008)
- **文档 ID**: `98008`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/exmail/useroption/update?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于更新用户的功能属性

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| userid | string | 是 | 用户UserID |
| type | int32 | 是 | 功能设置属性类型 1: 强制启用安全登录 2: IMAP/SMTP服务 3: POP/SMTP服务 4: 是否启用安全登录 |
| value | string | 是 | 1表示启用，0表示关闭 |

### 请求示例

```json
{"userid":"zhangsan","option":{"list":[{"type":1,"value":"0"},{"type":2,"value":"1"},{"type":3,"value":"0"}]}}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
