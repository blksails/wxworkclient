# 预定Rooms会议室 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98791](https://developer.work.weixin.qq.com/document/path/98791)
- **文档 ID**: `98791`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/rooms/book?access_token=ACCESS_TOKEN`

## 接口描述

对成功预定的会议添加Rooms会议室，支持为同一个会议预定多个Rooms会议室。Rooms会议室预定对会议时长有硬性要求，会议时长不得大于24小时，且不支持周期性会议。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议ID |
| meeting_room_id_list | string[] | 是 | Rooms会议室 ID 列表 |
| subject_visible | bool | 否 | 在会议开始前的一小时内，是否在 Room 上显示会议主题， 默认值为 true。true：显示false：不显示。说明：该参数并不影响预定时间晚过当前时间一个小时以上的会议。超过一小时的会议默认不显示会议主题。 |

### 请求示例

```json
{"meetingid":"MEETINGID","meeting_room_id_list":["RoomId001","RoomId002"],"subject_visible":true}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| meeting_room_list | object[] | Rooms会议室对象列表，详见MeetingRoom |
| meeting_room_id | string | Rooms会议室 ID |
| meeting_room_name | string | Rooms会议室名称 |
| meeting_room_location | string | Rooms会议室地址 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","meeting_room_list":[{"meeting_room_id":"100001","meeting_room_name":"会议室测试1","meeting_room_location":"广州-大厦-10"},{"meeting_room_id":"100002","meeting_room_name":"会议室测试2","meeting_room_location":"广州-大厦-11"}]}
```
