- **文档 ID**: `91623`

## 接口描述

医疗行业的企业，可通过语音电话提醒员工查看应用推送的重要消息，以便尽快处理紧急事件。

## 其他说明

### 开始开发

管理员进入企业微信“管理后台” — “应用管理”—“紧急通知”，启用应用。（注：目前仅医疗行业的企业可开启该应用）充值购买电话分钟数。设置可调用紧急通知的自建应用。进入自建应用详情，查看并妥善保存调用紧急通知接口所需的重要参数（secret），然后通过此应用的secret调用获取access_token得到接口调用凭证。调用发起语音电话接口，提醒员工查看应用推送的重要消息。语音电话模板：“XX（员工姓名），您好。XX（应用名称）给您发来一条重要消息，请打开XX（企业简称）的企业微信查看。”调用获取接听状态接口，了解员工是否已接听语音电话。
