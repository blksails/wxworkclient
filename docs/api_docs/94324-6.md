# ww.checkJsApi

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94324](https://developer.work.weixin.qq.com/document/path/94324)
- **文档 ID**: `94324`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

判断当前客户端版本是否支持指定 JS 接口。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| params.jsApiList | array | 是 | JS 接口列表 |
| params.success | Function | 否 | 成功回调 |
| params.fail | Function | 否 | 失败回调 |
| params.cancel | Function | 否 | 取消回调 |
| params.complete | Function | 否 | 完成回调 |

### 请求示例

```text
ww.checkJsApi({
  jsApiList: ['chooseImage']
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errMsg | string | 通用错误信息 |
| errCode | number | 通用错误码 |
| checkResult | Object | 检查结果  以键值对的形式返回，接口可用时为 true，不可用为 false |

### 响应示例

```text
wx.checkJsApi({
  jsApiList: ['chooseImage'],
  success(res) {
    console.log(res)
  }
})
```
