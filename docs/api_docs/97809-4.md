# 修改日程事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97809](https://developer.work.weixin.qq.com/document/path/97809)
- **文档 ID**: `97809`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/schedule/modify_schedule?access_token=ACCESS_TOKEN`

## 接口描述

应用通过API创建日程后，当日程管理员修改了日程时，触发该事件。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName | string | 是 | 企业微信CorpID |
| FromUserName | string | 是 | 成员UserID |
| CreateTime | uint32 | 是 | 消息创建时间，unix时间戳 |
| MsgType | string | 是 | 消息类型，固定为：`event` |
| Event | string | 是 | 事件类型，固定为：`modify_schedule` |
| CalId | string | 是 | 日历ID |
| ScheduleId | string | 是 | 日程ID |

### 请求示例

```xml
<xml>
   <ToUserName><![CDATA[toUser]]></ToUserName>
   <FromUserName><![CDATA[fromUser]]></FromUserName>
   <CreateTime>1348831860</CreateTime>
   <MsgType><![CDATA[event]]></MsgType>
   <Event><![CDATA[modify_schedule]]></Event>
   <CalId><![CDATA[wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA]]></CalId>
   <ScheduleId><![CDATA[17c7d2bd9f20d652840f72f59e796AAA]]></ScheduleId>
</xml>
```
