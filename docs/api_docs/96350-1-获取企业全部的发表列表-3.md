# 关于客户朋友圈

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96350](https://developer.work.weixin.qq.com/document/path/96350)
- **文档 ID**: `96350`
- **API 名称**: `获取企业全部的发表列表`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/get_publish_list`
- **分组信息**: 第 1 个接口，共 3 个

## 接口描述

成员可发表内容到微信客户的朋友圈，还可与客户评论互动。企业也可统一创建内容，由成员确认后发表。客户朋友圈可分为企业发表和个人发表两类。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| publish_list | object[] | 发表列表 |
| publish_list[].type | string | 发表类型 |
| publish_list[].content | string | 发表内容 |

## 其他说明

### 关于客户朋友圈
