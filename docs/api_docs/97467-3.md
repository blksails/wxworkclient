# 接收事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97467](https://developer.work.weixin.qq.com/document/path/97467)
- **文档 ID**: `97467`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webhook/receive_event`

## 接口描述

企业微信推送事件给第三方应用的回调URL，第三方应用需要接收并解析这些事件。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| event_type | string | 事件类型，如：user_create、user_update |
| event_time | uint32 | 事件发生的时间戳 |
| event_detail | object | 事件详情，具体字段根据不同事件类型而定 |

## 其他说明

### 概述

API创建的文档允许管理员对该文档进行修改成员/删除操作。为了让企业实时获取用户的操作，当用户进行了某种操作后，企业微信会将相应的事件推送给应用配置的回调URL。
