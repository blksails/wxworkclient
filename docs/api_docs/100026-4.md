# 会话名称搜索 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100026](https://developer.work.weixin.qq.com/document/path/100026)
- **文档 ID**: `100026`

## 接口描述

可调用该接口对企业的群会话名称进行搜索权限说明

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 自建应用 | 自建应用 | 否 | 需具备「数据与智能专区权限」 |
| 代开发应用 | 代开发应用 | 否 | 需具备「数据与智能专区权限」 |
| 第三方应用 | 第三方应用 | 否 | 需具备「数据与智能专区权限」 |
| query_word |  | 是 | 搜索的文本，要求大于等于2个UTF8字符。 |
| limit |  | 否 | 用于分页查询，每次请求返回的数据上限。最大为100，不填默认为50 |
| cursor |  | 否 | 用于分页查询的游标，字符串类型，由上一次调用返回，首次调用可不填 |

### 请求示例

```json
{"query_word":"VIP","limit":50,"cursor":"CURSOR"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 错误码 |
| errmsg |  | 错误码说明 |
| has_more |  | 是否还有更多数据未获取，不填默认为00: 否1: 是 |
| next_cursor |  | 下一次请求的cursor值 |
| chat_list |  | 搜索结果的群列表 |
| chatid |  | 群聊会话id |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","has_more":1,"next_cursor":"CURSOR","chat_list":[{"chatid":"CHATID1"},{"chatid":"CHATID2"}]}
```
