# 设备数据授权变更回调 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96103](https://developer.work.weixin.qq.com/document/path/96103)
- **文档 ID**: `96103`
- **请求方法**: `POST`

## 接口描述

当用户授权设备数据给应用发生变更时，企业微信回调该指令回调到应用配置的数据回调url中。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId |  | 否 | 第三方应用的SuiteId |
| AuthCorpId |  | 否 | 授权企业的CorpID |
| InfoType |  | 否 | 事件类型，device_data_auth_change 设备数据授权变更 |
| TimeStamp |  | 否 | 消息创建时间（整型） |
| AgentID |  | 否 | 企业应用的id，整型。可在应用的设置页面查看 |

### 请求示例

```xml
</xml><SuiteId><![CDATA[ww4asffe99e54cxxxx]]></SuiteId><AuthCorpId><![CDATA[wxf8b4f85f3a79xxxx]]></AuthCorpId><InfoType><![CDATA[device_data_auth_change]]></InfoType><TimeStamp>1403610513</TimeStamp><AgentID>1000001</AgentID></xml>
```
