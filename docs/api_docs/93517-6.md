# 获取上报事件分类统计 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93517](https://developer.work.weixin.qq.com/document/path/93517)
- **文档 ID**: `93517`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/report/resident/category_statistic?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/report/resident/category_statistic?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证。“获取access_token” |
| category_id |  | 否 | 分类ID，不传此字段，能拉取到所有一级分类的数据，然后传一级分类的category_id能拉取到该一级分类下的所有二级分类的数据 |
| 自建应用 | 自建应用 | 否 | 配置到「居民上报 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"category_id":"category_id"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| dashboard_list.category_id |  | 分类ID |
| dashboard_list.category_name |  | 分类名称 |
| dashboard_list.category_level |  | 分类等级，1、一级分组；2、二级分类 |
| dashboard_list.total_case |  | 累计上报 |
| dashboard_list.total_solved |  | 累计办结 |
| dashboard_list.category_type |  | 分类类型，0：其他，1：有效分类 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","dashboard_list":[{"category_id":"category_id","category_name":"category name","category_level":1,"total_case":100,"total_solved":100,"category_type":1}]}
```
