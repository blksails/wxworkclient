# 获取邮件未读数 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95514](https://developer.work.weixin.qq.com/document/path/95514)
- **文档 ID**: `95514`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/exmail/mail/get_newcount?access_token=ACCESS_TOKEN`

## 接口描述

获取指定成员邮箱当前未读邮件数量权限说明

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| userid |  | 是 | 成员UserID |

### 请求示例

```json
{"userid":"member"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| count |  | 成员邮箱中邮件未读数 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","count":1}
```
