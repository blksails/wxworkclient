| 参数名 | 类型 | 必填 | 说明 |

## 其他说明

### ww.setShareAttr([params])

兼容性：企业微信 3.1.8 开始支持
功能描述设置私密消息。
ww.setShareAttr({withShareTicket:true,state:'STATE'})
使用限制本接口必须使用应用身份进行注册
参数说明params: Object属性类型必填说明withShareTicketboolean否分享消息是否携带 share ticketstatestring否详见私密消息successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

设置私密消息。

### 使用限制

本接口必须使用应用身份进行注册

### params: Object

属性类型必填说明withShareTicketboolean否分享消息是否携带 share ticketstatestring否详见私密消息successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 示例代码

wx.invoke("setShareAttr",{withShareTicket:true,state:"STATE",},function(res){if(res.err_msg=="setShareAttr:ok"){}});
