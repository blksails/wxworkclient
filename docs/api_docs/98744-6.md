# 获取带参授权链接

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98744](https://developer.work.weixin.qq.com/document/path/98744)
- **文档 ID**: `98744`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/get_customized_auth_url?provider_access_token=PROVIDER_ACCESS_TOKEN`

## 接口描述

该API用于获取代开发自建应用授权链接，用于生成带参临时二维码。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | provider_access_token，获取方法参见服务商的凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| state | string | 是 | state值，可以填写a-zA-Z0-9，长度不可超过32个字节。当扫带参二维码授权代开发模版时，获取企业永久授权码接口会返回该state值 |
| templateid_list | array | 是 | 代开发自建应用模版ID列表，数量不能超过9个 |

### 请求示例

```json
{
	"state":"test",
	"templateid_list": ["dk1","dk2"]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误描述 |
| qrcode_url | string | 可用来生成二维码的授权url，需要开发者自行生成为二维码 |
| expires_in | uint32 | 有效期（秒）。10天过期。 |

### 响应示例

```json
{
	"errcode":0 ,
	"errmsg":"ok" ,
	"qrcode_url":"xxxxx",
	"expires_in":864000
}
```
