# 发起班级收款

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94804](https://developer.work.weixin.qq.com/document/path/94804)
- **文档 ID**: `94804`
- **请求方法**: `wx.qy.createSchoolPayment`

## 接口描述

该接口可以用于老师对学生家长发起付款请求。接口调用成功后会通过家校通知发送付款小程序给家长。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| projectName | String | 否 | 收款项目名称，最多32个字。若为空或者超出最大长度，唤起原生收款页面时，客户端会忽略该字段，由用户填写 |
| amount | Int | 否 | 收款金额，每个学生需付费的金额，单位为分。可为空。限额100000。若非法客户端会忽略该字段 |
| payers.students | Array | 否 | 需要收款的学生列表 |
| payers.departments | Array | 否 | 需要收款的家校通讯录部门列表、支持班级，年级，校区 |

### 请求示例

```text
wx.qy.createSchoolPayment({
    projectName: '1班班费',
    amount: 100,
    payers: {
        students: ['zhangshan', 'lisi'],
        departments: [1, 2]
    }
},
success: function(res) {
    var paymentId = res.paymentId;
});
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errMsg | String | 返回结果说明 |
| paymentId | String | 收款项目id，可使用该id调用“获取学生付款结果”接口 |

### 响应示例

```json
{
    "errMsg": "createSchoolPayment:ok",
    "paymentId": "xxxxx"
}
```

## 其他说明

### 接口权限说明

1. 仅企业微信客户端3.1.10版本及以上可用（mac端暂不支持）
2. 必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态
3. 仅支持配置在家长可使用范围内的应用；
4. 企业必须已验证或者已认证才允许调用；
5. 发起的用户必顨在应用可见范围，且实名。
