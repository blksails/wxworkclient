## 其他说明

### ww.saveApprovalSelectedItems(params)

兼容性：企业微信 4.0.18 开始支持
功能描述保存用户选择的审批选项。用户在网页中修改审批选项时，调用该接口保存用户的选择。
ww.saveApprovalSelectedItems({key:'key',selectedData:[{key:'item-1',value:'选项1'},{key:'item-2',value:'选项2'}]})
使用限制本接口必须使用应用身份进行注册应用必须具有审批权限
参数说明params: Object属性类型必填说明keystring是从 URL 中获取到的 keyselectedDataObject[]是选中的选项属性类型必填说明keystring是表示选项的唯一识别 IDvaluestring是选项展示给用户的名字successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码
使用说明接口仅用于审批设置外部选项场景，请勿用作其他场景网页应该做好深色模式适配

### 功能描述

保存用户选择的审批选项。
用户在网页中修改审批选项时，调用该接口保存用户的选择。

### 使用限制

本接口必须使用应用身份进行注册应用必须具有审批权限

### params: Object

属性类型必填说明keystring是从 URL 中获取到的 keyselectedDataObject[]是选中的选项属性类型必填说明keystring是表示选项的唯一识别 IDvaluestring是选项展示给用户的名字successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 使用说明

接口仅用于审批设置外部选项场景，请勿用作其他场景网页应该做好深色模式适配
