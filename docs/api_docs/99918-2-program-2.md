## 接口描述

服务商可在专区中上传部署程序，用于分析企业授权给服务商的数据，并将分析结果回调通知服务商的应用。

## 其他说明

### 上传程序信息

上传程序时，需要完善以下程序信息：
- 基本信息：程序名称、程序Logo、程序描述
- 镜像文件：上传程序的镜像文件，并设置启动命令和启动参数
- 程序能力：支持添加一到多项程序能力，每项程序能力包括能力名称、能力ID、输入协议、输出协议
