# 开始开发

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96252](https://developer.work.weixin.qq.com/document/path/96252)
- **文档 ID**: `96252`
- **API 名称**: `通讯录同步secret`
- **请求方法**: `GET`
- **接口地址**: `https://developer.work.weixin.qq.com/resource/devtool`
- **分组信息**: 第 1 个接口，共 4 个

## 接口描述

企业如果需要从自有的系统同步通讯录到企业微信，需要使用通讯录同步secret。如果企业委托授权第三方服务商将通讯录从其他系统同步到企业微信，则需要企业授权“通讯录编辑授权”给服务商。

## 请求信息

### 请求示例

```text
1. 进入企业微信管理后台，在“管理工具” -- “通讯录同步”开启“API接口同步"
2. 开启后，可设置通讯录API的权限：读取或者编辑通讯录
3. 配置企业可信IP，仅所配IP可调用通讯录API
4. 使用通讯录同步的secret进行开发。点击查看 示例代码
```
