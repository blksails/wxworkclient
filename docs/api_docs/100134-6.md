# 获取成员多次收消息详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100134](https://developer.work.weixin.qq.com/document/path/100134)
- **文档 ID**: `100134`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/customer_acquisition/get_chat_info`

## 接口描述

企业和服务商可通过此接口获取成员多次收消息情况，如次数、客户id等信息。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| chat_key | string | 是 | [成员多次收消息事件](#42924/%E6%88%90%E5%91%98%E5%A4%9A%E6%AC%A1%E6%94%B6%E6%B6%88%E6%81%AF%E4%BA%8B%E4%BB%B6)中回调的会话信息凭据ChatKey，回调后30分钟内有效 |

### 请求示例

```json
{
  "chat_key":"CHAT_KEY"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| userid | string | 成员的userid |
| external_userid | string | 客户id |
| chat_info | object |  |
| chat_info.recv_msg_cnt | int32 | 成员收到的此客户的消息次数 |
| chat_info.link_id | string | 成员添加客户的获客链接id |
| chat_info.state | string | 成员添加客户的state |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"userid":"zhangsan",
	"external_userid":"woAJ2GCAAAXtWyujaWJHDDGi0mACAAA",
	"chat_info":
	{
			"recv_msg_cnt":3,
			"link_id":"caXXXXXXX",
			"state":"STATE"
	}
}
```
