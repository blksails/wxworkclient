## 其他说明

### 设置日志打印级别

设置指定模型或程序的日志打印级别。
请求方式:POST(HTTPS)
请求地址:https://qyapi.weixin.qq.com/cgi-bin/chatdata/set_log_level?access_token=ACCESS_TOKEN
请求示例
{"program_id":"xxxx","log_level":2}
参数说明:
参数必须类型说明access_token是string调用接口凭证program_id是string应用关联的程序idlog_level是uint32日志级别指定后，仅会存储不高于改级别的日志。例如指定级别为2，那么只会存储级别为1或2的日志默认级别为2取值范围：1 - ERR2 - INFO3 - DBG
权限说明:
应用需具有数据专区权限指定的程序program_id需与应用有授权关系官方模型不支持
返回结果:
{"errcode":0,"errmsg":"ok"}
参数说明:
参数类型说明errcodeint32返回码errmsgstring错误码描述
