# 使用 `@wecom/jssdk` 初始化企业微信登录组件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98171](https://developer.work.weixin.qq.com/document/path/98171)
- **文档 ID**: `98171`

- **请求方法**: `N/A`
- **接口地址**: `N/A`
- **分组信息**: 第 1 个接口，共 2 个

## 接口描述

为了满足网站定制化的需求，支持将企业微信登录组件内嵌到开发者的网站中。用户使用企业微信登录授权后，登录组件将 `auth code` 返回给网站。

## 其他说明

### 开始使用

使用企业微信 JS-SDK：[@wecom/jssdk](#24229) `>=2.3.2`
