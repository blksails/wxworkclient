# 查询订单 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97323](https://developer.work.weixin.qq.com/document/path/97323)
- **文档 ID**: `97323`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/miniapppay/get_order?access_token=ACCESS_TOKEN`

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 商户号 | string[1,32] | 是 | 二级商户号，由企业微信生成并下发。 |
| 商户订单号 | string[6,32] | 是 | 商户系统内部订单号，只能是数字、大小 写字母_-*且在同一个商户号下唯一。 |

### 请求示例

```json
{"mchid":"xxx","out_trade_no":"xxx"}
```
