# 分享设置 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97889](https://developer.work.weixin.qq.com/document/path/97889)
- **文档 ID**: `97889`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_setting?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于文件的分享设置。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fileid | string | 是 | 文件fileid |
| auth_scope | uint32 | 是 | 权限范围：1:指定人 2:企业内 3:企业外 4: 企业内需管理员审批（仅有管理员时可设置） 5: 企业外需管理员审批（仅有管理员时可设置） |
| auth | uint32 | 否 | 权限信息普通文档： 1:仅浏览（可下载) 4:仅预览（仅专业版企业可设置）；如果不填充此字段为保持原有状态微文档： 1:仅浏览（可下载）；如果不填充此字段为保持原有状态 |

### 请求示例

```json
{"fileid":"FILDID","auth_scope":1,"auth":1}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
