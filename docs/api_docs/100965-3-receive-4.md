# 取消授权通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100965](https://developer.work.weixin.qq.com/document/path/100965)
- **文档 ID**: `100965`
- **API 名称**: `receive`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`
- **分组信息**: 第 3 个接口，共 3 个

## 接口描述

当授权方（即授权企业）在企业微信管理端的授权管理中，取消了对应用的授权托管后，企业微信后台会推送取消授权通知。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 | 消息签名 |
| timestamp | string | 是 | 时间戳 |
| nonce | string | 是 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 第三方应用的SuiteId或者代开发应用模板id |
| InfoType | string | 是 | cancel_auth |
| TimeStamp | string | 是 | 时间戳 |
| AuthCorpId | string | 是 | 授权方企业的corpid |

### 请求示例

```xml
<xml>
	<SuiteId><![CDATA[ww4asffe99e54cxxxx]]></SuiteId>
	<InfoType><![CDATA[cancel_auth]]></InfoType>
	<TimeStamp>1403610513</TimeStamp>
	<AuthCorpId><![CDATA[wxf8b4f85fxx794xxx]]></AuthCorpId>
</xml>
```
