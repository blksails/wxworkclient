# 获取会话内容存档开启成员列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91614](https://developer.work.weixin.qq.com/document/path/91614)
- **文档 ID**: `91614`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/msgaudit/get_permit_user_list?access_token=ACCESS_TOKEN`

## 接口描述

企业可通过此接口，获取企业开启会话内容存档的成员列表。调用频率不可超过1000次/分钟。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| type |  | 否 | 拉取对应版本的开启成员列表。1表示办公版；2表示服务版；3表示企业版。非必填，不填写的时候返回全量成员列表。 |

### 请求示例

```json
{"type":1}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| ids |  | 设置在开启范围内的成员的userid列表 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","ids":["userid_111","userid_222","userid_333"]}
```
