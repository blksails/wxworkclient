# 进入应用购买页面 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99118](https://developer.work.weixin.qq.com/document/path/99118)
- **文档 ID**: `99118`

## 接口描述

接口说明第三方应用可以使用该接口提供按钮，让用户可快速进入应用购买流程。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| ios |  | 否 | ≥4.1.6 |
| android |  | 否 | ≥4.1.6 |
| win |  | 否 | ≥4.1.6 |
| mac |  | 否 | ≥4.1.6 |
| 自建应用 | 自建应用 | 否 | 不支持 |
| 代开发应用 | 代开发应用 | 否 | 不支持 |
| 第三方应用 | 第三方应用 | 否 | 支持 |

### 请求示例

```json
wx.qy.openAppPurchase({success:function(res){// 调用成功},fail:function(err){// 调用失败}});
```
