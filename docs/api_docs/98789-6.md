# 挂断 MRA 呼叫 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98789](https://developer.work.weixin.qq.com/document/path/98789)
- **文档 ID**: `98789`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/mra/hangup?access_token=ACCESS_TOKEN`

## 接口描述

会议中对 MRA 的呼叫进行挂断操作。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议ID |

### 请求示例

```json
{"meetingid":"MEETINGID","mra":{"tmp_openid":"TMP_OPENID"}}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| mra.tmp_openid | string | 被操作 mra 设备 的会中临时ID |
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
