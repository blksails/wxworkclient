# 获取会话内容存档内部群信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92951](https://developer.work.weixin.qq.com/document/path/92951)
- **文档 ID**: `92951`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/msgaudit/groupchat/get?access_token=ACCESS_TOKEN`

## 接口描述

企业可通过此接口，获取会话内容存档本企业的内部群信息，包括群名称、群主id、公告、群创建时间以及所有群成员的id与加入时间。调用频率不可超过2000次/分钟。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| roomid |  | 是 | 待查询的群id |

### 请求示例

```json
{"roomid":"wrNplhCgAAIVZohLe57zKnvIV7xBKrig"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| roomname |  | roomid对应的群名称 |
| creator |  | roomid对应的群创建者，userid |
| room_create_time |  | roomid对应的群创建时间 |
| notice |  | roomid对应的群公告 |
| members |  | roomid对应的群成员列表 |
| memberid |  | roomid群成员的id，userid |
| jointime |  | roomid群成员的入群时间 |
| 301052 |  | 会话存档已过期 |

### 响应示例

```json
{"roomname":"蓦然回首","creator":"ZhangWenChao","room_create_time":1592361604,"notice":"","members":[{"memberid":"ZhangWenChao","jointime":1592361605},{"memberid":"xujinsheng","jointime":1592377076}],"errcode":0,"errmsg":"ok"}
```
