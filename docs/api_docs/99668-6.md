# 获取授权存档的成员列表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99668](https://developer.work.weixin.qq.com/document/path/99668)
- **文档 ID**: `99668`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/get_auth_user_list`

## 接口描述

可通过该接口获取授权企业所有生效中的成员列表。（企业会话内容中授权的生效成员）

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | String | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| cursor | String | 否 | 上一次调用时返回的next_cursor，第一次拉取可以不填 |
| limit | Interger | 否 | 本次查询返回的最大条数。不超过1000，默认200条 |

### 请求示例

```json
{
	"cursor":"JMgigeong",
	"limit":2
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | Integer | 错误码 |
| errmsg | String | 错误码说明 |
| auth_user_list | Object[] | 生效成员列表，详见 AuthUser |
| next_cursor | String | 下一次查询时使用，将值填到请求包的cursor字段中 |
| has_more | Integer | 是否还有更多未拉取的数据，1：是；0：否 |
| auth_user_list[].userid | String | 授权成员的userid |
| auth_user_list[].edition_list | Integer | 生效的版本列表，1：内部会话；2：内外部会话；3：内外部会话以及语音通话 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"auth_user_list": [{
		"userid": "woxxxjigegen",
		"edition_list":[1]
	},
	{
		"userid": "woaaaaaaaaaa",
		"edition_list":[1,2,3]
	}
	],
	"next_cursor":"xjigeojgeigwj",
	"has_more":true
}
```
