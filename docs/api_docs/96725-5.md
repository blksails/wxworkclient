# 获取可使用的家长范围 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96725](https://developer.work.weixin.qq.com/document/path/96725)
- **文档 ID**: `96725`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/agent/get_allow_scope?access_token=ACCESS_TOKEN&agentid=AGENTID`

## 接口描述

获取可在微信「学校通知-学校应用」使用该应用的家长范围，以学生或部门列表的形式返回。应用只能给该列表下的家长发送「学校通知」。注意该范围只能由学校的系统管理员在「管理端-家校沟通-配置」配置。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| agentid |  | 是 | 应用id |
| 自建应用 | 自建应用 | 否 |  |
| 代开发应用 | 代开发应用 | 否 | 无 |
| 第三方应用 | 第三方应用 | 否 |  |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 出错返回码，为0表示成功，非0表示调用失败 |
| errmsg |  | 返回码提示语 |
| allow_scope |  | 可在微信「学校通知-学校应用」使用该应用的家长范围 |
| allow_scope.students.userid |  | 家长可在微信「学校通知-学校应用」使用该应用的学生列表 |
| allow_scope.departments.partyid |  | 家长可在微信「学校通知-学校应用」使用该应用的部门列表 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","allow_scope":{"students":{"userid":["student1","student2"]},"departments":{"partyid":[1,2]}}}
```
