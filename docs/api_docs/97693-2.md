# 查看日程闲忙状态 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97693](https://developer.work.weixin.qq.com/document/path/97693)
- **文档 ID**: `97693`

## 接口描述

此接口用于查看企业内成员在指定时间内日程闲忙状态

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| start_time | Uint32 | 是 | 开始时间，Unix时间戳 |
| end_time | Uint32 | 是 | 结束时间，Unix时间戳 |
| users | Array | 否 | 成员userid列表，第三方应用与代开发应用须传入open_userid列表。不填或列表为空则仅查看用户本人的日程闲忙状态 |
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| ios |  | 否 | ≥4.0.20 |
| android |  | 否 | ≥4.0.20 |
| win |  | 否 | ≥4.0.20 |
| mac |  | 否 | ≥4.0.20 |
| 自建应用 | 自建应用 | 否 | 需具有“日程”权限 |
| 第三方应用 | 第三方应用 | 否 | 需具有“日程”权限 |
| 代开发应用 | 代开发应用 | 否 | 需具有“日程”权限 |

### 请求示例

```json
wx.qy.checkSchedule({start_time:1667232000,end_time:1667318400,users:["jack","jason"],success:function(res){// 回调}});
```
