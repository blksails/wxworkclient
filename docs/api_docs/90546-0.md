# 企业微信JS-SDK

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90546](https://developer.work.weixin.qq.com/document/path/90546)
- **文档 ID**: `90546`

## 接口描述

企业微信JS-SDK是企业微信面向网页开发者提供的基于企业微信内的网页开发工具包。通过使用企业微信JS-SDK，网页开发者可借助企业微信高效地使用拍照、选图、语音、位置等手机系统的能力，同时可以直接使用企业微信分享、扫一扫等企业微信特有的能力，为企业微信用户提供更优质的网页体验。
