# 删除文档事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97840](https://developer.work.weixin.qq.com/document/path/97840)
- **文档 ID**: `97840`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

当文档管理员删除API创建的文档、表格后，触发该事件。

## 请求信息

### 请求示例

```xml
<xml>
   <ToUserName><![CDATA[toUser]]></ToUserName>
   <FromUserName><![CDATA[fromUser]]></FromUserName>
   <CreateTime>1348831860</CreateTime>
   <MsgType><![CDATA[event]]></MsgType>
   <Event><![CDATA[doc_change]]></Event>
   <ChangeType><![CDATA[delete_doc]]></ChangeType>
   <DocId><![CDATA[wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA]]></DocId>
   <DocId><![CDATA[wcjgewCwAAqeJcPI1d8Pwbjt7nttzBBB]]></DocId>
</xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| ToUserName | string | 企业微信CorpID |
| FromUserName | string | 成员UserID |
| CreateTime | uint32 | 消息创建时间，unix时间戳 |
| MsgType | string | 消息类型，固定为：`event` |
| Event | string | 事件类型，固定为：`doc_change` |
| ChangeType | string | 事件类型，固定为：`delete_doc` |
| DocId | string | 文档ID列表 |
