# 空间变更事件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97933](https://developer.work.weixin.qq.com/document/path/97933)
- **文档 ID**: `97933`

## 接口描述

当接口指定的管理员在企业微信客户端，对应用创建的空间进行变更，触发该事件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 此事件该值固定为sys，表示该消息由系统生成 |
| CreateTime |  | 否 | 消息创建时间，unix时间戳 |
| MsgType |  | 否 | 消息类型，此时固定为：event |
| Event |  | 否 | 事件类型，此时固定为：wedrive_space_change |
| ChangeType |  | 否 | 变更类型，此处目前固定为：dismiss_space |
| SpaceId |  | 否 | 空间spaceid |

### 请求示例

```xml
<xml><ToUserName><![CDATA[toUser]]></ToUserName><FromUserName><![CDATA[sys]]></FromUserName><CreateTime>1348831860</CreateTime><MsgType><![CDATA[event]]></MsgType><Event><![CDATA[wedrive_space_change]]></Event><ChangeType><![CDATA[dismiss_space]]></ChangeType><SpaceId><![CDATA[spxxxxxxxxxxxxx]]></SpaceId></xml>
```
