# 获取分享链接 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97924](https://developer.work.weixin.qq.com/document/path/97924)
- **文档 ID**: `97924`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_share?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于获取文件的分享链接。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fileid | string | 是 | 文件fileid |

### 请求示例

```json
{"fileid":"FILEID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| share_url | string | 分享文件的链接 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","share_url":"SHARE_URL"}
```
