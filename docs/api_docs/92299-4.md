# 获取部门列表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92299](https://developer.work.weixin.qq.com/document/path/92299)
- **文档 ID**: `92299`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/department/list`

## 接口描述

获取部门列表及其下的子部门。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| id | string | 否 | 部门id。如果不填，默认获取全量组织架构 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| departments | object[] | 部门列表数据 |
| departments[].id | int32 | 部门id，根部门固定为1 |
| departments[].type | int32 | 部门类型，1表示班级，2表示年级，3表示学段，4表示校区，5表示学校（根部门） |
| departments[].name | string | 部门名称 |
| departments[].parentid | int32 | 父亲部门id，根部门该项为0 |
| departments[].register_year | string | 入学年份，仅标准年级返回，格式为YYYY |
| departments[].standard_grade | int32 | 标准年级，当为部门为标准年级时返回 |
| departments[].order | int32 | 在父部门中的次序值，order值大的排序靠前。有效的值范围是[0, 2^32)，仅在API设置后才返回 |
| departments[].department_admins[].userid | string | 部门管理员的userid |
| departments[].department_admins[].type | int32 | 部门管理员的类型，1表示校区负责人，2表示年级负责人，3表示班主任，4表示任课老师，5表示学段负责人 |
| departments[].department_admins[].subject | string | 教师或班主任的科目 |
| departments[].open_group_chat | int32 | 是否开启班级群，1表示开启，0表示关闭。仅部门类型为班级时才返回该字段 |
| departments[].group_chat_id | string | 班级群id。仅部门类型为班级时且open_group_chat为1时才返回该字段 |
| departments[].is_graduated | int32 | 是否是已毕业，1表示是，0表示不是。仅部门类型为年级时才返回该字段 |
