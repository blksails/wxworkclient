# 更新会议受邀成员列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98162](https://developer.work.weixin.qq.com/document/path/98162)
- **文档 ID**: `98162`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/set_invitees?access_token=ACCESS_TOKEN`

## 接口描述

可根据会议 ID 更新会议受邀成员列表，最多可以传入2000名受邀成员，与企业所购在线会议室最大方数相关。管理员必须在受邀成员列表中。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议ID |
| invitees | object[] | 否 | 受邀成员列表。最大支持2000人。与企业所购在线会议室最大方数相关。 |

### 请求示例

```json
{"meetingid":"meetingid","invitees":[{"userid":"zhangsan"},{"userid":"lisi"}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| invitees.userid | string | 受邀成员userid |
| errcode |  | int32 |
| errmsg |  | string |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
