# 语音转文字接口

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92416](https://developer.work.weixin.qq.com/document/path/92416)
- **文档 ID**: `92416`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/translateVoice`

## 接口描述

企业微信手机端2.7.5及以后版本支持的语音转文字接口。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| filePath | String | 否 | 需要识别的语音文件的文件路径，由录音相关接口获得，仅支持由wx.startRecord录制的语音文件 |
| isShowProgressTips | number | 否 | 默认为1，显示进度提示 |

### 请求示例

```text
wx.qy.translateVoice({
  filePath: '', // 需要播放的语音文件的文件路径，由录音相关接口获得
  isShowProgressTips: 1, // 默认为1，显示进度提示
  success: (res) => {
      var result  = res.translateResult
  },
  fail: (res) => {
    console.log(JSON.stringify(res))
  }
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| translateResult | String | 语音识别的结果 |
