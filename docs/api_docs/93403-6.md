# 获取应用共享信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93403](https://developer.work.weixin.qq.com/document/path/93403)
- **文档 ID**: `93403`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/corpgroup/corp/list_app_share_info?access_token=ACCESS_TOKEN`

## 接口描述

局校互联中的局端或者上下游中的上游企业通过该接口可以获取某个应用分享给的所有企业列表。特别注意，对于有敏感权限的应用，需要下级/下游企业确认后才能共享成功，若下级/下游企业未确认，则不会存在于该接口的返回列表

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证，上级/上游企业应用access_token |
| business_type |  | 否 | 填0则为企业互联/局校互联，填1则表示上下游企业 |
| agentid |  | 是 | 上级/上游企业应用agentid |
| corpid |  | 否 | 下级/下游企业corpid，若指定该参数则表示拉取该下级/下游企业的应用共享信息 |
| limit |  | 否 | 返回的最大记录数，整型，最大值100，默认情况或者值为0表示下拉取全量数据，建议分页拉取或者通过指定corpid参数拉取。 |
| cursor |  | 否 | 用于分页查询的游标，字符串类型，由上一次调用返回，首次调用可不填 |

### 请求示例

```json
{"agentid":1111,"business_type":1,"corpid":"wwcorp","limit":100,"cursor":"xxxxxx"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| ending |  | 1表示拉取完毕，0表示数据没有拉取完 |
| next_cursor |  | 分页游标，再下次请求时填写以获取之后分页的记录，如果已经没有更多的数据则返回空 |
| corp_list |  | 应用共享信息 |
| corp_list.corpid |  | 下级/下游企业corpid |
| corp_list.corp_name |  | 下级/下游企业名称 |
| corp_list.agentid |  | 下级/下游企业应用id |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","ending":0"corp_list":[{"corpid":"wwcorpid1","corp_name":"测试企业1""agentid":1111},{"corpid":"wwcorpid2","corp_name":"测试企业2","agentid":1112}],"next_cursor":"next_cursor1111"}
```
