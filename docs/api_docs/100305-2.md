# 取消订单通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100305](https://developer.work.weixin.qq.com/document/path/100305)
- **文档 ID**: `100305`

## 接口描述

当服务商取消订单时，企业微信后台会推送“退款结果通知”到服务商的系统事件接收URL（应用管理 -通用开发参数-系统事件接收URL）。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ServiceCorpId |  | 否 | 服务商CorpID |
| InfoType |  | 否 | 事件类型，此时固定为advanced_api_cancel |
| AuthCorpId |  | 否 | 客户企业CorpID |
| AdvanceApiType |  | 否 | 高级接口类型，1-会话内容数据接口 |
| OrderId |  | 否 | 订单号 |
| TimeStamp |  | 否 | 时间戳 |

### 请求示例

```xml
<xml><ServiceCorpId><![CDATA[xxxx]]></ServiceCorpId><InfoType><![CDATA[advanced_api_cancel]]></InfoType><AuthCorpId><![CDATA[yyyy]]></AuthCorpId><AdvanceApiType><![CDATA[1]]></AdvanceApiType><OrderId><![CDATA[zzzz]]></OrderId><TimeStamp>1403610513</TimeStamp></xml>
```
