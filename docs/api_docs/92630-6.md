# 复制/更新模板到企业

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92630](https://developer.work.weixin.qq.com/document/path/92630)
- **文档 ID**: `92630`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/oa/approval/copytemplate`

## 接口描述

第三方应用开发者可通过安装在企业的第三方应用的AccessToken调用此接口，将服务商后台创建并审核通过的审批模板，复制或更新到指定企业以便以在不同企业使用相同的审批模板进行其他接口操作。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。必须使用第三方应用accesstoken获取，获取方式参考：文档-获取access_token（accesstoken决定了此服务商模板复制/更新的目标企业，请务必保证准确） |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| open_template_id | string | 是 | 服务商审批模板的唯一标识id。可在“获取审批单据详情”、“审批状态变化回调通知”中获得，也可在服务商后台-应用管理-审批模板的模板编辑页面中获得。 |

### 请求示例

```json
{
        “open_template_id”: "1Bs2gZz5csQ1jLdCY8A3WHN9GtFaSYWJDK2S4FU2Fp",
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| template_id | string | 企业模板id，即将服务商模板复制/更新到企业后，企业审批应用中目标模板的id ，同一个服务商模板复制到不同企业，将获得不同的模板id |

### 响应示例

```json
{
    "template_id" : "open_Bs5M5HHaB8HiW9vs5MKHQRm7mhW1mv8UkptTJDeoc"
}
```

## 其他说明

### 企业内表现

调用接口后，在企业管理后台将出现“第三方模板”分类，分类下将显示服务商复制到企业的模板。 企业可查看审批申请记录、查看（不可编辑）表单控件、设置审批规则。企业设置的审批规则，仅在服务商调用“提交审批申请”接口，且使用“复用管理后台流程”模式时生效。

### 错误说明

| 错误码 | 说明 |
| --- | --- |
| 301025 | 参数错误，请求template_id非法 |
| 301026 | 通用错误码，拉取审批模板内部接口失败 |
