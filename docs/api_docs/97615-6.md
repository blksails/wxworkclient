# 停止发表企业朋友圈

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97615](https://developer.work.weixin.qq.com/document/path/97615)
- **文档 ID**: `97615`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/cancel_moment_task`

## 接口描述

企业和第三方应用可调用此接口，停止尚未发送的企业朋友圈发送任务。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| moment_id | string | 是 | 朋友圈id，可通过获取客户朋友圈企业发表的列表接口获取朋友圈企业发表的列表 |

### 请求示例

```json
{
	"moment_id":"momXXXXXXXXXX"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok"
}
```

## 其他说明

### 权限说明

企业需要使用配置到“可调用应用”列表中的自建应用secret所获取的accesstoken来调用。第三方应用或代开发自建应用调用需要企业授权客户朋友圈下发表到成员客户的朋友圈的权限

### 注意事项

注意，此接口无法撤回已经发表到客户朋友圈的信息。
