# 获取单位居民上报数据统计 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97154](https://developer.work.weixin.qq.com/document/path/97154)
- **文档 ID**: `97154`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/report/resident/get_corp_status?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/report/resident/get_corp_status?access_token=ACCESS_TOKEN

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。“获取access_token” |

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| grid_id | string | 否 | 网格id，不传的话获取整个企业的概况 |
| 自建应用 | 自建应用 | 否 | 配置到「居民上报 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"grid_id":"grid_id"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| processing |  | 办理中 |
| added_today |  | 今日上报 |
| solved_today |  | 今日办结 |
| pending |  | 待受理 |
| total_case |  | 累计上报 |
| total_accepted |  | 累计受理 |
| total_solved |  | 累计办结 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","processing":1,"added_today":1,"solved_today":1,"pending":10"total_case":1,"total_accepted":1,"total_solved":1,}
```
