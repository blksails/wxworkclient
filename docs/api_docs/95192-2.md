# 进入应用客服会话 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95192](https://developer.work.weixin.qq.com/document/path/95192)
- **文档 ID**: `95192`

## 接口描述

接口说明：第三方应用可以使用该接口提供按钮，让用户可快速打开应用客服的会话。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| ios |  | 否 | ≥3.1.18 |
| android |  | 否 | ≥3.1.18 |
| win |  | 否 | ≥4.1.6 |
| mac |  | 否 | ≥4.1.6 |
| 自建应用 | 自建应用 | 否 | 不支持 |
| 第三方应用 | 第三方应用 | 否 | 提前配置了客服 |

### 请求示例

```json
wx.qy.openThirdAppServiceChat({success:function(res){// 回调}});
```
