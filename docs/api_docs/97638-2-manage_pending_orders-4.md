# 管理待付款订单

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97638](https://developer.work.weixin.qq.com/document/path/97638)
- **文档 ID**: `97638`
- **API 名称**: `manage_pending_orders`
- **请求方法**: `POST`
- **接口地址**: `https://wework.qpic.cn/wwpic/800107_suC5mLegQB2MhRn_1669799699/0`
- **分组信息**: 第 2 个接口，共 10 个

## 接口描述

展示所有已创建的待支付订单，区分客户下单/服务商创建两个列表，服务商可点击查看订单详情并可获取支付二维码/链接发送给客户进行支付，针对客户下单的订单，服务商可进行改价，最低折扣为原价的7折。
