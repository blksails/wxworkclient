# 批量外呼 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98823](https://developer.work.weixin.qq.com/document/path/98823)
- **文档 ID**: `98823`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/phone/callout?access_token=ACCESS_TOKEN`

## 接口描述

电话入会支持成员通过手机或座机拨打对应号码进入会议，也可在会中输入电话号码呼叫其他成员加入，通话质量更稳定。此接口用于创建批量电话入会呼叫。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议ID |
| phone_numbers | object | 是 | 外呼的电话号码对象数组。详见PhoneNumber。 |
| area | int32 | 是 | 国家/地区代码。（例如：中国是86） |
| phone | string | 是 | 电话号码或固定电话总机号 |
| extension_number | string | 否 | 固定电话分机号 |

### 请求示例

```json
{"meetingid":"MEETINGID","phone_numbers":[{"area":86,"phone":"13333333333","extension_number":"12345678"}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| phone_numbers | object[] | 成功外呼的电话号码对象数组。详见SuccessPhoneNumber。 |
| invalid_phone_numbers | object[] | 不合法的外呼电话号码对象数组。详见PhoneNumber。 |
| area | int32 | 国家/地区代码。（例如：中国是86） |
| phone | string | 电话号码或固定电话总机号 |
| extension_number | string | 固定电话分机号 |
| status | string | 外呼状态 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","phone_numbers":[{"area":86,"phone":"13333333333","extension_number":"12345678","status":"STATUS"}],"invalid_phone_numbers":[{"area":86,"phone":"13333333333","extension_number":"12345678"}]}
```
