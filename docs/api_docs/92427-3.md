# 第三方小程序登录流程

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92427](https://developer.work.weixin.qq.com/document/path/92427)
- **文档 ID**: `92427`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wxqy/third_party_mini_program_login`

## 接口描述

当小程序在企业微信端运行时，需要通过对应的登录接口获取到当前企业微信用户在当前企业的员工身份信息。开发者需特别关注，当小程序在微信端运行时由微信派发和验证code参数，当小程序在企业微信端运行时由企业微信派发和验证code参数。两个平台的登录流程不同。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
