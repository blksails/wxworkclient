# 关于发送成功列表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96350](https://developer.work.weixin.qq.com/document/path/96350)
- **文档 ID**: `96350`
- **API 名称**: `获取朋友圈发表后的可见客户列表`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/get_success_customers`
- **分组信息**: 第 3 个接口，共 3 个

## 接口描述

获取某个成员发送成功客户列表

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| success_customers | string[] | 发送成功客户列表 |
