# 成员对外信息的添加方法

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96252](https://developer.work.weixin.qq.com/document/path/96252)
- **文档 ID**: `96252`
- **API 名称**: `添加对外成员属性`
- **请求方法**: `POST`
- **分组信息**: 第 3 个接口，共 4 个

## 接口描述

登录管理后台->外部联系人管理->成员对外信息，点击编辑来添加对外成员的属性，支持文本、链接、小程序类型
