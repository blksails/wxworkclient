# 获取空间信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97874](https://developer.work.weixin.qq.com/document/path/97874)
- **文档 ID**: `97874`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/space_info`

## 接口描述

该接口用于获取空间的成员列表、信息、权限等信息。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spaceid | string | 是 | 空间spaceid |

### 请求示例

```json
{
    "spaceid": "SPACEID"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| space_info | object | 空间信息 |
| space_info.spaceid | string | 空间spaceid |
| space_info.space_name | string | 空间名称 |
| space_info.auth_list | object[] | 空间成员列表 |
| space_info.auth_list.auth_info | object[] | 空间成员信息 |
| space_info.auth_list.auth_info.type | uint32 | 成员类型 1:个人 2:部门 |
| space_info.auth_list.auth_info.userid | string | 成员userid,字符串 |
| space_info.auth_list.auth_info.departmentid | uint32 | 部门departmentid, 32位整型范围是[0, 2^32) |
| space_info.auth_list.auth_info.auth | uint32 | 成员权限 1:仅下载 4:可预览 7:应用空间管理员 |
| space_info.auth_list.quit_userid | string[] | 空间无权限成员userid (成员在一个有权限的部门中, 自己退出空间或者被移除权限) |
| space_info.space_sub_type | uint32 | 空间类型 0:普通 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "space_info": {
        "spaceid": "SPACEID",
        "space_name": "SPACE_NAME",
        "auth_list": {
            "auth_info": [{
                "type": 1,
                "userid": "USERID1",
                "auth": 3
            }, {
                "type": 1,
                "userid": "USERID2",
                "auth": 2
            }, {
                "type": 2,
                "departmentid": DEPARTMENTID1,
                "auth": 1
            }],
            "quit_userid": ["USERID3","USERID4"]
        },
        "space_sub_type":0
    }
}
```
