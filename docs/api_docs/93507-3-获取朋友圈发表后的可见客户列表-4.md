# 关于发送成功列表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93507](https://developer.work.weixin.qq.com/document/path/93507)
- **文档 ID**: `93507`
- **API 名称**: `获取朋友圈发表后的可见客户列表`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/get_success_customer_list?access_token=ACCESS_TOKEN`
- **分组信息**: 第 3 个接口，共 3 个

## 接口描述

获取某个成员发送成功客户列表

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 接口调用凭证 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| success_customer_list | object[] | 发送成功客户列表 |

## 其他说明

### 关于发送成功列表

可通过接口获取朋友圈发表后的可见客户列表，获取某个成员发送成功客户列表。
