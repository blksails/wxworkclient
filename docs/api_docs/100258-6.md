# 取消订单

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100258](https://developer.work.weixin.qq.com/document/path/100258)
- **文档 ID**: `100258`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/advanced_api/cancel_order`

## 接口描述

取消「数据与智能专区」中的会话内容数据接口订单，只可取消未支付且未失效的订单。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 应用服务商的接口调用凭证，获取方法参见服务商的凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| order_id | string | 是 | 订单id |

### 请求示例

```json
{
	"order_id":"xxxxx"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok"
}
```
