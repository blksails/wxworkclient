# 提示原跟进成员和接替成员在最近一年内需要登录过至少一次企业微信。

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92125](https://developer.work.weixin.qq.com/document/path/92125)
- **文档 ID**: `92125`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/transfer_customer?access_token=ACCESS_TOKEN`

## 接口描述

企业可通过此接口，转接在职成员的客户给其他成员。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| handover_userid |  | 是 | 原跟进成员的userid |
| takeover_userid |  | 是 | 接替成员的userid |
| external_userid |  | 是 | 客户的external_userid列表，每次最多分配100个客户 |
| transfer_success_msg |  | 否 | 转移成功后发给客户的消息，最多200个字符，不填则使用默认文案 |

### 请求示例

```json
{"handover_userid":"zhangsan","takeover_userid":"lisi","external_userid":["woAJ2GCAAAXtWyujaWJHDDGi0mACAAAA","woAJ2GCAAAXtWyujaWJHDDGi0mACBBBB"],"transfer_success_msg":"您好，您的服务已升级，后续将由我的同事李四@腾讯接替我的工作，继续为您服务。"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| customer.external_userid |  | 客户的external_userid |
| customer.errcode |  | 对此客户进行分配的结果, 具体可参考全局错误码,0表示成功发起接替,待24小时后自动接替,并不代表最终接替成功 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","customer":[{"external_userid":"woAJ2GCAAAXtWyujaWJHDDGi0mACAAAA","errcode":40096},{"external_userid":"woAJ2GCAAAXtWyujaWJHDDGi0mACBBBB","errcode":0}]}
```
