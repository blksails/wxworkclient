# 会话内容导出完成通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99867](https://developer.work.weixin.qq.com/document/path/99867)
- **文档 ID**: `99867`

## 接口描述

当会话内容导出任务完成时，企业微信专区后台会向有企业会话内容权限的应用关联的程序推送相应的事件消息。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| event_type |  | 否 | 事件类型，此时固定为chat_archive_export_finished |

### 请求示例

```json
{"event_type":"chat_archive_export_finished","timestamp":1408091189,"chat_archive_export_finished":{"jobid":"xxx"}}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| jobid |  | 24小时内有效，用于调用获取会话内容导出任务结果，不超过64个字符 |
| timestamp |  | 时间戳 |

## 其他说明
