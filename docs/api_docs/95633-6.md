# 获取对接规则详情 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95633](https://developer.work.weixin.qq.com/document/path/95633)
- **文档 ID**: `95633`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/corpgroup/rule/get_rule_info?access_token=ACCESS_TOKEN`

## 接口描述

上下游系统应用可通过该接口获取企业上下游规则详情

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| chain_id |  | 是 | 上下游id |
| rule_id |  | 是 | 上下游规则id |
| 自建应用 | 自建应用 | 否 | 配置到「上下游- 可调用接口的应用」中 |

### 请求示例

```json
{"chain_id":"Chxxxxxx","rule_id":1}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| rule_info |  | 上下游关系规则的详情 |
| rule_info.owner_corp_range |  | 上游企业的对接人规则（下游企业可以看到并联系的成员或部门） |
| rule_info.owner_corp_range.departmentids |  | 部门id |
| rule_info.owner_corp_range.userids |  | 用户id |
| rule_info.member_corp_range |  | 下游企业规则范围 |
| rule_info.member_corp_range.groupids |  | 分组id |
| rule_info.member_corp_range.corpids |  | 企业id |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","rule_info":{"owner_corp_range":{"departmentids":["departmentid1","departmentid2"],"userids":["userid1","userid2"]},"member_corp_range":{"groupids":["groupid1","groupid2"],"corpids":["corpid1","corpid2"]}}}
```
