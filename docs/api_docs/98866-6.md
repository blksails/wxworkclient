# 批量删除布局 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98866](https://developer.work.weixin.qq.com/document/path/98866)
- **文档 ID**: `98866`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/advanced_layout/batch_delete?access_token=ACCESS_TOKEN`

## 接口描述

根据布局 ID 批量删除布局，可以删除基础布局和高级布局。正在被应用的布局无法删除，请先设置成其他布局或恢复成默认原始布局后再行删除。接口不做布局是否存在的校验，删除不存在的布局不会有提示。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议 ID |
| layout_id_list | string[] | 是 | 布局 ID 列表，要删除的一个或多个布局 ID（最多支持20个）。当前正在应用的布局不能被删除 |

### 请求示例

```json
{"meetingid":"MEETINGID","layout_id_list":["LAYOUTID1","LAYOUTID2"]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
