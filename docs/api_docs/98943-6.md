# 小程序接入对外收款

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98943](https://developer.work.weixin.qq.com/document/path/98943)
- **文档 ID**: `98943`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalpayment/access?access_token=ACCESS_TOKEN`

## 接口描述

在由企业微信向微信发出的小程序中使用对外收款账户进行收款。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| app_id | string | 是 | 小程序的 AppID |
| mchid | string | 是 | 商户号 |
| sub_mchid | string | 是 | 子商户号 |
| notify_url | string | 是 | 支付结果通知地址 |

### 请求示例

```json
{
  "app_id": "wx1234567890",
  "mchid": "123456",
  "sub_mchid": "789012",
  "notify_url": "https://example.com/callback"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |

### 响应示例

#### 示例 1: 成功响应

```json
{
  "errcode": 0,
  "errmsg": "ok"
}
```

#### 示例 2: 失败响应

```json
{
  "errcode": 40001,
  "errmsg": "invalid credentials"
}
```

## 其他说明

### 调用权限说明

调用的应用需要满足如下的权限：
- 自建应用：配置到「对外收款 - 可调用接口的应用」中
- 代开发应用：暂不支持
- 第三方应用：暂不支持

注：从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 [查看详情](https://developer.work.weixin.qq.com/document/51165)

### 开发前准备

1、在使用本接口前，需要在手机端或企业微信管理后台对外收款中申请收款账户；
2、在手机端「对外收款-小程序」中绑定需要接入的小程序和对外收款账户；
3、将要调用接口的自建应用配置到「对外收款 - 可调用接口的应用」中；
4、在企业微信管理后台为要调用接口的自建应用设置企业可信IP，仅企业可信IP可调用本目录相关接口；
5、在企业微信管理后台为要调用接口的自建应用设置接收事件服务器，用于接收支付和退款相关的回调通知；
