## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 自建应用 | 自建应用 | 否 | 需具备「数据与智能专区权限」 |
| 代开发应用 | 代开发应用 | 否 | 需具备「数据与智能专区权限」 |
| 第三方应用 | 第三方应用 | 否 | 需具备「数据与智能专区权限」 |
| program_id |  | 是 | 应用关联的程序id |
| ability_id |  | 是 | 程序关联的能力id |
| request_data |  | 是 | 请求的输入JSON，要求与配置的格式匹配 |
| jobid |  | 是 | 任务id |

### 请求示例

#### 示例 1: 请求包体：

```json
{"program_id":"xxx","ability_id":"xxx","request_data":"{\"input\":\"xxx\"}"}
```

#### 示例 2: 请求包体：

```json
{"jobid":"JOBID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 错误码 |
| errmsg |  | 错误码说明 |
| jobid |  | 任务id |
| errcode |  | 错误码 |
| errmsg |  | 错误码说明 |
| response_errcode |  | 上报异步任务结果中上报的errcode。代表专区程序返回的错误码 |
| response_data |  | 上报异步任务结果中上报的result。代表专区程序的输出结果，为自定义的JSON字符串，要求与管理端配置的输出协议格式匹配 |

### 响应示例

#### 示例 1: 返回结果：

```json
{"errcode":0,"errmsg":"ok","jobid":"JOBID"}
```

#### 示例 2: 返回结果：

```json
{"errcode":0,"errmsg":"ok","response_errcode":0,"response_data":"{\"output\":\"xxx\"}"}
```

## 其他说明

### 创建专区程序调用任务

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/chatdata/async_program_task?access_token=ACCESS_TOKEN
请求包体：
{"program_id":"xxx","ability_id":"xxx","request_data":"{\"input\":\"xxx\"}"}
参数说明：
参数是否必须说明program_id是应用关联的程序idability_id是程序关联的能力idrequest_data是请求的输入JSON，要求与配置的格式匹配
返回结果：
{"errcode":0,"errmsg":"ok","jobid":"JOBID"}
参数说明：
参数说明errcode错误码errmsg错误码说明jobid任务id

### 获取专区程序任务结果

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/chatdata/async_program_result?access_token=ACCESS_TOKEN
请求包体：
{"jobid":"JOBID"}
参数说明：
参数是否必须说明jobid是任务id
返回结果：
{"errcode":0,"errmsg":"ok","response_errcode":0,"response_data":"{\"output\":\"xxx\"}"}
参数说明：
参数说明errcode错误码errmsg错误码说明response_errcode上报异步任务结果中上报的errcode。代表专区程序返回的错误码response_data上报异步任务结果中上报的result。代表专区程序的输出结果，为自定义的JSON字符串，要求与管理端配置的输出协议格式匹配
