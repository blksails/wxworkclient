# ID转换接口

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99375](https://developer.work.weixin.qq.com/document/path/99375)
- **文档 ID**: `99375`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/finish_openid_migration?provider_access_token=ACCESS_TOKEN`

## 接口描述

服务商完成企业下所有第三方应用的新旧id（userid/corpid/external_userid)迁移后，即可主动将该企业设置为“迁移完成”。设置后，第三方应用获取到的是升级后的id。注意，该接口需要使用provider_access_token来调用。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 应用提供商的provider_access_token，获取方法参见服务商的凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| corpid | string | 是 | 企业corpid |
| openid_type | array | 是 | id类型：1-userid与corpid; 3-external_userid |

### 请求示例

```json
{
    "corpid":"xxxxx",
    "openid_type":[
        1,3
    ]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
  "errcode":0,
  "errmsg":"ok"
}
```

## 其他说明

### 参数说明

| 参数 | 必须 | 说明 |
| --- | --- | --- |
| provider_access_token | 是 | 应用提供商的provider_access_token，获取方法参见[服务商的凭证](#11791/%E6%9C%8D%E5%8A%A1%E5%95%86%E7%9A%84token) |
| corpid | 是 | 企业corpid |
| openid_type | 是 | id类型：1-userid与corpid; 3-external_userid |

### 权限说明

该企业授权了该服务商的第三方应用
