# 接口不足通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100306](https://developer.work.weixin.qq.com/document/path/100306)
- **文档 ID**: `100306`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webhook/callback`

## 接口描述

当企业授权范围人数超过服务商购买的高级接口账号数量时，企业微信后台会推送“接口不足通知”到服务商的系统事件接收URL。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ServiceCorpId | string | 是 | 服务商CorpID |
| InfoType | string | 是 | 事件类型，固定为advanced_api_insufficient |
| AuthCorpId | string | 是 | 客户企业CorpID |
| AdvanceApiType | string | 是 | 高级接口类型，1-会话内容数据接口 |
| ChatArchiveApi.Edition | string | 是 | 会话内容数据接口的账号版本。2-内外部会话，3-内外部会话及语音通话 |
| TimeStamp | string | 是 | 时间戳 |

### 请求示例

```xml
<xml>
	<ServiceCorpId><![CDATA[xxxx]]></ServiceCorpId>
	<InfoType><![CDATA[advanced_api_insufficient]]></InfoType>
	<AuthCorpId><![CDATA[yyyy]]></AuthCorpId>
	<AdvanceApiType><![CDATA[1]]></AdvanceApiType>
	<ChatArchiveApi>
		<Edition><![CDATA[2]]></Edition>
	</ChatArchiveApi>
	<TimeStamp>1403610513</TimeStamp>
</xml>
```
