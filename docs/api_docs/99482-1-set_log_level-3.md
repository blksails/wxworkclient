# 提交申请

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99482](https://developer.work.weixin.qq.com/document/path/99482)
- **文档 ID**: `99482`
- **API 名称**: `set_log_level`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/set_log_level?access_token=ACCESS_TOKEN`
- **分组信息**: 第 1 个接口，共 2 个

## 接口描述

注册成为企业微信服务商，在「服务商助手-工具-通用配置-获客助手组件」处提交申请。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |
