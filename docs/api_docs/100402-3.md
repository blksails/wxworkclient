## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
|  | string | 是 | 需要识别的音频的本地 ID通过 stopRecord 接口获得，音频时长不能超过 60 秒 |
|  | boolean | 否 | 是否显示进度提示 |
|  | Function | 否 | 成功回调 |
|  | Function | 否 | 失败回调 |
|  | Function | 否 | 取消回调 |
|  | Function | 否 | 完成回调 |

### 请求示例

#### 示例 1: 功能描述

```json
ww.translateVoice({localId:localId,isShowProgressTips:true})
```

#### 示例 2: 示例代码

```json
wx.translateVoice({localId:'',// 需要识别的音频的本地Id，由录音相关接口获得，音频时长不能超过60秒isShowProgressTips:1,// 默认为1，显示进度提示success:function(res){alert(res.translateResult);// 语音识别的结果}});
```

## 其他说明

### ww.translateVoice(params)

兼容性：企业微信 iOS、Android 2.7.5 开始支持
功能描述语音转文字。
ww.translateVoice({localId:localId,isShowProgressTips:true})
参数说明params: Object属性类型默认值必填说明localIdstring是需要识别的音频的本地 ID通过 stopRecord 接口获得，音频时长不能超过 60 秒isShowProgressTipsbooleantrue否是否显示进度提示successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码translateResultstring是识别结果

### 功能描述

语音转文字。

### params: Object

属性类型默认值必填说明localIdstring是需要识别的音频的本地 ID通过 stopRecord 接口获得，音频时长不能超过 60 秒isShowProgressTipsbooleantrue否是否显示进度提示successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码translateResultstring是识别结果

wx.translateVoice({localId:'',// 需要识别的音频的本地Id，由录音相关接口获得，音频时长不能超过60秒isShowProgressTips:1,// 默认为1，显示进度提示success:function(res){alert(res.translateResult);// 语音识别的结果}});
