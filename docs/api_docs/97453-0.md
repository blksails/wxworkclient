## 其他说明

### 接口预定会议表现

接口创建的会议，将在会议详情标识出应用创建的信息通过接口创建、修改、删除会议后，会议应用将自动向会议相关成员推送通知。

### 配置可使用会议的应用

应用在调用会议接口前，需要先获得会议的使用权限。自建应用登录企业微信管理端，进入「协作」-「会议」-「API」，配置「可调用接口的应用」。第三方应用第三方服务商创建应用时，需要开启「会议接口权限」。企业授权应用之后，应用即拥有会议权限。代开发应用服务商为企业配置代开发应用时，需要开启「会议接口权限」，企业管理员确认之后，应用即拥有会议权限。

### 注意事项

企业微信需要更新到4.0.20及以上版本应用仅可对本应用创建的会议进行操作预约会议高级管理中以下3个接口与预约会议基础管理的接口名相同，但拥有更丰富的参数字段创建预约会议修改预约会议获取会议详情预约会议基础管理接口创建的会议，不可与预约会议高级管理接口混用。高级管理接口创建的会议可与基础管理接口混用
