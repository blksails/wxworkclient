# 退款结果通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97197](https://developer.work.weixin.qq.com/document/path/97197)
- **文档 ID**: `97197`

## 接口描述

当服务商提交退款申请的订单发生状态变更时，企业微信后台会推送“退款结果通知”到服务商的系统事件接收URL（应用管理 -通用开发参数-系统事件接收URL）。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ServiceCorpId |  | 否 | 服务商CorpID |
| InfoType |  | 否 | 事件类型，此时固定为license_refund |
| AuthCorpId |  | 否 | 客户企业CorpID |
| OrderId |  | 否 | 订单号 |
| OrderStatus |  | 否 | 订单状态，1:退款成功，2:退款被拒绝。 |
| TimeStamp |  | 否 | 时间戳 |

### 请求示例

```xml
<xml><ServiceCorpId><![CDATA[xxxx]]></ServiceCorpId><InfoType><![CDATA[license_refund]]></InfoType><AuthCorpId><![CDATA[yyyy]]></AuthCorpId><OrderId><![CDATA[zzzz]]></OrderId><OrderStatus>1</OrderStatus><TimeStamp>1403610513</TimeStamp></xml>
```
