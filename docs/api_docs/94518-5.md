# 打开已有群聊并可选发送一条链接消息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94518](https://developer.work.weixin.qq.com/document/path/94518)
- **文档 ID**: `94518`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

兼容性：企业微信 3.1.8 开始支持。打开已有群聊并可选发送一条链接消息（link消息）。支持打开企业内部群、外部群、互联群。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| params.chatId | string | 是 | 打开的群聊 ID |
| params.msg | Object | 否 | 发送的链接消息。不传则表示仅进入群聊，不发消息。 |
| params.msg.msgtype | string | 是 | 消息类型 |
| params.msg.link | Object | 是 |  |
| params.msg.link.title | string | 否 | H5消息标题 |
| params.msg.link.imgUrl | string | 否 | H5消息封面图片URL |
| params.msg.link.desc | string | 否 | H5消息摘要 |
| params.msg.link.url | string | 是 | H5消息页面url |
| params.success | Function | 否 | 成功回调 |
| params.fail | Function | 否 | 失败回调 |
| params.cancel | Function | 否 | 取消回调 |
| params.complete | Function | 否 | 完成回调 |

### 请求示例

```text
ww.openExistedChatWithMsg({
  chatId: 'chatId123',
  msg: {
    msgtype: 'link',
    link: {
      title: 'title1',
      desc: 'desc1',
      url: 'link1',
      imgUrl: 'imgurl1'
    }
  }
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| Promise<Object>.errMsg | string | 通用错误信息 |
| Promise<Object>.errCode | number | 通用错误码 |

## 其他说明

### 功能描述

打开已有群聊并可选发送一条链接消息（link消息）。支持打开企业内部群、外部群、互联群。

### 使用限制

本接口必须使用应用身份进行注册

### 错误信息

| errMsg | 说明 |
| --- | --- |
| openExistedChatWithMsg:ok | 执行成功 |
| openExistedChatWithMsg:fail_unsupported_msgtype | msgtype不合法 |
| openExistedChatWithMsg:fail_msg_link_missing_url | msg.link.url未传入 |
