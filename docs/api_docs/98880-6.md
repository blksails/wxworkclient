# 导入网络研讨会报名信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98880](https://developer.work.weixin.qq.com/document/path/98880)
- **文档 ID**: `98880`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/webinar/enroll/import?access_token=ACCESS_TOKEN`

## 接口描述

在指定网络研讨会（Webinar）中导入报名信息。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| enroll_list.userid | string | 否 | (从请求示例中推测) |
| enroll_list.area | string | 否 | (从请求示例中推测) |
| enroll_list.phone_number | string | 否 | (从请求示例中推测) |
| enroll_list.nick_name | string | 否 | (从请求示例中推测) |

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| meetingid | string | 是 | 网络研讨会ID |
| enroll_list | object[] | 是 | 报名成员列表。详见EnrollRequest。 |
| userid | string | 否 | 报名成员 userid |
| area | string | 否 | 国家/地区代码。若使用 phone_number 的方式导入，该字段必填（例如：中国传86，不是+86）。 |
| phone_number | string | 否 | 手机号。 |
| nick_name | string | 否 | 报名的昵称。此处的修改不会影响会中展示的昵称，仅展示在导出的报名信息。 |

### 请求示例

```json
{"meetingid":"meetingid11234","enroll_list":[{"userid":"USERID","area":"86","phone_number":"13333333333","nick_name":"NICKNAME"}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| total_count | uint32 | 成功导入的报名信息条数 |
| enroll_list | object[] | 报名成员列表。详见EnrollResponse。 |
| enroll_id | string | 报名ID |
| userid | string | 报名成员 userid |
| area | string | 国家/地区代码。若使用 phone_number 的方式导入，该字段必填（例如：中国传86，不是+86）。 |
| phone_number | string | 手机号。 |
| nick_name | string | 报名的昵称。 |
| enroll_code | string | 报名码，用于电话入会的凭证。 |
| enroll_list.enroll_id | string | (从响应示例中推测) |
| enroll_list.enroll_code | string | (从响应示例中推测) |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","total_count":1,"enroll_list":[{"enroll_id":"123456","userid":"USERID","area":"86","phone_number":"13333333333","nick_name":"NICKNAME","enroll_code":"CODE"}]}
```
