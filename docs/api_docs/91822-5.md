# 打开个人信息页接口

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91822](https://developer.work.weixin.qq.com/document/path/91822)
- **文档 ID**: `91822`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

唤起成员或外部联系人的个人信息页面。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| params.type | int32 | 是 | 用户类型。合法值：1 - 企业成员，2 - 外部联系人 |
| params.userid | string | 是 | 企业成员或外部联系人的用户 ID |
| params.success | Function | 否 | 成功回调 |
| params.fail | Function | 否 | 失败回调 |
| params.cancel | Function | 否 | 取消回调 |
| params.complete | Function | 否 | 完成回调 |

### 请求示例

```text
ww.openUserProfile({
  type: 1,
  userid: 'wmEAlECwAAHrbWYDetiu3Af13xlYDAAA'
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errMsg | string | 通用错误信息 |
| errCode | int32 | 通用错误码 |

## 其他说明

### 功能描述

唤起成员或外部联系人的个人信息页面。

### 使用限制

第三方应用调用，需使用应用签名（agentConfig）进行注册，否则调用时会报“no permission”错误
