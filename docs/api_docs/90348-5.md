# 删除标签

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90348](https://developer.work.weixin.qq.com/document/path/90348)
- **文档 ID**: `90348`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/tag/delete`

## 接口描述

删除标签

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| tagid | string | 是 | 标签ID |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "deleted"
}
```

## 其他说明

### 权限说明

调用的应用必顡是指定标签的创建者。
