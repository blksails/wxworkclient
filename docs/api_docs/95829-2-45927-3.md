# 获取上下游互联群ID

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95829](https://developer.work.weixin.qq.com/document/path/95829)
- **文档 ID**: `95829`
- **API 名称**: `45927`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/get_interconnected_group_id?access_token=ACCESS_TOKEN`
- **分组信息**: 第 2 个接口，共 3 个

## 接口描述

获取上下游互联群的ID。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| group_id | string | 上下游互联群的ID |
