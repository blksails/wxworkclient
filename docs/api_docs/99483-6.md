# 查询链接使用详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99483](https://developer.work.weixin.qq.com/document/path/99483)
- **文档 ID**: `99483`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/customer_acquisition/statistic`

## 接口描述

服务商可通过此接口查询指定组件授权的获客链接在指定时间范围内的访问情况。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| link_id | string | 是 | 获客链接的id |
| start_time | uint32 | 是 | 统计起始时间戳 |
| end_time | uint32 | 是 | 统计结束时间戳 |

### 请求示例

```json
{
   "link_id":"caxxxxxxx",
   "start_time":1688140800,
   "end_time":1688486400
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| click_link_customer_cnt | uint32 | 点击链接客户数 |
| new_customer_cnt | uint32 | 新增客户数 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "ok",
   "click_link_customer_cnt":1000,
   "new_customer_cnt":500
}
```

## 其他说明

### 参数说明

1.统计范围的最小粒度为日，将自动转换为时间戳所在日进行统计，区间为闭区间。
2.仅可查询最近180天内的使用记录，起始和结束时间相差不可超过30天
3.使用的链接需要携带组件的comp_scene参数才会计算为组件的数据

### 权限说明

获客助手组件调用，仅可查询授权给获客助手组件的链接
