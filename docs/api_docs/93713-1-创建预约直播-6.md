# API

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93713](https://developer.work.weixin.qq.com/document/path/93713)
- **文档 ID**: `93713`
- **API 名称**: `创建预约直播`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/live/create`
- **分组信息**: 第 1 个接口，共 8 个

## 接口描述

创建预约直播接口用于预约一个直播会议。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| meeting_title | string | 是 | 直播主题 |
| start_time | uint32 | 是 | 直播开始时间 |
| end_time | uint32 | 是 | 直播结束时间 |
| anchor_userid | string | 是 | 主播的userid |

### 请求示例

```json
{
  "meeting_title": "Weekly Meeting",
  "start_time": 1608313200,
  "end_time": 1608316800,
  "anchor_userid": "zhangsan"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| meeting_id | string | 直播ID |

### 响应示例

```json
{
  "errcode": 0,
  "errmsg": "ok",
  "meeting_id": "123456"
}
```
