## 其他说明

### ww.onWifiConnected(callback)

兼容性：企业微信 iOS、Android 2.4.16 开始支持
功能描述监听 Wi-Fi 连接成功。
ww.onWifiConnected(function(event){console.log(event)})
参数说明callback: Function参数event: Object属性类型必填说明wifiObject是Wi-Fi 信息属性类型必填说明SSIDstring是Wi-Fi 的 SSIDBSSIDstring是Wi-Fi 的 BSSIDsecureboolean是Wi-Fi 是否安全signalStrengthnumber是Wi-Fi 信号强度

### 功能描述

监听 Wi-Fi 连接成功。

参数

属性类型必填说明wifiObject是Wi-Fi 信息属性类型必填说明SSIDstring是Wi-Fi 的 SSIDBSSIDstring是Wi-Fi 的 BSSIDsecureboolean是Wi-Fi 是否安全signalStrengthnumber是Wi-Fi 信号强度
