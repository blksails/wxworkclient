# 邮箱获取userid - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95895](https://developer.work.weixin.qq.com/document/path/95895)
- **文档 ID**: `95895`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/user/get_userid_by_email?access_token=ACCESS_TOKEN`

## 接口描述

通过邮箱获取其所对应的userid。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证，授权企业的token（通过获取企业凭证获取）或上游获取的下游企业的token（通过获取下级/下游企业的access_token获取） |
| email |  | 是 | 邮箱 |
| email_type |  | 否 | 邮箱类型：1-企业邮箱（默认）；2-个人邮箱 |

### 请求示例

```json
{"email":"12345@qq.com","email_type":1}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| userid |  | 成员UserID。注意：已升级openid的代开发或第三方，获取的是密文userid |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","userid":"zhangsan"}
```
