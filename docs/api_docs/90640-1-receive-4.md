# 新增部门事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90640](https://developer.work.weixin.qq.com/document/path/90640)
- **文档 ID**: `90640`
- **API 名称**: `receive`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`
- **分组信息**: 第 1 个接口，共 3 个

## 接口描述

新增部门事件触发时的回调通知。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 | 消息签名 |
| timestamp | uint32 | 是 | 时间戳 |
| nonce | uint32 | 是 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 第三方应用ID |
| AuthCorpId | string | 是 | 授权企业的CorpID |
| InfoType | string | 是 | 固定为change_contact |
| TimeStamp | uint32 | 是 | 时间戳 |
| ChangeType | string | 是 | 固定为create_party |
| Id | uint32 | 是 | 部门Id |
| Name | string | 否 | 部门名称，部分情况下不返回 |
| ParentId | uint32 | 是 | 父部门id |
| Order | uint32 | 是 | 部门排序 |
