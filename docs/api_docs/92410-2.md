# wx.qy.startNFCReader - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92410](https://developer.work.weixin.qq.com/document/path/92410)
- **文档 ID**: `92410`

## 接口描述

打开 NFC 模块，仅支持安卓系统

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | function | 否 | 接口调用成功的回调函数 |
| fail | function | 否 | 接口调用失败的回调函数 |
| complete | function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| 0 |  | 否 | 正常 |
| 13000 |  | 否 | 当前设备不支持NFC |
| 13001 |  | 否 | 当前设备支持NFC，但系统NFC开关未开启 |

### 请求示例

```json
wx.qy.startNFCReader({success:(res)=>{wx.showModal({title:'NFC-Start',content:JSON.stringify(res),})},fail:(res)=>{console.log(JSON.stringify(res))}})
```
