# 下载微盘文件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98021](https://developer.work.weixin.qq.com/document/path/98021)
- **文档 ID**: `98021`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/oa/journal/download_wedrive_file?access_token=ACCESS_TOKEN`

## 接口描述

该接口仅可用于下载汇报中关联的微盘文件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| journaluuid | string | 是 | 汇报记录id |
| fileid | string | 是 | 微盘fileid。获取汇报记录详情返回的微盘附件fileid。必须是journaluuid对应的汇报关联的wedrive_files。 |
| 自建应用 | 自建应用 | 否 | 需要配置到「汇报 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"journaluuid":"JOURNAL_UUID","fileid":"FILEID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| download_url | string | 下载请求url (有效期2个小时) |
| cookie_name | string | 下载请求带cookie的key |
| cookie_value | string | 下载请求带cookie的value |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","download_url":"DOWNLOAD_URL","cookie_name":"COOKIE_NAME","cookie_value":"COOKIE_VALUE"}
```
