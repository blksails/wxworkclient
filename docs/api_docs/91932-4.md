# 退款通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91932](https://developer.work.weixin.qq.com/document/path/91932)
- **文档 ID**: `91932`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`

## 接口描述

当某个客户发起有效的退款，经服务商在管理端同意后（或过期自动完成退款），会触发回调事件给服务商，服务商收到退款事件后，需自行通过相关接口重新获取这个客户最新的购买状态

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 | 消息签名 |
| timestamp | string | 是 | 时间戳 |
| nonce | string | 是 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 第三方应用的SuiteId |
| PaidCorpId | string | 是 | 购买方corpid |
| InfoType | string | 是 | 固定为refund |
| TimeStamp | string | 是 | 时间戳 |
| OrderId | string | 是 | 订单号，付费订单的唯一标志。服务商可据此拉取购买信息。该OrderId由企业微信生成，不超过32个字符 |

### 请求示例

```xml
<xml>
    <SuiteId><![CDATA[ww4asffe99e54c0aaa]]></SuiteId>
    <PaidCorpId><![CDATA[wxf8b4f85f3a794aaa]]></PaidCorpId>
    <InfoType><![CDATA[refund]]></InfoType>
    <TimeStamp>1403610513</TimeStamp>
    <OrderId><![CDATA[ORDERID]]></OrderId>
</xml>
```
