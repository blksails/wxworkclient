# 获取Rooms会议室列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98795](https://developer.work.weixin.qq.com/document/path/98795)
- **文档 ID**: `98795`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/rooms/list?access_token=ACCESS_TOKEN`

## 接口描述

获取企业下的Rooms会议室列表。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meeting_room_name | string | 否 | Rooms会议室名称（支持模糊匹配查找） |
| cursor | string | 否 | 分页查询用，将上一个请求返回的next_cursor字段传入。第一次查询时可不传值 |
| limit | uint32 | 否 | 分页大小，从1开始，最大50，默认20 |

### 请求示例

```json
{"meeting_room_name":"ROOMNAME","cursor":"CURSOR","limit":50}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| has_more | bool | 是否还有更多Rooms会议室列表 |
| next_cursor | string | 分页用，has_more为true时，下一次拉取列表将该字段填入cursor字段 |
| meeting_room_list | object[] | Rooms会议室对象列表，详见MeetingRoom |
| meeting_room_id | string | Rooms会议室 ID |
| meeting_room_name | string | Rooms会议室名称 |
| meeting_room_location | string | Rooms会议室地址 |
| account_type | uint32 | 账号类型：0：普通1：专款2：试用 |
| active_code | string | 激活码 |
| participant_number | uint32 | 容纳人数 |
| meeting_room_status | uint32 | Rooms会议室状态：0：未激活1：未绑定2：空闲3：使用中4：离线 |
| scheduled_status | uint32 | 预定状态：0：未开放预定1：开放预定 |
| is_allow_call | bool | 是否允许被呼叫：true：是false：否 |
