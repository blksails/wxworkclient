# 从企业微信管理端单点登录

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98170](https://developer.work.weixin.qq.com/document/path/98170)
- **文档 ID**: `98170`
- **API 名称**: `admin_sso`
- **请求方法**: `GET`
- **接口地址**: `https://open.work.weixin.qq.com/wwopen/developer`
- **分组信息**: 第 2 个接口，共 2 个

## 接口描述

企业微信管理员可从第三方应用的‘业务设置’入口跳转到第三方网站，流程是：企业 -> 企业微信管理端 -> 第三方服务商网站 -> 企业微信API

## 请求信息

### 请求示例

```text
1. 管理员登录企业微信管理端，点击应用中的“业务设置”
2. 跳转到第三方服务商的业务设置URL，服务商据此得到登录授权码
3. 利用登录授权码调用相关API
```

## 其他说明

### 注意事项

1. 使用该功能之前，服务商需要在第三方应用设置-“业务设置”，设置链接。
2. 若服务商未开启登录授权应用，仍然使用旧的服务商单点登录，则仍然需要通过旧接口换取登录授权信息。若服务商开启登录授权应用后，可切换为获取登录用户身份接口。
