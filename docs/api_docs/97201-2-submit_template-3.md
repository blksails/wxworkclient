# 提交代开发模版上线

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97201](https://developer.work.weixin.qq.com/document/path/97201)
- **文档 ID**: `97201`
- **API 名称**: `submit_template`
- **请求方法**: `POST`
- **接口地址**: `https://wework.qpic.cn/wwpic/73709_vJ6u4-QPQj6DB1E_1626354280/0`
- **分组信息**: 第 2 个接口，共 3 个

## 接口描述

提交代开发模版上线审核，审核通过后手动提交上线。

## 请求信息

### 请求示例

```text
POST https://wework.qpic.cn/wwpic/73709_vJ6u4-QPQj6DB1E_1626354280/0
```

## 其他说明

### 提交代开发模版上线流程

1. 点击提交上线按钮，选择需要提交上线审核的代开发模版
2. 提交后可在代开发应用上线列表中，查看模版上线审核单
3. 审核通过后，审核状态将变为“待上线”，进入审核详情页，点击“提交上线”
