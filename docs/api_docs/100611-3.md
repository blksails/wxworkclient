## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
|  | string | 是 | 需要下载的图片的服务器端 ID由 uploadImage 接口获得 |
|  | boolean | 否 | 显示进度提示 |
|  | Function | 否 | 成功回调 |
|  | Function | 否 | 失败回调 |
|  | Function | 否 | 取消回调 |
|  | Function | 否 | 完成回调 |

### 请求示例

#### 示例 1: 功能描述

```json
ww.downloadImage({serverId:serverId,isShowProgressTips:true})
```

#### 示例 2: 示例代码

```json
wx.downloadImage({serverId:'',// 需要下载的图片的服务器端ID，由uploadImage接口获得isShowProgressTips:1,// 默认为1，显示进度提示success:function(res){varlocalId=res.localId;// 返回图片下载后的本地ID}});
```

## 其他说明

### ww.downloadImage(params)

功能描述下载图片。
ww.downloadImage({serverId:serverId,isShowProgressTips:true})
参数说明params: Object属性类型默认值必填说明serverIdstring是需要下载的图片的服务器端 ID由 uploadImage 接口获得isShowProgressTipsbooleantrue否显示进度提示successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码localIdstring是返回图片下载后的本地 ID

### 功能描述

下载图片。

### params: Object

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码localIdstring是返回图片下载后的本地 ID
