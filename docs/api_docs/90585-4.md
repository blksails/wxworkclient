# 注册完成回调事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90585](https://developer.work.weixin.qq.com/document/path/90585)
- **文档 ID**: `90585`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/service/receive`

## 接口描述

推送到服务商设置的接收事件回调url上

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 | 消息签名 |
| timestamp | string | 是 | 时间戳 |
| nonce | string | 是 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ServiceCorpId | string | 是 | 服务商corpid |
| InfoType | string | 是 | 事件类型，固定为register_corp，注册回调事件 |
| TimeStamp | string | 是 | 回调事件的时间戳 |
| RegisterCode | string | 是 | 创建企业对应的注册码 |
| AuthCorpId | string | 是 | 注册成功的企业corpid |
| ContactSync | object | 否 | 通讯录迁移的凭证信息。仅当注册推广包开启通讯录迁移接口时返回该参数 |
| AuthUserInfo | object | 是 | 授权管理员的信息 |
| State | string | 否 | 用户自定义的状态值，参数值由接口 获取注册码 指定。若未指定，则无该字段 |
| TemplateId | string | 是 | 推广包ID |
