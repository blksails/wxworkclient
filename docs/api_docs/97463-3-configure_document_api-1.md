# 配置可调用文档接口的应用

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97463](https://developer.work.weixin.qq.com/document/path/97463)
- **文档 ID**: `97463`
- **API 名称**: `configure_document_api`
- **请求方法**: `POST`
- **分组信息**: 第 3 个接口，共 3 个

## 接口描述

应用在调用文档接口前，需要先获得文档的使用权限。自建应用、第三方应用和代开发应用都需要配置相应的权限。
