### 功能描述

发表内容到客户朋友圈。

### 使用限制

本接口必须使用应用身份进行注册应用需有客户联系功能权限当前成员必须配置了客户联系功能当前成员必须在客户朋友圈使用范围当前成员必须具备外部沟通管理成员使用权限

### params: Object

属性类型必填说明textObject否文本消息属性类型必填说明contentstring否消息文本内容使用限制最多支持 2000 字attachmentsObject[]否附件使用限制最多支持 9 张图片、1 个视频或 1 个链接可能为以下类型属性类型必填说明msgtype"image"是消息类型imageObject是属性类型必填说明mediaidstring否图片的素材 ID可以通过素材管理接口获得，暂不支持公众平台的 mediaidimgUrlstring否图片的 url，跟图片素材 ID 填其中一个即可或属性类型必填说明msgtype"link"是消息类型linkObject是属性类型必填说明titlestring否H5消息标题imgUrlstring否H5消息封面图片URLdescstring否H5消息摘要urlstring是H5消息页面url或属性类型必填说明msgtype"video"是消息类型videoObject是属性类型必填说明mediaidstring是视频的素材 ID可以通过素材管理接口获得，暂不支持公众平台的 mediaidsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码
