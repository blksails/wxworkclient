# 修改日历事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97808](https://developer.work.weixin.qq.com/document/path/97808)
- **文档 ID**: `97808`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/calendar/modify_calendar?access_token=ACCESS_TOKEN`

## 接口描述

当日历管理员修改了API创建的日历时，触发该事件。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName | string | 是 | 企业微信CorpID |
| FromUserName | string | 是 | 成员UserID |
| CreateTime | uint32 | 是 | 消息创建时间，unix时间戳 |
| MsgType | string | 是 | 消息类型，固定为：`event` |
| Event | string | 是 | 事件类型，固定为：`modify_calendar` |
| CalId | string | 是 | 日历ID |

### 请求示例

```xml
<xml>
   <ToUserName><![CDATA[toUser]]></ToUserName>
   <FromUserName><![CDATA[fromUser]]></FromUserName>
   <CreateTime>1348831860</CreateTime>
   <MsgType><![CDATA[event]]></MsgType>
   <Event><![CDATA[modify_calendar]]></Event>
   <CalId><![CDATA[wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA]]></CalId>
</xml>
```
