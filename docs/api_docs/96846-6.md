# 添加成员/部门 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96846](https://developer.work.weixin.qq.com/document/path/96846)
- **文档 ID**: `96846`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/space_acl_add?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于对指定空间添加成员/部门，可一次性添加多个。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spaceid | string | 是 | 空间spaceid |
| auth_info | obj[] | 是 | 被添加的空间成员信息 |
| type | uint32 | 是 | 成员类型 1:个人 2:部门 |
| userid | string | 是 | 成员userid,字符串 (type为1时填写) |
| departmentid | uint32 | 是 | 部门departmentid, 32位整型范围是[0, 2^32) (type为2时填写) |
| auth | uint32 | 是 | 1:仅下载 7:应用空间管理员(连同已经设置的管理员，最多可指定三个,不支持设置部门) |

### 请求示例

```json
{"spaceid":"SPACEID","auth_info":[{"type":1,"userid":"USERID1","auth":7},{"type":2,"departmentid":1,"auth":1}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
