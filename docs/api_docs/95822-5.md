# 移除企业 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95822](https://developer.work.weixin.qq.com/document/path/95822)
- **文档 ID**: `95822`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/corpgroup/corp/remove_corp?access_token=ACCESS_TOKEN`

## 接口描述

上级/上游企业通过该接口移除下游企业

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证，上游企业应用access_token |
| chain_id |  | 是 | 上下游id |
| corpid |  | 否 | 需要移除的下游企业corpid |
| pending_corpid |  | 否 | 需要移除的未加入下游企业corpid，corpid和pending_corpid至少填一个，都填corpid生效 |
| 自建应用 | 自建应用 | 否 | 配置到「上下游- 可调用接口的应用」中 |

### 请求示例

```json
{"chain_id":"xxxx","corpid":"xxxxx","pending_corpid":"xxxx"}
```

## 响应信息

### 响应示例

```json
{"errcode":0,"errmsg":"ok",}
```
