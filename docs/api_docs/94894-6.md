# 获取选人ticket对应的用户

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94894](https://developer.work.weixin.qq.com/document/path/94894)
- **文档 ID**: `94894`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/user/list_selected_ticket_user`

## 接口描述

当企业以成员授权的方式安装了第三方应用，成员在企业微信终端通过选人jsapi选择通讯录，应用获取到selectedTicket后，可调用该接口获取SelectedTicket对应的用户open_userid列表。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| selected_ticket | string | 是 | 选人jsapi返回的selectedTicket |

### 请求示例

```json
{
	"selected_ticket": "xxxx"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| operator_open_userid | string | 选人用户的open_userid |
| open_userid_list | array | 此次选人操作中，在应用可见范围内的open_userid列表 |
| open_userid_list[]. | string | open_userid列表中的元素 |
| unauth_open_userid_list | array | 此次选人操作中，不在应用可见范围内的open_userid列表 |
| unauth_open_userid_list[]. | string | 不在应用可见范围内的open_userid列表中的元素 |
| total | int32 | 用户选择的总人数 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"operator_open_userid":"xxx",
	"open_userid_list":["xxxx","xxxx"],
	"unauth_open_userid_list":["yyyy", "yyy"],
	"total":10
}
```
