# 取消高级功能账号 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99317](https://developer.work.weixin.qq.com/document/path/99317)
- **文档 ID**: `99317`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/exmail/vip/batch_del?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于撤销分配应用可见范围的企业成员的高级功能。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| userid_list |  | 是 | 要撤销分配高级功能的企业成员userid列表，单次操作最多限制100个 |

### 请求示例

```json
{"userid_list":["zhangsan","lisi","wangwu"]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| succ_userid_list |  | 撤销分配成功的用户列表 |
| fail_userid_list |  | 撤销分配失败的用户列表 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","succ_userid_list":["zhangsan","lisi"],"fail_userid_list":["wangwu"]}
```
