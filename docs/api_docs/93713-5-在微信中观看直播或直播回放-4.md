# API

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93713](https://developer.work.weixin.qq.com/document/path/93713)
- **文档 ID**: `93713`
- **API 名称**: `在微信中观看直播或直播回放`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/live/watch`
- **分组信息**: 第 5 个接口，共 8 个

## 接口描述

在微信中观看直播或直播回放接口用于在微信中观看直播或直播回放。
