### ww.getBeacons([params])

兼容性：企业微信
功能描述获取所有已搜索到的 iBeacon 设备。
ww.getBeacons()
参数说明params: Object属性类型必填说明successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码beaconsObject[]是iBeacon 设备列表属性类型必填说明uuidstring是iBeacon 设备广播的 uuidmajorstring是iBeacon 设备的主 idminorstring是iBeacon 设备的次 idproximitynumber是表示设备距离的枚举值, 0 信号太弱不足以计算距离, 或非 iOS 设备;	1	十分近; 2	比较近; 3	远;accuracynumber是iBeacon 设备的距离rssinumber是表示设备的信号强度

### 功能描述

获取所有已搜索到的 iBeacon 设备。

### params: Object

属性类型必填说明successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
