# 配置调试参数

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100082](https://developer.work.weixin.qq.com/document/path/100082)
- **文档 ID**: `100082`
- **API 名称**: `设置接收事件服务器`
- **请求方法**: `GET`
- **接口地址**: `https://wework.qpic.cn/wwpic3az/316719_cIpPbRmnQyyH9Z-_1721652044/0`
- **分组信息**: 第 2 个接口，共 6 个

## 接口描述

点击「接收事件服务器-设置」，可以设置部署了调试程序的URL地址。开启调试模式后，通过应用同步调用专区程序或者应用异步调用专区程序主动发起的程序调用，以及专区程序接收事件通知的回调，都会转发到此URL。
