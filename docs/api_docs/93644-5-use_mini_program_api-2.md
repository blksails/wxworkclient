# 使用小程序接口

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93644](https://developer.work.weixin.qq.com/document/path/93644)
- **文档 ID**: `93644`
- **API 名称**: `use_mini_program_api`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/use_mini_program_api?access_token=ACCESS_TOKEN`
- **分组信息**: 第 5 个接口，共 6 个

## 接口描述

与普通企业小程序登录流程的不同点在于：需要使用获取下级企业的小程序session来转为下级企业的登录session。
