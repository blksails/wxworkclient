# 接口购买到期通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100269](https://developer.work.weixin.qq.com/document/path/100269)
- **文档 ID**: `100269`

## 接口描述

当服务商购买的高级接口账号过期时，企业微信后台会推送“接口购买到期通知”到服务商的系统事件接收URL（应用管理 -通用开发参数-系统事件接收URL）。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ServiceCorpId |  | 否 | 服务商CorpID |
| InfoType |  | 否 | 事件类型，此时固定为advanced_api_expired |
| AuthCorpId |  | 否 | 客户企业CorpID |
| AdvanceApiType |  | 否 | 高级接口类型，1-会话内容数据接口 |
| ChatArchiveApi |  | 否 | 会话内容数据接口相关信息，详见ChatArchiveApi |
| TimeStamp |  | 否 | 时间戳 |
| Edition |  | 否 | 到期的账号版本。2-内外部会话，3-内外部会话及语音通话 |

### 请求示例

```xml
<xml><ServiceCorpId><![CDATA[xxxx]]></ServiceCorpId><InfoType><![CDATA[advanced_api_expired]]></InfoType><AuthCorpId><![CDATA[yyyy]]></AuthCorpId><AdvanceApiType><![CDATA[1]]></AdvanceApiType><ChatArchiveApi><Edition><![CDATA[2]]></Edition></ChatArchiveApi><TimeStamp>1403610513</TimeStamp></xml>
```
