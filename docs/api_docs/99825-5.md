# 获取内部群信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99825](https://developer.work.weixin.qq.com/document/path/99825)
- **文档 ID**: `99825`
- **请求方法**: `SDK调用`

## 接口描述

通过此接口，获取本企业的内部群信息，包括群主id、群创建时间以及所有群成员的id与加入时间。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| chatid | string | 是 | 待查询的群id |

### 请求示例

```json
{
	"chatid":"wrNplhCgAAIVZohLe57zKnvIV7xBKrig"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| creator | string | chatid对应的群创建者，userid |
| room_create_time | uint32 | chatid对应的群创建时间 |
| members | object[] | chatid对应的群成员列表 |
| members[].memberid | string | chatid群成员的id，userid、external_userid、机器人id |
| members[].jointime | uint32 | chatid群成员的入群时间 |
| members[].type | uint32 | chatid群成员的类型 1：员工 2：外部联系人 3：机器人 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "creator": "woAAAAAAA",
    "room_create_time": 1592361604,
    "members": [
        {
            "memberid": "woAAAAAAAA",
            "jointime": 1592361605,
			"type": 1
        },
        {
            "memberid": "woBBBBBBBB",
            "jointime": 1592377076,
			"type": 2
        }
    ]
}
```

## 其他说明

### 权限说明

> 内部群中至少存在一个员工在授权范围内

| 应用类型 | 权限要求 |
| --- | --- |
| 自建应用 | 需具备「数据与智能专区权限」 |
| 代开发应用 | 需具备「数据与智能专区权限」 |
| 第三方应用 | 需具备「数据与智能专区权限」 |

### 错误说明

| 返回码 | 说明 |
| --- | --- |
| 301052 | 会话存档已过期 |
