# 获取「联系客户统计」数据

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92275](https://developer.work.weixin.qq.com/document/path/92275)
- **文档 ID**: `92275`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/get_user_behavior_data`

## 接口描述

企业可通过此接口获取成员联系客户的数据，包括发起申请数、新增客户数、聊天数、发送消息数和删除/拉黑成员的客户数等指标。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| userid | string[] | 否 | 成员ID列表，最多100个 |
| partyid | int32[] | 否 | 部门ID列表，最多100个 |
| start_time | uint32 | 是 | 数据起始时间 |
| end_time | uint32 | 是 | 数据结束时间 |

### 请求示例

```json
{
    "userid": [
        "zhangsan",
        "lisi"
    ],
	"partyid":
	[
		1001,
		1002
	],
    "start_time": 1536508800,
	"end_time": 1536595200
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| behavior_data | object[] | 行为数据列表 |
| behavior_data[].stat_time | uint32 | 数据日期，为当日0点的时间戳 |
| behavior_data[].new_apply_cnt | uint32 | 发起申请数 |
| behavior_data[].new_contact_cnt | uint32 | 新增客户数 |
| behavior_data[].chat_cnt | uint32 | 聊天总数 |
| behavior_data[].message_cnt | uint32 | 发送消息数 |
| behavior_data[].reply_percentage | float | 已回复聊天占比 |
| behavior_data[].avg_reply_time | uint32 | 平均首次回复时长 |
| behavior_data[].negative_feedback_cnt | uint32 | 删除/拉黑成员的客户数 |
