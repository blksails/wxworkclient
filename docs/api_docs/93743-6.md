# 删除直播回放 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93743](https://developer.work.weixin.qq.com/document/path/93743)
- **文档 ID**: `93743`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/living/delete_replay_data?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/living/delete_replay_data?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| livingid |  | 是 | 直播id |
| 自建应用 | 自建应用 | 否 | 配置到「上课直播/直播 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 具有「直播」权限 |
| 第三方应用 | 第三方应用 | 否 | 具有「直播」权限 |

### 请求示例

```json
{"livingid":"XXXXXXXXX"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
