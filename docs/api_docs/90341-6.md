# 创建部门

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90341](https://developer.work.weixin.qq.com/document/path/90341)
- **文档 ID**: `90341`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/department/create`

## 接口描述

创建部门

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| name | string | 是 | 部门名称。同一个层级的部门名称不能重复。长度限制为1~64个UTF-8字符，字符不能包括":*?"<>｜ |
| name_en | string | 否 | 英文名称。同一个层级的部门名称不能重复。需要在管理后台开启多语言支持才能生效。长度限制为1~64个字符，字符不能包括":*?"<>｜ |
| parentid | int32 | 是 | 父部门id，32位整型 |
| order | uint32 | 否 | 在父部门中的次序值。order值大的排序靠前。有效的值范围是[0, 2^32) |
| id | uint32 | 否 | 部门id，32位整型，指定时必须大于1。若不填该参数，将自动生成id |

### 请求示例

```json
{
   "name": "广州研发中心",
   "name_en": "RDGZ",
   "parentid": 1,
   "order": 1,
   "id": 2
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| id | uint32 | 创建的部门id |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "created",
   "id": 2
}
```

## 其他说明

### 权限说明

第三方仅通讯录应用可以调用。注意，部门的最大层级为15层；部门总数不能超过3万个；每个部门下的节点不能超过3万个。建议保证创建的部门和对应部门成员是串行化处理。
