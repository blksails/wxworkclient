# 改单通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91930](https://developer.work.weixin.qq.com/document/path/91930)
- **文档 ID**: `91930`
- **请求方法**: `POST`

## 接口描述

当服务商管理员修改订单价格之后，企业微信后台会推送改单通知到服务商的指令回调URL。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId |  | 否 | 第三方应用的SuiteId |
| PaidCorpId |  | 否 | 购买方corpid |
| InfoType |  | 否 | 固定为 change_order |
| TimeStamp |  | 否 | 时间戳 |
| OldOrderId |  | 否 | 原订单号 |
| NewOrderId |  | 否 | 改单后新的订单号，服务商可据此拉取购买信息。该OrderId由企业微信生成，不超过32个字符，每次修改价格都会产生新的订单号 |
