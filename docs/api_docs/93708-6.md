# 获取会议详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93708](https://developer.work.weixin.qq.com/document/path/93708)
- **文档 ID**: `93708`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/get_info`

## 接口描述

该接口用于获取指定会议的详情内容。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| meetingid | string | 是 | 会议id |

### 请求示例

```json
{
   "meetingid": "XXXXXXXXX"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| admin_userid | string | 会议管理员的userId |
| title | string | 会议的标题，最大60个字节 |
| meeting_start | int32 | 会议开始时间的unix时间戳 |
| meeting_duration | int32 | 会议时长 |
| description | string | 会议的描述，最大600字节 |
| location | string | 会议地点,最多128个字符 |
| main_department | int32 | 发起人所在部门 |
| status | int32 | 会议的状态，1：待开始，2：会议中，3：已结束，4：已取消，5：已过期 |
| attendees | object | 会议成员。快速会议仅返回已参与成员列表。 |
| attendees.member | object[] | 企业内部成员 |
| attendees.member.userid | string | 企业内部成员的userid |
| attendees.member.status | int32 | 与会状态。1为已参与，2为未参与。 |
| attendees.member.first_join_time | int32 | 参会人首次加入会议时间的unix时间戳 |
| attendees.member.last_quit_time | int32 | 参会人最后一次离开会议时间的unix时间戳 |
| attendees.member.total_join_count | int32 | 参会人入会次数 |
| attendees.member.cumulative_time | int32 | 参会人累计参会时长，单位为秒 |
| attendees.tmp_external_user | object[] | 会中参会的外部联系人 `注意：微信入会的用户不会返回` |
| attendees.tmp_external_user.tmp_external_userid | string | 会中入会的外部用户临时id。同一个用户在不同会议中返回的该id不一致。 可进一步通过tmp_external_userid的转换接口转换成external_userid，方便识别外部参会人身份。 |
| attendees.tmp_external_user.status | int32 | 与会状态。1为已参与，2为未参与 |
| attendees.tmp_external_user.first_join_time | int32 | 参会人首次进入会议时间的unix时间戳 |
| attendees.tmp_external_user.last_quit_time | int32 | 参会人最后一次离开会议时间的unix时间戳 |
| attendees.tmp_external_user.total_join_count | int32 | 参会人入会次数 |
| attendees.tmp_external_user.cumulative_time | int32 | 参会人累计参会时长，单位为秒 |
| settings | object | 会议配置，详见Settings |
| reminders | object | 重复会议相关配置，详见Reminders |
| meeting_code | string | 会议号 |
| meeting_link | string | 入会链接 |
