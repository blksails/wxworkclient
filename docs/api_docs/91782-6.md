# 获取会话同意情况 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91782](https://developer.work.weixin.qq.com/document/path/91782)
- **文档 ID**: `91782`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/msgaudit/check_room_agree?access_token=ACCESS_TOKEN`

## 接口描述

企业可通过下述接口，获取会话中外部成员的同意情况。单聊请求调用频率不可超过2500次/分钟，群聊请求调用频率不可超过1500次/分钟。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| userid |  | 是 | 内部成员的userid |
| exteranalopenid |  | 是 | 外部成员的exteranalopenid |
| access_token |  | 是 | 调用接口凭证 |
| roomid |  | 是 | 待查询的roomid |

### 请求示例

#### 示例 1: 请求示例：

```json
{"info":[{"userid":"XuJinSheng","exteranalopenid":"wmeDKaCQAAGd9oGiQWxVsAKwV2HxNAAA"},{"userid":"XuJinSheng","exteranalopenid":"wmeDKaCQAAIQ_p7ACn_jpLVBJSGocAAA"},{"userid":"XuJinSheng","exteranalopenid":"wmeDKaCQAAPE_p7ABnxkpLBBJSGocAAA"}]}
```

#### 示例 2: 请求示例：

```json
{"roomid":"wrjc7bDwAASxc8tZvBErFE02BtPWyAAA"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| info |  | 待查询的会话信息，数组 |
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| agreeinfo |  | 同意情况 |
| agree_status |  | 同意:"Agree"，不同意:"Disagree" |
| status_change_time |  | 同意状态改变的具体时间，utc时间 |
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| agreeinfo |  | 同意情况 |
| agree_status |  | 同意:"Agree"，不同意:"Disagree" |
| status_change_time |  | 同意状态改变的具体时间，utc时间 |
| exteranalopenid |  | 群内外部联系人的externalopenid |
