- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98723](https://developer.work.weixin.qq.com/document/path/98723)
- **文档 ID**: `98723`

## 接口描述

本接口适用于在 由企业微信向微信发出的小程序中使用对外收款账户进行收款。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 自建应用 | 自建应用 | 否 | 配置到「对外收款 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

## 其他说明

### 开发前准备

1、在使用本接口前，需要在手机端或企业微信管理后台对外收款中申请收款账户；2、 在手机端「对外收款-小程序」中绑定需要接入的小程序和对外收款账户；3、将要调用接口的自建应用配置到「对外收款 - 可调用接口的应用」中 ；4、在企业微信管理后台为要调用接口的自建应用设置企业可信IP，仅企业可信IP可调用本目录相关接口；5、在企业微信管理后台为要调用接口的自建应用设置接收事件服务器，用于接收支付和退款相关的回调通知；
