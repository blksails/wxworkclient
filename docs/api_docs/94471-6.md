# 获取订单详情 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94471](https://developer.work.weixin.qq.com/document/path/94471)
- **文档 ID**: `94471`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/get_trade?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/school/get_trade?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| payment_id |  | 是 | 收款项目id，由发起班级收款接口返回 |
| trade_no |  | 是 | 订单号，由获取学生付款结果返回 |

### 请求示例

```json
{"payment_id":"xxxx","trade_no":"xxxx"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| transaction_id |  | 微信交易单号 |
| pay_time |  | 交易时间 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","transaction_id":"xxxxx","pay_time":12345}
```
