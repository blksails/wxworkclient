# 获取「学校通知」二维码 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92320](https://developer.work.weixin.qq.com/document/path/92320)
- **文档 ID**: `92320`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/get_subscribe_qr_code?access_token=ACCESS_TOKEN`

## 接口描述

学校可通过此接口获取「学校通知」二维码，家长可通过扫描此二维码关注「学校通知」并接收学校推送的消息。请求方式：GET（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/externalcontact/get_subscribe_qr_code?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| 自建应用 | 自建应用 | 否 | 配置到「家校沟通-读取和编辑家校通讯录的应用」或「家校沟通-读取家校通讯录的应用」或“家长可使用应用”列表中 |
| 代开发应用 | 代开发应用 | 否 | 具有「家校沟通」权限 |
| 第三方应用 | 第三方应用 | 否 | 具有「家校沟通」使用权限 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| qrcode_big |  | 1200px的大尺寸二维码 |
| qrcode_middle |  | 430px的中尺寸二维码 |
| qrcode_thumb |  | 258px的小尺寸二维码 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","qrcode_big":"http://p.qpic.cn/wwhead/XXXX","qrcode_middle":"http://p.qpic.cn/wwhead/XXXX","qrcode_thumb":"http://p.qpic.cn/wwhead/XXXX"}
```
