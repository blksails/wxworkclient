## 接口描述

wx.qy.updateEnterpriseChat(Object object)

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| chatId | String | 否 | 指定的群会话id |
| userIdsToAdd | String | 否 | 参与会话的企业成员列表，格式为userid1;userid2;...，用分号隔开 |
| success | function | 否 | 接口调用成功的回调函数 |
| fail | function | 否 | 接口调用失败的回调函数 |
| complete | function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```json
wx.qy.updateEnterpriseChat({chatId:"CHATID",//通过企业微信创建群聊接口返回的chatIduserIdsToAdd:'zhangshan;lisi;wangwu',//参与会话的企业成员列表，格式为userid1;userid2;...，用分号隔开。success:(res)=>{console.log(JSON.stringify(res))},fail:(res)=>{console.log(JSON.stringify(res))}})
```
