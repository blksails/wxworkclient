# 更新成员事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92654](https://developer.work.weixin.qq.com/document/path/92654)
- **文档 ID**: `92654`
- **API 名称**: `receive`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`
- **分组信息**: 第 2 个接口，共 3 个

## 接口描述

更新成员事件通知接口

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 | 消息签名 |
| timestamp | string | 是 | 时间戳 |
| nonce | string | 是 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 第三方应用ID |
| AuthCorpId | string | 是 | 授权企业的CorpID |
| InfoType | string | 是 | 固定为change_contact |
| TimeStamp | string | 是 | 时间戳 |
| ChangeType | string | 是 | 固定为update_user |
| UserID | string | 是 | 变更信息的成员UserID |
| OpenUserID | string | 是 | 全局唯一标识 |
| NewUserID | string | 是 | 新的UserID |
| Name | string | 是 | 成员名称 |
| Department | string | 是 | 更新后成员所在部门列表 |
| MainDepartment | string | 是 | 主部门 |
| IsLeaderInDept | string | 是 | 部门领导信息 |
| DirectLeader | string | 是 | 直属上级UserID |
| Mobile | string | 是 | 手机号码 |
| Position | string | 是 | 职位信息 |
| Gender | int32 | 是 | 性别 |
| Email | string | 是 | 邮箱 |
| BizMail | string | 是 | 企业邮箱 |
| Avatar | string | 是 | 头像url |
| Alias | string | 是 | 成员别名 |
| Telephone | string | 是 | 座机 |
| ExtAttr | object[] | 是 | 扩展属性 |

### 请求示例

```xml
<xml>
	<SuiteId><![CDATA[ww4asffe99e54c0f4c]]></SuiteId>
	...

```
