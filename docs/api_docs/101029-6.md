# 修改群聊会话 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/101029](https://developer.work.weixin.qq.com/document/path/101029)
- **文档 ID**: `101029`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartsheet/groupchat/update?access_token=ACCESS_TOKEN`

## 接口描述

该接口可以修改通过智能表格自动化创建的群聊的群成员。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| docid |  | 是 | 智能表格ID。需为当前应用创建的智能表格 |
| chat_id |  | 是 | 群聊ID，需为对应智能表自动规则创建群聊的ID |
| owner |  | 否 | 新群主的id。若不需更新，请忽略此参数。del_user_list包含群主时本字段必填 |
| add_user_list |  | 否 | 添加成员的id列表，一次最多传入500人 |
| del_user_list |  | 否 | 踢出成员的id列表，一次最多传入500人 |

### 请求示例

```json
{"docid":"DOCID","chat_id":"CHATID","owner":"userid","add_user_list":["userid1","userid2","userid3"],"del_user_list":["userid3","userid4"]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
