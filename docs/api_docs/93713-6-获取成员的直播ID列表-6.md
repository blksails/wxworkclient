# API

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93713](https://developer.work.weixin.qq.com/document/path/93713)
- **文档 ID**: `93713`
- **API 名称**: `获取成员的直播ID列表`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/live/get_member_liveid`
- **分组信息**: 第 6 个接口，共 8 个

## 接口描述

获取成员的直播ID列表接口用于获取指定成员的直播ID列表。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| userid | string | 是 | 成员的userid |

### 请求示例

```json
{
  "userid": "zhangsan"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| liveid_list | string[] | 直播ID列表 |

### 响应示例

```json
{
  "errcode": 0,
  "errmsg": "ok",
  "liveid_list": ["123456", "789012"]
}
```
