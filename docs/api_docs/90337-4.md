# 获取部门成员详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90337](https://developer.work.weixin.qq.com/document/path/90337)
- **文档 ID**: `90337`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/user/list`

## 接口描述

应用只能获取可见范围内的成员信息，且每种应用获取的字段有所不同。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| department_id | string | 是 | 获取的部门id |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| userlist | object[] | 成员列表 |
| userlist[].userid | string | 成员UserID。对应管理端的账号 |
| userlist[].name | string | 成员名称 |
| userlist[].mobile | string | 手机号码 |
| userlist[].department | int32[] | 成员所属部门id列表 |
| userlist[].order | int32[] | 部门内的排序值 |
| userlist[].position | string | 职务信息 |
| userlist[].gender | string | 性别 |
| userlist[].email | string | 邮箱 |
| userlist[].biz_mail | string | 企业邮箱 |
| userlist[].is_leader_in_dept | int32[] | 在所在的部门内是否为部门负责人 |
| userlist[].direct_leader | string[] | 直属上级UserID |
| userlist[].avatar | string | 头像url |
| userlist[].thumb_avatar | string | 头像缩略图url |
| userlist[].telephone | string | 座机 |
| userlist[].alias | string | 别名 |
| userlist[].extattr | object | 扩展属性 |
| userlist[].status | int32 | 激活状态 |
| userlist[].qr_code | string | 员工个人二维码 |
| userlist[].external_profile | object | 成员对外属性 |
| userlist[].external_position | string | 对外职务 |
| userlist[].address | string | 地址 |
| userlist[].open_userid | string | 全局唯一 |
| userlist[].main_department | int32 | 主部门 |
