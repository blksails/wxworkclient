# 网络研讨会暖场上传结果 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98773](https://developer.work.weixin.qq.com/document/path/98773)
- **文档 ID**: `98773`

## 接口描述

对于API创建的网络研讨会，通过 API 进行暖场配置上传暖场图片或视频时，触发该事件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 操作者UserID。此事件为系统触发事件，固定为sys |
| CreateTime |  | 否 | 消息创建时间，unix时间戳 |
| MsgType |  | 否 | 消息类型，此时固定为：event |
| Event |  | 否 | 事件类型，此时固定为：meeting_change |
| ChangeType |  | 否 | 事件类型，此时固定为：webinar_warm_up_upload |
| MeetingId |  | 否 | 会议ID |
| WarmUpInfo |  | 否 | 暖场配置信息 |

### 请求示例

```xml
<xml><ToUserName><![CDATA[toUser]]></ToUserName><FromUserName><![CDATA[sys]]></FromUserName><CreateTime>1348831860</CreateTime><MsgType><![CDATA[event]]></MsgType><Event><![CDATA[meeting_change]]></Event><ChangeType><![CDATA[webinar_warm_up_upload]]></ChangeType><MeetingId><![CDATA[wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA]]></MeetingId><WarmUpInfo><WarmUpPicture><![CDATA[https://image.qq.com/12519.png]]</WarmUpPicture><WarmUpVideo><![CDATA[https://image.qq.com/a135.mp4]]</WarmUpVideo><UploadStatus>1</UploadStatus><ErrorMsg><![CDATA[low-resolution]]</ErrorMsg></WarmUpInfo></xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| WarmUpInfo.WarmUpPicture |  | 暖场图片地址 |
| WarmUpInfo.WarmUpVideo |  | 暖场视频地址 |
| WarmUpInfo.UploadStatus |  | 上传结果。1: 成功0: 失败 |
| WarmUpInfo.ErrorMsg |  | 如果上传失败，则该字段返回失败原因 |
