# 专区产品框架

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99864](https://developer.work.weixin.qq.com/document/path/99864)
- **文档 ID**: `99864`
- **API 名称**: `area_framework`
- **分组信息**: 第 1 个接口，共 3 个

## 接口描述

数据与智能专区包括企业数据和分析工具两块，其中：

- 企业数据：专区中存储企业会话内容和企业知识集等数据。
- 分析工具：企业授权后，服务商可使用分析工具帮企业分析数据，按照能力和来源分析工具共包括以下工具：
  + 服务商程序：服务商需在专区中上传部署分析程序，服务商应用后台可通过专区通道调用程序。
  + 服务商模型：服务商可在专区中上传部署模型，服务商应用后台可通过专区通道调用模型。
  + 企业微信模型：企业微信提供了基础模型，包括话术推荐、客户标签、会话摘要、情感分析等模型能力，以及自定义提示词的通用模型能力，服务商可通过在专区部署程序之后进行调用。

服务商的应用后台可通过专区通道使用分析工具来分析企业数据，并获得数据分析结果.
