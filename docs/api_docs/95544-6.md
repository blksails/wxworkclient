# 获取企业的账号列表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95544](https://developer.work.weixin.qq.com/document/path/95544)
- **文档 ID**: `95544`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/license/list_actived_account?provider_access_token=ACCESS_TOKEN`

## 接口描述

查询指定企业下的平台能力服务账号列表。若为上下游场景，corpid指定的为上游企业，仅返回上游企业激活的账号；若corpid指定为下游企业，若激活码为上游企业分享过来的且已绑定，也会返回。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 应用服务商的接口调用凭证，获取方法参见服务商的凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| corpid | string | 是 | 企业corpid。若为上下游场景，为上游企业corpid |
| limit | uint32 | 否 | 返回的最大记录数，整型，最大值1000，默认值500 |
| cursor | string | 否 | 用于分页查询的游标，字符串类型，由上一次调用返回，首次调用可不填 |

### 请求示例

```json
{
	"corpid" : "CORPID",
	"limit":500,
	"cursor":"xxxx"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| next_cursor | string | 分页游标，再下次请求时填写以获取之后分页的记录，如果已经没有更多的数据则返回空 |
| has_more | bool | 是否结束 |
| account_list | object[] | 已激活成员列表，已激活过期的也会返回 |
| account_list.userid | string | 企业的成员userid。返回加密的userid |
| account_list.type | int32 | 激活码账号类型：1:基础账号，2:互通账号 |
| account_list.expire_time | uint32 | 过期时间 |
| account_list.active_time | uint32 | 激活时间 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"next_cursor":"xxx",
	"has_more":1,
	"account_list":[
		{
			"userid": "user1",
			"type": 1,
			"expire_time":1500000000,
			"active_time":1500000000
		},
		{
			"userid": "user2",
			"type": 1,
			"expire_time":1500000000,
			"active_time":1500000000
		}
	]
}
```
