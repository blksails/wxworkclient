# SDK生成notify_id

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100040](https://developer.work.weixin.qq.com/document/path/100040)
- **文档 ID**: `100040`
- **API 名称**: `spec_gen_notify_id`
- **分组信息**: 第 2 个接口，共 2 个

## 请求信息

### 请求示例

```text

std::string notify_id = GenerateNotifyId();
```

