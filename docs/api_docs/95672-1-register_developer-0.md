# 成为企业微信开发者

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95672](https://developer.work.weixin.qq.com/document/path/95672)
- **文档 ID**: `95672`
- **API 名称**: `register_developer`
- **分组信息**: 第 1 个接口，共 3 个

## 接口描述

帮助开发者快速了解企业微信开发体系及如何基于企业微信快速创建个人主体的第三方应用。

## 其他说明

### 背景

企业微信保持完全开放的第三方合作策略，致力于和所有的合作伙伴携手为用户提供最优质的企业服务。企业微信开发者，是在企业微信内提供应用开发服务的开发伙伴，面向任何有意向、有能力的程序开发人员。本接入说明将帮助企业微信开发者快速了解企业微信开发体系及如何基于企业微信快速创建个人主体的第三方应用。企业微信开发者仅支持网页应用接入。

### 成为企业微信开发者

企业微信开发者基于企业微信账号体系创建，需先成为企业微信的用户，即创建一个新企业或加入一个已有企业。前往企业微信开发者中心，申请成为企业微信开发者。

### 企业微信开发者中心

企业微信开发者中心，是面向开发者提供集工具、社区和文档一体的开发者平台。功能包括工具、社区和文档。
