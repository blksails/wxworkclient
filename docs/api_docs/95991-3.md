# 获取设备数据授权

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95991](https://developer.work.weixin.qq.com/document/path/95991)
- **文档 ID**: `95991`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

应用通过调用该接口，唤起设备选择列表，企业管理员选择之后，应用后续可通过云端接口获取到设备上报的数据

## 请求信息

### 请求示例

```text
wx.qy.openAppDeviceDataAuth({
    success: function(res){
        // 调用成功
    },
    fail: function(err){
        // 调用失败
    }
});
```

## 其他说明

### 调用前提

必须先调用过wx.qy.login，并且调用时传入suiteId参数

### 调用权限

发起调用的第三方应用，必须勾选了“智慧硬件接口权限”

### 平台版本说明

| 平台 | 企业微信版本要求 |
| --- | --- |
| ios | ≥ 4.0.12 |
| android | ≥ 4.0.12 |
| win | ≥ 4.0.12 |
| mac | ≥ 4.0.12 |

### 应用权限说明

| 应用类型 | 是否支持 |
| --- | --- |
| 自建应用 | 不支持 |
| 第三方应用 | 支持 |
| 代开发自建应用 | 不支持 |
