# 获取可使用的家长范围

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94960](https://developer.work.weixin.qq.com/document/path/94960)
- **文档 ID**: `94960`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/agent/get_allow_scope`

## 接口描述

获取可在微信「学校通知-学校应用」使用该应用的家长范围，以学生或部门列表的形式返回。应用只能给该列表下的家长发送「学校通知」。注意该范围只能由学校的系统管理员在「管理端-家校沟通-配置」配置。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| agentid | string | 是 | 应用id |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 出错返回码，为0表示成功，非0表示调用失败 |
| errmsg | string | 返回码提示语 |
| allow_scope | object | 可在微信「学校通知-学校应用」使用该应用的家长范围 |
| allow_scope.students.userid | array | 家长可在微信「学校通知-学校应用」使用该应用的学生列表 |
| allow_scope.departments.partyid | array | 家长可在微信「学校通知-学校应用」使用该应用的部门列表 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "ok",
   "allow_scope": {
       "students": {
             "userid": ["student1", "student2"]
       },
	   "departments": {
             "partyid": [1, 2]
	   }
    }
}
```

## 其他说明

### 权限说明

调用的应用需要满足如下的权限

自建应用：配置到「家校沟通- 读取和编辑家校通讯录的应用」或「家校沟通- 读取家校通讯录的应用」或“家长可使用应用”列表中
代开发应用：具有「家校沟通->家校通讯录读取权限”权限
第三方应用：具有 「家校沟通」使用权限

注：从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 查看详情
