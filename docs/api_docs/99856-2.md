## 接口描述

专区程序接收事件通知，企业微信通过HTTP请求回调到专区，专区需要启动一个HTTP服务并监听8080端口。

## 其他说明

### http请求body

http服务接收到的http的body为加密的，需要调用sdk进行解密，解密后拿到消息原文，各个通知事件的协议参见本目录下各个文档。
