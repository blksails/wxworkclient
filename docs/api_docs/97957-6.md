# 删除文件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97957](https://developer.work.weixin.qq.com/document/path/97957)
- **文档 ID**: `97957`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_delete`

## 接口描述

该接口用于删除指定文件。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fileid | string[] | 是 | 文件fileid |

### 请求示例

```json
{
    "fileid": ["FILEID1", "FILEID2"]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok"
}
```

## 其他说明

### 权限说明

* 自建应用需配置到“可调用应用”列表中的应用secret所获取的accesstoken来调用（accesstoken如何获取？）
* 第三方应用需具有“微盘”权限
* 代开发自建应用需具有“微盘”权限
