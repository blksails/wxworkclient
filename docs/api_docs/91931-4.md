# 支付成功通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91931](https://developer.work.weixin.qq.com/document/path/91931)
- **文档 ID**: `91931`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`

## 接口描述

当企业对某一个订单完成付款后，企业微信后台会推送付款成功通知到服务商的指令回调URL。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 | 消息签名 |
| timestamp | string | 是 | 时间戳 |
| nonce | string | 是 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 第三方应用的SuiteId |
| PaidCorpId | string | 是 | 购买方corpid |
| InfoType | string | 是 | 固定为 pay_for_app_success |
| TimeStamp | string | 是 | 时间戳 |
| OrderId | string | 是 | 订单号，付费订单的唯一标志。服务商可据此拉取购买信息。该OrderId由企业微信生成，不超过32个字符 |

### 请求示例

```xml
<xml>
    <SuiteId><![CDATA[ww4asffe99e54c0aaa]]></SuiteId>
    <PaidCorpId><![CDATA[wxf8b4f85f3a794aaa]]></PaidCorpId>
    <InfoType><![CDATA[pay_for_app_success]]></InfoType>
    <TimeStamp>1403610513</TimeStamp>
    <OrderId><![CDATA[ORDERID]]></OrderId>
</xml>
```
