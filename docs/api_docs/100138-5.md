# 充值账户余额查询

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100138](https://developer.work.weixin.qq.com/document/path/100138)
- **文档 ID**: `100138`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/get_account_balance`

## 接口描述

可以通过该接口查询服务商充值账户余额。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 应用服务商的接口调用凭证，获取方法参见服务商的凭证 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| balance | int32 | 充值账户余额，单位为分 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"balance": 1000090
}
```
