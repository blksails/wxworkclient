# 打开应用管理页面 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95525](https://developer.work.weixin.qq.com/document/path/95525)
- **文档 ID**: `95525`

## 接口描述

接口说明小程序应用可以使用该接口提供按钮，让企业管理员打开应用的管理页面，快速进行修改可见范围等操作。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| page | String | 否 | 不指定时，表示为跳转至应用管理首页。指定时，表示跳转的应用二级页。目前支持的二级页有：permission：对应为应用权限详情页。datazone_permission：对应为数据与智能专区权限授权页。 |
| suiteId | String | 否 | 不指定时，表示为跳转agentConfig指定的应用管理页。指定时，表示为跳转的其他应用管理页。目前仅支持会话内容存档应用。指定的suiteid必须是同服务商的应用，且当前企业有授权关系，同时当前用户必须是超管 |
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| ios |  | 否 | ≥4.0.2 |
| android |  | 否 | ≥4.0.2 |
| win |  | 否 | ≥4.0.2 |
| mac |  | 否 | ≥4.0.2 |
| 自建应用 | 自建应用 | 否 | 支持 |
| 第三方应用 | 第三方应用 | 否 | 支持 |
| 代开发自建应用 | 代开发自建应用 | 否 | 支持 |

### 请求示例

```json
wx.qy.openAppManage({page:"permission",// 非必填，指定跳转的页面地址suiteId:"wwabcdefghijk",// 非必填，指定跳转的关联应用success:function(res){// 调用成功},fail:function(err){// 调用失败}});
```
