## 其他说明

### ww.setClipboardData(params)

兼容性：企业微信 iOS、Android 2.4.16 开始支持，企业微信 PC、Mac 3.1.2 开始支持
功能描述设置系统剪贴板的内容。
ww.setClipboardData({data:'data'})
参数说明params: Object属性类型必填说明datastring是剪贴板的内容successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

设置系统剪贴板的内容。

### params: Object

属性类型必填说明datastring是剪贴板的内容successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 示例代码

wx.setClipboardData({data:'data',// 设置的success:function(res){console.log(res.errMsg)}})
