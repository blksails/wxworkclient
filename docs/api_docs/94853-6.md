# 导出标签成员 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94853](https://developer.work.weixin.qq.com/document/path/94853)
- **文档 ID**: `94853`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/export/taguser?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/export/taguser?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| tagid |  | 是 | 需要导出的标签 |
| encoding_aeskey |  | 是 | Base64编码后的加密密钥。长度固定为43，从a-z, A-Z, 0-9共62个字符中选取，是AESKey的Base64编码。解码后即为32字节长的AESKey。加密方式采用AES-256-CBC方式，数据采用PKCS#7填充至32字节的倍数；IV初始向量大小为16字节，取AESKey前16字节，详见：https://datatracker.ietf.org/doc/html/rfc2315 |
| block_size |  | 否 | 每块数据的人员数和部门数之和，支持范围[104,106]，默认值为106 |

### 请求示例

```json
{"tagid":1,"encoding_aeskey":"IJUiXNpvGbODwKEBSEsAeOAPAhkqHqNCF6g19t9wfg2","block_size":1000000}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| jobid |  | 任务ID，可通过获取导出结果接口查询任务结果 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","jobid":"jobid_xxxxxxxxx"}
```
