# 客户同意进行聊天内容存档事件回调

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99852](https://developer.work.weixin.qq.com/document/path/99852)
- **文档 ID**: `99852`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webhook/receive_chat_archive_audit_approved_event?access_token=ACCESS_TOKEN`

## 接口描述

配置了客户联系功能的成员添加外部联系人同意进行聊天内容存档时，回调该事件。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| event_type | string | 是 | 事件类型。当客户在单聊中同意存档时，该字段值为 chat_archive_audit_approved_single；当客户在群聊中同意存档时，该字段值为 chat_archive_audit_approved_room。 |
| timestamp | uint32 | 是 | 时间戳 |
| chat_archive_audit_approved.userid | string | 否 | 单聊时返回，企业服务人员的UserID |
| chat_archive_audit_approved.external_userid | string | 是 | 外部联系人的userid，注意不是企业成员的账号 |
| chat_archive_audit_approved.chatid | string | 否 | 群聊时返回，所在群聊的ChatId |

### 请求示例

```json
{
    "event_type": "chat_archive_audit_approved_single",
    "timestamp": 1408091189,
    "chat_archive_audit_approved": {
        "userid": "xxxxxxx",
        "external_userid": "zzzzz",
        "chatid":"xxx"
    }
}
```
