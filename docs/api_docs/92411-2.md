## 接口描述

关闭 NFC 模块，仅支持安卓系统

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | function | 否 | 接口调用成功的回调函数 |
| fail | function | 否 | 接口调用失败的回调函数 |
| complete | function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| 0 |  | 否 | 正常 |

### 请求示例

```json
wx.qy.stopNFCReader({success:(res)=>{wx.showModal({title:'NFC-Stop',content:JSON.stringify(res),})},fail:(res)=>{console.log(JSON.stringify(res))}})
```
