# 获取注册码

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90581](https://developer.work.weixin.qq.com/document/path/90581)
- **文档 ID**: `90581`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/get_register_code?provider_access_token=PROVIDER_ACCESS_TOKEN`

## 接口描述

根据注册推广包生成注册码（register_code）

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 服务商provider_access_token，获取方法参见服务商的凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| template_id | string | 是 | 推广包ID，最长为128个字节。在服务商管理端-应用管理-推广二维码，创建的推广码详情可查看。 |
| corp_name | string | 否 | 企业名称 |
| admin_name | string | 否 | 管理员姓名 |
| admin_mobile | string | 否 | 管理员手机号 |
| state | string | 否 | 用户自定义的状态值。只支持英文字母和数字，最长为128字节。若指定该参数，接口查询注册状态及注册完成回调事件会相应返回该字段值 |
| follow_user | string | 否 | 跟进人的userid，必须是服务商所在企业的成员。若配置该值，则由该注册码创建的企业，在服务商管理后台，该企业的报备记录会自动标注跟进人员为指定成员 |

### 请求示例

```json
{
    "template_id":"TEMPLATEID" ,
    "corp_name":"腾讯科技",
    "admin_name":"张三",
    "admin_mobile":"12345678901",
    "state":"TestState123",
    "follow_user": "lisi"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码对应的描述 |
| register_code | string | 注册码，只能消费一次。在访问注册链接时消费。最长为512个字节 |
| expires_in | uint32 | register_code有效期，生成链接需要在有效期内点击跳转 |

### 响应示例

```json
{
    "errcode":0,
    "errmsg":"ok",
    "register_code":"pIKi3wRPNWCGF-pyP-YU5KWjDDD",
    "expires_in": 600
}
```
