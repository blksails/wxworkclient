# 删除日历 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97718](https://developer.work.weixin.qq.com/document/path/97718)
- **文档 ID**: `97718`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/oa/calendar/del?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于删除指定日历。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | Query参数，示例值: ACCESS_TOKEN |

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| cal_id | string | 是 | 日历ID |

### 请求示例

```json
{"cal_id":"wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
