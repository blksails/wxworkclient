# 获取外部联系人详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91670](https://developer.work.weixin.qq.com/document/path/91670)
- **文档 ID**: `91670`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/get?access_token=ACCESS_TOKEN&external_userid=EXTERNAL_USERID`

## 接口描述

学校可通过此接口，根据外部联系人的userid，拉取外部联系人详情。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| external_userid | string | 是 | 外部联系人的userid，注意不是学校成员的账号 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| external_contact.external_userid | string | 外部联系人的userid |
| external_contact.name | string | 外部联系人的姓名或别名 |
| external_contact.position | string | 外部联系人的职位 |
| external_contact.avatar | string | 外部联系人头像，第三方不可获取 |
| external_contact.corp_name | string | 外部联系人所在企业的简称 |
| external_contact.corp_full_name | string | 外部联系人所在企业的主体名称 |
| external_contact.type | int32 | 外部联系人的类型，1表示微信用户，2表示企业微信用户 |
| external_contact.gender | int32 | 外部联系人性别 0-未知 1-男性 2-女性 |
| external_contact.unionid | string | 外部联系人在微信开放平台的唯一身份标识 |
| external_contact.is_subscribe | int32 | 外部联系人是否关注了“学校通知”，1-已关注 |
| external_contact.subscriber_info.tag_id | string[] | 学校为家长设置的标签id |
| external_contact.subscriber_info.remark_mobiles | string[] | 学校为家长备注的手机号 |
| external_contact.subscriber_info.remark | string | 学校对家长的备注信息 |
| external_contact.external_profile.external_attr | object[] | 外部联系人的自定义展示信息 |
| follow_user | object[] | 返回的跟进人follow_user仅包含应用可见范围之内的成员 |
| follow_user[].userid | string | 添加了此外部联系人的企业成员userid |
| follow_user[].remark | string | 该成员对此外部联系人的备注 |
| follow_user[].description | string | 该成员对此外部联系人的描述 |
| follow_user[].createtime | int32 | 该成员添加此外部联系人为客户的时间 |
| follow_user[].tags.group_name | string | 该成员添加此外部联系人所打标签的分组名称 |
| follow_user[].tags.tag_name | string | 该成员添加此外部联系人所打标签名称 |
| follow_user[].tags.type | int32 | 该成员添加此外部联系人所打标签类型, 1-企业设置, 2-用户自定义 |
| follow_user[].remark_corp_name | string | 该成员对此外部联系人备注的企业名称 |
| follow_user[].remark_mobiles | int32[] | 该成员对此外部联系人备注的手机号码 |
| follow_user[].state | string | 该成员添加此外部联系人的渠道 |
