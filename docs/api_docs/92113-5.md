# 获取客户列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92113](https://developer.work.weixin.qq.com/document/path/92113)
- **文档 ID**: `92113`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/list?access_token=ACCESS_TOKEN&userid=USERID`

## 接口描述

企业可通过此接口获取指定成员添加的客户列表。客户是指配置了客户联系功能的成员所添加的外部联系人。没有配置客户联系功能的成员，所添加的外部联系人将不会作为客户返回。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 自建应用 | 自建应用 | 否 | 配置到「客户联系 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 具有「客户基础信息」权限 |
| 第三方应用 | 第三方应用 | 否 | 具有「客户基础信息」权限；营销获客类应用具有「建联客户信息」权限 |
| access_token |  | 是 | 调用接口凭证 |
| userid |  | 是 | 企业成员的userid |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| external_userid |  | 外部联系人的userid列表 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","external_userid":["woAJ2GCAAAXtWyujaWJHDDGi0mACAAA","wmqfasd1e1927831291723123109rAAA"]}
```
