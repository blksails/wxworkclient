### 请求示例

#### 示例 1: 功能描述

```json
ww.openAppManage({page:"permission",suiteId:"wwabcdefghijk",})
```

#### 示例 2: 跳转应用管理

```text
ww.openAppManage();
```

#### 示例 3: 跳转应用权限详情页

```json
ww.openAppManage({page:"permission"});
```

#### 示例 4: 跳转数据与智能专区权限授权页

```json
ww.openAppManage({page:"datazone_permission"});
```

#### 示例 5: 示例代码

```json
wx.invoke('openAppManage',{"page":"permission",// 非必填，指定跳转的页面地址"suiteId":"wwabcdefghijk",// 非必填，指定跳转的关联应用},function(res){if(res.err_msg=="openAppManage:ok"){// 调用成功}if(res.err_msg=="openAppManage:fail:no permission"){// 调用人身份不符合}if(res.err_msg=="openAppManage:fail:unknown app"){// 应用信息获取失败}if(res.err_msg=="openAppManage:fail:unsupported app type"){// 应用类型不符合要求}if(res.err_msg=="openAppManage:fail"){// 其它错误}})
```

## 其他说明

### ww.openAppManage([params])

兼容性：企业微信 4.0.2 开始支持
功能描述打开应用管理页面。应用可以使用该接口提供按钮，让企业管理员快速打开应用的管理页面。
ww.openAppManage({page:"permission",suiteId:"wwabcdefghijk",})
使用限制本接口必须使用应用身份进行注册当前用户需要是企业超级管理员，或具有应用管理权限
参数说明params: Object属性类型必填说明pagestring否指定跳转的页面地址，不指定时跳转至应用管理首页合法值说明permission应用权限详情页datazone_permission数据与智能专区权限授权页需要满足：企业访问者身份为超级管理员应用需要满足勾选了“数据与智能专区权限”（注：该权限目前灰度开放）应用类型为第三方应用/代开发应用（注：不支持上下游共享应用）suiteIdstring否指定跳转的关联应用不指定时，表示为跳转到 agentConfig 注册应用的管理页指定时，表示为跳转的其他应用管理页，目前仅支持会话内容存档应用使用限制服务商应用调用时，指定的 suiteid 必须为同服务商的应用，应用与当前企业需要有授权关系，且当前用户必须是企业超级管理员。successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

打开应用管理页面。
应用可以使用该接口提供按钮，让企业管理员快速打开应用的管理页面。

### 使用限制

本接口必须使用应用身份进行注册当前用户需要是企业超级管理员，或具有应用管理权限

### params: Object

属性类型必填说明pagestring否指定跳转的页面地址，不指定时跳转至应用管理首页合法值说明permission应用权限详情页datazone_permission数据与智能专区权限授权页需要满足：企业访问者身份为超级管理员应用需要满足勾选了“数据与智能专区权限”（注：该权限目前灰度开放）应用类型为第三方应用/代开发应用（注：不支持上下游共享应用）suiteIdstring否指定跳转的关联应用不指定时，表示为跳转到 agentConfig 注册应用的管理页指定时，表示为跳转的其他应用管理页，目前仅支持会话内容存档应用使用限制服务商应用调用时，指定的 suiteid 必须为同服务商的应用，应用与当前企业需要有授权关系，且当前用户必须是企业超级管理员。successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 跳转应用管理

当需要应用信息查看或者修改（如：可见范围/购买付费/应用评价），可跳转至该管理页进行操作。
ww.openAppManage();

### 跳转应用权限详情页

查看应用已授权权限项，及更新应用最新权限（如应用有权限变更时）。
ww.openAppManage({page:"permission"});

### 跳转数据与智能专区权限授权页

授权数据与智能专区权限，若已授权则为修改授权内容。
ww.openAppManage({page:"datazone_permission"});
