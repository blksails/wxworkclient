# 发送消息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94700](https://developer.work.weixin.qq.com/document/path/94700)
- **文档 ID**: `94700`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/kf/send_msg?access_token=ACCESS_TOKEN`

## 接口描述

当微信客户处于“新接入待处理”或“由智能助手接待”状态下，可调用该接口给用户发送消息。注意仅当微信客户在主动发送消息给客服后的48小时内，企业可发送消息给客户，最多可发送5条消息；若用户继续发送消息，企业可再次下发消息。支持发送消息类型：文本、图片、语音、视频、文件、图文、小程序、菜单消息、地理位置、获客链接。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| touser | string | 是 | 指定接收消息的客户UserID |
| open_kfid | string | 是 | 指定发送消息的客服账号ID |
| msgid | string | 否 | 指定消息ID |
| msgtype | string | 是 | 消息类型，固定为：text/image/voice/video/file/link/miniprogram/msgmenu/location/ca_link |
| text.content | string | 否 | 消息内容，最长不超过2048个字节 |
| image.media_id | string | 否 | 图片文件id，可以调用上传临时素材接口获取 |
| voice.media_id | string | 否 | 语音文件id，可以调用上传临时素材接口获取 |
| video.media_id | string | 否 | 视频媒体文件id，可以调用上传临时素材接口获取 |
| file.media_id | string | 否 | 文件id，可以调用上传临时素材接口获取 |
| link.title | string | 否 | 标题，不超过128个字节，超过会自动截断 |
| link.desc | string | 否 | 描述，不超过512个字节，超过会自动截断 |
| link.url | string | 否 | 点击后跳转的链接。最长2048字节，请确保包含了协议头(http/https) |
| link.thumb_media_id | string | 否 | 缩略图的media_id，可以通过素材管理接口获得。此处thumb_media_id即上传接口返回的media_id |
| miniprogram.appid | string | 否 | 小程序appid |
| miniprogram.title | string | 否 | 小程序消息标题，最多64个字节，超过会自动截断 |
| miniprogram.thumb_media_id | string | 否 | 小程序消息封面的mediaid，封面图建议尺寸为520*416 |
| miniprogram.pagepath | string | 否 | 点击消息卡片后进入的小程序页面路径。注意路径要以.html为后缀，否则在微信中打开会提示找不到页面 |
| ca_link.link_url | string | 否 | 通过获客助手创建的获客链接 |

### 请求示例

#### 示例 1: 文本消息示例

```json
{
   "touser" : "EXTERNAL_USERID",
   "open_kfid": "OPEN_KFID",
   "msgid": "MSGID",
   "msgtype" : "text",
   "text" : {
       "content" : "你购买的物品已发货，可点击链接查看物流状态http://work.weixin.qq.com/xxxxxx"
   }
}
```

#### 示例 2: 图片消息示例

```json
{
   "touser" : "EXTERNAL_USERID",
   "open_kfid": "OPEN_KFID",
   "msgid": "MSGID",
   "msgtype" : "image",
   "image" : {
        "media_id" : "MEDIA_ID"
   }
}
```

#### 示例 3: 语音消息示例

```json
{
   "touser" : "EXTERNAL_USERID",
   "open_kfid": "OPEN_KFID",
   "msgtype" : "voice",
   "voice" : {
        "media_id" : "MEDIA_ID"
   }
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |
| msgid | string | 消息ID。如果请求参数指定了msgid，则原样返回，否则系统自动生成并返回。若指定msgid，开发者需确保客服账号内唯一，否则接口返回错误。 不多于32字节 字符串取值范围(正则表达式)：[0-9a-zA-Z_-]* |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "msgid": "MSG_ID"
}
```

## 其他说明

### 消息类型

支持发送消息类型：文本、图片、语音、视频、文件、图文、小程序、菜单消息、地理位置、获客链接。
