# 删除家长

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92079](https://developer.work.weixin.qq.com/document/path/92079)
- **文档 ID**: `92079`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/user/delete_parent`

## 接口描述

家校通信录中删除家长

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| userid | string | 是 | 家校通信录中家长的userid |

### 请求示例

```json
{
	"access_token": "ACCESS_TOKEN",
    "userid": "USERID"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
	"errcode": 0,
    "errmsg": "deleted"
}
```
