# 获取表格行列信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98031](https://developer.work.weixin.qq.com/document/path/98031)
- **文档 ID**: `98031`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/spreadsheet/get_sheet_properties`

## 接口描述

该接口用于获取在线表格的工作表、行数、列数等。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 是 | 在线表格的docid |

### 请求示例

```json
{
	"docid": "DOCID"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| properties | object[] | 工作表属性 |
| properties[].sheet_id | string | 工作表ID，工作表的唯一标识 |
| properties[].title | string | 工作表名称 |
| properties[].row_count | uint32 | 表格的总行数 |
| properties[].column_count | uint32 | 表格的总列数 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"properties": [
		{
			"sheet_id": "ABCDE",
			"title": "XXXXXX",
			"row_count": 100,
			"column_count": 100
		}
	]
}
```
