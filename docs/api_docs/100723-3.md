## 其他说明

### ww.getApprovalSelectedItems(params)

兼容性：企业微信 4.0.18 开始支持
功能描述获取 saveApprovalSelectedItems 保存的审批选项。当用户打开网页后，应该先调用一次该接口获取用户已经选择的数据作为初始数据。获取到初始数据后，应该恢复已经选择的选项。
ww.getApprovalSelectedItems({key:'key'})
使用限制本接口必须使用应用身份进行注册所签名的应用必须具有审批权限
参数说明params: Object属性类型必填说明keystring是从 URL 获取的 keysuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码selectedDataObject[]是选中的选项属性类型必填说明keystring是表示选项的唯一识别 IDvaluestring是选项展示给用户的名字
使用说明网页应该做好深色模式适配接口仅用于审批设置外部选项场景，请勿用作其他场景

### 功能描述

获取 saveApprovalSelectedItems 保存的审批选项。
当用户打开网页后，应该先调用一次该接口获取用户已经选择的数据作为初始数据。获取到初始数据后，应该恢复已经选择的选项。

### 使用限制

本接口必须使用应用身份进行注册所签名的应用必须具有审批权限

### params: Object

属性类型必填说明keystring是从 URL 获取的 keysuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码selectedDataObject[]是选中的选项属性类型必填说明keystring是表示选项的唯一识别 IDvaluestring是选项展示给用户的名字

### 使用说明

网页应该做好深色模式适配接口仅用于审批设置外部选项场景，请勿用作其他场景
