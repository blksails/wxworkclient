# 获取内部群信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99825](https://developer.work.weixin.qq.com/document/path/99825)
- **文档 ID**: `99825`

## 接口描述

可通过此接口，获取本企业的内部群信息，包括群主id、群创建时间以及所有群成员的id与加入时间。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 自建应用 | 自建应用 | 否 | 需具备「数据与智能专区权限」 |
| 代开发应用 | 代开发应用 | 否 | 需具备「数据与智能专区权限」 |
| 第三方应用 | 第三方应用 | 否 | 需具备「数据与智能专区权限」 |
| chatid |  | 是 | 待查询的群id |

### 请求示例

```json
{"chatid":"wrNplhCgAAIVZohLe57zKnvIV7xBKrig"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| creator |  | chatid对应的群创建者，userid |
| room_create_time |  | chatid对应的群创建时间 |
| members |  | chatid对应的群成员列表 |
| memberid |  | chatid群成员的id，userid、external_userid、机器人id |
| jointime |  | chatid群成员的入群时间 |
| type |  | chatid群成员的类型1：员工2：外部联系人3：机器人 |
| 301052 |  | 会话存档已过期 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","creator":"woAAAAAAA","room_create_time":1592361604,"members":[{"memberid":"woAAAAAAAA","jointime":1592361605,"type":1},{"memberid":"woBBBBBBBB","jointime":1592377076,"type":2}]}
```
