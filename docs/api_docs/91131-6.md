# 全量覆盖成员

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91131](https://developer.work.weixin.qq.com/document/path/91131)
- **文档 ID**: `91131`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/batch/replaceuser`

## 接口描述

本接口以userid为主键，全量覆盖企业的通讯录成员，任务完成后企业的通讯录成员与提交的文件完全保持一致。请先下载CSV文件，根据需求填写文件内容。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| media_id | string | 是 | 上传的csv文件的media_id |
| to_invite | bool | 否 | 是否邀请新建的成员使用企业微信，默认值为true |
| callback.url | string | 否 | 企业应用接收企业微信推送请求的访问协议和地址，支持http或https协议 |
| callback.token | string | 否 | 用于生成签名 |
| callback.encodingaeskey | string | 否 | 用于消息体的加密，是AES密钥的Base64编码 |

### 请求示例

```json
{
	"media_id":"xxxxxx",
	"to_invite": true,
	"callback":
	{
	 	"url": "xxx",
	 	"token": "xxx",
	 	"encodingaeskey": "xxx"
	}
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| jobid | string | 异步任务id，最大长度为64字节 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"jobid": "xxxxx"
}
```
