# 获取应用的接口许可状态

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97194](https://developer.work.weixin.qq.com/document/path/97194)
- **文档 ID**: `97194`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/license/get_app_license_info?provider_access_token=ACCESS_TOKEN`

## 接口描述

服务商可获取某个授权企业的应用接口许可试用期，免费试用期为企业首次安装应用后的90天。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| corpid | string | 是 | 企业id |
| suite_id | string | 是 | 套件id |
| appid | uint32 | 否 | 旧的多应用套件中的应用id，新开发者请忽略 |

### 请求示例

```json
{
	"corpid" : "xxxx",
	"suite_id": "xxxx",
	"appid": 1
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| license_status | int32 | license检查开启状态。0：未开启license检查状态或未达到拦截时间的历史授权的第三方应用或代开发应用。1：已开启license检查状态。若开启且已过试用期，则需要为企业购买license账号才可以使用 |
| trail_info | object | 应用license试用期信息。仅当license_status为1且应用有试用期时返回该字段。服务商测试企业、历史迁移应用无试用期。 |
| trail_info.start_time | uint32 | 接口许可试用开始时间 |
| trail_info.end_time | uint32 | 接口许可试用到期时间。若企业多次安装卸载同一个第三方应用，以第一次安装的时间为试用期开始时间，第一次安装完90天后为结束试用时间。 |
| license_check_time | uint32 | 接口开启拦截校验时间。开始拦截校验后，无接口许可将会被拦截，有接口许可将不会被拦截。 |

### 响应示例

```json
{
	"errcode":0,
	"errmsg":"ok",
	"license_status":1,
	"trail_info":
	{
		"start_time": 1651752000,
		"end_time": 1659700800
	},
	"license_check_time": 1684166400
}
```
