# 获取健康上报使用统计 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93676](https://developer.work.weixin.qq.com/document/path/93676)
- **文档 ID**: `93676`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/health/get_health_report_stat?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/health/get_health_report_stat?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证。 |
| date |  | 是 | 具体某天的使用统计，最长支持获取30天前数据 |
| 自建应用 | 自建应用 | 否 | 配置到「健康上报 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"date":"2020-03-27"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| pv |  | 应用使用次数 |
| uv |  | 应用使用成员数 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","pv":100,"uv":50}
```
