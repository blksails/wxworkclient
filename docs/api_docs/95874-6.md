# 查询企业的许可自动激活状态

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95874](https://developer.work.weixin.qq.com/document/path/95874)
- **文档 ID**: `95874`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/license/get_auto_active_status?provider_access_token=ACCESS_TOKEN`

## 接口描述

服务商可以调用该接口查询授权企业的许可自动激活状态。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 应用服务商的接口调用凭证，获取方法参见服务商的凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| corpid | string | 是 | 查询的企业corpid。要求服务商为企业购买过接口许可才有查询结果。 |

### 请求示例

```json
{
	"corpid": "CORPID1"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| auto_active_status | uint32 | 许可自动激活状态。0：关闭，1：打开 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"auto_active_status": 1
}
```
