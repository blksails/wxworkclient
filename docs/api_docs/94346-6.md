# 获取launch_code

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94346](https://developer.work.weixin.qq.com/document/path/94346)
- **文档 ID**: `94346`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/get_launch_code?access_token=ACCESS_TOKEN`

## 接口描述

获取唤起页面的code，用于URI scheme客户端协议的launch_code参数。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| operator_userid | string | 是 | 当前操作者的userid |
| single_chat.userid | string | 是 | 需要发起单聊对象的userid，仅支持企业内部成员 |

### 请求示例

```json
{
   "operator_userid":"zhangshan",
   "single_chat":{
		"userid":"lisi"
	}
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| launch_code | string | 唤起页面的code，5分钟内有效，只能消费一次。用于URI scheme客户端协议的launch_code参数 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
	"launch_code":"xxxx"
}
```
