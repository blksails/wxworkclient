# 获取访问用户身份

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91711](https://developer.work.weixin.qq.com/document/path/91711)
- **文档 ID**: `91711`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/getuserinfo3rd`

## 接口描述

获取访问用户的身份信息。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| suite_access_token | string | 是 | 第三方应用的 suite_access_token |
| code | string | 是 | 通过成员授权获取到的 code，最大为512字节。每次成员授权带上的 code 将不一样，code 只能使用一次，5分钟未被使用自动过期。 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| CorpId | string | 用户所属企业的 corpid |
| UserId | string | 用户在企业内的 UserID |
| DeviceId | string | 手机设备号(由企业微信在安装时随机生成，删除重装会改变，升级不受影响) |

### 响应示例

#### 示例 1: 用户属于某个企业示例

```json
{
   "errcode": 0,
   "errmsg": "ok",
   "CorpId":"CORPID",
   "UserId":"USERID",
   "DeviceId":"DEVICEID"
}
```

#### 示例 2: 用户为学校家长示例

```json
{
   "errcode": 0,
   "errmsg": "ok",
   "CorpId":"wxCorpId1",
   "DeviceId":"DEVICEID",
   "external_userid": "zhangsan_external",
   "parents":[
		{
			"corpid": "wxCorpId1",
			"parent_userid": "zhangsan",
			"external_userid": "zhangsan_external"
		},
		{
			"corpid": "wxCorpId2",
			"parent_userid": "lisi",
			"external_userid": "lisi_external"
		}
   ]
}
```

#### 示例 3: 用户不属于任何企业示例

```json
{
   "errcode": 0,
   "errmsg": "ok",
   "OpenId":"OPENID",
   "DeviceId":"DEVICEID"
}
```

#### 示例 4: 出错示例

```json
{
   "errcode": 40029,
   "errmsg": "invalid code"
}
```

## 其他说明

### 权限说明

跳转的域名须完全匹配 access_token 对应第三方应用的可信域名，否则会返回 50001 错误。
