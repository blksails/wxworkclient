# 导出任务完成通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96298](https://developer.work.weixin.qq.com/document/path/96298)
- **文档 ID**: `96298`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webhook/callback`

## 接口描述

用于接收任务执行完毕的结果通知，事件回调到提交任务的应用配置的回调地址。

## 请求信息

### 请求示例

```xml
<xml>
	<ToUserName><![CDATA[wx28dbb14e3720FAKE]]></ToUserName>
	<FromUserName><![CDATA[FromUser]]></FromUserName>
	<CreateTime>1425284517</CreateTime>
	<MsgType><![CDATA[event]]></MsgType>
	<Event><![CDATA[batch_job_result]]></Event>
	<BatchJob>
		<JobId><![CDATA[jobid_S0MrnndvRG5fadSlLwiBqiDDbM143UqTmKP3152FZk4]]></JobId>
		<JobType><![CDATA[export_user]]></JobType>
		<ErrCode>0</ErrCode>
		<ErrMsg><![CDATA[ok]]></ErrMsg>
	</BatchJob>
</xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| ToUserName | string | 企业微信CorpID |
| FromUserName | string | 此时固定为sys |
| CreateTime | int32 | 消息创建时间（整型） |
| MsgType | string | 消息类型，此时固定为：event |
| Event | string | 事件类型：batch_job_result |
| BatchJob.JobId | string | 异步任务id，最大长度为64字符 |
| BatchJob.JobType | string | 操作类型，字符串，在异步导出的场景下分别有：export_user(导出成员详情)、 export_simple_user(导出成员）、export_department(导出部门）、export_tag(导出标签成员) |
| BatchJob.ErrCode | int32 | 返回码 |
| BatchJob.ErrMsg | string | 对返回码的文本描述内容 |
