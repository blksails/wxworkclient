#### 示例 1: 功能描述

```json
ww.createChatWithMsg({selectedOpenUserIds:['zhangsan','lisi'],selectedTickets:['tick1','token2'],chatName:'discussName',msg:{msgtype:'link',link:{title:'title1',desc:'desc1',url:'link1',imgUrl:'imgurl1'}}})
```

#### 示例 2: 接口定义

```json
wx.invoke("createChatWithMsg",{selectedOpenUserIds:["zhangsan","lisi"],selectedTickets:["tick1","token2"],chatName:"discussName",msg:{msgtype:"link",link:{title:"title1",desc:"desc1",url:"link1",imgUrl:"imgurl1"}}},function(res){if(res.err_msg=="createChatWithMsg:ok"){varchatId=res.chatId;// 新建的会话ID，当会话为单聊时不返回此字段}});
```

### 功能描述

向用户申请给指定范围发送消息。
调用接口后，用户可在选人界面对群聊范围进行修改，当创建群聊成功时会返回新建的群聊 ID。

### 使用限制

仅第三方应用（非通讯录应用）与代开发应用可调用本接口必须使用应用身份进行注册

### params: Object

属性类型必填说明selectedOpenUserIdsstring否群聊指定的用户 OpenUserID 列表selectedTicketsstring[]否群聊指定的 selectedTicket 列表可通过 selectPrivilegedContact 接口获取chatNamestring是新建群聊指定的群名msgObject否发送的链接消息属性类型必填说明msgtype"link"是消息类型linkObject是属性类型必填说明titlestring否H5消息标题imgUrlstring否H5消息封面图片URLdescstring否H5消息摘要urlstring是H5消息页面urlsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
