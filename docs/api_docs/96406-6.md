# 修改客服账号 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96406](https://developer.work.weixin.qq.com/document/path/96406)
- **文档 ID**: `96406`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/kf/account/update?access_token=ACCESS_TOKEN`

## 接口描述

修改已有的客服账号，可修改客服名称和头像。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| open_kfid | string | 是 | 要修改的客服账号ID。不多于64字节 |
| name | string | 否 | 新的客服名称，如不需要修改可不填。不多于16个字符 |
| media_id | string | 否 | 新的客服头像临时素材，如不需要修改可不填。可以调用上传临时素材接口获取。不多于128个字节 |
| 自建应用 | 自建应用 | 否 | 配置到「微信客服- 可调用接口的应用」中 |
| 第三方应用 | 第三方应用 | 否 | 具有“微信客服->管理账号、分配会话和收发消息”权限 |
| 代开发自建应用 | 代开发自建应用 | 否 | 具有“微信客服->管理账号、分配会话和收发消息”权限 |

### 请求示例

```json
{"open_kfid":"wkAJ2GCAAAZSfhHCt7IFSvLKtMPxyJTw","name":"修改客服名","media_id":"294DpAog3YA5b9rTK4PjjfRfYLO0L5qpDHAJIzhhQ2jAEWjb9i661Q4lk8oFnPtmj"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
