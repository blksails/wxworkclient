# 使用系统浏览器打开指定 URL

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100602](https://developer.work.weixin.qq.com/document/path/100602)
- **文档 ID**: `100602`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

使用系统浏览器打开指定 URL，支持传入 oauth2 链接，实现在系统浏览器内免登录的效果。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| params.url | string | 是 | 在默认浏览器打开的 URL。若 URL 为 oauth2 链接，跳转时将附带上 code 参数。 |
| params.success | Function | 否 | 成功回调 |
| params.fail | Function | 否 | 失败回调 |
| params.cancel | Function | 否 | 取消回调 |
| params.complete | Function | 否 | 完成回调 |

### 请求示例

```text
ww.openDefaultBrowser({
  url: 'https://work.weixin.qq.com/'
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errMsg | string | 通用错误信息 |
| errCode | number | 通用错误码 |
