# 企业微信上的多场景调试

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92383](https://developer.work.weixin.qq.com/document/path/92383)
- **文档 ID**: `92383`
- **API 名称**: `multi_scene_debug`
- **请求方法**: `GET`
- **分组信息**: 第 2 个接口，共 2 个

## 接口描述

在企业微信聊天工具栏、聊天会话等场景打开小程序时，默认都是打开小程序正式包，没法对debug包进行调试预览。企业微信从3.0.28版本开始支持通过以下方式开启多场景调试，开启之后，在附件栏、工具栏、会话、工作台等场景打开此小程序时，都将使用本次扫码的开发版小程序来运行。

## 响应信息

### 响应示例

```text
1. 在微信小程序开发者工具生成小程序的预览二维码，然后使用企业微信扫码打开小程序
2. 点击小程序右上角菜单，找到“开启多场景调试”并点击开启
```
