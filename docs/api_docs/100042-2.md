## 接口描述

专区程序接收事件通知，企业微信通过 HTTP 请求回调到专区，专区需要启动一个 HTTP 服务并监听 8080 端口。

## 其他说明

### 概述

如果用户在企业微信操作，产生了相关事件，企业微信会通过 HTTP 请求回调到专区。专区需要启动一个 HTTP 服务，并监听 8080 端口。回调到该服务的根路径。专区接收到请求后，需要将 HTTP 请求的 header 以及 body 传给 SDK 解密接口。

### http请求body

HTTP 服务接收到的 HTTP 的 body 为加密的，需要调用 SDK 进行解密，解密后拿到消息原文，各个通知事件的协议参见本目录下各个文档。
