### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| single_chat_tools |  | 否 | 从单聊会话的工具栏进入 |
| group_chat_tools |  | 否 | 从群聊会话的工具栏进入 |

## 其他说明

### 概述

企业可将应用页面配置到聊天工具栏，方便成员在与客户的聊天中查看和使用，提高服务效率。从聊天工具栏打开的页面，除了一般的JSAPI之外，可以使用有会话属性的JSAPI：
获取当前客户ID获取当前客户群的群ID分享消息到当前会话

### 支持的入口环境

开发者首先需要调用ww.getContext接口获取当前入口entry值，在聊天工具栏里返回的entry值是single_chat_tools或group_chat_tools，具体如下：
值说明支持的最低版本single_chat_tools从单聊会话的工具栏进入3.0.24group_chat_tools从群聊会话的工具栏进入3.0.24
