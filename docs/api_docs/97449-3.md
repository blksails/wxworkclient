## 其他说明

### ww.checkSchedule(params)

兼容性：企业微信 4.0.20 开始支持
功能描述查看其他成员某段时间内日程中的闲忙状态。
ww.checkSchedule({start_time:1667232000,end_time:1667318400,users:['jack','jason']})
使用限制本接口必须使用应用身份进行注册当前成员必须在应用可见范围内应用需具有日程使用权限
参数说明params: Object属性类型必填说明start_timenumber是开始时间，unix 时间戳end_timenumber是结束时间，unix 时间戳usersstring[]否成员 userid 列表第三方应用与代开发应用须传入 open_userid 列表，不填或列表为空则仅查看用户本人的日程闲忙状态successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

查看其他成员某段时间内日程中的闲忙状态。

### 使用限制

本接口必须使用应用身份进行注册当前成员必须在应用可见范围内应用需具有日程使用权限

### params: Object

属性类型必填说明start_timenumber是开始时间，unix 时间戳end_timenumber是结束时间，unix 时间戳usersstring[]否成员 userid 列表第三方应用与代开发应用须传入 open_userid 列表，不填或列表为空则仅查看用户本人的日程闲忙状态successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 示例代码

wx.invoke('checkSchedule',{"start_time":1667232000,"end_time":1667318400,"users":["jack","jason"]},function(res){if(res.err_msg!="checkSchedule:ok"){//错误处理}});
