## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| notify_id |  | 否 | 可指定notify_id，见SDK生成notify_id。如果不指定，则由接口自动生成。不超过64字节。 |
| notify_scene |  | 否 | 通知场景值，可用于标识此次通知对应的事件类型。取值范围：0~255，满足条件则通过应用接收专区通知回调给开发者，否则忽略该字段。 |

### 请求示例

#### 示例 1: 请求包体：

```json
{"notify_id":"NOTIFY_ID","notify_scene":1}
```

#### 示例 2: Python

```text
fromwwspecapisdkimportspec_gen_notify_idnotify_id=spec_gen_notify_id()
```

#### 示例 3: C++

```text
#include"cpp_sdk.h"std::string notify_id=GenerateNotifyId();
```

#### 示例 4: Java

```text
importcom.tencent.wework.SpecUtil;Stringnotify_id=SpecUtil.GenerateNotifyId();
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 错误码 |
| errmsg |  | 错误码说明 |
| notify_id |  | 10分钟内有效，同时会通过应用接收专区通知回调给开发者。如果请求包体指定了notify_id，则直接采用并返回，否则由接口自动生成。 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","notify_id":"NOTIFY_ID"}
```
