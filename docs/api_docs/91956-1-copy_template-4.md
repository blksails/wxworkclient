# 复制/更新模板到企业

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91956](https://developer.work.weixin.qq.com/document/path/91956)
- **文档 ID**: `91956`
- **API 名称**: `copy_template`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/xxx/copy_template?access_token=ACCESS_TOKEN`
- **分组信息**: 第 1 个接口，共 5 个

## 接口描述

将服务商后台配置好的审批模板，作为第三方应用模板，安装到指定企业的审批应用中。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |
