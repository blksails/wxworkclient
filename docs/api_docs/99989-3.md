# 上报异步任务结果 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99989](https://developer.work.weixin.qq.com/document/path/99989)
- **文档 ID**: `99989`

## 接口描述

可以通过这个接口上报异步任务的结果。

## 请求信息

### 请求示例

```json
{"errcode":0,"result":"aaaabbbccc"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 错误码，为0表示任务执行成功 |
| result |  | 任务结果。返回的结果必须符合模型或程序的输出协议。 |
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
