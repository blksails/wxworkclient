# 获取当前客户群的群ID

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93593](https://developer.work.weixin.qq.com/document/path/93593)
- **文档 ID**: `93593`
- **请求方法**: `wx.qy.getCurExternalChat`
- **接口地址**: `wx.qy.getCurExternalChat`

## 接口描述

获取当前客户群的群ID

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| Object object | object | 是 | 参数对象 |
| object.success | Function | 否 | 接口调用成功的回调函数 |
| object.fail | Function | 否 | 接口调用失败的回调函数 |
| object.complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```text
wx.qy.getCurExternalChat ({
  success: function(res) {
    var chatId = res.chatId //返回当前外部群的群聊ID
  }
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| res | object | 返回结果对象 |
| res.chatId | String | 返回当前外部群的群聊ID |

## 其他说明

### 调用前提

必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态
当前成员必须在应用的可见范围。
需要从特定入口进入页面才可调用，否则会报错：without context of external contact，可先通过调用wx.qy.getContext来判断进入小程序的入口，允许调用的入口说明参见“不同入口所需的权限说明”。
从不同的入口进入的页面，应用需要满足相应的权限，否则会报“no permission”错误，所需的权限参见“不同入口所需的权限说明”。
「营销获客」类应用仅可获取该应用带来的客户群

### 不同入口所需的权限说明

| 入口 | getContext的entry值 | 自建应用 | 第三方应用 | 用户 | 支持的最低版本 |
| --- | --- | --- | --- | --- | --- |
| 外部群聊工具栏 | group_chat_tools | 需有客户联系功能权限 | 需有“企业客户权限->客户基础信息”权限 | 配置了客户联系功能的成员 | 3.0.16 |
| 班级群的聊天工具栏 | group_chat_tools | 所有 | 需有「家校沟通」使用权限 | 所有 | 3.0.36 |
