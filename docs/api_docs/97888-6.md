# 删除成员 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97888](https://developer.work.weixin.qq.com/document/path/97888)
- **文档 ID**: `97888`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_acl_del?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于删除指定文件的成员。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | Query参数，示例值: ACCESS_TOKEN |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| auth_info.type | int | 否 | (从请求示例中推测) |
| auth_info.userid | string | 否 | (从请求示例中推测) |

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fileid | string | 是 | 文件fileid |
| auth_info | obj[] | 是 | 被移除的成员信息 |
| type(后续将废弃) | uint32 | 是 | 成员类型 1:个人 2:部门 |
| userid | string | 是 | 成员userid,字符串 |
| departmentid(后续将废弃) | uint32 | 是 | 部门departmentid, 32位整型范围是[0, 2^32) (type为2时填写) |

### 请求示例

```json
{"fileid":"FILEID","auth_info":[{"type":1,"userid":"USERID1"},{"type":2,"departmentid":1}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
