## 其他说明

### 企业互联共享应用事件回调

本事件触发时机为：
集团企业把第三方应用共享给下属企业使用集团企业把下属企业从共享的第三方应用中移除
请求包体：
<xml><SuiteId><![CDATA[ww4asffe99exxx0f4c]]></SuiteId><InfoType><![CDATA[share_agent_change]]></InfoType><TimeStamp>1403610513</TimeStamp><AppId>11</AppId><CorpId>![CDATA[ww11111]]</CorpId><AgentId>11</AgentId></xml>
参数说明：
参数说明SuiteId第三方应用的SuiteIdInfoTypeshare_agent_changeTimeStamp时间戳AppId旧的多应用套件中的对应应用id，新开发者请忽略CorpId上级企业corpidAgentId上级企业应用id

### 上下游共享应用事件回调

本事件触发时机为：
上游企业把第三方应用共享给下游企业使用上游企业把下游企业从共享的第三方应用中移除
请求包体：
<xml><SuiteId><![CDATA[ww4asffe99exxx0f4c]]></SuiteId><InfoType><![CDATA[share_chain_change]]></InfoType><TimeStamp>1403610513</TimeStamp><AppId>11</AppId><CorpId>![CDATA[ww11111]]</CorpId><AgentId>11</AgentId></xml>
参数说明：
参数说明SuiteId第三方应用的SuiteIdInfoTypeshare_chain_changeTimeStamp时间戳AppId旧的多应用套件中的对应应用id，新开发者请忽略CorpId上游企业corpidAgentId上游企业应用id
