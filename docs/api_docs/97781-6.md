# 修改文档通知范围及权限 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97781](https://developer.work.weixin.qq.com/document/path/97781)
- **文档 ID**: `97781`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/mod_doc_member?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于修改文档、表格、智能表格通知范围列表，可以新增文档、表格、智能表格通知范围并设置权限、修改已有范围的权限以及删除文档、表格、智能表格通知范围内的人员

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 是 | 操作的文档id |
| update_file_member_list | obj[] | 否 | 更新文档通知范围的列表, 批次大小最大100 |
| type | uint32 | 是 | 文档通知范围的类型 1:用户。文档通知范围仅支持按人配置 |
| auth | uint32 | 是 | 文档通知范围内人员获得的权限 1:只读权限 2:读写权限（目前仅智能表可设置为读写权限） 7:管理员权限 |
| userid | string | 否 | 企业内成员的ID |
| type | uint32 | 是 | 文档通知范围的类型 1:用户。文档通知范围仅支持按人配置 |
| userid | string | 否 | 企业内成员的ID |

### 请求示例

```json
{"docid":"DOCID","update_file_member_list":[{"type":1,"auth":7,"userid":"USERID1"}],"del_file_member_list":[{"type":1,"userid":"USERID2"},{"type":1,"tmp_external_userid":"TMP_EXTERNAL_USERID2"}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| tmp_external_userid | string | 外部用户临时id。同一个用户在不同的文档中返回的该id不一致。 |
| del_file_member_list | obj[] | 删除的文档通知范围列表，批次大小最大一百 |
| tmp_external_userid | string | 外部用户临时id。同一个用户在不同的文档中返回的该id不一致。 |
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
