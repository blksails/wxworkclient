# 回调通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95846](https://developer.work.weixin.qq.com/document/path/95846)
- **文档 ID**: `95846`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webdrive/callback?access_token=ACCESS_TOKEN`

## 接口描述

企业微信会将特定的事件推送给微盘所属应用配置的回调URL。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌，用于验证身份 |

### 请求示例

```json
{
  "event": "file_upload",
  "timestamp": 1641025200,
  "file_id": "file_id_123",
  "file_name": "example.txt"
}
```

## 其他说明

### 概述

为了让企业/开发者实时获取授权企业微盘的状态，企业微信会将特定的事件推送给微盘所属应用配置的回调URL。
