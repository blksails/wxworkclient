# 使用JSAPI接口

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93644](https://developer.work.weixin.qq.com/document/path/93644)
- **文档 ID**: `93644`
- **API 名称**: `use_jsapi`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/use_jsapi?access_token=ACCESS_TOKEN`
- **分组信息**: 第 4 个接口，共 6 个

## 接口描述

需要使用下级企业 jsapi_ticket 来进行jsapi签名，并用下级企业的身份调用agentConfig。
