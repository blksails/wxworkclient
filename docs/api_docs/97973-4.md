# 空间变更事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97973](https://developer.work.weixin.qq.com/document/path/97973)
- **文档 ID**: `97973`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webdrive/space_change_notify`

## 接口描述

当接口指定的管理员在企业微信客户端，对应用创建的空间进行变更，触发该事件。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName | string | 是 | 企业微信CorpID |
| FromUserName | string | 是 | 此事件该值固定为`sys`，表示该消息由系统生成 |
| CreateTime | uint32 | 是 | 消息创建时间，unix时间戳 |
| MsgType | string | 是 | 消息类型，此时固定为：`event` |
| Event | string | 是 | 事件类型，此时固定为：`wedrive_space_change` |
| ChangeType | string | 是 | 变更类型，此处目前固定为：`dismiss_space` |
| SpaceId | string | 是 | 空间spaceid |

### 请求示例

```xml
<xml>
   <ToUserName><![CDATA[toUser]]></ToUserName>
   <FromUserName><![CDATA[sys]]></FromUserName>
   <CreateTime>1348831860</CreateTime>
   <MsgType><![CDATA[event]]></MsgType>
   <Event><![CDATA[wedrive_space_change]]></Event>
   <ChangeType><![CDATA[dismiss_space]]></ChangeType>
   <SpaceId><![CDATA[spxxxxxxxxxxxxx]]></SpaceId>
</xml>
```
