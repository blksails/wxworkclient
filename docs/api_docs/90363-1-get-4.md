# 获取指定的应用详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90363](https://developer.work.weixin.qq.com/document/path/90363)
- **文档 ID**: `90363`
- **API 名称**: `get`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/agent/get`
- **分组信息**: 第 1 个接口，共 2 个

## 接口描述

获取指定的应用详情

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| agentid | string | 是 | 应用id |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 出错返回码，为0表示成功，非0表示调用失败 |
| errmsg | string | 返回码提示语 |
| agentid | int32 | 企业应用id |
| name | string | 企业应用名称 |
| square_logo_url | string | 企业应用方形头像 |
| description | string | 企业应用详情 |
| allow_userinfos | object | 企业应用可见范围（人员），其中包括userid |
| allow_userinfos.user | object[] | 用户列表 |
| allow_userinfos.user[].userid | string | 用户ID |
| allow_partys | object | 企业应用可见范围（部门） |
| allow_partys.partyid | int32[] | 部门ID列表 |
| allow_tags | object | 企业应用可见范围（标签） |
| allow_tags.tagid | int32[] | 标签ID列表 |
| close | int32 | 企业应用是否被停用。0：未被停用；1：被停用 |
| redirect_domain | string | 企业应用可信域名 |
| report_location_flag | int32 | 企业应用是否打开地理位置上报 0：不上报；1：进入会话上报； |
| isreportenter | int32 | 是否上报用户进入应用事件。0：不接收；1：接收 |
| home_url | string | 应用主页url |
| customized_publish_status | int32 | 代开发自建应用返回该字段，表示代开发发布状态。0：待开发（企业已授权，服务商未创建应用）；1：开发中（服务商已创建应用，未上线）；2：已上线（服务商已上线应用且不存在未上线版本）；3：存在未上线版本（服务商已上线应用但存在未上线版本） |
