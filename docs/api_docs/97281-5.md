# 家校通讯录批量变更事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97281](https://developer.work.weixin.qq.com/document/path/97281)
- **文档 ID**: `97281`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webhook/callback`

## 接口描述

当学校在家校通讯录中，成员变更或部门变更时，若同一个学校短时间内触发多个变更事件，为了提高回调效率，则会将这些变更合并为一个批量的事件回调到第三方应用/套件的指令回调URL。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 第三方应用ID |
| AuthCorpId | string | 是 | 授权企业的CorpID |
| InfoType | string | 是 | 固定为change_school_contact_batch |
| TimeStamp | string | 是 | 最终回调的时间戳 |
| ChangeList | object[] | 是 | 变更列表。目前最大支持合并1000条，后续可能会根据情况调整，以实际收到的数据为准。 |

### 请求示例

```xml
<xml>
  <SuiteId>wwSuiteId</SuiteId>
  <AuthCorpId>wxAuthCorpId</AuthCorpId>
  <InfoType>change_school_contact_batch</InfoType>
  <TimeStamp>1403610513</TimeStamp>
  <ChangeList>
    <TimeStamp>1403610513</TimeStamp>
    <ChangeType>create_student</ChangeType>
    <Id>zhangsan</Id>
  </ChangeList>
  ...
</xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| SuiteId | string | 第三方应用ID |
| AuthCorpId | string | 授权企业的CorpID |
| InfoType | string | 固定为change_school_contact_batch |
| TimeStamp | string | 最终回调的时间戳 |
| ChangeList | object[] | 变更列表 |
| ChangeList[].ChangeType | string | 此处固定为create_student |
| ChangeList[].Id | string | 学生的家校通讯录userid或者家长的id或者家校通讯录部门id |
| ChangeList[].NewId | string | 当ChangeType为update_student或者update_parent时，变更后的学生的家校通讯录userid或者家长的id |
| ChangeList[].TimeStamp | string | 产生变更的时间戳 |
