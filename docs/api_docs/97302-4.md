# 客服账号授权变更事件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97302](https://developer.work.weixin.qq.com/document/path/97302)
- **文档 ID**: `97302`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webhook/callback`

## 接口描述

当应用授权的客服账号发生变化（新增/取消客服账号的授权），触发该回调。

## 请求信息

### 请求示例

```xml
<xml>
   <ToUserName><![CDATA[toUser]]></ToUserName>
   <FromUserName><![CDATA[sys]]></FromUserName>
   <CreateTime>1348831860</CreateTime>
   <MsgType><![CDATA[event]]></MsgType>
   <Event><![CDATA[kf_account_auth_change]]></Event>
   <AuthAddOpenKfId><![CDATA[wkxxxx1]]></AuthAddOpenKfId>
   <AuthDelOpenKfId><![CDATA[wkxxxx2]]></AuthDelOpenKfId>
</xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| ToUserName | string | 企业微信CorpID |
| FromUserName | string | 此事件该值固定为`sys`，表示该消息由系统生成 |
| CreateTime | uint32 | 消息创建时间，unix时间戳 |
| MsgType | string | 消息类型，此时固定为：`event` |
| Event | string | 事件类型，此时固定为：`kf_account_auth_change` |
| AuthAddOpenKfId | string | 新增授权的客服账号列表，多个`AuthAddOpenKfId`节点表示多个新增账号 |
| AuthDelOpenKfId | string | 取消授权的客服账号列表，多个`AuthDelOpenKfId`节点表示多个取消账号 |
