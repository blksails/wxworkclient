# 查询成员用户是否已授权

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94514](https://developer.work.weixin.qq.com/document/path/94514)
- **文档 ID**: `94514`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/user/check_member_auth`

## 接口描述

当企业当前授权模式为成员授权时，可调用该接口查询成员用户是否已授权

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 企业调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| open_userid | string | 是 | 企业成员的 open_userid。全局唯一，对于同一个服务商不同的应用 open_userid 是相同的；同一用户，对于不同服务商 open_userid 是不同的 |

### 请求示例

```json
{
	"open_userid": "xxx"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| is_member_auth | bool | 成员授权模式下，该成员已授权 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"is_member_auth": true
}
```

## 其他说明

### 权限说明

仅企业为成员授权模式下可调用
