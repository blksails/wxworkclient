# 选择文档 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99187](https://developer.work.weixin.qq.com/document/path/99187)
- **文档 ID**: `99187`

## 接口描述

接口说明：调用该接口，可唤起选择器窗口，选择一个或多个文档。选中后返回对应文档的url。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| selectedFileNum | 是 | 否 | 选择文件的数量。1表示单选，大于1表示多选，上限为50 |
| success | 否 | 否 | 接口调用成功的回调函数 |
| fail | 否 | 否 | 接口调用失败的回调函数 |
| complete | 否 | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```json
wx.qy.wedocSelectDoc({selectedFileNum:1,success:function(res){varselectedFileUrls=res.result.selectedFileUrls;// 后续废弃varselectedFileInfos=res.result.selectedFileInfos;},fail:function(res){console.log(res)},});
```

## 响应信息
