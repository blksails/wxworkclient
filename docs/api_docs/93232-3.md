## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
|  | string | 是 | 通过企业微信创建群聊接口返回的 chatId |
|  | string | string[] | 是 | 参与会话的企业成员列表 |
|  | Function | 否 | 成功回调 |
|  | Function | 否 | 失败回调 |
|  | Function | 否 | 取消回调 |
|  | Function | 否 | 完成回调 |

### 请求示例

#### 示例 1: 功能描述

```json
ww.updateEnterpriseChat({chatId:'CHATID',userIdsToAdd:['zhangsan','lisi']})
```

#### 示例 2: 示例代码

```json
wx.invoke("updateEnterpriseChat",{chatId:"CHATID",//通过企业微信创建群聊接口返回的chatIduserIdsToAdd:'zhangshan;lisi;wangwu',//参与会话的企业成员列表，格式为userid1;userid2;...，用分号隔开。},function(res){if(res.err_msg=="updateEnterpriseChat:ok"){}});
```

## 其他说明

### ww.updateEnterpriseChat(params)

兼容性：企业微信 iOS、Android、PC 3.0.36 开始支持
功能描述变更群成员。
ww.updateEnterpriseChat({chatId:'CHATID',userIdsToAdd:['zhangsan','lisi']})
使用限制本接口必须使用应用身份进行注册目前仅支持添加企业内部成员仅支持客户群调用
参数说明params: Object属性类型必填说明chatIdstring是通过企业微信创建群聊接口返回的 chatIduserIdsToAddstring | string[]是参与会话的企业成员列表successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

变更群成员。

### 使用限制

本接口必须使用应用身份进行注册目前仅支持添加企业内部成员仅支持客户群调用

### params: Object

属性类型必填说明chatIdstring是通过企业微信创建群聊接口返回的 chatIduserIdsToAddstring | string[]是参与会话的企业成员列表successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 示例代码

wx.invoke("updateEnterpriseChat",{chatId:"CHATID",//通过企业微信创建群聊接口返回的chatIduserIdsToAdd:'zhangshan;lisi;wangwu',//参与会话的企业成员列表，格式为userid1;userid2;...，用分号隔开。},function(res){if(res.err_msg=="updateEnterpriseChat:ok"){}});
