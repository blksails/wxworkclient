# ID查询接口

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96078](https://developer.work.weixin.qq.com/document/path/96078)
- **文档 ID**: `96078`
- **API 名称**: `get_userid_list`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/user/get_userid_list`
- **分组信息**: 第 1 个接口，共 14 个

## 接口描述

获取企业所有成员的open_userid

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| userid_list | string[] | 成员的open_userid列表 |
