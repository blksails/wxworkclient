# 授权组织架构权限通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97378](https://developer.work.weixin.qq.com/document/path/97378)
- **文档 ID**: `97378`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/callback/authorize_org_arch`

## 接口描述

当企业在管理端后台点击同意授权第三方应用的“组织架构”权限时，会触发该事件的回调。该事件会推送到第三方应用的指令回调URL上。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 第三方应用ID |
| AuthCorpId | string | 是 | 授权企业的CorpID |
| InfoType | string | 是 | 事件类型，此时固定为corp_arch_auth |
| TimeStamp | string | 是 | 时间戳 |

### 请求示例

```xml
<xml>
	<SuiteId><![CDATA[ww4asffe99e54c0fxxxx]]></SuiteId>
	<AuthCorpId><![CDATA[wxf8b4f85f3a79xxxx]]></AuthCorpId>
	<InfoType><![CDATA[corp_arch_auth]]></InfoType>
	<TimeStamp>1403610513</TimeStamp>
</xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
