# 推送suite_ticket - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97173](https://developer.work.weixin.qq.com/document/path/97173)
- **文档 ID**: `97173`
- **请求方法**: `POST`

## 接口描述

企业微信服务器会定时（每十分钟）向指令回调url推送ticket，suite_ticket实际有效期为30分钟，ticket会实时变更，并用于后续接口的调用。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId |  | 否 | 第三方应用的SuiteId或者代开发应用模板id |
| InfoType |  | 否 | suite_ticket |
| TimeStamp |  | 否 | 时间戳 |
| SuiteTicket |  | 否 | Ticket内容，最长为512字节 |

### 请求示例

```xml
<xml><SuiteId><![CDATA[ww4asffe99e54c0fxxxx]]></SuiteId><InfoType><![CDATA[suite_ticket]]></InfoType><TimeStamp>1403610513</TimeStamp><SuiteTicket><![CDATA[asdfasfdasdfasdf]]></SuiteTicket></xml>
```
