# 获取空间信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97858](https://developer.work.weixin.qq.com/document/path/97858)
- **文档 ID**: `97858`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/space_info?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于获取空间的成员列表、信息、权限等信息。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spaceid | string | 是 | 空间spaceid |
| userid | string | 否 | 成员userid,字符串 |

### 请求示例

```json
{"spaceid":"SPACEID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| spaceid | string | 空间spaceid |
| space_name | string | 空间名称 |
| auth_list | obj[] | 空间成员列表 |
| auth_info | obj[] | 空间成员信息 |
| type | uint32 | 成员类型 1:个人 2:部门 |
| departmentid | uint32 | 部门departmentid, 32位整型范围是[0, 2^32) |
| auth | uint32 | 成员权限 1:仅下载 4:可预览 7:应用空间管理员 |
| quit_userid | string[] | 空间无权限成员userid (成员在一个有权限的部门中, 自己退出空间或者被移除权限) |
| space_sub_type | uint32 | 空间类型 0:普通 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","space_info":{"spaceid":"SPACEID","space_name":"SPACE_NAME","auth_list":{"auth_info":[{"type":1,"userid":"USERID1","auth":3,},{"type":1,"userid":"USERID2","auth":2},{"type":2,"departmentid":DEPARTMENTID1,"auth":1}],"quit_userid":["USERID3","USERID4"]}"space_sub_type":0}}
```
