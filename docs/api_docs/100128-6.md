# 截屏/录屏管理 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100128](https://developer.work.weixin.qq.com/document/path/100128)
- **文档 ID**: `100128`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/security/get_screen_oper_record?access_token=ACCESS_TOKEN`

## 接口描述

启用了 “截屏/录屏管理”的企业可以通过本接口查询截屏操作记录。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 应用调用凭证 |
| start_time | int | 是 | 开始时间 |
| end_time | int | 是 | 结束时间，开始时间到结束时间的范围不能超过14天 |
| userid_list | array | 否 | 需要查询的截屏操作者的userid，单次最多可以传100个用户。设置的userid需要在应用的可见范围内 |
| department_id_list | array | 否 | 需要查询的截屏操作者部门的department_id，单次最多可以传100个部门id。设置的department_id需要在应用的可见范围内 |
| screen_shot_type | int | 否 | 截屏内容的类型，不设置默认为全部1: 聊天2: 通讯录3: 邮件4: 文件5: 日程6: 其他 |
| cursor | string | 否 | 由企业微信后台返回，第一次调用可不填 |
| limit | int | 否 | 限制返回的条数，最多设置为1000 |
| 自建应用 | 自建应用 | 否 | 配置到「截屏/录屏管理 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"start_time":166666666,"end_time":166666667,"userid_list":["zhangsan","lisi"],"department_id_list":[2,3],"screen_shot_type":1,"cursor":"ngLgjieajgieo","limit":100}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| has_more | bool | 是否还有更多数据 |
| next_cursor | string | 仅has_more值为true时返回该字段，下一次调用将该值填到cursor字段，以实现分页查询 |
| record_list.time | int | 操作时间 |
| record_list.userid | string | 企业用户账号id |
| record_list.department_id | int | 企业用户部门id。仅当department_id在应用可见范围内才返回 |
| record_list.screen_shot_type | int | 截屏内容的类型1: 聊天2: 通讯录3: 邮件4: 文件5: 日程6: 其他 |
| record_list.screen_shot_content | string | 截屏内容 |
| record_list.system | string | 企业用户的操作系统，目前支持返回的类型：iOS、iPad |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","has_more":true,"next_cursor":"gejMjgLjgeigoejg","record_list":[{"time":16666666666,"userid":"zhangsan","department_id":2,"screen_shot_type":1,"screen_shot_content":"聊天","system":"iPad"},{"time":16666666666,"userid":"lisi","department_id":3,"screen_shot_type":4,"screen_shot_content":"文件（防泄露文件：test.pdf）","system":"iOS"}]}
```
