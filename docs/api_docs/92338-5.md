# 获取部门学生详情 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92338](https://developer.work.weixin.qq.com/document/path/92338)
- **文档 ID**: `92338`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/user/list?access_token=ACCESS_TOKEN&department_id=DEPARTMENT_ID`

## 接口描述

请求方式：GET（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/school/user/list?access_token=ACCESS_TOKEN&department_id=DEPARTMENT_ID

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| department_id |  | 是 | 获取的部门id |
| 自建应用 | 自建应用 | 否 | 配置到「家校沟通- 读取和编辑家校通讯录的应用」或「家校沟通- 读取家校通讯录的应用」或“家长可使用应用”列表中 |
| 代开发应用 | 代开发应用 | 否 | 具有「家校沟通」权限 |
| 第三方应用 | 第三方应用 | 否 | 具有「家校沟通」使用权限 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| students |  | 学生列表 |
| students.student_userid |  | 学生UserID |
| students.name |  | 学生名字 |
| students.department |  | 学生所在的班级id列表,不超过20个 |
| students.parents |  | 学生家长列表 |
| students.parents.parent_userid |  | 家长的userid |
| students.parents.relation |  | 学生与家长的关系 |
| students.parents.mobile |  | 家长手机号，第三方不可获取 |
| students.parent.is_subscribe |  | 家长是否关注了“学校通知”，0-未关注，1-已关注 |
| students.parent.external_userid |  | 家长的external_userid,仅当家长已关注才返回。 |
