# 读取收集表答案

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97827](https://developer.work.weixin.qq.com/document/path/97827)
- **文档 ID**: `97827`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/get_form_answer`

## 接口描述

该接口用于读取收集表的答案

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| repeated_id | string | 是 | 操作的收集表周期id |
| answer_ids | uint64[] | 是 | 需要拉取的答案列表，批次大小最大100 |

### 请求示例

```json
{
    "repeated_id":"REPEATED_ID1",
    "answer_ids":[
        1
    ]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| answer | object | 答案 |
| answer_list | object[] | 答案列表 |
| answer_id | uint64 | 答案id |
| user_name | string | 用户名 |
| ctime | uint64 | 创建时间 |
| mtime | uint64 | 修改时间 |
| reply | object | 该用户的答案明细 |
| items | object[] | 每个问题的答案 |
| question_id | uint64 | 问题id |
| text_reply | string | 答案 |
| option_reply | uint32[] | 选择题答案，多选题有多个答案 |
| option_extend_reply | object[] | 选择题，其他选项列表 |
| file_extend_reply | object[] | 文件题答案列表 |
| department_reply | object | 部门题答案 |
| member_reply | object | 成员题答案 |
| duration_reply | object | 时长题答案 |
| answer_status | uint32 | 答案状态 1:正常 3:统计者移除此答案或删除 |
| tmp_external_userid | string | 外部用户临时id，匿名填写不返回，同一个用户在不同的收集表中返回的该id不一致。 可进一步通过[tmp_external_userid的转换](#46252)接口转换成外部联系人的external_userid，方便识别外部填写人的身份。 |

### 响应示例

```json
{
  "errcode": 0,
  "errmsg": "ok",
  "answer": {
    "answer_list": [
      {
        "answer_id": 15,
        "user_name": "USER_NAME1",
        "ctime": 1668430580,
        "mtime": 1668430580,
        "reply": {
          "items": [
            {
              "question_id": 1,
              "text_reply": "Ndjnd"
            },
            ...
          ]
        },
        "answer_status": 1,
        "tmp_external_userid": "TMP_EXTERNAL_USERID1"
      }
    ]
  }
}
```
