# 手机号转外部联系人ID

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92506](https://developer.work.weixin.qq.com/document/path/92506)
- **文档 ID**: `92506`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/batch_to_external_userid`

## 接口描述

家校沟通基础接口，用于将家长手机号转换为外部联系人的userid。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| mobiles | array | 是 | 家长手机号列表，最多支持100个 |

### 请求示例

```json
{
    "mobiles": [
        "10000000000",
        "10000000001"
    ]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| success_list | array | 转换成功列表 |
| success_list.mobile | string | 手机号 |
| success_list.external_userid | string | 外部联系人的userid(家长关注后才会返回该字段) |
| fail_list | array | 转换失败列表 |
| fail_list.errcode | int32 | 返回码 |
| fail_list.errmsg | string | 对返回码的文本描述内容 |
| fail_list.mobile | string | 手机号 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "success_list": [
        {
            "mobile": "18900000000",
            "external_userid": "aaabbbcccc"
        }
    ],
    "fail_list": [
        {
            "errcode": 60136,
            "errmsg": "record not found",
            "mobile": "10000000001"
        }
    ]
}
```
