# 邀请成员 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90975](https://developer.work.weixin.qq.com/document/path/90975)
- **文档 ID**: `90975`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/batch/invite?access_token=ACCESS_TOKEN`

## 接口描述

企业可通过接口批量邀请成员使用企业微信，邀请后将通过微信或邮件下发通知。请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/batch/invite?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| user |  | 否 | 成员ID列表, 最多支持1000个。 |
| party |  | 否 | 部门ID列表，最多支持100个。 |
| tag |  | 否 | 标签ID列表，最多支持100个。 |

### 请求示例

```json
{"user":["UserID1","UserID2","UserID3"],"party":[1,2,3],"tag":[101,102,103]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| invaliduser |  | 非法成员列表 |
| invalidparty |  | 非法部门列表 |
| invalidtag |  | 非法标签列表 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","invaliduser":["UserID1","UserID2"],"invalidparty":[1,2],"invalidtag":[101,102]}
```
