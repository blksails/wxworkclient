# 扩展属性的添加方法

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90329](https://developer.work.weixin.qq.com/document/path/90329)
- **文档 ID**: `90329`
- **API 名称**: `添加自定义信息`
- **请求方法**: `POST`
- **接口地址**: `https://p.qpic.cn/pic_wework/23479275/aea815a539bde258a2462e21ac55c4ec8080cda051ee0cb1/0`
- **分组信息**: 第 2 个接口，共 4 个

## 接口描述

登录管理后台->我的企业->通讯录管理->成员资料显示栏，点击修改按钮，会看到修改页面，点击“添加自定义信息”按钮，即可添加。
