# 删除视图

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100200](https://developer.work.weixin.qq.com/document/path/100200)
- **文档 ID**: `100200`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartsheet/delete_views`

## 接口描述

本接口用于在 smartsheet 中的某个子表里删除若干个视图。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 是 | 文档的docid |
| sheet_id | string | 是 | Smartsheet 子表ID |
| view_ids | string[] | 是 | 要删除的视图ID列表 |

### 请求示例

```json
{
	"docid": "DOCID",
	"sheet_id": "123Abc",
	"view_ids": [
		"VIEWID1", "VIEWID2"
	]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok"
}
```

## 其他说明

### 权限说明

* 自建应用需配置到“可调用应用”列表中的应用secret所获取的accesstoken来调用（accesstoken如何获取？）
* 第三方应用需具有“文档”权限
* 代开发自建应用需具有“文档”权限
