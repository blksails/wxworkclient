# 获取客户详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96315](https://developer.work.weixin.qq.com/document/path/96315)
- **文档 ID**: `96315`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/get?access_token=ACCESS_TOKEN&external_userid=EXTERNAL_USERID&cursor=CURSOR`

## 接口描述

企业可通过此接口，根据外部联系人的userid，拉取客户详情。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| external_userid | string | 是 | 外部联系人的userid，注意不是企业成员的账号 |
| cursor | string | 否 | 上次请求返回的next_cursor |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| external_contact | object | 外部联系人信息 |
| external_contact.external_userid | string | 外部联系人的userid |
| external_contact.name | string | 外部联系人的名称 |
| external_contact.position | string | 外部联系人的职位 |
| external_contact.avatar | string | 外部联系人头像 |
| external_contact.corp_name | string | 外部联系人所在企业的简称 |
| external_contact.corp_full_name | string | 外部联系人所在企业的主体名称 |
| external_contact.type | int32 | 外部联系人的类型，1表示微信用户，2表示企业微信用户 |
| external_contact.gender | int32 | 外部联系人性别，0-未知，1-男性，2-女性 |
| external_contact.unionid | string | 外部联系人在微信开放平台的唯一身份标识 |
| external_contact.external_profile | object | 外部联系人的自定义展示信息 |
| follow_user | object[] | 跟进人信息列表 |
| follow_user.userid | string | 添加了此外部联系人的企业成员userid |
| follow_user.remark | string | 该成员对此外部联系人的备注 |
| follow_user.description | string | 该成员对此外部联系人的描述 |
| follow_user.createtime | int32 | 该成员添加此外部联系人的时间 |
| follow_user.tags | object[] | 标签列表 |
| follow_user.tags.group_name | string | 标签分组名称 |
| follow_user.tags.tag_name | string | 标签名称 |
| follow_user.tags.type | int32 | 标签类型, 1-企业设置，2-用户自定义，3-规则组标签 |
| follow_user.tags.tag_id | string | 企业标签的id |
| follow_user.remark_corp_name | string | 该成员对此微信客户备注的企业名称 |
| follow_user.remark_mobiles | string[] | 该成员对此客户备注的手机号码 |
| follow_user.add_way | int32 | 该成员添加此客户的来源 |
| follow_user.wechat_channels | object | 视频号信息 |
| follow_user.wechat_channels.nickname | string | 视频号名称 |
| follow_user.wechat_channels.source | int32 | 视频号添加场景, 0-未知 1-视频号主页 2-视频号直播间 3-视频号留资服务 |
| follow_user.oper_userid | string | 发起添加的userid |
| follow_user.state | string | 企业自定义的state参数 |
| next_cursor | string | 分页的cursor |
