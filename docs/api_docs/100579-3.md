## 其他说明

### ww.startAutoLBS(params)

兼容性：企业微信 2.4.20 开始支持
功能描述打开持续定位。
ww.startAutoLBS({type:'gcj02'})
参数说明params: Object属性类型必填说明typestring是合法值说明wgs84gps 坐标gcj02火星坐标successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

打开持续定位。

### params: Object

属性类型必填说明typestring是合法值说明wgs84gps 坐标gcj02火星坐标successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码
