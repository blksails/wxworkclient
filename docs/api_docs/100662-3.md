ww.openLocation({latitude:0,longitude:0,name:'name',address:'address',scale:1})

wx.openLocation({latitude:0,// 纬度，浮点数，范围为90 ~ -90longitude:0,// 经度，浮点数，范围为180 ~ -180。name:'',// 位置名address:'',// 地址详情说明scale:1,// 地图缩放级别,整形值,范围从1~28。默认为16});

## 其他说明

### ww.openLocation(params)

功能描述使用企业微信内置地图查看位置。
ww.openLocation({latitude:0,longitude:0,name:'name',address:'address',scale:1})
参数说明params: Object属性类型默认值必填说明latitudenumber是纬度范围为 90 ~ -90longitudenumber是经度范围为 180 ~ -180namestring否位置名addressstring否地址详情说明scalenumber16否地图缩放级别范围为 1 ~ 28infoUrlstring否successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

使用企业微信内置地图查看位置。

### params: Object

属性类型默认值必填说明latitudenumber是纬度范围为 90 ~ -90longitudenumber是经度范围为 180 ~ -180namestring否位置名addressstring否地址详情说明scalenumber16否地图缩放级别范围为 1 ~ 28infoUrlstring否successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 示例代码

wx.openLocation({latitude:0,// 纬度，浮点数，范围为90 ~ -90longitude:0,// 经度，浮点数，范围为180 ~ -180。name:'',// 位置名address:'',// 地址详情说明scale:1,// 地图缩放级别,整形值,范围从1~28。默认为16});
