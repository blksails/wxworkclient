## 接口描述

判断当前设备是否支持 NFC 能力

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | function | 否 | 接口调用成功的回调函数 |
| fail | function | 否 | 接口调用失败的回调函数 |
| complete | function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| 0 |  | 否 | 正常 |
| 13000 |  | 否 | 当前设备不支持NFC |

### 请求示例

```json
wx.qy.getNFCReaderState({success:(res)=>{console.log(JSON.stringify(res))// todo what you want},fail:(res)=>{wx.showModal({title:'NFC-Not-Support',content:JSON.stringify(res),})}})
```
