# 修改会议录制共享设置 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98208](https://developer.work.weixin.qq.com/document/path/98208)
- **文档 ID**: `98208`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/record/update_sharing_config?access_token=ACCESS_TOKEN`

## 接口描述

根据会议录制 ID 修改共享等配置，支持修改共享权限、共享密码、共享有效期等信息

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meeting_record_id | string | 是 | 会议录制ID |
| meetingid | string | 是 | 会议ID |
| sharing_config | object | 否 | 共享配置，详见SharingConfig |
| enable_sharing | bool | 否 | 共享开关，是否开启共享，默认为 true。true：开启false：未开启说明：- 未开启时不允许设置以下参数。- 修改为 false 关闭共享后，之前设置的共享设置将不保存。 |
| sharing_auth_type | int32 | 否 | 共享权限类型。0：仅允许登录成员查看1：仅企业内成员可查看2：仅参会成员可查看3：全部成员可查看4：通过权限审批的成员可查看（Rooms 不支持）5：获取微信特邀链接的成员可查看（Rooms 不支持） |
| enable_password | bool | 否 | 是否开启密码，默认为 true。true：开启false：不开启 |
| password | string | 否 | 共享密码，默认随机生成。说明：当 enable_password = true 时，必传；当 enable_password = false 时，不可传。 |
| enable_sharing_expire | bool | 否 | 是否开启共享链接有效期，默认为 false。true：开启false：不开启 |
| sharing_expire | int64 | 否 | 共享链接有效期，unix 时间戳（单位毫秒），默认为空。说明：当 enable_sharing_expire = true 时，必传；当 enable_sharing_expire = false 时，不可传。 |
| allow_download | bool | 否 | 是否允许下载，默认为 false。true：允许下载false：不允许下载 |

### 请求示例

```json
{"meeting_record_id":"record","meetingid":"MEETINGID","sharing_config":{"enable_sharing":true,"sharing_auth_type":0,"enable_password":true,"password":"123352","enable_sharing_expire":true,"sharing_expire":1999999999,"allow_download":true}}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
