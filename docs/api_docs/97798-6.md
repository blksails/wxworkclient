# 获取日程详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97798](https://developer.work.weixin.qq.com/document/path/97798)
- **文档 ID**: `97798`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/oa/schedule/get`

## 接口描述

该接口用于获取指定的日程详情。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| schedule_id_list[] | string | 是 | 日程ID列表。一次最多拉取1000条 |

### 请求示例

```json
{
	"schedule_id_list": [
		"17c7d2bd9f20d652840f72f59e796AAA"
	]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |
| schedule_list | object[] | 日程列表 |
| schedule_list[].schedule_id | string | 日程ID |
| schedule_list[].admins | string | 管理员userid列表 |
| schedule_list[].attendees | object[] | 日程参与者列表 |
| schedule_list[].attendees.userid | string | 日程参与者ID |
| schedule_list[].attendees.response_status | uint32 | 日程参与者的接受状态。 0 - 未处理 1 - 待定 2 - 全部接受 3 - 仅接受一次 4 - 拒绝 |
| schedule_list[].summary | string | 日程标题 |
| schedule_list[].description | string | 日程描述 |
| schedule_list[].reminders | object | 提醒相关信息 |
| schedule_list[].reminders.is_remind | int32 | 是否需要提醒。0-否；1-是 |
| schedule_list[].reminders.is_repeat | int32 | 是否重复日程。0-否；1-是 |
| schedule_list[].reminders.remind_before_event_secs | uint32 | 日程开始前多少秒提醒 |
| schedule_list[].reminders.remind_time_diffs | int32[] | 提醒时间与日程开始时间的差值 |
| schedule_list[].reminders.repeat_type | uint32 | 重复类型 |
| schedule_list[].reminders.repeat_until | uint32 | 重复结束时刻 |
| schedule_list[].reminders.is_custom_repeat | uint32 | 是否自定义重复 |
| schedule_list[].reminders.repeat_interval | uint32 | 重复间隔 |
| schedule_list[].reminders.repeat_day_of_week | uint32[] | 每周周几重复 |
| schedule_list[].reminders.repeat_day_of_month | uint32[] | 每月哪几天重复 |
| schedule_list[].reminders.timezone | uint32 | 时区 |
| schedule_list[].reminders.exclude_time_list | object[] | 重复日程不包含的日期列表 |
| schedule_list[].location | string | 日程地址 |
| schedule_list[].status | uint32 | 日程状态 |
| schedule_list[].start_time | uint32 | 日程开始时间 |
| schedule_list[].end_time | uint32 | 日程结束时间 |
| schedule_list[].is_whole_day | uint32 | 是否全天日程 |
| schedule_list[].cal_id | string | 日程所属日历ID |
