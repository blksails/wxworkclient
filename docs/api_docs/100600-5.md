## 接口描述

批量显示功能按钮。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| params.menuList | string[] | 是 | 要显示的菜单项 |
| params.success | Function | 否 | 成功回调 |
| params.fail | Function | 否 | 失败回调 |
| params.cancel | Function | 否 | 取消回调 |
| params.complete | Function | 否 | 完成回调 |

### 请求示例

```json
ww.showMenuItems({
  menuList: ['menuItem:setFont']
})
```
