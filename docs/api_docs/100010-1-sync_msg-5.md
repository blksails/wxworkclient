# 专区请求企微后台

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100010](https://developer.work.weixin.qq.com/document/path/100010)
- **文档 ID**: `100010`
- **API 名称**: `sync_msg`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/sync_msg?access_token=ACCESS_TOKEN`
- **分组信息**: 第 1 个接口，共 2 个

## 接口描述

用于拉取用户会话id记录、模型分析等功能的接口。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| cursor | string | 是 | 游标 |
| token | string | 是 | 令牌 |
| limit | int32 | 是 | 限制数量 |

### 请求示例

```json
{"cursor":"RMTJID","token":"NGEJKGOEGJKOEGNEOAGOEGOKEGHOEEEEE","limit":200}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| data | string | 数据 |
