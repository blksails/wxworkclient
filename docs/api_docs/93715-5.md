# 获取直播详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93715](https://developer.work.weixin.qq.com/document/path/93715)
- **文档 ID**: `93715`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/living/get_living_info`

## 接口描述

获取直播详情，只能获取本应用创建的直播。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| livingid | string | 是 | 直播ID |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| living_info | object | 直播信息 |
| living_info.theme | string | 直播主题 |
| living_info.living_start | int32 | 直播开始时间戳 |
| living_info.living_duration | int32 | 直播时长，单位为秒 |
| living_info.status | int32 | 直播的状态，0：预约中，1：直播中，2：已结束，3：已过期，4：已取消 |
| living_info.reserve_start | int32 | 直播预约的开始时间戳 |
| living_info.reserve_living_duration | int32 | 直播预约时长，单位为秒 |
| living_info.description | string | 直播的描述，最多支持100个汉字 |
| living_info.anchor_userid | string | 主播的userid |
| living_info.main_department | int32 | 主播所在主部门id |
| living_info.viewer_num | int32 | 观看直播总人数 |
| living_info.comment_num | int32 | 评论数 |
| living_info.mic_num | int32 | 连麦发言人数 |
| living_info.open_replay | int32 | 是否开启回放，1表示开启，0表示关闭 |
| living_info.replay_status | int32 | open_replay为1时才返回该字段。0表示生成成功，1表示生成中，2表示回放已删除，3表示生成失败 |
| living_info.type | int32 | 直播的类型，0：通用直播，1：小班课，2：大班课，3：企业培训，4：活动直播 |
| living_info.push_stream_url | string | 推流地址，仅直播类型为活动直播并且直播状态是待开播返回该字段 |
| living_info.online_count | int32 | 当前在线观看人数 |
| living_info.subscribe_count | int32 | 直播预约人数 |
