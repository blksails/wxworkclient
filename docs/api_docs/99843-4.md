# 应用接收专区通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99843](https://developer.work.weixin.qq.com/document/path/99843)
- **文档 ID**: `99843`
- **请求方法**: `POST`
- **接口地址**: `https://wework.qpic.cn/wwpic3az/819586_HUPnc3hpTgSqQAN_1718099460/0`

## 接口描述

应用收到回调数据后，可以用notifyid请求应用同步调用专区程序接口调用专区程序

## 请求信息

### 请求示例

```xml
<xml>
   <ToUserName><![CDATA[toUser]]></ToUserName>
   <FromUserName><![CDATA[fromUser]]></FromUserName>
   <CreateTime>1348831860</CreateTime>
   <MsgType><![CDATA[event]]></MsgType>
   <Event><![CDATA[program_notify]]></Event>
   <NotifyId><![CDATA[data]]></NotifyId>
   <NotifyScene>1</NotifyScene>
</xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| ToUserName | string | 企业微信CorpID |
| FromUserName | string | 此处固定为sys |
| CreateTime | int32 | 消息创建时间，unix时间戳 |
| MsgType | string | 消息类型，此时固定为：`event` |
| Event | string | 事件类型，此时固定为：`program_notify` |
| NotifyId | string | 通知id，专区通过获取NotifyId接口生成。10分钟内有效，不超过128字节。 |
| NotifyScene | uint32 | 通知场景值，为调用sdk接口[专区调用应用](#53408)时指定的`notify_scene` |
