## 接口描述

小程序插件接入步骤，包括申请插件、引用插件、配置组件等。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 消息发送状态 |
| name | string | 推送的客服人员姓名 |
| headurl | string | 推送的客服人员头像 |
| notifytype | int32 | 消息通知方式：0表示消息通知的方式，1表示二维码的方式 |
