# 设置家校通讯录自动同步模式 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92345](https://developer.work.weixin.qq.com/document/path/92345)
- **文档 ID**: `92345`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/set_arch_sync_mode?access_token=ACCESS_TOKEN`

## 接口描述

企业和第三方可通过此接口修改家校通讯录与班级标签之间的自动同步模式，注意，一旦设置禁止自动同步，将无法再次开启。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| arch_sync_mode |  | 是 | 家校通讯录同步模式：1-禁止将标签同步至家校通讯录，2-禁止将家校通讯录同步至标签，3-禁止家校通讯录和标签相互同步 |
| 自建应用 | 自建应用 | 否 | 配置到「家校沟通-读取和编辑家校通讯录的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 需拥有「家校沟通」可使用家校通讯录-家校通讯录编辑权限 |
| 第三方应用 | 第三方应用 | 否 | 需拥有「家校沟通」使用和编辑权限 |

### 请求示例

```json
{"arch_sync_mode":1}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
