# 重命名文档

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97745](https://developer.work.weixin.qq.com/document/path/97745)
- **文档 ID**: `97745`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/rename_doc`

## 接口描述

该接口用于对指定文档、表格、智能表格及收集表进行重命名。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 否 | 文档docid（docid、formid只能填其中一个），仅可修改应用自己创建的文档 |
| formid | string | 否 | 收集表id（docid、formid只能填其中一个），仅可修改应用自己创建的收集表 |
| new_name | string | 是 | 重命名后的文档名（注意：文档名最多填255个字符, 英文算1个, 汉字算2个, 超过255个字符会被截断） |

### 请求示例

```json
{
	"docid": "DOCID",
	"formid": "FORMID",
	"new_name": "NEW_NAME"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok"
}
```
