# 成员申请的终止回调 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99877](https://developer.work.weixin.qq.com/document/path/99877)
- **文档 ID**: `99877`

## 接口描述

当企业开启通过接口设置高级功能的申请流程时，成员的申请流程流转到结束状态( 如管理员驳回、高级账号分配、成员删除申请 )后，将会向「可调用接口的应用」设置的消息服务器发送成员终止事件回调。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 此处固定为sys |
| CreateTime |  | 否 | 操作时间 |
| MsgType |  | 否 | 消息的类型，此时固定为event |
| Event |  | 否 | 事件的类型，此时固定为finish_vip_account_approval |
| FinishType |  | 否 | 状态改变类型：1-管理员驳回 2-高级账号分配 3-成员删除申请 |
| AgentID |  | 否 | 企业应用的id，整型。可在应用的设置页面查看 |
| ApplyId |  | 否 | 申请id，字符串 |

### 请求示例

```xml
<xml><ToUserName><![CDATA[toUser]]></ToUserName><FromUserName><![CDATA[sys]]></FromUserName><CreateTime>1715681559</CreateTime><MsgType><![CDATA[event]]></MsgType><Event><![CDATA[finish_vip_account_approval]]></Event><FinishType><1></FinishType><ApplyId><![CDATA[ApplyId]]></ApplyId><AgentID>1</AgentID></xml>
```
