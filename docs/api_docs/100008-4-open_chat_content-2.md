# 如何接入

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100008](https://developer.work.weixin.qq.com/document/path/100008)
- **文档 ID**: `100008`
- **API 名称**: `open_chat_content`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/open_chat_content?access_token=ACCESS_TOKEN`
- **分组信息**: 第 4 个接口，共 13 个

## 接口描述

企业开启会话内容，若授权时企业还未开启「企业会话内容」，授权页将展示开启指引。
