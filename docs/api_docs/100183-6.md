# 获取客户端专用密码列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100183](https://developer.work.weixin.qq.com/document/path/100183)
- **文档 ID**: `100183`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/exmail/publicmail/get_auth_code_list?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于获取公共邮箱创建的客户端专用密码列表，包括客户端专用密码ID、创建时间、最后使用时间、备注。需要注意的是，该接口不返回客户端专用密码，客户端专用密码仅会在创建时候返回，参考创建公共邮箱、更新公共邮箱

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | uint32 | 是 | 公共邮箱ID |

### 请求示例

```json
{"id":1}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| auth_code_list.auth_code_id | uint32 | 客户端专用密码ID |
| auth_code_list.create_time | uint32 | 客户端专用密码创建时间戳 |
| auth_code_list.last_use_time | uint32 | 客户端专用密码最后使用时间，如果未使用过则返回0 |
| auth_code_list.remark | string | 客户端专用密码备注 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","auth_code_list":[{"auth_code_id":1,"create_time":1724743240,"last_use_time":1724743241,"remark":"客户端专用密码备注"},{"auth_code_id":2,"create_time":1724743240,"last_use_time":1724743241,"remark":"客户端专用密码备注"}]}
```
