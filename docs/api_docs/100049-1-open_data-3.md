# 会话展示组件

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100049](https://developer.work.weixin.qq.com/document/path/100049)
- **文档 ID**: `100049`
- **API 名称**: `open_data`
- **请求方法**: `GET`
- **接口地址**: `https://open.work.weixin.qq.com/wwopen/js/jwxwork-1.0.0.js`
- **分组信息**: 第 1 个接口，共 2 个

## 接口描述

展示单个名称

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| type | string | 是 | 名称类型，可选值：userName，chatName |
| openid | string | 是 | 名称对应的 id |

## 其他说明

### 参数说明

组件容器，可以是实际的 DOM 元素或 CSS 选择器字符串。

### 参数说明

组件自定义数据，可以在模板中访问。

### 参数说明

组件模板，用于自定义组件内的展示内容。语法参考 WXML。目前支持以下组件：block, view, scroll-view, span, image, checkbox, ww-open-data, ww-open-message, ww-open-button, ww-open-result-link, ww-open-result-text, ww-suspense, ww-open-data-export-button

### 参数说明

组件模板样式表，语法是 CSS 的子集，目前仅支持以下语法：Class Selector, Pseudo, @media, @supports, @font-face, @page, @keyframes, @counter-style
