# 上传临时素材

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96483](https://developer.work.weixin.qq.com/document/path/96483)
- **文档 ID**: `96483`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/media/upload?access_token=accesstoken001&type=file`

## 接口描述

上传临时素材

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |
| type | string | 是 | 文件类型 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| media | object | 是 | 媒体文件 |

### 请求示例

#### 示例 1: 构造HTTP请求包示例

```text
--acebdf13572468
Content-Disposition: form-data; name="media";filename="wework.txt"; filelength=6
Content-Type: application/octet-stream

mytext
--acebdf13572468--
```

#### 示例 2: 设置HTTP头部信息示例

```text
POST https://qyapi.weixin.qq.com/cgi-bin/media/upload?access_token=accesstoken001&type=file HTTP/1.1
Content-Type: multipart/form-data; boundary=acebdf13572468
Content-Length: 168

--acebdf13572468
Content-Disposition: form-data; name="media";filename="wework.txt"; filelength=6
Content-Type: application/octet-stream

mytext
--acebdf13572468--
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
