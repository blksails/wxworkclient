# 从会话选择文件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95026](https://developer.work.weixin.qq.com/document/path/95026)
- **文档 ID**: `95026`

## 接口描述

从企业微信客户端会话选择文件

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| video |  | 否 | 只能选择视频文件 |
| image |  | 否 | 只能选择图片文件 |
| file |  | 否 | 可以选择除了图片和视频之外的其它的文件 |
| video_and_image |  | 否 | 可以同时选择视频与图片 |
| path | String | 否 | 本地临时文件路径 (本地路径) |
| size | Number | 否 | 本地临时文件大小，单位 B |
| name | String | 否 | 选择的文件名称 |
| type | String | 否 | 选择的文件类型 |
| time | Number | 否 | 选择的文件的会话发送时间，Unix时间戳 |
| video |  | 否 | 视频文件 |
| image |  | 否 | 图片文件 |
| file |  | 否 | 除图片和视频的文件 |
| ios |  | 否 | ≥3.1.18 |
| android |  | 否 | ≥3.1.18 |
| win |  | 否 | 不支持 |
| mac |  | 否 | 不支持 |

### 请求示例

```json
wx.qy.chooseMessageFile({count:10,type:'image',success(res){// tempFilePath可以作为img标签的src属性显示图片consttempFilePaths=res.tempFiles}})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| count | Number | 最多可以选择的文件个数，可以 1～100，超出该范围时报错parameter error: invalid parameter.count |
| type | String | 所选的文件的类型，默认值是'video_and_image'，合法的类型值参见object.type说明，不合法时报错parameter error: invalid parameter.type |
| success | Function | 接口调用成功的回调函数 |
| fail | Function | 接口调用失败的回调函数 |
| complete | Function | 接口调用结束的回调函数（调用成功、失败都会执行） |
| tempFiles | Array[Object] | 返回选择的文件的本地临时文件对象数组 |

## 其他说明

### wx.qy.chooseMessageFile(Object object)

调用前提：必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态当前成员必须在应用的可见范围
从企业微信客户端会话选择文件
