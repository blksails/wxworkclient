# 获取跳转小程序商城的直播观众信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94578](https://developer.work.weixin.qq.com/document/path/94578)
- **文档 ID**: `94578`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/living/get_living_share_info`

## 接口描述

通过此接口，开发者可获取跳转小程序商城的直播间(“推广产品”直播)观众id、邀请人id及对应直播间id，以打通卖货直播的“人货场”信息闭环。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ww_share_code | string | 是 | “推广产品”直播观众跳转小程序商城时会在小程序path中带上ww_share_code=xxxxx参数，ww_share_code五分钟内有效 |

### 请求示例

```json
{
	"ww_share_code": "CODE"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| livingid | string | 直播id |
| viewer_userid | string | 观众的userid，观众为企业内部成员时返回 |
| viewer_external_userid | string | 观众的external_userid，观众为非企业内部成员时返回 |
| invitor_userid | string | 邀请人的userid，邀请人为企业内部成员时返回（观众首次进入直播时，其使用的直播卡片/二维码所对应的分享人） |
| invitor_external_userid | string | 邀请人的external_userid，邀请人为非企业内部成员时返回 （观众首次进入直播时，其使用的直播卡片/二维码所对应的分享人） |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "ok",
   "livingid": "livingid",
   "viewer_userid": "viewer_userid",
   "viewer_external_userid": "viewer_external_userid",
   "invitor_userid": "invitor_userid",
   "invitor_external_userid": "invitor_external_userid"
}
```
