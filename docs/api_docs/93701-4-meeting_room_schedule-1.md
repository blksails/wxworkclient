# 「会议室」应用可调用的日程接口

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93701](https://developer.work.weixin.qq.com/document/path/93701)
- **文档 ID**: `93701`
- **API 名称**: `meeting_room_schedule`
- **请求方法**: `GET`
- **分组信息**: 第 4 个接口，共 4 个

## 接口描述

「会议室」应用可调用的日程接口包括：创建日程、更新日程、更新重复日程、新增日程参与者、删除日程参与者、获取日程详情、取消日程。
