# 支付成功通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91931](https://developer.work.weixin.qq.com/document/path/91931)
- **文档 ID**: `91931`
- **请求方法**: `POST`

## 接口描述

当企业对某一个订单完成付款后，企业微信后台会推送付款成功通知到服务商的指令回调URL。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId |  | 否 | 第三方应用的SuiteId |
| PaidCorpId |  | 否 | 购买方corpid |
| InfoType |  | 否 | 固定为 pay_for_app_success |
| TimeStamp |  | 否 | 时间戳 |
| OrderId |  | 否 | 订单号，付费订单的唯一标志。服务商可据此拉取购买信息。该OrderId由企业微信生成，不超过32个字符 |

### 请求示例

```xml
<xml><SuiteId><![CDATA[ww4asffe99e54c0aaa]]></SuiteId><PaidCorpId><![CDATA[wxf8b4f85f3a794aaa]]></PaidCorpId><InfoType><![CDATA[pay_for_app_success]]></InfoType><TimeStamp>1403610513</TimeStamp><OrderId><![CDATA[ORDERID]]></OrderId></xml>
```
