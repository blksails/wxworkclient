# 会话内容导出完成通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100047](https://developer.work.weixin.qq.com/document/path/100047)
- **文档 ID**: `100047`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webdoc/chat_archive_export_finished`

## 接口描述

当会话内容导出任务完成时，企业微信专区后台会向有企业会话内容权限的应用关联的程序推送相应的事件消息。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| event_type | string | 是 | 事件类型，此时固定为 chat_archive_export_finished |
| jobid | string | 是 | 24小时内有效，用于调用获取会话内容导出任务结果，不超过64个字符 |
| timestamp | uint32 | 是 | 时间戳 |

### 请求示例

```json
{
    "event_type": "chat_archive_export_finished",
    "timestamp": 1408091189,
    "chat_archive_export_finished": {
        "jobid": "xxx"
    }
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| event_type | string | 事件类型，此时固定为 chat_archive_export_finished |
| jobid | string | 24小时内有效，用于调用获取会话内容导出任务结果，不超过64个字符 |
| timestamp | uint32 | 时间戳 |
