# 删除部门

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90343](https://developer.work.weixin.qq.com/document/path/90343)
- **文档 ID**: `90343`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/department/delete`

## 接口描述

删除部门

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| id | string | 是 | 部门id。（注：不能删除根部门；不能删除含有子部门、成员的部门） |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "deleted"
}
```

## 其他说明

### 权限说明

应用须拥有指定部门的管理权限。第三方仅通讯录应用可以调用。
