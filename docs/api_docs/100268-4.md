# 取消订单通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100268](https://developer.work.weixin.qq.com/document/path/100268)
- **文档 ID**: `100268`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webhook/callback/order_cancel`

## 接口描述

当服务商取消订单时，企业微信后台会推送“退款结果通知”到服务商的系统事件接收URL。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ServiceCorpId | string | 是 | 服务商CorpID |
| InfoType | string | 是 | 事件类型，此时固定为advanced_api_cancel |
| AuthCorpId | string | 是 | 客户企业CorpID |
| AdvanceApiType | string | 是 | 高级接口类型，1-会话内容数据接口 |
| OrderId | string | 是 | 订单号 |
| TimeStamp | uint32 | 是 | 时间戳 |

### 请求示例

```xml
<xml>
	<ServiceCorpId><![CDATA[xxxx]]></ServiceCorpId>
	<InfoType><![CDATA[advanced_api_cancel]]></InfoType>
	<AuthCorpId><![CDATA[yyyy]]></AuthCorpId>
	<AdvanceApiType><![CDATA[1]]></AdvanceApiType>
	<OrderId><![CDATA[zzzz]]></OrderId>
	<TimeStamp>1403610513</TimeStamp>
</xml>
```
