# 获取客服账号链接 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96416](https://developer.work.weixin.qq.com/document/path/96416)
- **文档 ID**: `96416`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/kf/add_contact_way?access_token=ACCESS_TOKEN`

## 接口描述

企业可通过此接口获取带有不同参数的客服链接，不同客服账号对应不同的客服链接。获取后，企业可将链接嵌入到网页等场景中，微信用户点击链接即可向对应的客服账号发起咨询。企业可依据参数来识别用户的咨询来源等。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| open_kfid | string | 是 | 客服账号ID |
| scene | string | 否 | 场景值，字符串类型，由开发者自定义。不多于32字节字符串取值范围(正则表达式)：[0-9a-zA-Z_-]* |
| 自建应用 | 自建应用 | 否 | 配置到「微信客服- 可调用接口的应用」中 |
| 第三方应用 | 第三方应用 | 否 | 具有“微信客服->获取基础信息”权限 |
| 代开发自建应用 | 代开发自建应用 | 否 | 具有“微信客服->获取基础信息”权限 |
| 微信客服组件应用 | 微信客服组件应用 | 否 | 具有“管理接入的微信客服->获取企业授权接入的客服账号->客服账号信息与链接”权限，仅可获取企业已授权的客服账号链接 |

### 请求示例

```json
{"open_kfid":"OPEN_KFID","scene":"12345"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| url | string | 客服链接，开发者可将该链接嵌入到H5页面中，用户点击链接即可向对应的微信客服账号发起咨询。开发者也可根据该url自行生成需要的二维码图片 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","url":"https://work.weixin.qq.com/kf/kfcbf8f8d07ac7215f?enc_scene=ENCGFSDF567DF"}
```
