# 命中关键词规则通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99995](https://developer.work.weixin.qq.com/document/path/99995)
- **文档 ID**: `99995`

## 接口描述

当企业会话内容授权范围内成员的会话命中关键词规则时，企业微信专区后台会向有企业会话内容权限的应用关联的程序推送相应的事件消息。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| event_type |  | 否 | 事件类型，此时固定为hit_keyword |
| timestamp |  | 否 | 时间戳 |
| token |  | 否 | 10分钟内有效，用于调用获取命中关键词规则的会话记录，不超过128字节 |

### 请求示例

```json
{"event_type":"hit_keyword","timestamp":1408091189,"hit_keyword":{"token":"xxx"}}
```
