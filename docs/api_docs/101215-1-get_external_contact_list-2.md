# 配置可使用客户联系功能的成员

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/101215](https://developer.work.weixin.qq.com/document/path/101215)
- **文档 ID**: `101215`
- **API 名称**: `get_external_contact_list`
- **请求方法**: `GET`
- **接口地址**: `/cgi-bin/externalcontact/list`
- **分组信息**: 第 1 个接口，共 9 个

## 接口描述

管理员设置哪些人可使用联系客户功能，如未配置，则无法调用后文提到的相关接口。
