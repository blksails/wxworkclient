# 获取设备数据授权 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95991](https://developer.work.weixin.qq.com/document/path/95991)
- **文档 ID**: `95991`

## 接口描述

接口说明应用通过调用该接口，唤起设备选择列表，企业管理员选择之后，应用后续可通过云端接口获取到设备上报的数据

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ios |  | 否 | ≥ 4.0.12 |
| android |  | 否 | ≥ 4.0.12 |
| win |  | 否 | ≥ 4.0.12 |
| mac |  | 否 | ≥ 4.0.12 |
| 自建应用 | 自建应用 | 否 | 不支持 |
| 第三方应用 | 第三方应用 | 否 | 支持 |
| 代开发自建应用 | 代开发自建应用 | 否 | 不支持 |

### 请求示例

```json
wx.qy.openAppDeviceDataAuth({success:function(res){// 调用成功},fail:function(err){// 调用失败}});
```
