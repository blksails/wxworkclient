# 获取未观看直播统计V2 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95795](https://developer.work.weixin.qq.com/document/path/95795)
- **文档 ID**: `95795`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/living/get_unwatch_stat_v2?access_token=ACCESS_TOKEN`

## 接口描述

通过该接口可以获取未观看直播的学生和家长统计，学生和学生的家长必须是已经关注「学校通知」才会纳入统计范围。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| livingid |  | 是 | 直播id |
| 自建应用 | 自建应用 | 否 | 配置到「上课直播 - 可调用接口的应用」和「家长可以使用的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 具有「直播」和「家校沟通」权限 |
| 第三方应用 | 第三方应用 | 否 | 具有「直播」和「家校沟通」权限 |

### 请求示例

```json
{"livingid":"livingid1","next_cursor":"NEXT_CURSOR"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| next_cursor |  | 上一次调用时返回的next_cursor，初次调用可以填"0" |
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| has_more |  | 是否结束。1：表示还有更多数据，需要继续拉取，0：表示已经拉取完所有数据。注意只能根据该字段判断是否已经拉完数据 |
| next_cursor |  | 当前数据最后一个cursor值，如果下次调用带上该值则从该cursor值往后拉，用于实现分页拉取 |
| stat_info |  | 统计信息列表 |
| stat_info.students |  | 未观看直播的学生列表 |
| stat_info.students.student_userid |  | 学生的userid |
| stat_info.students.partyids |  | 学生所在的班级id列表 |
| stat_info.parents |  | 未观看直播的家长列表 |
| stat_info.parents.parent_userid |  | 家长的userid |
| stat_info.parents.student_userid |  | 家长对应学生的userid |
| stat_info.parents.partyids |  | 家长对应学生所在的班级id列表 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","has_more":1,"next_cursor":"NEXT_CURSOR","stat_info":{"students":[{"student_userid":"zhansan_child","partyids":[10,11]}，{"student_userid":"lisi_child","partyids":[5]}],"parents":[{"parent_userid":"zhangsan","student_userid":"zhansan_child","partyids":[10,11]}，{"parent_userid":"lisi","student_userid":"lisi_child","partyids":[5]}]}}
```
