# 更新应用邮箱账号 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97373](https://developer.work.weixin.qq.com/document/path/97373)
- **文档 ID**: `97373`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/exmail/app/update_email_alias?access_token=ACCESS_TOKEN`

## 接口描述

应用调用此接口，可以更新应用邮箱账号，原有的应用邮箱账号将会作为别名邮箱，具有收信能力

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| new_email | string | 是 | 修改后的应用邮箱账号 |

### 请求示例

```json
{"new_email":"youxiangceshi@ceshi.wecom.work"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
