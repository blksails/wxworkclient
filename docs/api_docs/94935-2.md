# 微信小程序进入填写学生资料页面 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94935](https://developer.work.weixin.qq.com/document/path/94935)
- **文档 ID**: `94935`

## 接口描述

在微信环境里，当开发者检测到用户不在家校通讯录指定班级中，在微信小程序里可调用小程序接口wx.navigateToMiniProgram（点击查看更详细的调用方法），唤起填写学生资料页面。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| appId |  | 是 | 固定填企业微信填写学生资料小程序appid: wx61866588582f5f71 |
| path |  | 是 | 跳转到小程序的路径，路径格式固定为：pages/index/index?strcorpid=${strcorpid}&strpartyid=${strpartyid} |
| strcorpid |  | 是 | 学校corpid |
| strpartyid |  | 否 | 班级的部门id，当传入班级部门id后，用户不能再修改校区和班级 |

### 请求示例

```json
wx.navigateToMiniProgram({appId:'wx61866588582f5f71',path:'pages/index/index?strcorpid=${strcorpid}&strpartyid=${strpartyid}'success(res){// 打开成功}})
```
