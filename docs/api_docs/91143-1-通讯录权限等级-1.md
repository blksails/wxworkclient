# 通讯录权限等级

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91143](https://developer.work.weixin.qq.com/document/path/91143)
- **文档 ID**: `91143`
- **API 名称**: `通讯录权限等级`
- **分组信息**: 第 1 个接口，共 2 个

## 接口描述

企业微信通讯录成员的信息按照其敏感性划分为基本信息和敏感信息，具体划分如下图所示。企业授权第三方应用后，第三方应用可以读取管理员所设置的应用使用范围内成员的信息，但读取成员的哪些信息由应用的通讯录权限等级决定。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| 单个基本信息只读 | bool | 不允许第三方应用读取授权企业的组织架构，仅允许通过OAuth方式读取当前进入应用的用户的基本信息 |
| 通讯录基本信息只读 | bool | 允许第三方应用读取授权企业的组织架构，仅允许读取授权应用使用范围内的组织架构和成员的基本信息 |

## 其他说明

### 配置策略建议

第三方应用可以读取通讯录时的权限等级由合作伙伴在合作伙伴管理后台登记应用时预先设置，企业微信团队审核。通讯录权限级别是与应用相关的，不允许跨应用、跨级别来调用通讯录相关接口。合作伙伴创建应用时，应遵循“权限最小化”原则，仅申请获取应用提供所必须的、最少属性的、最小范围的通讯录数据。
