# 如何使用

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100008](https://developer.work.weixin.qq.com/document/path/100008)
- **文档 ID**: `100008`
- **API 名称**: `prepare_work`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/prepare_work?access_token=ACCESS_TOKEN`
- **分组信息**: 第 6 个接口，共 13 个

## 接口描述

开发者需要准备一个程序，该程序首先需要引用专区SDK，实现获取会话记录等SDK接口的调用。

## 其他说明

### 注意

如果需要在程序中实时接收企业微信专区后台的回调通知，需要在程序中起动一个http服务并监听8080端口。
