# 创建日历

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93704](https://developer.work.weixin.qq.com/document/path/93704)
- **文档 ID**: `93704`
- **API 名称**: `create_calendar`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/calendar/create?access_token=ACCESS_TOKEN`
- **分组信息**: 第 1 个接口，共 3 个

## 接口描述

API创建的日历，仅允许管理员对该日历进行删除/修改信息/修改通知范围等操作。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |
