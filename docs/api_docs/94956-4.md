# 导出任务完成通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94956](https://developer.work.weixin.qq.com/document/path/94956)
- **文档 ID**: `94956`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webhook/batch_job_result`

## 接口描述

用于接收任务执行完毕的结果通知，事件回调到提交任务的第三方套件配置的回调地址。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| SuiteId | string | 是 | 套件ID |
| CreateTime | int32 | 是 | 消息创建时间（整型） |
| InfoType | string | 是 | 事件类型：batch_job_result |
| JobId | string | 是 | 异步任务id，最大长度为64字符 |
| JobType | string | 是 | 操作类型，字符串，在异步导出的场景下分别有：export_user(导出成员详情)、 export_simple_user(导出成员）、export_department(导出部门）、export_taguser(导出标签成员) |
| ErrCode | int32 | 是 | 返回码 |
| ErrMsg | string | 是 | 对返回码的文本描述内容 |

### 请求示例

```xml
<xml>
	<SuiteId><![CDATA[ww543f52cb62809e5a]]></SuiteId>
	<InfoType><![CDATA[batch_job_result]]></InfoType>
	<TimeStamp>1628653600</TimeStamp>
	<BatchJob>
		<JobId><![CDATA[jobid_FTR983N1FNGMsoZKXogNzmLwS8cwu150GTufJB]]></JobId>
		<JobType><![CDATA[export_user]]></JobType>
		<ErrCode>0</ErrCode>
		<ErrMsg><![CDATA[ok]]></ErrMsg>
	</BatchJob>
</xml>
```
