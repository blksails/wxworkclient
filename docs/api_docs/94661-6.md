# 获取客服账号列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94661](https://developer.work.weixin.qq.com/document/path/94661)
- **文档 ID**: `94661`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/kf/account/list?access_token=ACCESS_TOKEN`

## 接口描述

获取客服账号列表，包括客服账号的客服ID、名称和头像。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| offset | uint32 | 否 | 分页，偏移量, 默认为0 |
| limit | uint32 | 否 | 分页，预期请求的数据量，默认为100，取值范围 1 ~ 100 |
| 自建应用 | 自建应用 | 否 | 配置到「微信客服- 可调用接口的应用」中 |
| 第三方应用 | 第三方应用 | 否 | 具有“微信客服->获取基础信息”权限 |
| 代开发自建应用 | 代开发自建应用 | 否 | 具有“微信客服->获取基础信息”权限 |
| 微信客服组件应用 | 微信客服组件应用 | 否 | 具有“管理接入的微信客服->获取企业授权接入的客服账号->客服账号信息与链接”权限，仅可获取企业已授权的客服账号 |

### 请求示例

```json
{"offset":0,"limit":100}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |
| account_list | obj[] | 账号信息列表 |
| account_list.open_kfid | string | 客服账号ID |
| account_list.name | string | 客服名称 |
| account_list.avatar | string | 客服头像URL |
| account_list.manage_privilege | bool | 当前调用接口的应用身份，是否有该客服账号的管理权限（编辑客服账号信息、分配会话和收发消息）。组件应用不返回此字段 |
