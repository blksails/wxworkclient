# 获取员工字段配置 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99131](https://developer.work.weixin.qq.com/document/path/99131)
- **文档 ID**: `99131`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/hr/get_fields?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：GET(HTTPS)请求地址：https://qyapi.weixin.qq.com/cgi-bin/hr/get_fields?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| group_id |  | 否 | 字段组的id |
| group_name |  | 否 | 字段组的名称 |
| field_list |  | 否 | 字段组所包含的所有字段信息，参考字段信息说明 |
| field_id |  | 否 | 字段的id |
| field_name |  | 否 | 字段的名称 |
| field_type |  | 否 | 字段的类型，参考字段类型说明 |
| value_type |  | 否 | 字段值的类型，参考字段值类型对照表 |
| is_must |  | 否 | 字段是否为必填 |
| 1 | 文本类型 | 否 | 字符串类型 或 电话号码类型 |
| 2 | 选项类型 | 否 | 32位非负整数类型 |
| 3 | 时间类型 | 否 | 64位非负整数类型 或 64位整数类型 |
| 4 | 图片类型 | 否 | 文件类型 |
| 5 | 单个文件类型 | 否 | 文件类型 |
| 6 | 多个文件类型 | 否 | 文件类型 |
| 1 | 字符串 | 否 | 字符串 |
| 2 | 64位非负整数 | 否 | 非负整数 |
| 3 | 32位非负整数 | 否 | 非负整数 |
| 4 | 64位整数 | 否 | 整数 |
| 5 | 电话号码 | 否 | 参考电话号码类型字段结构 |
| 6 | 文件 | 否 | 参考文件类型字段结构 |
| value_country_code |  | 否 | 字符串，表示电话号码的区号 |
| value_mobile |  | 否 | 字符串，表示电话号码 |
| media_id |  | 否 | 列表，内容为字符串，可在获取临时素材接口下载对应文件 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述 |
| group_list |  | 字段组的配置信息，参考字段组配置信息说明 |
| option_list |  | 字段为选项类型时，每个选项的枚举值和对应的文本仅当field_type字段为2时返回 |
| option_list.id |  | 选项的枚举值 |
| option_list.value |  | 选项的文本内容 |
