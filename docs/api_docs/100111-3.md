# 通讯录管理

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100111](https://developer.work.weixin.qq.com/document/path/100111)
- **文档 ID**: `100111`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/user/update?access_token=ACCESS_TOKEN`

## 接口描述

成员自定义属性需要先在WEB管理端添加，否则忽略未知属性的赋值。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| extattr.type | uint32 | 是 | 属性类型: 0-文本 1-网页 |
| extattr.name | string | 是 | 属性名称。在新增或者更新操作时，需要先确保在管理端有创建该属性，否则会忽略 |
| extattr.text.value | string | 是 | 文本属性内容，长度限制64个UTF8字符 |
| extattr.web.url | string | 是 | 网页的url,必须包含http或者https头 |
| extattr.web.title | string | 是 | 网页的展示标题,长度限制12个UTF8字符 |

## 其他说明

### 字段内容

"extattr": {
		"attrs": [
			{
				"type": 0,
				"name": "文本名称",
				"text": {
					"value": "文本"
				}
			},
			{
				"type": 1,
				"name": "网页名称",
				"web": {
					"url": "http://www.test.com",
					"title": "标题"
				}
			}
		]
	}

### 字段说明

| 参数 | 说明 | 设置时是否必填 |
| --- | --- | --- |
| extattr | 否 | 自定义字段 |
| extattr.type | 是 | 属性类型: 0-文本 1-网页 |
| extattr.name | 是 | 属性名称。 `在新增或者更新操作时，需要先确保在管理端有创建该属性，否则会忽略` |
| extattr.text | 否 | 文本类型的属性 |
| extattr.text.value | 是 | 文本属性内容，长度限制64个UTF8字符 |
| extattr.web | 否 | 网页类型的属性，url和title字段要么同时为空表示清除该属性，要么同时不为空 |
| extattr.web.url | 是 | 网页的url,必须包含http或者https头 |
| extattr.web.title | 是 | 网页的展示标题,长度限制12个UTF8字符 |
