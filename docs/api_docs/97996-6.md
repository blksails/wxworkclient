# 删除邮件群组 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97996](https://developer.work.weixin.qq.com/document/path/97996)
- **文档 ID**: `97996`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/exmail/group/delete?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于删除已有的邮件群组。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| groupid | string | 是 | 邮件群组ID，邮箱格式 |

### 请求示例

```json
{"groupid":"zhangsangroup@gzdev.com"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
