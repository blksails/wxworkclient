# 应用同步调用专区程序

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100019](https://developer.work.weixin.qq.com/document/path/100019)
- **文档 ID**: `100019`
- **API 名称**: `应用同步调用专区程序`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/xxx/yyy`
- **分组信息**: 第 1 个接口，共 2 个

## 接口描述

开发者若要触发专区程序执行指定任务时，可通过应用同步调用专区程序发起程序调用。专区程序需要监听8080端口接收企微后台调用专区的请求。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| request_data | object | 是 | 应用调用专区程序接口时，request_data 字段必须匹配当前能力设置的输入协议。否则报错 790016。 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| response_data | object | 专区程序返回的结果会填到 response_data 字段，程序返回的结果必须匹配当前能力设置的输出协议。否则报错 790014。 |
