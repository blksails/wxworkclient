## 接口描述

参数Object object

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| filePath | String | 否 | 需要识别的语音文件的文件路径，由录音相关接口获得，仅支持由wx.startRecord录制的语音文件 |
| isShowProgressTips | number | 否 | 默认为1，显示进度提示 |
| success | function | 否 | 接口调用成功的回调函数 |
| fail | function | 否 | 接口调用失败的回调函数 |
| complete | function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```json
wx.qy.translateVoice({filePath:'',// 需要播放的语音文件的文件路径，由录音相关接口获得isShowProgressTips:1,// 默认为1，显示进度提示success:(res)=>{varresult=res.translateResult},fail:(res)=>{console.log(JSON.stringify(res))}})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| translateResult | String | 语音识别的结果 |
