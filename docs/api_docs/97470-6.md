# 新建文档 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97470](https://developer.work.weixin.qq.com/document/path/97470)
- **文档 ID**: `97470`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/create_doc?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于新建文档、表格及智能表格，新建收集表可前往收集表管理查看。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spaceid | string | 否 | 空间spaceid。若指定spaceid，则fatherid也要同时指定 |
| fatherid | string | 否 | 父目录fileid, 在根目录时为空间spaceid |
| doc_type | uint32 | 是 | 文档类型, 3:文档 4:表格 10:智能表格 |
| doc_name | string | 是 | 文档名字（注意：文件名最多填255个字符, 超过255个字符会被截断） |
| admin_users | string[] | 否 | 文档管理员userid |

### 请求示例

```json
{"spaceid":"SPACEID","fatherid":"FATHERID","doc_type":3,"doc_name":"DOC_NAME","admin_users":["USERID1","USERID2","USERID3"]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| url | string | 新建文档的访问链接 |
| docid | string | 新建文档的docid。docid仅在创建时返回，需要开发者妥善保存 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","url":"URL","docid":"DOCID"}
```
