# 上传临时文件到专区 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100174](https://developer.work.weixin.qq.com/document/path/100174)
- **文档 ID**: `100174`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/upload_media?access_token=ACCESS_TOKEN&type=TYPE`

## 接口描述

该接口将文件上传到数据专区临时存储，得到media_id。

## 请求信息

### 请求格式

```http
POST https://qyapi.weixin.qq.com/cgi-bin/chatdata/upload_media?access_token=accesstoken001&type=file HTTP/1.1
Content-Type: multipart/form-data; boundary=-------------------------acebdf13572468
Content-Length: 220

---------------------------acebdf13572468
Content-Disposition: form-data; name="media";filename="wework.txt"; filelength=6
Content-Type: application/octet-stream

mytext
---------------------------acebdf13572468--
```

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| type |  | 是 | 文件类型，目前仅支持普通文件：file |

### 请求示例

```http
POSThttps://qyapi.weixin.qq.com/cgi-bin/chatdata/upload_media?access_token=accesstoken001&type=fileHTTP/1.1Content-Type:multipart/form-data;boundary=-------------------------acebdf13572468Content-Length:220---------------------------acebdf13572468Content-Disposition:form-data;name="media";filename="wework.txt";filelength=6Content-Type:application/octet-streammytext---------------------------acebdf13572468--
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| type |  | 文件类型，目前仅支持普通文件：file |
| media_id |  | 文件上传后获取的唯一标识，3天内有效 |
| created_at |  | 文件上传时间戳 |

### 响应示例

```json
{"errcode":0,"errmsg":"","type":"file","media_id":"1G6nrLmr5EC3MMb_-zK1dDdzmd0p7cNliYu9V5w7o8K0","created_at":"1380000000"}
```

## 其他说明

### 上传的文件限制

所有文件size必须介于：6B~60MB
