# 获取用户负责及参与的网格列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97140](https://developer.work.weixin.qq.com/document/path/97140)
- **文档 ID**: `97140`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/report/grid/get_user_grid_info?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/report/grid/get_user_grid_info?access_token=ACCESS_TOKEN

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。“获取access_token” |

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| userid | string | 是 | 需要查询的成员userid |
| 自建应用 | 自建应用 | 否 | 配置到「居民联系 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 支持，需要勾选「居民联系权限」 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"userid":"zhangsan"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| manage_grids | array[object] | 作为管理员管理的网格信息 |
| manage_grids.grid_id | string | 网格id |
| manage_grids.grid_name | string | 网格名称 |
| joined_grids | array[object] | 作为成员参与的grid信息 |
| joined_grids.grid_id | string | 网格id |
| joined_grids.grid_name | string | 网格名称 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","manage_grids":[{"grid_id":"grid_id1","grid_name":"grid_name1"},{"grid_id":"grid_id2","grid_name":"grid_name2"}],"joined_grids":[{"grid_id":"grid_id1","grid_name":"grid_name1"},{"grid_id":"grid_id2","grid_name":"grid_name2"}]}
```
