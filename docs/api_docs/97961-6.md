# 获取分享链接

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97961](https://developer.work.weixin.qq.com/document/path/97961)
- **文档 ID**: `97961`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_share`

## 接口描述

该接口用于获取文件的分享链接。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fileid | string | 是 | 文件fileid |

### 请求示例

```json
{
    "fileid": "FILEID"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| share_url | string | 分享文件的链接 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "share_url": "SHARE_URL"
}
```
