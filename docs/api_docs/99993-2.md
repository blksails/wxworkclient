# 客户同意进行聊天内容存档事件回调 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99993](https://developer.work.weixin.qq.com/document/path/99993)
- **文档 ID**: `99993`

## 接口描述

配置了客户联系功能的成员添加外部联系人同意进行聊天内容存档时，回调该事件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| event_type |  | 否 | 事件类型当客户在单聊中同意存档时，该字段值为chat_archive_audit_approved_single当客户在群聊中同意存档时，该字段值为chat_archive_audit_approved_room |
| timestamp |  | 否 | 时间戳 |
| userid |  | 否 | 单聊时返回，企业服务人员的UserID |
| external_userid |  | 否 | 外部联系人的userid，注意不是企业成员的账号 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| chatid |  | 群聊时返回，所在群聊的ChatId |
