## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93823](https://developer.work.weixin.qq.com/document/path/93823)
- **文档 ID**: `93823`

## 接口描述

此接口用于参会人主动点击进入在线会议室。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| meetingId | string | 是 | 会议ID，可通过创建预约会议获取 |
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```json
wx.qy.startMeeting({"meetingId":"MEETINGID",success:function(res){},fail:function(res){console.log(res)},});
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| res.errMsg | string | 错误信息，如果调用成功返回 ${apiName}:ok |
| qy.startMeeting:ok |  | 执行成功 |
| qy.startMeeting:fail no session or expired |  | 没有调用 wx.qy.login，或者 session_key 已过期 |
| qy.startMeeting:fail no permission |  | 应用需具有会议权限 |
| qy.startMeeting:fail user not in allow list |  | 当前成员不在应用可见范围 |
| qy.startMeeting:fail invalid meeting id |  | 不可跨企业使用会议ID |
| qy.startMeeting:fail not allow to cross corp |  | 不可跨企业使用会议ID |
| qy.startMeeting:fail not allow to cross app |  | 不可跨应用使用会议ID |
| qy.startMeeting:fail invalid parameter |  | 参数不合法 |
| qy.startMeeting:fail api freq out of limit |  | 创建会议超过频率限制。单应用每天不可创建超过1w个会议 |

## 其他说明

### 权限说明

此接口仅在企业微信4.0及以后版本支持必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态当前成员必须在应用的可见范围应用需具有会议使用权限，见配置可使用会议的应用

### 示例代码

wx.qy.startMeeting({"meetingId":"MEETINGID",success:function(res){},fail:function(res){console.log(res)},});
