# 会话消息搜索

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99818](https://developer.work.weixin.qq.com/document/path/99818)
- **文档 ID**: `99818`
- **请求方法**: `SDK调用`

## 接口描述

可调用该接口对企业会话的内容进行搜索

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| query_word | string | 是 | 搜索的关键词，要求大于等于2个UTF8字符 |
| chat_info.chat_type | uint32 | 否 | 指定会话类型，1：单聊；2：群聊 |
| chat_info.id_list | object[] | 否 | 会话双方的id列表，长度必须为2 |
| chat_info.chat_id | string | 否 | 指定的群聊会话id |
| chat_info.msg_type_list | uint32[] | 否 | 指定搜索的消息类型 |
| chat_info.sender | object | 否 | 指定消息的发送者 |
| chat_info.sender_list | object[] | 否 | 指定消息的发送者列表，最多100个 |
| start_time | uint32 | 否 | 搜索会话范围的起始时间，Unix时间戳 |
| end_time | uint32 | 否 | 搜索会话范围的结束时间，Unix时间戳 |
| skip_stop_words | bool | 否 | 是否去掉部分常见停用词 |
| limit | uint32 | 否 | 每次请求返回的数据上限，最大为100 |
| cursor | string | 否 | 用于分页查询的游标 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| query_word | string | 是 | 搜索的关键词，要求大于等于2个UTF8字符 |
| chat_info | object | 否 | 指定要搜索的客户会话范围 |
| start_time | uint32 | 否 | 指定搜索会话范围的起始时间，Unix时间戳 |
| end_time | uint32 | 否 | 指定搜索会话范围的结束时间，Unix时间戳 |
| skip_stop_words | bool | 否 | 是否在搜索时去掉部分常见停用词 |
| limit | uint32 | 否 | 每次请求返回的数据上限 |
| cursor | string | 否 | 用于分页查询的游标 |

### 请求示例

```json
{
	"query_word": "欢迎",
	"chat_info": {
		"chat_type": 1,
		"id_list": [ {
				"open_userid": "OPEN_USERID"
				}, {
				"external_userid": "EXTERNALID1"
			}
		],
		"chat_id": "CHAT_ID",
		"msg_type_list":[1],
		"sender": {
			"open_userid": "OPEN_USERID"
		},
		"sender_list": [
			{
				"open_userid":"OPEN_USERID"
			}
		]
	},
	"start_time": 1668355200,
	"end_time": 1668528000,
	"skip_stop_words": true,
	"limit": 50,
	"cursor": "CURSOR"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| has_more | int32 | 是否还有更多数据未获取，0: 否, 1: 是 |
| next_cursor | string | 下一次请求的cursor值 |
| msg_list | object[] | 搜索结果的消息列表，按消息发送时间降序排序 |
| msg_list[].msgid | string | 每条消息对应的msgid |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"has_more": 1,
	"next_cursor": "CURSOR",
	"msg_list":[
		{
			"msgid": "xxxmsgid"
		},
		{
			"msgid": "yyymsgid"
		}
	]
}
```

## 其他说明

### 权限说明

自建应用、代开发应用、第三方应用需具备「数据与智能专区权限」

### 支持搜索的消息

仅支持对部分消息内容进行搜索，包括：
- 文本消息：消息内容
- 链接消息：消息标题
- 小程序消息：消息标题
- 文件消息：文件名称
- 在线文档消息：文档名称
- 图文混合消息：文本内容
- 微盘文件消息：文件名称

### 请求方法

通过SDK调用，具体方式参考[专区程序使用指引](#54006)

### IdInfo说明

成员ID：会话者的身份类型为企业成员时填
客户ID：会话者的身份类型为客户时填
