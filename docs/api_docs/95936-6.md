# 获取收款项目的商户单号

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95936](https://developer.work.weixin.qq.com/document/path/95936)
- **文档 ID**: `95936`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalpay/get_payment_info`

## 接口描述

获取对外收款项目中每笔收款的商户单号

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| payment_id | string | 是 | 收款项目单号。在发起对外收款中返回。 |

### 请求示例

```json
{
   "payment_id":"xxxx"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| bill_list | object[] | 收款单号列表。每笔支付对应一个收款单号 |
| bill_list[].out_trade_no | string | 收款单号 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"bill_list": [
		{
			"out_trade_no":"xxxx"
		},
		{
			"out_trade_no":"yyyy"
		}
	]
}
```

## 其他说明

### 权限说明

调用的应用需要满足如下的权限，且只允许获取由应用本身创建的收款项目的收款单号列表

- 自建应用：配置到「对外收款- 可调用接口的应用」中
- 代开发应用：具有「对外收款」权限
- 第三方应用：具有「对外收款」权限

*注：* 从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 [查看详情](https://developer.work.weixin.qq.com/document/51165)
