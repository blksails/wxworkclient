# 创建第三方应用审批模板

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93798](https://developer.work.weixin.qq.com/document/path/93798)
- **文档 ID**: `93798`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/corp/getopenapprovaldata?access_token=ACCESS_TOKEN`

## 接口描述

开发者可在“服务商后台-标准应用服务-网页应用-应用详情-审批接口”中，创建审批模板。用户在安装第三方应用后，可在“企业管理后台-应用与小程序-应用详情”中，设置审批流程。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### 请求示例

```json
{
	"thirdNo": "201806010001"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| ThirdNo | string | 审批单编号，由开发者在发起申请时自定义 |
| OpenTemplateId | string | 审批模板id |
| OpenSpName | string | 审批模板名称 |
| OpenSpstatus | int32 | 申请单当前审批状态：1-审批中；2-已通过；3-已驳回；4-已取消 |
| ApplyTime | int32 | 提交申请时间 |
| ApplyUsername | string | 提交者姓名 |
| ApplyUserId | string | 提交者userid |
| ApplyUserParty | string | 提交者所在部门 |
| ApplyUserImage | string | 提交者头像 |
| ApprovalNodes | object | 审批流程信息 |
| NotifyNodes | object | 抄送信息，可能有多个抄送人 |
| approverstep | int32 | 当前审批节点：0-第一个审批节点；1-第二个审批节点…以此类推 |
