# 获取文件信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97958](https://developer.work.weixin.qq.com/document/path/97958)
- **文档 ID**: `97958`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_info`

## 接口描述

该接口用于获取指定文件的信息。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fileid | string | 是 | 文件fileid |

### 请求示例

```json
{
    "fileid": "FILEID"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| file_info | object | 文件信息 |
| file_info.fileid | string | 文件fileid |
| file_info.file_name | string | 文件名字 |
| file_info.spaceid | string | 文件所在的空间spaceid |
| file_info.fatherid | string | 文件所在的目录fileid, 在根目录时为fileid |
| file_info.file_size | uint64 | 文件大小 |
| file_info.ctime | uint64 | 文件创建时间 |
| file_info.mtime | uint64 | 文件最后修改时间 |
| file_info.file_type | uint32 | 文件类型，1: 文件夹, 2: 文件, 3: 文档(文档), 4: 文档(表格), 5: 文档(收集表), 6: 文档(幻灯片) |
| file_info.file_status | uint32 | 文件状态, 1: 正常, 2: 删除 |
| file_info.sha | string | 文件sha。可用于确认是否跟与上传的文件一致，或避免重复上传相同的文件 |
| file_info.md5 | string | 文件md5。可用于确认是否跟与上传的文件一致，或避免重复上传相同的文件 |
| file_info.url | string | 仅微文档类型返回访问链接 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "file_info": {
        "fileid": "FILEID",
        "file_name": "FILE_NAME",
        "spaceid": "SPACEID",
        "fatherid": "FATHERID",
        "file_size": FILE_SIZE,
        "ctime": CTIME,
        "mtime": MTIME,
        "file_type": FILE_TYPE,
        "file_status": FILE_STATUS,
        "sha": "SHA",
        "md5": "MD5",
        "url": "URL"
    }
}
```
