# 获取导出结果

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96297](https://developer.work.weixin.qq.com/document/path/96297)
- **文档 ID**: `96297`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/export/get_result`

## 接口描述

获取导出任务的结果

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| jobid | string | 是 | 导出任务接口成功后返回的任务ID |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| status | int32 | 任务状态: 0-未处理，1-处理中，2-完成，3-异常失败 |
| data_list | object[] | 数据文件列表 |
| data_list.url | string | 数据下载链接,支持指定Range头部分段下载。有效期2个小时 |
| data_list.size | int32 | 密文数据大小 |
| data_list.md5 | string | 密文数据md5 |
