# 获取内部群信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99671](https://developer.work.weixin.qq.com/document/path/99671)
- **文档 ID**: `99671`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chat/get`

## 接口描述

获取内部群信息接口用于获取企业内部群的相关信息。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| chat_info | object | 群信息 |
| chat_info.chat_id | string | 群ID |
| chat_info.name | string | 群名称 |
| chat_info.owner | string | 群主 |
| chat_info.create_time | uint32 | 创建时间 |
