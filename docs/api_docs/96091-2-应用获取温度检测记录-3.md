# 获取硬件产生的数据

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96091](https://developer.work.weixin.qq.com/document/path/96091)
- **文档 ID**: `96091`
- **API 名称**: `应用获取温度检测记录`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/hardware/get_temperature_records`
- **分组信息**: 第 2 个接口，共 7 个

## 接口描述

应用获取温度检测记录接口用于获取企业下绑定的温度检测设备的记录。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| temperature_records | object[] | 温度记录列表 |
| temperature_records[].device_id | string | 设备ID |
| temperature_records[].temperature | float | 温度值 |
| temperature_records[].time | uint32 | 记录时间戳 |
