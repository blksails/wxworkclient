# 获取数据与智能专区授权信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100237](https://developer.work.weixin.qq.com/document/path/100237)
- **文档 ID**: `100237`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/get_corp_auth_info`

## 接口描述

第三方服务商应用可通过该接口获取企业数据与智能专区的授权详情。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | String | 是 | 调用接口凭证 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | String | 对返回码的文本描述内容 |
| auth_edition_list | object[] | 企业授权的会话版本。如果授权了多个会话版本，则返回授权的多个会话版本。 |
| auth_edition_list.edition | int32 | 企业授权的会话版本类型 1-内部会话 2-内外部会话 3-内外部会话及语音通话 |
| auth_edition_list.auth_scope.userid_list | String[] | 企业授权存档的人员列表。注意，实际生效人员取决于服务商购买的人数 |
| auth_edition_list.auth_scope.department_id_list | int32[] | 企业授权存档的部门列表。注意，实际生效人员取决于服务商购买的人数 |
| auth_edition_list.auth_scope.tag_id_list | int32[] | 企业授权存档的标签列表。注意，实际生效人员取决于服务商购买的人数 |
| auth_edition_list.status | int32 | 会话版本的当前状态 1-试用中 2-试用已到期 3-付费使用中 4-付费使用已到期 5-免费使用中 6-免费使用已到期 7-付费待生效 |
| auth_edition_list.begin_time | int32 | 版本状态对应的生效时间 |
| auth_edition_list.end_time | int32 | 版本状态对应的结束时间 |
| auth_edition_list.msg_duration_days | int32 | 会话授权的时长，单位：天（一年按365天计） |
| auth_edition_list.auth_user_count | int32 | 企业授权存档的去重人数 |

## 其他说明

### 权限说明

调用接口的应用需要满足如下的权限：

- 自建应用：不支持
- 代开发应用：需具备「数据与智能专区」权限
- 第三方应用：需具备「数据与智能专区」权限
