# 查询付款记录 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90279](https://developer.work.weixin.qq.com/document/path/90279)
- **文档 ID**: `90279`
- **请求方法**: `POST`

## 接口描述

说明：「向员工付款」功能暂不支持新增，已开通商户的功能使用不受影响。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 随机字符串 | String(32) | 是 | 随机字符串，不长于32位 |
| 微信支付签名 | String(32) | 是 | 参见“签名算法” |
| 商户订单号 | String(28) | 是 | 商户调用企业微信企业付款API时使用的商户订单号 |
| 商户号 | String(32) | 是 | 微信支付分配的商户号 |
| Appid | String(128) | 是 | 商户号的appid |
