# 更新记录

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100207](https://developer.work.weixin.qq.com/document/path/100207)
- **文档 ID**: `100207`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartsheet/update_records`

## 接口描述

本接口用于更新 Smartsheet 中的某个子表里的一行或多行记录。单次更新建议在500行内。不能通过更新记录接口给创建时间、最后编辑时间、创建人和最后编辑人四种类型的字段更新记录。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 是 | 文档的docid |
| sheet_id | string | 是 | Smartsheet 子表ID |
| key_type | string | 否 | 返回记录中单元格的key类型。枚举类型：CELL_VALUE_KEY_TYPE_FIELD_TITLE（key用字段标题表示）、CELL_VALUE_KEY_TYPE_FIELD_ID（key用字段 ID 表示） |
| records | object[] | 是 | 由需要更新的记录组成的 JSON 数组 |

### 请求示例

```json
{
	"docid": "DOCID",
	"sheet_id": "123Abc",
	"key_type": "CELL_VALUE_KEY_TYPE_FIELD_TITLE",
	"records": [
	]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| records | object[] | 由更新成功的记录的具体内容组成的 JSON 数组 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok",
    "records": [
    ]
}
```

## 其他说明

### 权限说明

自建应用需配置到“可调用应用”列表中的应用secret所获取的accesstoken来调用。第三方应用需具有“文档”权限。代开发自建应用需具有“文档”权限。
