# 第三方服务商如何帮助其他企业管理微信客服

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94683](https://developer.work.weixin.qq.com/document/path/94683)
- **文档 ID**: `94683`
- **API 名称**: `how_third_party_help_manage_wechat_customer_service`
- **请求方法**: `GET`
- **接口地址**: `https://work.weixin.qq.com/kf`
- **分组信息**: 第 2 个接口，共 4 个

## 接口描述

第三方服务商在创建第三方应用时，可以根据提供的功能，在“微信客服权限”勾选相应的权限。

## 其他说明

### 大体可分为两类：

1、服务商会基于微信客服做深入开发，帮企业管理指定的客服账号，包括编辑账号信息、分配会话和收发信息。
- 此种开发方式，服务商在创建第三方应用时，需勾选“管理账号、分配会话和收发消息”权限
- 企业管理员安装应用后，须再通过「微信客服」应用下发的授权通知，或在管理后台「应用管理-微信客服」处，选择微信客服账号授权给该应用，该应用才可获得此权限。
- 第三方通过API创建的微信客服账号，将自动拥有该客服账号的管理权限。企业历史（企业微信4.0.20版本前）已授权给第三方的客服账号，该第三方默认可继续管理。
- 一个客服账号同一时间，只能授权给一个第三方来管理。
- 企业将指定的客服账号授权给第三方后，该账号原生设置的接待规则将暂不生效，此时只能由第三方帮企业做会话消息管理，第三方需及时处理避免企业无法正常使用。

2、服务商只基于微信客服做一些简单开发（如获取带参数的客服链接、做客服详情页等业务页面帮助接待人员提效），但不介入管理企业客服账号的会话消息，企业依旧使用微信客服原生能力完成接待。
