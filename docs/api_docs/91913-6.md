# 延长试用期

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91913](https://developer.work.weixin.qq.com/document/path/91913)
- **文档 ID**: `91913`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/prolong_try?suite_access_token=SUITE_ACCESS_TOKEN`

## 接口描述

一个应用可以多次延长试用，但是试用总天数不能超过60天。仅限时试用或试用过期状态下的应用可以延长试用期。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| buyer_corpid | string | 是 | 购买方corpid |
| prolong_days | int32 | 是 | 延长天数 |
| appid | int32 | 否 | 仅旧套件需要填此参数 |

### 请求示例

```json
{
	"buyer_corpid" : "wx7da9abf8ac62baaa",
	"prolong_days" : 7,
	"appid" : 1
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| try_end_time | int32 | 延长后的试用到期时间（UNIX时间戳） |

### 响应示例

```json
{
    "errcode" : 0,
    "errmsg" : "ok",
	"try_end_time" : 1565152189
}
```
