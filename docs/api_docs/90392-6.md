# 上传图片

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90392](https://developer.work.weixin.qq.com/document/path/90392)
- **文档 ID**: `90392`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/media/uploadimg`

## 接口描述

上传图片得到图片URL，该URL永久有效。返回的图片URL，仅能用于图文消息正文中的图片展示，或者给客户发送欢迎语等；若用于非企业微信环境下的页面，图片将被屏蔽。每个企业每月最多可上传3000张图片，每天最多可上传1000张图片。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### 请求示例

```text
---------------------------acebdf13572468
Content-Disposition: form-data; name="fieldNameHere"; filename="20180103195745.png"
Content-Type: image/png
Content-Length: 220

<@INCLUDE *C:\Users\abelzhu\Pictures\企业微信截图_20180103195745.png*@>
---------------------------acebdf13572468--
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| url | string | 上传后得到的图片URL。永久有效 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "",
   "url" : "http://p.qpic.cn/pic_wework/3474110808/7a7c8471673ff0f178f63447935d35a5c1247a7f31d9c060/0"
}
```

## 其他说明

### 上传的图片大小限制

图片文件大小应在 5B ~ 2MB 之间
