# 修改会议基础布局 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98846](https://developer.work.weixin.qq.com/document/path/98846)
- **文档 ID**: `98846`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/layout/update?access_token=ACCESS_TOKEN`

## 接口描述

根据布局 ID 对设置好的会议基础布局进行修改。用户座次设置区分会前和会中两种方式：会前只允许设置邀请者成员，会中只允许设置参会成员。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议 ID |
| layout_id | string | 是 | 布局 ID |
| page_list | object[] | 是 | 布局对象列表，详见LayoutPage |
| enable_set_default | bool | 否 | 是否设置为会议应用的布局，默认不设置 |
| layout_template_id | string | 是 | 布局模板 ID |
| user_seat_list | object[] | 否 | 用户座次对象列表，详见UserSeat |
| grid_id | string | 是 | 宫格 ID。若多次传入同一宫格ID的对象，仅第一次出现的对象生效。 |
| grid_type | uint32 | 是 | 宫格类型：1. 视频画面2. 共享画面3. 拓展应用（目前一页仅可添加一个应用）。添加的应用需满足以下条件：1) 与会议绑定。2) 开启网页服务。3) 同企业下的仅企业内可见应用或外部企业可见应用。 |
| userid | string | 否 | 当场会议的企业成员的userid |
| tmp_openid | string | 否 | 当场会议的用户临时 ID |
| nick_name | string | 否 | 当宫格类型(grid_type)为1时必填，作为视频画面展示 |
| tool_sdkid | string | 否 | 拓展应用 ID |

### 请求示例

```json
{"meetingid":"MEETINGID","layout_id":"LAYOUTID1","page_list":[{"layout_template_id":"TEMID1","user_seat_list":[{"grid_id":"10","grid_type":1,"userid":"USERID","tmp_openid":"68j2xkYT1","nick_name":"NICK","tool_sdkid":"SDKID"}]}],"enable_set_default":false}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
