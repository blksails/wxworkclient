# 企微后台调用专区

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99948](https://developer.work.weixin.qq.com/document/path/99948)
- **文档 ID**: `99948`
- **API 名称**: `企微后台调用专区`
- **分组信息**: 第 3 个接口，共 5 个

## 接口描述

用户需要自行监听8080端口（IP使用0.0.0.0）接收企微后台调用专区的请求；收到请求后需要使用本SDK对请求进行验签和解密，处理完成后的回包也需要用本SDK进行加密和签名。

## 其他说明

### Python版本

收到请求后，需要将请求的header和body放到`WWToSpecHTTPReqDecoder`中去验签和解密。

### C++版本

收到请求后，需要将请求的header和body放到`SpecCallbackSDK`中去验签和解密。

### Java版本

收到请求后，需要将请求的header和body放到`SpecCallbackSDK`中去验签和解密。
