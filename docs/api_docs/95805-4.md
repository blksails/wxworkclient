# 退款结果通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95805](https://developer.work.weixin.qq.com/document/path/95805)
- **文档 ID**: `95805`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webhook/callback`

## 接口描述

当服务商提交退款申请的订单发生状态变更时，企业微信后台会推送“退款结果通知”到服务商的系统事件接收URL。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ServiceCorpId | string | 是 | 服务商CorpID |
| InfoType | string | 是 | 事件类型，此时固定为license_refund |
| AuthCorpId | string | 是 | 客户企业CorpID |
| OrderId | string | 是 | 订单号 |
| OrderStatus | int32 | 是 | 订单状态，1:退款成功，2:退款被拒绝。 |
| TimeStamp | uint32 | 是 | 时间戳 |

### 请求示例

```xml
<xml>
	<ServiceCorpId><![CDATA[xxxx]]></ServiceCorpId>
	<InfoType><![CDATA[license_refund]]></InfoType>
	<AuthCorpId><![CDATA[yyyy]]></AuthCorpId>
	<OrderId><![CDATA[zzzz]]></OrderId>
	<OrderStatus>1</OrderStatus>
	<TimeStamp>1403610513</TimeStamp>
</xml>
```
