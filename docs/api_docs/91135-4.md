# 异步任务完成通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91135](https://developer.work.weixin.qq.com/document/path/91135)
- **文档 ID**: `91135`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/batch/sync_job_result`

## 接口描述

本事件是成员在使用异步任务接口时，用于接收任务执行完毕的结果通知。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName | string | 是 | 企业微信CorpID |
| FromUserName | string | 是 | 成员UserID |
| CreateTime | int32 | 是 | 消息创建时间（整型） |
| MsgType | string | 是 | 消息类型，固定为：event |
| Event | string | 是 | 事件类型：batch_job_result |
| BatchJob.JobId | string | 是 | 异步任务id，最大长度为64字符 |
| BatchJob.JobType | string | 是 | 操作类型，字符串，目前分别有：sync_user(增量更新成员)、 replace_user(全量覆盖成员）、invite_user(邀请成员关注）、replace_party(全量覆盖部门) |
| BatchJob.ErrCode | int32 | 是 | 返回码 |
| BatchJob.ErrMsg | string | 是 | 对返回码的文本描述内容 |

### 请求示例

```xml
<xml><ToUserName><![CDATA[wx28dbb14e3720FAKE]]></ToUserName>
<FromUserName><![CDATA[FromUser]]></FromUserName>

<BatchJob><JobId><![CDATA[S0MrnndvRG5fadSlLwiBqiDDbM143UqTmKP3152FZk4]]></JobId>

</BatchJob>
</xml>
```
