# boolean wx.qy.canIUse

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92394](https://developer.work.weixin.qq.com/document/path/92394)
- **文档 ID**: `92394`
- **请求方法**: `GET`
- **接口地址**: `https://wx.qy.canIUse`

## 接口描述

判断企业微信专有接口、回调、参数等是否在当前版本可用。注意wx.canIUse判断的是微信小程序通用接口（即wx.[接口名]）是否在当前版本可用。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| schema | string | 是 | 使用 `${API}.${method}.${param}.${options}` 或者 `${component}.${attribute}.${option}` 方式来调用 |

### 请求示例

```text
wx.qy.canIUse('login.success.code')
wx.qy.canIUse('getNFCReaderState')
wx.qy.canIUse("getAvatar.success.avatar")
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| boolean | boolean | 当前版本是否可用 |
