# 获取预授权码 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90601](https://developer.work.weixin.qq.com/document/path/90601)
- **文档 ID**: `90601`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/get_pre_auth_code?suite_access_token=SUITE_ACCESS_TOKEN`

## 接口描述

该API用于获取预授权码。预授权码用于企业授权时的第三方服务商安全验证。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| suite_access_token |  | 是 | 第三方应用access_token,最长为512字节 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| pre_auth_code |  | 预授权码,最长为512字节 |
| expires_in |  | 有效期（秒） |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","pre_auth_code":"Cx_Dk6qiBE0Dmx4EmlT3oRfArPvwSQ-oa3NL_fwHM7VI08r52wazoZX2Rhpz1dEw","expires_in":1200}
```
