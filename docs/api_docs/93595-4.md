# wx.qy.shareToExternalChat

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93595](https://developer.work.weixin.qq.com/document/path/93595)
- **文档 ID**: `93595`
- **请求方法**: `wx.qy.shareToExternalChat`
- **接口地址**: `wx.qy.shareToExternalChat`

## 接口描述

该接口支持在小程序环境中，具有客户联系权限的企业成员将文本内容和附件传递到客户群群发、发送到客户群。为了防止滥用，每次群发最多可选2000个最近活跃的客户群，同一个成员每日向一个客户群最多可群发一条消息，可在企业微信-群发助手-设置群发规则中调整为每周接收七条或每月可接收条数为当月天数。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| chatIds | Array | 否 | 指定的客户群ID列表，从4.1.10版本开始支持（mac端不支持） |
| text.content | string | 否 | 消息文本内容 |
| attachments | Array | 是 | 附件，最多支持添加9个附件 |
| attachments.msgtype | string | 是 | 附件类型，可选image、link、miniprogram或者video |
| attachments.image.mediaid | string | 否 | 图片的素材id，可通过素材管理接口获得 |
| attachments.image.imgUrl | string | 否 | 图片的链接，仅可使用上传图片接口得到的链接，跟图片mediaid填其中一个即可 |
| attachments.link.title | string | 否 | 图文消息标题，最多64个字节 |
| attachments.link.desc | string | 否 | 图文消息摘要，最多512个字节 |
| attachments.link.url | string | 否 | 图文消息 |
| attachments.link.imgUrl | string | 否 | 图文消息封面图片URL |
| attachments.miniprogram.appid | string | 是 | 当前小程序appid，必须是关联到企业的小程序应用 |
| attachments.miniprogram.title | string | 是 | 小程序消息标题，最多64个字节 |
| attachments.miniprogram.imgUrl | string | 是 | 小程序消息封面的URL |
| attachments.miniprogram.page | string | 否 | 小程序打开后的路径，注意要以.html作为后缀，否则在微信端打开会提示找不到页面 |
| attachments.video.mediaid | string | 是 | 视频的素材id，可通过素材管理接口获得 |
| attachments.file.mediaid | string | 是 | 文件的素材id，可通过素材管理接口获得 |
| success | Function | 否 | 群发成功的回调函数 |
| fail | Function | 否 | 群发失败的回调函数 |
| complete | Function | 否 | 调用结束的回调函数（群发成功、失败都会执行） |
