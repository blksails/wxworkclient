# 删除成员

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90334](https://developer.work.weixin.qq.com/document/path/90334)
- **文档 ID**: `90334`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/user/delete`

## 接口描述

删除成员

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| userid | string | 是 | 成员UserID。对应管理端的账号 |

### 请求示例

```json
{
   "errcode": 0,
   "errmsg": "deleted"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

## 其他说明

### 权限说明

仅通讯录同步助手或第三方通讯录应用可调用。若是绑定了腾讯企业邮，则会同时删除邮箱账号。
