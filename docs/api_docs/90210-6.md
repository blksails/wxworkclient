# 创建标签 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90210](https://developer.work.weixin.qq.com/document/path/90210)
- **文档 ID**: `90210`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/tag/create?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/tag/create?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| tagname |  | 是 | 标签名称，长度限制为32个字以内（汉字或英文字母），标签名不可与其他标签重名。 |
| tagid |  | 否 | 标签id，非负整型，指定此参数时新增的标签会生成对应的标签id，不指定时则以目前最大的id自增。 |

### 请求示例

```json
{"tagname":"UI","tagid":12}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| tagid |  | 标签id |

### 响应示例

```json
{"errcode":0,"errmsg":"created""tagid":12}
```
