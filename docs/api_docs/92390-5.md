# 校验用户当前 session_key 是否有效

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92390](https://developer.work.weixin.qq.com/document/path/92390)
- **文档 ID**: `92390`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

校验用户当前 session_key 是否有效。如果有重新登录，需要调用 jscode2session 接口后，session_key 才会有效。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| Object object | object | 是 | 参数对象 |
| Object object.success | Function | 否 | 接口调用成功的回调函数，session_key 未过期 |
| Object object.fail | Function | 否 | 接口调用失败的回调函数，session_key 已过期 |
| Object object.complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```text
wx.qy.checkSession({
  success: function(){
    //session_key 未过期，并且在本生命周期一直有效
  },
  fail: function(){
    // session_key 已经失效，需要重新执行登录流程
    wx.qy.login() //重新登录
  }
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| Object res | object | 回调函数参数 |
| Object res.errMsg | String | 调用结果 |
| Object res.expireIn | Number | session过期时间 |
