# 概述

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93394](https://developer.work.weixin.qq.com/document/path/93394)
- **文档 ID**: `93394`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/oa/journal/get_record_detail?access_token=ACCESS_TOKEN`

## 接口描述

目录

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| journaluuid | string | 是 | -不多于256字节 |
| 自建应用 | 自建应用 | 否 | 需要配置到「汇报 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |
| new_money |  | 否 | 金额内容，即申请人在此控件填写的金额内容 |
| date |  | 否 | 日期/日期+时间内容，即申请人在此控件选择的 日期/日期+时间内容 |
| └ type |  | 否 | 时间展示类型：day-日期；hour-日期+时间；month-月份；minute-时分 |
| └ s_timestamp |  | 否 | 时间戳，字符串类型。若type为minute，则为从0点的偏移秒数 |
| selector |  | 否 | 选择内容，即申请人在此控件选择的选项内容 |
| └ type |  | 否 | 选择类型：single-单选；multi-多选 |
| └ options |  | 否 | 申请人所选择的选项，多选情况下可能有多个（仅包含申请人所选择的选项，并非所有选项，若需要了解所有选项，需使用“获取审批模板详情”接口） |
| └ └ key |  | 否 | 选项key，选项的唯一id，可通过“获取审批模板详情”接口获得（2022-08-25后新创建的企业，不再支持关联审批模板） |
| └ └ value |  | 否 | 选项值 |
| members |  | 否 | 成员内容，即申请人在此控件选择的成员，多选模式下可能有多个 |
| └ userid |  | 否 | 成员的userid |
| departments |  | 否 | 部门内容，即申请人在此控件选择的部门，多选模式下可能有多个 |
| └ openapi_id |  | 否 | 部门id |
| files |  | 否 | 文件内容，即申请人在此控件上传的文件内容，可能有多个 |
| └ file_id |  | 否 | 文件的media_id，具体使用请参考：文档-获取临时素材 |
| children |  | 否 | 明细内容，一个明细控件可能包含多个子明细 |
| └ list |  | 否 | 子明细列表，包含了申请人在子明细的所有子控件中填写的内容，子控件的数据接口同一般控件 |
| date_range |  | 否 | 时长组件 |
| └ type |  | 否 | 时间展示类型：halfday-日期；hour-日期+时间 |
| └ new_begin |  | 否 | 开始时间,unix时间戳 |
| └ new_end |  | 否 | 结束时间，unix时间戳 |
| └ new_duration |  | 否 | 时长范围， 单位秒 |
| latitude |  | 否 | 纬度，精确到6位小数 |
| longitude |  | 否 | 经度，精确到6位小数 |
| title |  | 否 | 地点标题 |
| address |  | 否 | 地点详情地址 |
| time |  | 否 | 选择地点的时间（已废弃） |
| value |  | 否 | 公式的值 |
| students |  | 否 | 学生内容，即申请人在此控件选择的 学生，多选模式下可能有多个 |
| └ name |  | 否 | 学生名 |
| classes |  | 否 | 班级内容，即申请人在此控件选择的班级，多选模式下可能有多个 |
| └ name |  | 否 | 班级名 |
| docs |  | 否 | 文档内容 |
| └ docid |  | 否 | 文档的docid |
| └ doc_url |  | 否 | 文档的访问链接注：企业微信4.0.20版本后，不会有新增的文档控件数据。开发者可通过此url进一步查看/导出文档 |
| wedrive_files |  | 否 | 微盘文件内容 |
| └ fileid |  | 否 | 微盘的fileid，可通过下载微盘文件接口下载文件 |
| 301065 |  | 否 | 无汇报应用数据拉取权限 |
| 301066 |  | 否 | 请求参数错误 |
| 301067 |  | 否 | 接口内部失败 |

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |
| info | obj | 汇报详情 |
| info.journal_uuid | string | 汇报记录id |
| info.template_name | string | 汇报表单名称 |
| info.template_id | string | 汇报表单id |
| info.report_time | int32 | 汇报时间 |

| info.readed_receivers | obj[] | 已读用户 |
| info.readed_receivers.userid | string | 已读用户id |
| info.apply_data | obj | 表单数据 |
| info.apply_data.contents | obj[] | 表单字段列表 |
| info.apply_data.contents.control | string | 控件类型：Text-文本；Textarea-多行文本；Number-数字；Money-金额；Date-日期/日期+时间；Selector-单选/多选；；Contact-成员/部门；Tips-说明文字；File-附件；Table-明细；DateRange-时长；Doc-文档；WedriveFile-微盘附件 |
