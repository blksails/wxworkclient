# 关于客户朋友圈

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93507](https://developer.work.weixin.qq.com/document/path/93507)
- **文档 ID**: `93507`
- **API 名称**: `获取企业全部的发表列表`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/get_publish_list?access_token=ACCESS_TOKEN`
- **分组信息**: 第 1 个接口，共 3 个

## 接口描述

企业统一创建，由成员确认后发表的内容

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 接口调用凭证 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| publish_list | object[] | 发表列表 |
| publish_list[].content | string | 内容 |
| publish_list[].confirm_member | string | 确认成员 |

## 其他说明

### 关于客户朋友圈

成员可发表内容到微信客户的朋友圈，还可与客户评论互动。企业也可统一创建内容，由成员确认后发表。客户朋友圈可分为以下两类：企业发表和个人发表。
