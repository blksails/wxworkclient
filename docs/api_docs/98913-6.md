# 修改群聊会话 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98913](https://developer.work.weixin.qq.com/document/path/98913)
- **文档 ID**: `98913`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/appchat/update?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/appchat/update?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| chatid |  | 是 | 群聊id |
| name |  | 否 | 新的群聊名。若不需更新，请忽略此参数。最多50个utf8字符，超过将截断 |
| owner |  | 否 | 新群主的id。若不需更新，请忽略此参数。课程群聊群主必须拥有课程群创建权限，del_user_list包含群主时本字段必填 |
| add_user_list |  | 否 | 添加成员的id列表 |
| del_user_list |  | 否 | 踢出成员的id列表 |

### 请求示例

```json
{"chatid":"CHATID","name":"NAME","owner":"userid2","add_user_list":["userid1","userid2","userid3"],"del_user_list":["userid3","userid4"]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
