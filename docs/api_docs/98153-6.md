# 取消预约会议 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98153](https://developer.work.weixin.qq.com/document/path/98153)
- **文档 ID**: `98153`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/cancel?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于取消一个指定的预约会议。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议id，仅允许取消预约状态下的会议 |
| sub_meetingid | string | 否 | 周期性子会议 ID。如果取消周期性会议且该字段不传，则会取消该系列的周期性会议。 |

### 请求示例

```json
{"meetingid":"XXXXXXXXX","sub_meetingid":"sub_meetingid1111"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
