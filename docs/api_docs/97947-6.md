# 管理空间权限

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97947](https://developer.work.weixin.qq.com/document/path/97947)
- **文档 ID**: `97947`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/space_acl_del`

## 接口描述

该接口用于对指定空间移除成员/部门，操作者为应用本身。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| spaceid | string | 是 | 空间spaceid |
| auth_info | object[] | 是 | 被移除的空间成员信息 |
| auth_info[].type | uint32 | 是 | 成员类型 1:个人 2:部门 |
| auth_info[].userid | string | 是 | 成员userid,字符串 (type为1时填写) |
| auth_info[].departmentid | uint32 | 是 | 部门departmentid, 32位整型范围是[0, 2^32) (type为2时填写) |

### 请求示例

```json
{
    "spaceid": "SPACEID",
    "auth_info": [{
        "type": 1,
        "userid": "USERID1"
    }, {
        "type": 2,
        "departmentid": 1
    }]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok"
}
```

## 其他说明

### 权限说明

* 自建应用需配置到“可调用应用”列表中的应用secret所获取的accesstoken来调用（accesstoken如何获取？）
* 第三方应用需具有“微盘”权限
* 代开发自建应用需具有“微盘”权限
