## 接口描述

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| deviceSn | String | 是 | 已绑定的设备SN |

### 请求示例

```text
wx.qy.openDeviceProfile({
    "deviceSn": "QYWX001", // 设备SN
},
success: (res) => {
    // 打开设备详情页成功
},
fail: (res) => {
    console.log(JSON.stringify(res))
}
);
```
