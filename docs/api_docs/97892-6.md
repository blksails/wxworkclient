# 修改文件安全设置 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97892](https://developer.work.weixin.qq.com/document/path/97892)
- **文档 ID**: `97892`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_secure_setting?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于修改文件安全设置，水印相关设置。仅支持在线文档类型。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | Query参数，示例值: ACCESS_TOKEN |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| watermark | object | 否 | (从请求示例中推测) |
| watermark.text | string | 否 | (从请求示例中推测) |
| watermark.margin_type | int | 否 | (从请求示例中推测) |
| watermark.show_visitor_name | bool | 否 | (从请求示例中推测) |
| watermark.show_text | bool | 否 | (从请求示例中推测) |

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fileid | string | 是 | 文件fileid |
| text | string | 否 | 水印文字，此字段不填则保持原样 |
| margin_type | uint32 | 否 | 水印类型。1：低密度水印， 2： 高密度水印，此字段不填则保持原样 |
| show_visitor_name | bool | 否 | 是否显示访问人名称，此字段不填则保持原样 |
| show_text | bool | 否 | 是否展示水印文本，此字段不填则保持原样 |

### 请求示例

```json
{"fileid":"FILEID","watermark":{"text":"WATERMARK_TEXT","margin_type":0,"show_visitor_name":false,"show_text":false}}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
