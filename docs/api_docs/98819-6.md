# 获取用户专属参会链接 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98819](https://developer.work.weixin.qq.com/document/path/98819)
- **文档 ID**: `98819`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/get_customer_short_url?access_token=ACCESS_TOKEN`

## 接口描述

可以获取指定会议的所有专属参会链接及customer_data。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议ID |

### 请求示例

```json
{"meetingid":"meetingid11234"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| meeting_short_url_customer_data_list | object[] | 用户专属参会链接对象列表。详见CustomerData。 |
| customer_data | string | 用户专属字段 |
| meeting_short_url | string | 用户专属参会链接 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","meeting_short_url_customer_data_list":[{"customer_data":"CUSTOMERDATA","meeting_short_url":"URL"}]}
```
