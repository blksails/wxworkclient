# 客户群opengid转换 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94822](https://developer.work.weixin.qq.com/document/path/94822)
- **文档 ID**: `94822`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/opengid_to_chatid?access_token=ACCESS_TOKEN`

## 接口描述

用户在微信里的客户群里打开小程序时，某些场景下可以获取到群的opengid，如果该群是企业微信的客户群，则企业或第三方可以调用此接口将一个opengid转换为客户群chat_id

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| opengid |  | 是 | 小程序在微信获取到的群ID，参见wx.getGroupEnterInfo |

### 请求示例

```json
{"opengid":"oAAAAAAA"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| chat_id |  | 客户群ID，可以用来调用获取客户群详情 |

### 响应示例

```text
｛"errcode":0,"errmsg":"ok","chat_id":"ooAAAAAAAAAAA"｝
```
