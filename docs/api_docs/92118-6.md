# 编辑客户企业标签 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92118](https://developer.work.weixin.qq.com/document/path/92118)
- **文档 ID**: `92118`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/mark_tag?access_token=ACCESS_TOKEN`

## 接口描述

企业可通过此接口为指定成员的客户添加上由企业统一配置的标签；注意，每个成员对同一个客户最多可以添加3000个标签。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| userid |  | 是 | 添加外部联系人的userid |
| external_userid |  | 是 | 外部联系人userid |
| add_tag |  | 否 | 要标记的标签列表 |
| remove_tag |  | 否 | 要移除的标签列表 |
| 自建应用 | 自建应用 | 否 | 配置到「客户联系 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 具有「客户基础信息」权限 |
| 第三方应用 | 第三方应用 | 否 | 具有「客户基础信息」权限 |

### 请求示例

```json
{"userid":"zhangsan","external_userid":"woAJ2GCAAAd1NPGHKSD4wKmE8Aabj9AAA","add_tag":["TAGID1","TAGID2"],"remove_tag":["TAGID3","TAGID4"]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok",}
```
