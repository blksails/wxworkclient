# 获取会议背景列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98856](https://developer.work.weixin.qq.com/document/path/98856)
- **文档 ID**: `98856`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/layout/list_background?access_token=ACCESS_TOKEN`

## 接口描述

根据会议 ID 返回会议背景列表信息。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议 ID |

### 请求示例

```json
{"meetingid":"MEETINGID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| selected_background_id | string | 会议应用的背景 ID |
| background_list | object[] | 背景对象列表，详见Background |
| background_id | string | 背景 ID |
| image_md5 | string | 背景图片 MD5（图片内容 MD5 的十六进制表示） |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","selected_background_id":"BACKID1","background_list":[{"background_id":"BACKID1","image_md5":"eax879asd3z"}]}
```
