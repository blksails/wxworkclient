# 使用二次验证 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99500](https://developer.work.weixin.qq.com/document/path/99500)
- **文档 ID**: `99500`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/user/tfa_succ?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/user/tfa_succ?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| userid |  | 是 | 用户的userid |
| tfa_code |  | 是 | 获取用户二次验证信息接口返回的tfa_codetfa_code五分钟内有效且只能使用一次 |

### 请求示例

```json
{"userid":"USERID","tfa_code":"TFA_CODE"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
