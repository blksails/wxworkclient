# 获取电话入会的成员ID - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98825](https://developer.work.weixin.qq.com/document/path/98825)
- **文档 ID**: `98825`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/phone/get_tmp_openid?access_token=ACCESS_TOKEN`

## 接口描述

此接口用于获取电话入会的成员ID，以tmp_openid返回。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议ID |
| phone_numbers | object[] | 是 | 外呼的电话号码对象数组，上限20个。详见PhoneNumber。 |
| area | uint32 | 是 | 国家/地区代码。（例如：中国是86） |
| phone | string | 是 | 电话号码或固定电话总机号 |
| extension_number | string | 否 | 固定电话分机号 |

### 请求示例

```json
{"meetingid":"MEETINGID","phone_numbers":[{"area":86,"phone":"13333333333","extension_number":"12345678"}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| tmp_openid_list | object[] | 返回的 tmp_openid 对象数组。详见TmpOpenId。 |
| area | uint32 | 国家/地区代码。（例如：中国是86） |
| phone | string | 电话号码或固定电话总机号 |
| extension_number | string | 固定电话分机号 |
| tmp_openid | string | 当场会议的成员临时 ID，可用于会控操作，适用于所有成员。 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","tmp_openid_list":[{"area":86,"phone":"13333333333","extension_number":"12345678","tmp_openid":"TMP_OPENID"}]}
```
