# 读取学生或家长

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92038](https://developer.work.weixin.qq.com/document/path/92038)
- **文档 ID**: `92038`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/user/get`

## 接口描述

读取学生或家长的信息。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| userid | string | 是 | 家校通讯录的userid，家长或者学生的userid。不区分大小写，长度为1~64个字节 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| user_type | int32 | 用户类型:1表示学生，2表示家长 |
| student | object | 学生字段，user_type为1时返回该字段 |
| student.student_userid | string | 学生UserID |
| student.name | string | 学生名字 |
| student.department | array | 学生所在的班级id列表,不超过20个 |
| student.parents | array | 学生家长列表 |
| student.parents.parent_userid | string | 家长的userid |
| student.parents.relation | string | 与家长的关系 |
| student.parents.mobile | string | 家长手机号，第三方不可获取；代开发自建应用需管理员授权手机号才返回 |
| student.parents.external_userid | string | 家长的external_userid,仅当家长已关注才返回。对同一个服务商来说，同一个家长微信在不同学校下返回的家长的external_userid是一样的 |
| parent | object | 学生家长，user_type为2时返回该字段 |
| parent.parent_userid | string | 家长的userid |
| parent.relation | string | 学生与家长的关系 |
| parent.mobile | string | 家长手机号，第三方不可获取；代开发自建应用需管理员授权手机号才返回 |
| parent.is_subscribe | int32 | 家长是否关注了“学校通知”，0-未关注，1-已关注 |
| parent.external_userid | string | 家长的external_userid,仅当家长已关注才返回。对同一个服务商来说，同一个家长微信在不同学校下返回的家长的external_userid是一样的 |
| parent.children | array | 家长孩子列表 |
| parent.children.student_userid | string | 学生的userid |
| parent.children.relation | string | 家长与孩子关系 |
