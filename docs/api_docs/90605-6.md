# 获取企业凭证

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90605](https://developer.work.weixin.qq.com/document/path/90605)
- **文档 ID**: `90605`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/get_corp_token?suite_access_token=SUITE_ACCESS_TOKEN`

## 接口描述

第三方服务商在取得企业的永久授权码后，通过此接口可以获取到企业的access_token。获取后可通过通讯录、应用、消息等企业接口来运营这些应用。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| auth_corpid | string | 是 | 授权方corpid |
| permanent_code | string | 是 | 永久授权码，通过get_permanent_code获取 |

### 请求示例

```json
{
	"auth_corpid": "auth_corpid_value",
	"permanent_code": "code_value"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| access_token | string | 授权方（企业）access_token,最长为512字节 |
| expires_in | uint32 | 授权方（企业）access_token超时时间 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"access_token": "xxxxxx",
	"expires_in": 7200
}
```
