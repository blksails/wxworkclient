# 微信进入居民上报小程序

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94942](https://developer.work.weixin.qq.com/document/path/94942)
- **文档 ID**: `94942`
- **请求方法**: `POST`
- **接口地址**: `https://developers.weixin.qq.com/miniprogram/dev/api/navigate/wx.navigateToMiniProgram.html`

## 接口描述

政府单位可在自己的小程序配置「居民上报」小程序跳转入口，居民点击后进入「居民上报」小程序反馈问题。在微信小程序里可按以下参数调用小程序接口 wx.navigateToMiniProgram，唤起「居民上报」小程序。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| appId | string | 是 | 固定填企业微信居民上报小程序appid: wx85c9a2814145dbeb |
| path | string | 是 | 跳转到小程序的路径，路径格式固定为：pages/feedback/feedback?strcorpid=${strcorpid}&userid=${userid} |
| strcorpid | string | 是 | 企业corpid |
| userid | string | 是 | 企业成员的userid |

### 请求示例

```text
wx.navigateToMiniProgram({
  appId: 'wx85c9a2814145dbeb',
  path: 'pages/feedback/feedback?strcorpid=${strcorpid}&userid=${userid}',
  success(res) {
    // 打开成功
  }
})
```
