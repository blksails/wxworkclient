## 接口描述

专区程序接收事件通知，企业微信通过HTTP请求回调到专区，专区需要启动一个HTTP服务，监听8080端口，并回调到根路径。专区接收到请求后，需要将HTTP请求的header以及body传给SDK解密接口。

## 请求信息

### 请求示例

```text
http服务接收到的HTTP的body为加密的，需要调用SDK进行解密，解密后拿到消息原文，各个通知事件的协议参见本目录下各个文档。
```
