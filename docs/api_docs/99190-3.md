# 选择可下载的文件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99190](https://developer.work.weixin.qq.com/document/path/99190)
- **文档 ID**: `99190`

## 接口描述

接口说明：调用该接口，可唤起微盘选择器窗口，选择一个或多个具有下载权限的文件（只能是微盘文件，不支持文档）。选中后返回文件的临时ticket，以操作用户的名义下载该文件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| selectedFileNum | 是 | 否 | 选择文件的数量。1表示单选，大于1表示多选，上限为50 |
| success | 否 | 否 | 接口调用成功的回调函数 |
| fail | 否 | 否 | 接口调用失败的回调函数 |
| complete | 否 | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| 自建应用 | 自建应用 | 否 | 不支持 |
| 第三方应用 | 第三方应用 | 否 | 需具有“微盘”权限 |
| 代开发应用 | 代开发应用 | 否 | 需具有“微盘”权限 |

### 请求示例

```json
wx.qy.wedriveSelectFileForDownload({selectedFileNum:1,success:function(res){varselectedTickets=res.result.selectedTickets;},fail:function(res){console.log(res)},});
```

## 其他说明

### 返回结果

Object res
参数名类型说明resObj返回信息res.errMsgstring执行结果。"wedriveSelectFileForDownload:ok" 表示执行成功，否则表示报错信息res.resultObj返回的数据res.result.selectedTicketsArray<string>选择的文件所生成的临时ticket列表
selectedTickets 说明：1. 有效期30分钟，不限制使用次数2. 可调用文件下载接口3. 每个文件对应一个ticket
