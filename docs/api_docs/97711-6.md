# 获取表格行列信息 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97711](https://developer.work.weixin.qq.com/document/path/97711)
- **文档 ID**: `97711`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/spreadsheet/get_sheet_properties?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于获取在线表格的工作表、行数、列数等。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 是 | 在线表格的docid |
| sheet_id | string | 否 | 工作表ID，工作表的唯一标识 |
| title | string | 否 | 工作表名称 |
| row_count | uint32 | 否 | 表格的总行数 |
| column_count | uint32 | 否 | 表格的总列数 |

### 请求示例

#### 示例 1: 请求包体

```json
{"docid":"DOCID"}
```

#### 示例 2: 示例

```json
{"sheet_id","ABCDE","title":"XXXXXX","row_count":100,"column_count":100}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| properties | object[](Properties) | 工作表属性 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","properties":[{...}]}
```
