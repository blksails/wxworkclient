# 获取待分配的离职成员列表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96330](https://developer.work.weixin.qq.com/document/path/96330)
- **文档 ID**: `96330`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/get_unassigned_list`

## 接口描述

企业和第三方可通过此接口，获取所有离职成员的客户列表，并可进一步调用分配离职成员的客户接口将这些客户重新分配给其他企业成员。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| cursor | string | 否 | 分页查询游标，字符串类型，适用于数据量较大的情况，如果使用该参数则无需填写page_id，该参数由上一次调用返回 |
| page_size | int32 | 否 | 每次返回的最大记录数，默认为1000，最大值为1000 |

### 请求示例

```json
{
  "cursor":"",
  "page_size":100
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| info | object[] | 信息列表 |
| info[].handover_userid | string | 离职成员的userid |
| info[].external_userid | string | 外部联系人userid |
| info[].dimission_time | uint32 | 成员离职时间 |
| is_last | bool | 是否是最后一条记录 |
| next_cursor | string | 分页查询游标,已经查完则返回空("")，使用`page_id`作为查询参数时不返回 |
