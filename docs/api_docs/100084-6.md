# 关闭专区调试模式

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100084](https://developer.work.weixin.qq.com/document/path/100084)
- **文档 ID**: `100084`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/close_debug_mode`

## 接口描述

关闭专区调试模式

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| program_id | string | 是 | 应用关联的程序id |

### 请求示例

```json
{
  "program_id": "xxx"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok"
}
```
