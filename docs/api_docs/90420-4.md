# 查询电子发票

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90420](https://developer.work.weixin.qq.com/document/path/90420)
- **文档 ID**: `90420`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/card/invoice/reimburse/getinvoiceinfo`

## 接口描述

报销方在获得用户选择的电子发票标识参数后，可以通过该接口查询电子发票的结构化信息，并获取发票PDF文件。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| card_id | string | 是 | 发票id |
| encrypt_code | string | 是 | 加密code |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| card_id | string | 发票id |
| begin_time | int32 | 发票的有效期起始时间 |
| end_time | int32 | 发票的有效期截止时间 |
| openid | string | 用户标识 |
| type | string | 发票类型，如广东增值税普通发票 |
| payee | string | 发票的收款方 |
| detail | string | 发票详情 |
| user_info | object | 发票的用户信息 |
| user_info.fee | int32 | 发票加税合计金额，以分为单位 |
| user_info.title | string | 发票的抬头 |
| user_info.billing_time | int32 | 开票时间，为十位时间戳 |
| user_info.billing_no | string | 发票代码 |
| user_info.billing_code | string | 发票号码 |
| user_info.tax | int32 | 税额,以分为单位 |
| user_info.fee_without_tax | int32 | 不含税金额，以分为单位 |
| user_info.detail | string | 发票详情，一般描述的是发票的使用说明 |
| user_info.pdf_url | string | 这张发票对应的PDF_URL |
| user_info.reimburse_status | string | 发报销状态INVOICE_REIMBURSE_INIT：发票初始状态，未锁定；INVOICE_REIMBURSE_LOCK：发票已锁定；INVOICE_REIMBURSE_CLOSURE：发票已核销 |
| user_info.check_code | string | 校验码 |
| user_info.info | object[] | 商品信息结构 |
| user_info.info[].name | string | 项目（商品）名称 |
| user_info.info[].num | int32 | 项目数量 |
| user_info.info[].unit | string | 项目单位 |
| user_info.info[].fee | int32 | 单价，以分为单位 |

## 其他说明

### 参数说明

| 参数 | 说明 | 是否必返回 |
| --- | --- | --- |
| card_id | 发票id | 是 |
| begin_time | 发票的有效期起始时间 | 是 |
| end_time | 发票的有效期截止时间 | 是 |
| openid | 用户标识 | 是 |
| type | 发票类型，如广东增值税普通发票 | 是 |
| payee | 发票的收款方 | 是 |
| detail | 发票详情 | 是 |
| user_info | 发票的用户信息，见user_info结构说明 | 是 |

### user_info 结构说明

| 参数 | 说明 | 是否必返回 |
| --- | --- | --- |
| fee | 发票加税合计金额，以分为单位 | 是 |
| title | 发票的抬头 | 是 |
| billing_time | 开票时间，为十位时间戳 | 是 |
| billing_no | 发票代码 | 是，全电发票发票号码是该字段 |
| billing_code | 发票号码 | 是（全电发票发票号码为billing_no字段） |
| tax | 税额,以分为单位 | 是 |
| fee_without_tax | 不含税金额，以分为单位 | 是 |
| detail | 发票详情，一般描述的是发票的使用说明 | 是 |
| pdf_url | 这张发票对应的PDF_URL | 是 |
| trip_pdf_url | 其它消费凭证附件对应的URL，如行程单、水单等 | 否 |
| check_code | 校验码 | 是 |
| buyer_number | 购买方纳税人识别号 | 否 |
| buyer_address_and_phone | 购买方地址、电话 | 否 |
| buyer_bank_account | 购买方开户行及账号 | 否 |
| seller_number | 销售方纳税人识别号 | 否 |
| seller_address_and_phone | 销售方地址、电话 | 否 |
| seller_bank_account | 销售方开户行及账号 | 否 |
| remarks | 备注 | 否 |
| cashier | 收款人，发票左下角处 | 否 |
| maker | 开票人，发票右下角处 | 否 |
| reimburse_status | 发报销状态INVOICE_REIMBURSE_INIT：发票初始状态，未锁定；INVOICE_REIMBURSE_LOCK：发票已锁定；INVOICE_REIMBURSE_CLOSURE：发票已核销 | 是 |
| info | 商品信息结构，见下方说明 | 否 |

### info 结构说明

| 参数 | 说明 | 是否必返回 |
| --- | --- | --- |
| name | 项目（商品）名称 | 是 |
| num | 项目数量 | 否 |
| unit | 项目单位 | 否 |
| price | 单价，以分为单位 | 是 |
