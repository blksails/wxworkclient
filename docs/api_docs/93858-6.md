# 获取观看直播统计

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93858](https://developer.work.weixin.qq.com/document/path/93858)
- **文档 ID**: `93858`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/living/get_watch_stat`

## 接口描述

通过该接口可以获取所有观看直播的人员统计

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| livingid | string | 是 | 直播的id |
| next_key | string | 否 | 上一次调用时返回的next_key，初次调用可以填"0" |

### 请求示例

```json
{
	"livingid": "livingid1",
	"next_key": "NEXT_KEY"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| ending | int32 | 是否结束。0：表示还有更多数据，需要继续拉取，1：表示已经拉取完所有数据。注意只能根据该字段判断是否已经拉完数据 |
| next_key | string | 当前数据最后一个key值，如果下次调用带上该值则从该key值往后拉，用于实现分页拉取 |
| stat_infoes | object | 统计信息列表 |
| stat_infoes.students | object[] | 观看直播的学生列表 |
| stat_infoes.students.student_userid | string | 学生的userid |
| stat_infoes.students.parent_userid | string | 家长的userid，直播参与者是家长身份才展示，学生身份加入直播不展示 |
| stat_infoes.students.partyids | int32[] | 学生所在的班级id列表 |
| stat_infoes.students.watch_time | int32 | 观看时长，单位为秒 |
| stat_infoes.students.enter_time | int32 | 首次进入直播时间 |
| stat_infoes.students.leave_time | int32 | 最后离开直播时间 |
| stat_infoes.students.is_comment | int32 | 是否评论，1表示评论，0表示没有评论 |
| stat_infoes.visitors | object[] | 观看直播的游客列表 |
| stat_infoes.visitors.nickname | string | 微信昵称 |
| stat_infoes.visitors.watch_time | int32 | 观看时长，单位为秒 |
| stat_infoes.visitors.enter_time | int32 | 首次进入直播时间 |
| stat_infoes.visitors.leave_time | int32 | 最后离开直播时间 |
| stat_infoes.visitors.is_comment | int32 | 是否评论，1表示评论，0表示没有评论 |
