## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
|  | string | 是 | 图片的local ID |
|  | Function | 否 | 成功回调 |
|  | Function | 否 | 失败回调 |
|  | Function | 否 | 取消回调 |
|  | Function | 否 | 完成回调 |

### 请求示例

#### 示例 1: 功能描述

```json
ww.getLocalImgData({localId:localId})
```

#### 示例 2: 示例代码

```json
wx.getLocalImgData({localId:'',// 图片的localIDsuccess:function(res){varlocalData=res.localData;// localData是图片的base64数据，可以用img标签显示}});
```

## 其他说明

### ww.getLocalImgData(params)

兼容性：企业微信 iOS 2.4.6 开始支持
功能描述获取本地图片内容。
ww.getLocalImgData({localId:localId})
使用限制仅在 iOS WKWebView 下支持。
参数说明params: Object属性类型必填说明localIdstring是图片的local IDsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码localDatastring是图片的 base64 数据

### 功能描述

获取本地图片内容。

### 使用限制

仅在 iOS WKWebView 下支持。

### params: Object

属性类型必填说明localIdstring是图片的local IDsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码localDatastring是图片的 base64 数据

### 示例代码

wx.getLocalImgData({localId:'',// 图片的localIDsuccess:function(res){varlocalData=res.localData;// localData是图片的base64数据，可以用img标签显示}});
