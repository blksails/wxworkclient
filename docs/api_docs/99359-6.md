# 取消收款订单

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99359](https://developer.work.weixin.qq.com/document/path/99359)
- **文档 ID**: `99359`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/paytool/close_order?provider_access_token=PROVIDER_ACCESS_TOKEN`

## 接口描述

服务商可以使用该接口取消指定的收款订单

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 服务商调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| order_id | string | 是 | 收款订单号 不多于64字节 |
| nonce_str | string | 是 | 随机字符串，长度要求在32字节以内，用于保证签名不可预测及防重放攻击。 需保证15分钟内不能重复，推荐随机字符串生成算法 |
| ts | uint64 | 是 | unix时间戳（中国时区）,精确到秒。 注意业务系统的机器时间与腾讯的时间相差不能超过15分钟 |
| sig | string | 是 | 数字签名。见签名算法。签名所需密钥获取路径： 工作台->企业微信服务商助手->工具->收银台->收银台API调用密钥 |

### 请求示例

```json
{
	"order_id": "T0Rkc2g3dWlQNzFLNTRXMDJBV25BNi95OVZZbjBPOE11N2FCWkc0STJRdz0=",
	"nonce_str": "129031823",
	"ts": 1548302135,
	"sig": "mPOwVW/vQ74xN+b+Yu1KMa9RrmhKJaJjAtXHTof+EpU="
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 错误码描述 |

### 响应示例

```json
{
    "errcode": 0,
    "errmsg": "ok"
}
```
