# 客户标签ID的转换

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97109](https://developer.work.weixin.qq.com/document/path/97109)
- **文档 ID**: `97109`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/idconvert/external_tagid`

## 接口描述

将企业主体下的客户标签ID转换成服务商主体下的客户标签ID。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证，第三方应用access_token或代开发应用access_token |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| external_tagid_list | array | 是 | 企业主体下的客户标签ID或标签组ID列表，最多不超过1000个 |

### 请求示例

```json
{
  "external_tagid_list":["TAG_ID1","TAG_ID2","TAG_ID3","TAG_ID4"]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| items | array | 客户标签转换结果 |
| items.external_tagid | string | 企业主体下的客户标签ID |
| items.open_external_tagid | string | 服务商主体下的客户标签ID，如果传入的external_tagid已经是服务商主体下的ID，则open_external_tagid与external_tagid相同。 |
| invalid_external_tagid_list | array | 无法转换的客户标签ID列表 |

### 响应示例

```json
{
 "errcode":0,
 "errmsg":"ok",
 "items":[
 	{
			"external_tagid":"TAG_ID1",
			"open_external_tagid":"OPEN_TAG_ID1"
	},
	{
			"external_tagid":"TAG_ID2",
			"open_external_tagid":"OPEN_TAG_ID2"
	},
	"invalid_external_tagid_list":["TAG_ID3","TAG_ID4"]
}
```
