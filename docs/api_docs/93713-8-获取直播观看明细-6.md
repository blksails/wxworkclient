# API

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93713](https://developer.work.weixin.qq.com/document/path/93713)
- **文档 ID**: `93713`
- **API 名称**: `获取直播观看明细`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/live/get_watch_stat`
- **分组信息**: 第 8 个接口，共 8 个

## 接口描述

获取直播观看明细接口用于获取直播的观看明细统计信息。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| meeting_id | string | 是 | 直播ID |

### 请求示例

```json
{
  "meeting_id": "123456"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| watch_stat | object | 观看统计信息 |

### 响应示例

```json
{
  "errcode": 0,
  "errmsg": "ok",
  "watch_stat": {
    "total_watch_num": 100,
    "average_watch_duration": 300
  }
}
```
