### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
|  | string | 是 | 已绑定的设备SN |
|  | Function | 否 | 成功回调 |
|  | Function | 否 | 失败回调 |
|  | Function | 否 | 取消回调 |
|  | Function | 否 | 完成回调 |

### 请求示例

#### 示例 1: 功能描述

```json
ww.openDeviceProfile({deviceSn:'QYWX001'})
```

#### 示例 2: 示例代码

```json
wx.invoke("openDeviceProfile",{"deviceSn":"QYWX001",// 设备SN},function(res){if(res.err_msg=="openDeviceProfile:ok"){// 打开设备详情页成功}});
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
|  | string | 通用错误信息 |
|  | number | 通用错误码 |
| openDeviceProfile:ok |  | 执行成功 |
| openDeviceProfile:fail_device_permission_denied |  | 管理员无设备管理权限 |
| openDeviceProfile:fail_device_not_found |  | 不存在此设备 |

## 其他说明

### ww.openDeviceProfile(params)

兼容性：企业微信 iOS、Android 2.8.2 开始支持
功能描述查看设备。
ww.openDeviceProfile({deviceSn:'QYWX001'})
使用限制调用者必须拥有指定 deviceSn 的管理权限。
参数说明params: Object属性类型必填说明deviceSnstring是已绑定的设备SNsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码
使用说明若开发者需要在 web 端引导跳转设备管理，可以构造链接跳转：https://work.weixin.qq.com/wework_admin/frame#hardware/device?sn={{DEVICESN}}。
错误信息errMsg说明openDeviceProfile:ok执行成功openDeviceProfile:fail_device_permission_denied管理员无设备管理权限openDeviceProfile:fail_device_not_found不存在此设备

### 功能描述

查看设备。

### 使用限制

调用者必须拥有指定 deviceSn 的管理权限。

### params: Object

属性类型必填说明deviceSnstring是已绑定的设备SNsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 使用说明

若开发者需要在 web 端引导跳转设备管理，可以构造链接跳转：https://work.weixin.qq.com/wework_admin/frame#hardware/device?sn={{DEVICESN}}。

### 错误信息

errMsg说明openDeviceProfile:ok执行成功openDeviceProfile:fail_device_permission_denied管理员无设备管理权限openDeviceProfile:fail_device_not_found不存在此设备

### 示例代码

wx.invoke("openDeviceProfile",{"deviceSn":"QYWX001",// 设备SN},function(res){if(res.err_msg=="openDeviceProfile:ok"){// 打开设备详情页成功}});
