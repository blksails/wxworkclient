# 获取公费电话拨打记录 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93662](https://developer.work.weixin.qq.com/document/path/93662)
- **文档 ID**: `93662`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/dial/get_dial_record?access_token=ACCESS_TOKEN`

## 接口描述

企业可通过此接口，按时间范围拉取成功接通的公费电话拨打记录。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| start_time |  | 否 | 查询的起始时间戳 |
| end_time |  | 否 | 查询的结束时间戳 |
| offset |  | 否 | 分页查询的偏移量 |
| limit |  | 否 | 分页查询的每页大小,默认为100条，如该参数大于100则按100处理 |
| 自建应用 | 自建应用 | 否 | 配置到「公费电话 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"start_time":1536508800,"end_time":1536940800,"offset":0,"limit":100}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| record.call_time |  | 拨出时间 |
| record.total_duration |  | 总通话时长，单位为分钟 |
| record.call_type |  | 通话类型，1-单人通话 2-多人通话 |
| record.caller.userid |  | 主叫用户的userid |
| record.caller.duration |  | 主叫用户的通话时长 |
| record.callee.userid |  | 被叫用户的userid，当被叫用户为企业内用户时返回 |
| record.callee.phone |  | 被叫用户的号码，当被叫用户为外部用户时返回 |
| record.callee.duration |  | 被叫用户的通话时长 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","record":[{"call_time":1536508800,"total_duration":10,"call_type":1,"caller":{"userid":"tony","duration":10},"callee":[{"phone":138000800,"duration":10}]},{"call_time":1536940800,"total_duration":20,"call_type":2,"caller":{"userid":"tony","duration":10},"callee":[{"phone":138000800,"duration":5},{"userid":"tom","duration":5}]}]}
```
