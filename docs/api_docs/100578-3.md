### 请求示例

#### 示例 1: 功能描述

```json
ww.getLocation({type:'wgs84'})
```

#### 示例 2: 示例代码

```json
wx.getLocation({type:'wgs84',// 默认为wgs84的gps坐标，如果要返回直接给openLocation用的火星坐标，可传入'gcj02'success:function(res){varlatitude=res.latitude;// 纬度，浮点数，范围为90 ~ -90varlongitude=res.longitude;// 经度，浮点数，范围为180 ~ -180。varspeed=res.speed;// 速度，以米/每秒计varaccuracy=res.accuracy;// 位置精度}});
```
