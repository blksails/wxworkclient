# 开始开发

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94444](https://developer.work.weixin.qq.com/document/path/94444)
- **文档 ID**: `94444`
- **API 名称**: `external_contact_mode`
- **请求方法**: `POST`
- **接口地址**: `https://work.weixin.qq.com/api/doc#14928/%E5%A4%96%E9%83%A8%E8%81%94%E7%B3%BB%E4%BA%BA%E9%80%89%E4%BA%BA%E6%8E%A5%E5%8F%A3`
- **分组信息**: 第 1 个接口，共 4 个

## 接口描述

家校应用开发者可以通过API接口设置“学校通知关注模式”，发送“学校通知”和管理“家校通讯录”等功能。

## 请求信息

### 请求示例

```text
管理员进入企业微信管理后台后点击“家校沟通”-“配置”页面进行配置。
```
