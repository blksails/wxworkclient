## 其他说明

### 概述

微信客服可接待的客户数，和企业是否已完成主体验证相关。
企业未完成主体验证时，微信客服仅可累计接待100位客户企业已验证时，微信客服可接待的客户数不受限制
开发者可获取状态后，在应用等地方提示企业去完成主体验证。

### 获取企业验证状态

可通过获取企业授权信息接口获取。
