# 会议室应答事件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98783](https://developer.work.weixin.qq.com/document/path/98783)
- **文档 ID**: `98783`

## 接口描述

API创建的会议对会议室发起的呼叫，当有应答结果时，触发该事件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 操作者UserID。此事件为系统触发事件，固定为sys |
| FromUserTmpOpenId |  | 否 | 操作者TmpOpenID。操作者的会中临时ID，仅成员操作事件有值。 |
| CreateTime |  | 否 | 消息创建时间，unix时间戳 |
| MsgType |  | 否 | 消息类型，此时固定为：event |
| Event |  | 否 | 事件类型，此时固定为：meeting_change |
| ChangeType |  | 否 | 事件类型，此时固定为：meeting_room_response |
| MeetingId |  | 否 | 会议ID |
| MeetingRoomId |  | 否 | 会议室ID，与MraAddress二选一 |
| MraAddress |  | 否 | MRA设备，与MeetingRoomId二选一 |

```xml
<xml><ToUserName><![CDATA[toUser]]></ToUserName><FromUserName><![CDATA[sys]]></FromUserName><CreateTime>1348831860</CreateTime><MsgType><![CDATA[event]]></MsgType><Event><![CDATA[meeting_change]]></Event><ChangeType><![CDATA[meeting_room_response]]></ChangeType><MeetingId><![CDATA[wcjgewCwAAqeJcPI1d8Pwbjt7nttzAAA]]></MeetingId><MeetingRoomId><![CDATA[mRidadc]]></MeetingRoomId><MraAddress><Protocol>1</Protocol><DialString><![CDATA[DialString]]></DialString></MraAddress><RoomResponseStatus>2</RoomResponseStatus></xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| MraAddress.Protocol |  | 信令协议。1: SIP2: H.323 |
| MraAddress.DialString |  | 信令地址 |
| RoomResponseStatus |  | 应答结果。0: 无应答2: 入会中3: 被拒绝5: 取消呼叫 |
