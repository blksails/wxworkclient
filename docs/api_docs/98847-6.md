# 设置会议默认布局 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98847](https://developer.work.weixin.qq.com/document/path/98847)
- **文档 ID**: `98847`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/layout/set_default?access_token=ACCESS_TOKEN`

## 接口描述

对API成功预定的会议设置默认布局。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议 ID |
| selected_layout_id | string | 是 | 会议应用的布局 ID（若送空""，表示恢复成会议自带的默认原始布局） |

### 请求示例

```json
{"meetingid":"MEETINGID","selected_layout_id":"3acdd90-b0b2"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
