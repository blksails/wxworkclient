# 删除文档 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97735](https://developer.work.weixin.qq.com/document/path/97735)
- **文档 ID**: `97735`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/del_doc?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于删除指定文档、表格、智能表格及收集表。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 否 | 文档docid（docid、formid只能填其中一个），仅可删除应用自己创建的文档 |
| formid | string | 否 | 收集表id（docid、formid只能填其中一个），仅可删除应用自己创建的收集表 |

### 请求示例

```json
{"docid":"DOCID","formid":"FORMID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
