## 其他说明

### ww.navigateToKfChat(params)

兼容性：企业微信 iOS、Android、PC 3.1.12 开始支持
功能描述进入微信客服消息界面。
ww.navigateToKfChat({openKfId:'wkAJ2GCAAAZSfhHCt7IFSvLKtMPxyAAA',externalUserId:'wmAJ2GCAAAZSfhHCt7IFSvLKtMPxyBBB'})
使用限制本接口必须使用应用身份进行注册应用必须具有“微信客服->获取基础信息”权限当前企业须已开启「微信客服」应用当前成员须是指定客服账号的坐席
参数说明params: Object属性类型必填说明最低版本openKfIdstring是客服账号 IDexternalUserIdstring否微信客户 ID若没指定，则跳转到客服账号的消息列表界面企业微信 3.1.18successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

进入微信客服消息界面。

### 使用限制

本接口必须使用应用身份进行注册应用必须具有“微信客服->获取基础信息”权限当前企业须已开启「微信客服」应用当前成员须是指定客服账号的坐席

### params: Object

属性类型必填说明最低版本openKfIdstring是客服账号 IDexternalUserIdstring否微信客户 ID若没指定，则跳转到客服账号的消息列表界面企业微信 3.1.18successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

wx.invoke('navigateToKfChat',{"openKfId":"wkAJ2GCAAAZSfhHCt7IFSvLKtMPxyAAA","externalUserId":"wmAJ2GCAAAZSfhHCt7IFSvLKtMPxyBBB"},function(res){});
