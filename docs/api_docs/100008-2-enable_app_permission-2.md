# 如何接入

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100008](https://developer.work.weixin.qq.com/document/path/100008)
- **文档 ID**: `100008`
- **API 名称**: `enable_app_permission`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chatdata/enable_app_permission?access_token=ACCESS_TOKEN`
- **分组信息**: 第 2 个接口，共 13 个

## 接口描述

在第三方应用或代开发应用的权限详情页中，勾选开启“数据与智能专区权限”。

## 其他说明

### 注意

勾选后，设置需要分析的企业数据、使用的模型、使用的程序，以及是否使用会话展示组件。
