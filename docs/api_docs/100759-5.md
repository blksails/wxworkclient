# 自定义转发到会话

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100759](https://developer.work.weixin.qq.com/document/path/100759)
- **文档 ID**: `100759`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

自定义转发到会话。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| params.title | string | 是 | 分享标题 |
| params.desc | string | 是 | 分享描述 |
| params.link | string | 是 | 分享链接 |
| params.imgUrl | string | 是 | 分享图标 |
| params.enableIdTrans | number | 否 | 企业微信 4.0.20 是否开启id转译, 1 为开启，0 为关闭，默认为0 仅供第三方应用使用 必须使用应用身份进行注册 |
| params.success | Function | 否 | 成功回调 |
| params.fail | Function | 否 | 失败回调 |
| params.cancel | Function | 否 | 取消回调 |
| params.complete | Function | 否 | 完成回调 |

### 请求示例

```text
ww.shareAppMessage({
  title: '企业微信',
  desc: '让每个企业都有自己的微信',
  link: 'https://work.weixin.qq.com/',
  imgUrl: 'https://res.mail.qq.com/node/ww/wwmng/style/images/index_share_logo$13c64306.png',
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errMsg | string | 通用错误信息 |
| errCode | number | 通用错误码 |
