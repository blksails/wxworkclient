# 跳转到认领班级的界面

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94543](https://developer.work.weixin.qq.com/document/path/94543)
- **文档 ID**: `94543`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

此接口仅在企业微信3.1.8及以后版本支持。必须先调用过wx.qy.login，且session_key未过期，开发者可调用checkSession 检查当前登录态。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |

### 请求示例

```text
wx.qy.claimClassAdmin({
    success: function(res) {
      // 回调成功
	  var departmentIds = res.departmentIds;  //数组类型
	  for (var i = 0; i < departmentIds.length; i++)
      {
		var departmentId = departmentIds[i];
      }
    },
    fail: function(res) {
      // 失败处理
	  console.log(res)
    }
});
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| departmentIds | Array | 认领的班级部门id列表 |

## 其他说明

### 错误列表

| res.err_msg | 说明 |
| --- | --- |
| claimClassAdmin:ok | 执行成功 |
| claimClassAdmin:fail no permission | 必须先成功调用agentConfig |
| claimClassAdmin:fail user not in allow list | 当前成员不在应用可见范围 |
