# secret的重置

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97163](https://developer.work.weixin.qq.com/document/path/97163)
- **文档 ID**: `97163`
- **API 名称**: `reset_secret`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/reset_secret?suite_access_token=SUITE_ACCESS_TOKEN`
- **分组信息**: 第 2 个接口，共 2 个

## 接口描述

服务商可在服务商管理端的代开发应用详情页点击“重新获取”来重置应用secret...
