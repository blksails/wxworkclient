# 支付成功通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97196](https://developer.work.weixin.qq.com/document/path/97196)
- **文档 ID**: `97196`

## 接口描述

当服务商购买接口调用许可账号并完成付款后，企业微信后台会推送“支付成功通知”到服务商的系统事件接收URL（应用管理 -通用开发参数-系统事件接收URL）。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ServiceCorpId |  | 否 | 服务商CorpID |
| InfoType |  | 否 | 事件类型，此时固定为license_pay_success |
| AuthCorpId |  | 否 | 客户企业CorpID |
| OrderId |  | 否 | 订单号。如果为多企业新购订单，该值为子订单号 |
| BuyerUserId |  | 否 | 服务商内下单用户UserID |
| TimeStamp |  | 否 | 时间戳 |

### 请求示例

```xml
<xml><ServiceCorpId><![CDATA[xxxx]]></ServiceCorpId><InfoType><![CDATA[license_pay_success]]></InfoType><AuthCorpId><![CDATA[yyyy]]></AuthCorpId><OrderId><![CDATA[zzzz]]></OrderId><BuyerUserId><![CDATA[USERID]]</BuyerUserId><TimeStamp>1403610513</TimeStamp></xml>
```
