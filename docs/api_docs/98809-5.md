# 获取Rooms会议室资源 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98809](https://developer.work.weixin.qq.com/document/path/98809)
- **文档 ID**: `98809`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/rooms/get_inventory?access_token=ACCESS_TOKEN`

## 接口描述

获取企业购买的Rooms会议室资源。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| normal_count | uint32 | 普通设备数 |
| special_count | uint32 | 专款设备数 |
| normal_used_count | uint32 | 普通设备使用数 |
| special_used_count | uint32 | 专款设备使用数 |
| normal_expired_count | uint32 | 普通设备过期数 |
| special_expired_count | uint32 | 专款设备过期数 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","normal_count":10,"special_count":2,"normal_used_count":2,"special_used_count":1,"normal_expired_count":0,"special_expired_count":0}
```
