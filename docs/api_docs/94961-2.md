## 接口描述

接口说明：在企业微信内快速跳转到微信客服消息的界面

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| openKfId | String | 是 | 客服账号ID |
| externalUserId | String | 否 | 微信客户ID，若没指定，则跳转到客服账号的消息列表界面 |
| success | Function | 否 | 接口调用成功的回调函数 |
| fail | Function | 否 | 接口调用失败的回调函数 |
| complete | Function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |
| ios |  | 否 | ≥3.1.12 |
| android |  | 否 | ≥3.1.12 |
| win |  | 否 | ≥3.1.12 |
| mac |  | 否 | 暂不支持 |
| 自建应用 | 自建应用 | 否 | 所有 |
| 第三方应用 | 第三方应用 | 否 | 需具有“微信客服->获取基础信息”权限 |

### 请求示例

```json
wx.qy.navigateToKfChat({openKfId:"wkAJ2GCAAAZSfhHCt7IFSvLKtMPxyAAA",externalUserId:"wmAJ2GCAAAZSfhHCt7IFSvLKtMPxyBBB",success:function(res){// 回调}});
```
