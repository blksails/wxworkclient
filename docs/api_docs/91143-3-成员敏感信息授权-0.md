# 成员敏感信息授权

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91143](https://developer.work.weixin.qq.com/document/path/91143)
- **文档 ID**: `91143`
- **API 名称**: `成员敏感信息授权`
- **分组信息**: 第 3 个接口，共 3 个

## 接口描述

第三方应用无法通过API接口获取成员的敏感信息，但如果确实需要用到成员的头像、个人二维码等敏感信息，可以申请成员授权敏感信息只读的权限。该权限和通讯录权限等级是相互独立的。
