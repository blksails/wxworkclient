# 代开发应用external_userid转换

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95195](https://developer.work.weixin.qq.com/document/path/95195)
- **文档 ID**: `95195`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/to_service_external_userid?access_token=ACCESS_TOKEN`

## 接口描述

企业同时授权了服务商的第三方应用与代开发应用，服务商可使用该接口将代开发应用获取到的external_userid转换为第三方应用的external_userid，从而进行关联映射。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 代开发自建应用的调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| external_userid | string | 是 | 代开发自建应用获取到的外部联系人ID |

### 请求示例

```json
{
  "external_userid":"wmAAAAAAA"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| external_userid | string | 该服务商第三方应用下的企业的外部联系人ID |

### 响应示例

```json
{
 "errcode":0,
 "errmsg":"ok",
 "external_userid":"ooAAAAAAAAAAA"
}
```

## 其他说明

### 权限说明

> 该企业授权了该服务商第三方应用,且授权的第三方应用具备“企业客户权限->客户基础信息”权限
> 相应客户的跟进人至少有一个在应用可见范围内 或 相应客户所属客户群群主至少有一个在应用可见范围内
> 应用需具备“企业客户权限->客户基础信息”权限
