## 其他说明

### ww.shareAppMessage(params)

兼容性：企业微信 2.4.5 开始支持
功能描述自定义转发到会话。
ww.shareAppMessage({title:'企业微信',desc:'让每个企业都有自己的微信',link:'https://work.weixin.qq.com/',imgUrl:'https://res.mail.qq.com/node/ww/wwmng/style/images/index_share_logo$13c64306.png',})
参数说明params: Object属性类型必填说明最低版本titlestring是分享标题descstring是分享描述linkstring是分享链接imgUrlstring是分享图标enableIdTransnumber否企业微信 4.0.20
是否开启id转译, 1 为开启，0 为关闭，默认为0
仅供第三方应用使用
必须使用应用身份进行注册successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码

### 功能描述

自定义转发到会话。

### params: Object

属性类型必填说明最低版本titlestring是分享标题descstring是分享描述linkstring是分享链接imgUrlstring是分享图标enableIdTransnumber否企业微信 4.0.20
是否开启id转译, 1 为开启，0 为关闭，默认为0
仅供第三方应用使用
必须使用应用身份进行注册successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码
