# 获取文件权限信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97962](https://developer.work.weixin.qq.com/document/path/97962)
- **文档 ID**: `97962`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/get_file_permission`

## 接口描述

该接口用于获取文件的权限信息。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fileid | string | 是 | 文件fileid |

### 请求示例

```json
{
    "fileid": "FILEID"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| share_range | object | 文件分享设置 |
| share_range.enable_corp_internal | bool | 是否为企业内可访问 |
| share_range.corp_internal_auth | uint32 | 企业内权限信息 普通文档： 1:仅浏览（可下载) 4:仅预览（仅专业版企业可设置）255:无权限或需要审批；如果不填充此字段为保持原有状态 微文档： 1:仅浏览（可下载）；如果不填充此字段为保持原有状态 |
| share_range.enable_corp_external | bool | 是否为企业外可访问 |
| share_range.corp_external_auth | uint32 | 企业外权限信息 普通文档： 1:仅浏览（可下载) 4:仅预览（仅专业版企业可设置） 255:无权限或需要审批；如果不填充此字段为保持原有状态 微文档： 1:仅浏览（可下载）；如果不填充此字段为保持原有状态 |
| share_range.corp_internal_approve_only_by_admin | bool | 是否开启企业内管理员审批 |
| share_range.corp_external_approve_only_by_admin | bool | 是否开启企业外管理员审批 |
| secure_setting | object | 文件安全配置 |
| secure_setting.enable_readonly_copy | bool | 是否开启只读备份 |
| secure_setting.modify_only_by_admin | bool | 是否只允许管理员进行修改 |
| secure_setting.enable_readonly_comment | bool | 是否开启只读评论 |
| secure_setting.ban_share_external | bool | 是否禁止分享到企业外部 |
| inherit_father_auth | object | 从文件父路径继承的权限 |
| inherit_father_auth.inherit | bool | 文件是否开启父路径权限继承 |
| file_member_list | object[] | 文件夹、文档成员 |
| file_member_list[].type | uint32 | 成员类型 1:个人 2:部门 |
| file_member_list[].userid | string | 成员userid,字符串 |
| file_member_list[].auth | uint32 | 成员权限 |
| watermark | object | 水印相关设置（除show_visitor_name字段外，其余字段仅文档可设置） |
| watermark.text | string | 水印文字 |
| watermark.margin_type | uint32 | 水印类型。1：低密度水印， 2： 高密度水印 |
| watermark.show_visitor_name | bool | 是否显示访问人名称 |
| watermark.force_by_admin | bool | 管理员是否强制要求使用水印 |
| watermark.show_text | bool | 是否展示水印文本 |
| watermark.force_by_space_admin | bool | 空间管理员是否强制要求使用水印 |
