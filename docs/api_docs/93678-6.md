# 获取健康上报任务详情 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93678](https://developer.work.weixin.qq.com/document/path/93678)
- **文档 ID**: `93678`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/health/get_report_job_info?access_token=ACCESS_TOKEN`

## 接口描述

通过此接口可以获取指定的健康上报任务详情。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| jobid |  | 是 | 任务ID |
| date |  | 是 | 具体某天任务详情，仅支持获取最近14天数据 |
| 自建应用 | 自建应用 | 否 | 配置到「健康上报 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"jobid":"jobid1","date":"2020-03-27"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| title |  | 任务名称 |
| creator |  | 发起人的userid |
| type |  | 任务类型。1：表示企业内部成员，2：表示家长 |
| apply_range |  | 适用范围 |
| apply_range.userids |  | type=1时表示企业成员的userid，type=2时表示学生的userid |
| apply_range.partyids |  | type=1时表示企业通讯录的部门id，type=2时表示学校通讯录的部门id |
| report_to.userids |  | 汇报对象的企业通讯录userid |
| report_type |  | 上报方式。1表示仅上报一次，2表示每天都上报 |
| skip_weekend |  | 非工作日是否需要上报。0表示周末需要上报，1表示周末不需要上报 |
| finish_cnt |  | 已填表人数 |
| question_templates |  | 健康上报问题列表 |
| question_templates.question_id |  | 问题的question_id |
| question_templates.title |  | 问题的title |
| question_templates.question_type |  | 问题类型，1：表示是填空题(包括地理位置和日期)，2：表示是单选题，3：表示多选题，4：表示收集文件/图片题，5：签名链接，6：行程卡截图 |
| question_templates.is_required |  | 问题是否必填，1：表示“是”，0：表示“否” |
| question_templates.option_list |  | 选项列表，仅当该题为单/多选题时才有该字段 |
| question_templates.option_list.option_id |  | 该选项的option_id |
| question_templates.option_list.option_text |  | 该选项的文案说明 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","job_info":{"title":"职工收集任务","creator"："creator_userid","type":1,"apply_range":{"userids":["userid1","userid2"],"partyids":[1,2]},"report_to":{"userids":["userid1","userid2"]}"report_type":1,"skip_weekend":0,"finish_cnt":10,"question_templates":[{"question_id":1,"title":"请问你有任何身体不适吗？","question_type":2,"is_required":1,"option_list":[{"option_id":1,"option_text":"有"},{"option_id":2,"option_text":"没有"}]},{"question_id":2,"title":"常驻地址","question_type":1,"is_required":0}]}}
```
