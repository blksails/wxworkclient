# 取消预约会议

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93709](https://developer.work.weixin.qq.com/document/path/93709)
- **文档 ID**: `93709`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/cancel`

## 接口描述

该接口用于取消一个指定的预约会议。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| meetingid | string | 是 | 会议id，仅允许取消预约状态下的会议 |

### 请求示例

```json
{
   "meetingid": "XXXXXXXXX"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "ok"
}
```

## 其他说明

### 权限说明

* 仅允许取消当前应用创建的会议。
* 自建应用需要配置在“可调用接口的应用”列表
