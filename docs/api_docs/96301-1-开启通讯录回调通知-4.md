# 开启通讯录回调通知

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96301](https://developer.work.weixin.qq.com/document/path/96301)
- **文档 ID**: `96301`
- **API 名称**: `开启通讯录回调通知`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/webhook/callback/update?access_token=ACCESS_TOKEN`
- **分组信息**: 第 1 个接口，共 3 个

## 接口描述

在企业微信管理后台的“管理工具-通讯录同步-设置接收事件服务器”处，进入配置页面，要求填写通讯录同步助手的URL、Token、EncodingAESKey三个参数。为保证企业数据安全，URL需要配置本企业主体的域名链接。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| URL | string | 是 | 通讯录同步助手的URL |
| Token | string | 是 | Token 参数 |
| EncodingAESKey | string | 是 | AES 加密密钥 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
