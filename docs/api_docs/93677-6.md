# 获取健康上报任务ID列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93677](https://developer.work.weixin.qq.com/document/path/93677)
- **文档 ID**: `93677`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/health/get_report_jobids?access_token=ACCESS_TOKEN`

## 接口描述

通过此接口可以获取企业当前正在运行的上报任务ID列表。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| offset |  | 否 | 分页，偏移量, 默认为0 |
| limit |  | 否 | 分页，预期请求的数据量，默认为100，取值范围 1 ~ 100 |
| 自建应用 | 自建应用 | 否 | 配置到「健康上报 - 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |

### 请求示例

```json
{"offset":0,"limit":100}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| ending |  | 是否结束。0：表示还有更多数据，需要继续拉取，1：表示已经拉取完所有数据 |
| jobids |  | 任务id |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","ending":1,"jobids":["jobid1","jobid2"]}
```
