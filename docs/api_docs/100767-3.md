### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|

|  | Function | 否 | 取消回调 |
|  | Function | 否 | 完成回调 |

### 请求示例

#### 示例 1: 功能描述

```json
ww.getLocalFileData({localId:'',})
```

#### 示例 2: 示例代码

```json
wx.invoke('getLocalFileData',{localId:'',// 文件的localID},function(res){// 这里是回调函数if(res.err_msg=="getLocalFileData:ok"){varlocalData=res.localData;// localData是文件的base64数据}});
```

## 其他说明

### ww.getLocalFileData(params)

兼容性：企业微信 iOS、Android 4.0.20 开始支持
功能描述获取 chooseMessageFile 返回的 localId 对应的文件内容。
ww.getLocalFileData({localId:'',})
使用限制本接口必须使用应用身份进行注册当前成员必须在应用的可见范围之中
参数说明params: Object属性类型必填说明localIdstring是本地临时文件 IDsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码localDatastring是文件内容的base64编码
错误信息errMsg说明getLocalFileData:ok执行成功no permission应用签名校验失败，或成员不在应用的可见范围内no such filelocalId不存在或者文件已删除file exceed size limit不支持超过20M的文件

### 功能描述

获取 chooseMessageFile 返回的 localId 对应的文件内容。

### 使用限制

本接口必须使用应用身份进行注册当前成员必须在应用的可见范围之中

### params: Object

属性类型必填说明localIdstring是本地临时文件 IDsuccessFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码localDatastring是文件内容的base64编码

### 错误信息

errMsg说明getLocalFileData:ok执行成功no permission应用签名校验失败，或成员不在应用的可见范围内no such filelocalId不存在或者文件已删除file exceed size limit不支持超过20M的文件
