# PSTN 外呼状态更新事件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98774](https://developer.work.weixin.qq.com/document/path/98774)
- **文档 ID**: `98774`

## 接口描述

对于API创建的会议，当 PSTN 外呼的状态发生变化时，触发该事件。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 操作者UserID。如果为系统触发事件，默认为sys；如果为成员操作事件，仅当是企业成员时有值。 |
| FromUserTmpOpenId |  | 否 | 操作者TmpOpenID。操作者的会中临时ID，仅成员操作事件有值。 |
| CreateTime |  | 否 | 消息创建时间，unix时间戳 |
| MsgType |  | 否 | 消息类型，此时固定为：event |
| Event |  | 否 | 事件类型，此时固定为：meeting_change |
| ChangeType |  | 否 | 事件类型，此时固定为：pstn_status_update |
| MeetingId |  | 否 | 会议ID |
| PstnStatus |  | 否 | PSTN外呼状态。START_INVITE: 开始邀请START_ACCEPT: 用户接听LEAVE_WITHOUT_ACCEPT: 用户无接听挂断或拒绝LEAVE_WITH_ACCEPTED: 用户已接听挂断CANCLE_INVITE: 呼叫中邀请人主动取消 |
