# 获取标签列表

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90352](https://developer.work.weixin.qq.com/document/path/90352)
- **文档 ID**: `90352`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/tag/list`

## 接口描述

获取标签列表

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| taglist | object[] | 标签列表 |
| taglist[].tagid | int32 | 标签id |
| taglist[].tagname | string | 标签名 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "ok",
   "taglist":[
      {"tagid":1,"tagname":"a"},
      {"tagid":2,"tagname":"b"}
   ]
}
```

## 其他说明

### 权限说明

自建应用或通讯同步助手可以获取所有标签列表；第三方应用仅可获取自己创建的标签。
