# 读取门禁通行规则

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96030](https://developer.work.weixin.qq.com/document/path/96030)
- **文档 ID**: `96030`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/devicedata/get_accesscontrol_rule`

## 接口描述

第三方应用获取企业已设置的门禁规则。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 应用调用接口权限 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| device_sn | string | 是 | 打卡设备SN |

### 请求示例

```json
{
	"device_sn":"SN"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码描述 |
| pass_rule | object | 门禁规则列表 |
| pass_rule.items.rule_id | string | 门禁规则编号 |
| pass_rule.items.name | string | 门禁规则名称 |
| pass_rule.items.rule_list | array | 门禁规则，一个rule_id下可存在多条规则 |
| pass_rule.items.effect_time | uint32 | 门禁生效时间 |
| pass_rule.items.effect_open_userid_list | object[] | 门禁规则适用成员的openuserid 列表 |
| pass_rule.items.effect_open_userid_list.open_userid | string | 生效成员的openuserid |
| pass_rule.items.effect_open_userid_list.user_type | int32 | 成员类型：0：企业员工 2：学生 不填则默认值为0 |
| remote_pass_rule | object | 远程开门规则列表，object内各字段含义与 pass_rule_list相同 |
