# 呼叫Rooms会议室 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98804](https://developer.work.weixin.qq.com/document/path/98804)
- **文档 ID**: `98804`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/rooms/call?access_token=ACCESS_TOKEN`

## 接口描述

会议可以通过Rooms会议室 ID 呼叫Rooms会议室邀请其入会。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 会议 ID |
| meeting_room_id | string | 否 | Rooms会议室 ID，与 mra_address 二选一 |
| mra_address | object | 否 | MRA 对象，详见MRA |
| protocol | uint32 | 是 | 信令协议。1：SIP2：H.323 |
| dial_string | string | 是 | 信令地址。如果是 H.323 类型，输入 IP 地址或 E.164 号码。如果是 SIP 类型，输入 IP 地址或 URI。 |

### 请求示例

```json
{"meetingid":"MEETINGID","meeting_room_id":"ROOMID","mra_address":{"protocol":2,"dial_string":"xxx"}}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| invite_id | string | 呼叫 ID |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","invite_id":"INVITEID"}
```
