# 提交图片 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98972](https://developer.work.weixin.qq.com/document/path/98972)
- **文档 ID**: `98972`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/miniapppay/upload_image?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST(HTTPS)请求地址：https://qyapi.weixin.qq.com/cgi-bin/miniapppay/upload_image?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 自建应用 | 自建应用 | 否 | 配置到「对外收款- 可调用接口的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 暂不支持 |
| 第三方应用 | 第三方应用 | 否 | 暂不支持 |
| access_token |  | 是 | 调用接口凭证 |

### 请求示例

```text
---------------------------acebdf13572468Content-Disposition:form-data;name="fieldNameHere";filename="20180103195745.png"Content-Type:image/pngContent-Length:220<@INCLUDE*C:\Users\abelzhu\Pictures\企业微信截图_20180103195745.png*@>---------------------------acebdf13572468--
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| open_wx_pay_media_id |  | 上传后得到的图片id，30天后过期 |

### 响应示例

```json
{"errcode":0,"errmsg":""，"open_wx_pay_media_id":"NF_LnvqOch94lvxMB5fUWw8fbKvYrF_EcH239fgIjIkG_q4Q5kyzEGpJtddhyfZY"}
```

## 其他说明

### 上传的图片大小限制

图片文件大小应在 5B ~ 2MB 之间

### 上传图片的类型限制

图片文件名后缀应为JPG/JPEG、PNG、BMP之一

### 图片ID的失效条件

以下条件满足其一，则图片ID失效
1、距离上传超过30天2、图片ID已经被用于提交申请单成功，且所有相关的申请单均已经走完流程
