# 更新网络研讨会嘉宾列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98872](https://developer.work.weixin.qq.com/document/path/98872)
- **文档 ID**: `98872`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/meeting/webinar/update_guest_list?access_token=ACCESS_TOKEN`

## 接口描述

更新指定的网络研讨会嘉宾列表。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| meetingid | string | 是 | 网络研讨会ID。 |
| guests | object[] | 是 | 网络研讨会嘉宾列表。详见GuestInfo。 |
| guest_type | uint32 | 是 | 嘉宾类型：1：内部嘉宾2：外部嘉宾 |
| userid | string | 否 | 嘉宾的userid， 内部嘉宾必填。 |
| area | string | 否 | 国家/地区代码，外部嘉宾必填（例如：中国传86，不是+86）。 |
| phone_number | string | 否 | 手机号，外部嘉宾必填。 |
| guest_name | string | 否 | 嘉宾名称，外部嘉宾必填（1~16位字符长度）。 |
| email | string | 否 | 邮箱，外部嘉宾选填。 |

### 请求示例

```json
{"meetingid":"MEETINGID","guests":[{"guest_type":1,"userid":"USERID","area":86,"phone_number":"13333333333","guest_name":"NAME","email":"xxxxx@yy.com"}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
