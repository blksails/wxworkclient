# 获取硬件产生的数据

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96091](https://developer.work.weixin.qq.com/document/path/96091)
- **文档 ID**: `96091`
- **API 名称**: `应用获取门禁通行记录`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/accesscontrol/get_access_records`
- **分组信息**: 第 3 个接口，共 7 个

## 接口描述

应用获取门禁通行记录接口用于获取企业下绑定的门禁设备的通行记录。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| access_records | object[] | 通行记录列表 |
| access_records[].user_id | string | 用户ID |
| access_records[].time | uint32 | 通行时间戳 |
| access_records[].location | string | 通行地点 |
