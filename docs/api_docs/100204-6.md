# 更新字段

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100204](https://developer.work.weixin.qq.com/document/path/100204)
- **文档 ID**: `100204`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/smartsheet/update_fields`

## 接口描述

本接口用于更新智能中的某个子表里的一个或多个字段的标题和字段属性信息。注意：该接口只能更新字段名、字段属性，不能更新字段类型。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 是 | 文档的docid |
| sheet_id | string | 是 | 表格ID |
| fields | object[] | 是 | 字段详情 |
| fields[].field_id | string | 是 | 字段 ID，更新字段属性时需要填写该字段，但字段 ID 不能被更新 |
| fields[].field_title | string | 否 | 字段标题，需要更新为的字段标题 |
| fields[].field_type | string | 是 | 字段类型，必须为原属性 |
| fields[].property_text | object | 否 | `文本` 类型的字段属性为空 |
| fields[].property_number | object(NumberFieldProperty) | 否 | `数字` 类型的字段属性 |
| fields[].property_checkbox | object(CheckboxFieldProperty) | 否 | `复选框` 类型的字段属性 |
| fields[].property_date_time | object(DateTimeFieldProperty) | 否 | `日期` 类型的字段属性 |
| fields[].property_attachment | object(AttachmentFieldProperty) | 否 | `文件` 类型的字段属性 |
| fields[].property_user | object(UserFieldProperty) | 否 | `人员` 类型的字段属性 |
| fields[].property_url | object(UrlFieldProperty) | 否 | `超链接` 类型的字段属性 |
| fields[].property_select | object(SelectFieldProperty) | 否 | `多选` 类型的字段属性 |
| fields[].property_created_user | object | 否 | `创建人` 类型的字段属性为空 |
| fields[].property_modified_user | object | 否 | `最后编辑人` 类型的字段属性为空 |
| fields[].property_created_time | object(CreatedTimeFieldProperty) | 否 | `创建时间` 类型的字段属性 |
| fields[].property_modified_time | object(ModifiedTimeFieldProperty) | 否 | `最后编辑时间` 类型的字段属性 |
| fields[].property_progress | object(ProgressFieldProperty) | 否 | `进度` 类型的字段属性 |
| fields[].property_single_select | object(SingleSelectFieldProperty) | 否 | `单选` 类型的字段属性 |
| fields[].property_reference | object(PropertyReference) | 否 | `引用` 类型的字段属性 |
| fields[].property_location | object(LocationFieldProperty) | 否 | `地理位置` 类型的字段属性 |
| fields[].property_auto_number | object(AutoNumberFieldProperty) | 否 | `自动编号` 类型的字段属性 |
| fields[].property_currency | object(CurrencyFieldProperty) | 否 | `货币` 类型的字段属性 |
| fields[].property_ww_group | object(WwGroupFieldProperty) | 否 | `群` 类型的字段属性 |
| fields[].property_percentage | object(PercentageFieldProperty) | 否 | `百分数` 类型的字段属性 |
| fields[].property_barcode | object(BarcodeFieldProperty) | 否 | `条码` 类型的字段属性 |

### 请求示例

```json
{
	"docid": "DOCID",
	"sheet_id": "SHEETID",
	"fields": [{
		"field_id": "FIELD_ID",
		"field_title": "TITLE",
		"field_type": "FIELD_TYPE_TEXT"
	}]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| fields | object[] | 字段详情 |
| fields[].field_id | string | 字段 ID |
| fields[].field_title | string | 字段标题 |
| fields[].field_type | string | 字段类型 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"fields": [{
		"field_id": "FIELDID",
		"field_title": "TITLE",
		"field_type": "FIELD_TYPE_TEXT"
	}]
}
```
