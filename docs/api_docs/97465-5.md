# 设置文档权限

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97465](https://developer.work.weixin.qq.com/document/path/97465)
- **文档 ID**: `97465`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedoc/doc_get_auth`

## 接口描述

该接口用于获取文档、表格、智能表格的查看规则、文档通知范围及权限、安全设置信息

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 访问令牌 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| docid | string | 是 | 文档id |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| access_rule | object | 文档的查看规则 |
| access_rule.enable_corp_internal | bool | 是否允许企业内成员浏览文档 |
| access_rule.corp_internal_auth | uint32 | 企业内成员主动查看文档后获得的权限类型 1:只读 2:读写（目前仅智能表可设置为读写） |
| access_rule.enable_corp_external | bool | 是否允许企业外成员浏览文档 |
| access_rule.corp_external_auth | uint32 | 企业内成员主动查看文档后获得的权限类型 1:只读 2:读写（目前仅智能表可设置为读写） |
| access_rule.corp_internal_approve_only_by_admin | bool | 企业内成员浏览文档是否必须由管理员审批，enable_corp_internal为false时，只能为true |
| access_rule.corp_external_approve_only_by_admin | bool | 企业外成员浏览文档是否必须由管理员审批，enable_corp_external和ban_share_external均为false时，该参数只能为true |
| access_rule.ban_share_external | bool | 是否允许企业外成员浏览文档 |
| secure_setting | object | 安全设置信息 |
| secure_setting.enable_readonly_copy | bool | 仅浏览权限的成员是否允许导出、复制、打印 |
| secure_setting.watermark | object | 文档水印设置 |
| secure_setting.watermark.margin_type | uint32 | 水印密度 1:稀疏 2:紧密 |
| secure_setting.watermark.show_visitor_name | bool | 是否展示访问者名字 |
| secure_setting.watermark.show_text | bool | 是否展示水印文字 |
| secure_setting.watermark.text | bytes | 水印文字 |
| secure_setting.enable_readonly_comment | bool | 是否允许仅浏览权限的成员评论 |
| doc_member_list | object[] | 文档通知范围及权限列表 |
| doc_member_list.type | uint32 | 文档通知范围成员种类 1:user, 只支持成员 |
| doc_member_list.userid | bytes | 企业成员的userid |
| doc_member_list.tmp_external_userid | string | 外部用户临时id。同一个用户在不同的文档中返回的该id不一致。 |
| doc_member_list.auth | uint32 | 该文档通知范围成员的权限 1:只读 2:读写（目前仅智能表可设置为读写） 7:管理员 |
| co_auth_list | object | 文档查看权限特定部门列表，可以直接浏览文档 |
| co_auth_list.type | uint32 | 特定部门列表 2:部门, 目前只支持部门 |
| co_auth_list.departmentid | uint64 | 特定部门id |
| co_auth_list.auth | uint32 | 权限类型 1:只读,2:读写（目前仅智能表可设置为读写） |

## 其他说明

### 权限说明

自建应用需配置到“可调用应用”列表中的应用secret所获取的accesstoken来调用（accesstoken如何获取？）
第三方应用需具有“文档”权限
代开发自建应用需具有“文档”权限
只能访问该应用创建的文档
