# 批量更新学生 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100150](https://developer.work.weixin.qq.com/document/path/100150)
- **文档 ID**: `100150`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/user/batch_update_student?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/school/user/batch_update_student?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| students |  | 否 | 学生列表，每次最多100个 |
| 自建应用 | 自建应用 | 否 | 配置到「家校沟通-读取和编辑家校通讯录的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 需拥有「家校沟通」可使用家校通讯录-家校通讯录编辑权限 |
| 第三方应用 | 第三方应用 | 否 | 需拥有「家校沟通」使用和编辑权限 |

### 请求示例

```json
{"students":[{"student_userid":"zhangsan","mobile":"10000000000","new_student_userid":"zhangsan_new","name":"张三","department":[1,2]},{"student_userid":"lisi","mobile":"10000000000","name":"李四","department":[3,4]}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| students.student_userid |  | 学生UserID。学校内必须唯一。不区分大小写，长度为1~64个字节。只能由数字、字母和“_-@.”四种字符组成，且第一个字符必须是数字或字母。 |
| students.mobile |  | 学生手机号 |
| students.new_student_userid |  | 要变更的学生UserID,不能与已存在的UserID相同。每个学生仅能修改一次。 |
| students.name |  | 学生姓名，长度为1~32个字符 |
| students.department |  | 学生所在的班级id列表,不超过20个 |
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| result_list |  | 失败的学生列表 |
| result_list.student_userid |  | 失败的学生userid |
| result_list.errcode |  | 返回码 |
| result_list.errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":1,"errmsg":"invalid student_userid: zhangsan","result_list":[{"student_userid":"zhangsan","errcode":1,"errmsg":"invalid student_userid: zhangsan"}]}
```
