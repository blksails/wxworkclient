## 其他说明

### 插件安装

开发者通过在微信小程序开发者工具(版本号1.02.1903211以上)安装企业微信插件，可以方便地开发企业微信小程序，详见企业微信小程序开发

### 企业微信上的多场景调试

在企业微信聊天工具栏、聊天会话等场景打开小程序时，默认都是打开小程序正式包，没法对debug包进行调试预览。企业微信从3.0.28版本开始支持通过以下方式开启多场景调试，开启之后，在附件栏、工具栏、会话、工作台等场景打开此小程序时，都将使用本次扫码的开发版小程序来运行。具体步骤如下：
在微信小程序开发者工具生成小程序的预览二维码，然后使用企业微信扫码打开小程序点击小程序右上角菜单，找到“开启多场景调试”并点击开启
