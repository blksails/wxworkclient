# 创建成员

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90331](https://developer.work.weixin.qq.com/document/path/90331)
- **文档 ID**: `90331`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/user/create`

## 接口描述

用于在企业微信通讯录中创建新成员。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证。获取方法查看“获取access_token” |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| userid | string | 是 | 成员UserID。对应管理端的账号，企业内必须唯一。长度为1~64个字节。只能由数字、字母和“_-@.”四种字符组成，且第一个字符必须是数字或字母。系统进行唯一性检查时会忽略大小写。 |
| name | string | 是 | 成员名称。长度为1~64个utf8字符 |
| alias | string | 否 | 成员别名。长度1~64个utf8字符 |
| mobile | string | 否 | 手机号码。企业内必须唯一，mobile/email二者不能同时为空，中国大陆手机号码可省略“+86”，其他国家或地区必须要带上国际码。 |
| department | int32[] | 否 | 成员所属部门id列表，不超过100个。当不填写department或id为0时，成员会放在其他（待设置部门）下，当填写的部门不存在时，会在在其他（待设置部门）下新建对应部门 |
| order | int32[] | 否 | 部门内的排序值，默认为0，成员次序以创建时间从小到大排列。个数必须和参数department的个数一致，数值越大排序越前面。有效的值范围是[0, 2^32) |
| position | string | 否 | 职务信息。长度为0~128个字符 |
| gender | string | 否 | 性别。1表示男性，2表示女性 |
| email | string | 否 | 邮箱。可填写企业已有的邮箱账号，方便同事获取成员的邮箱账号以发邮件。长度6~64个字节，且为有效的email格式。企业内必须唯一，mobile/email二者不能同时为空。境外成员可用此邮箱登录企业微信。 |
| biz_mail | string | 否 | 如果企业已开通腾讯企业邮（企业微信邮箱），设置该值可创建企业邮箱账号。长度6~64个字节，且为有效的企业邮箱格式。企业内必须唯一。未填写则系统会为用户生成默认企业邮箱（由系统生成的邮箱可修改一次） |
| telephone | string | 否 | 座机。32字节以内，由纯数字、“-”、“+”或“,”组成。 |
| is_leader_in_dept | int32[] | 否 | 个数必须和参数department的个数一致，表示在所在的部门内是否为部门负责人。1表示为部门负责人，0表示非部门负责人。在审批（自建、第三方）等应用里可以用来标识上级审批人 |
| direct_leader | string[] | 否 | 直属上级UserID，设置范围为企业内成员，可以设置最多1个上级 |
| avatar_mediaid | string | 否 | 成员头像的mediaid，通过素材管理接口上传图片获得的mediaid |
| enable | int32 | 否 | 启用/禁用成员。1表示启用成员，0表示禁用成员 |
| extattr.attrs[].type | int32 | 否 | 扩展属性类型。0表示文本，1表示网页 |
| extattr.attrs[].name | string | 否 | 属性名称 |
| extattr.attrs[].text.value | string | 否 | 文本值 |
| extattr.attrs[].web.url | string | 否 | 网页链接 |
| extattr.attrs[].web.title | string | 否 | 网页标题 |
| to_invite | bool | 否 | 是否邀请该成员使用企业微信。默认值为true。 |
| external_position | string | 否 | 对外职务，如果设置了该值，则以此作为对外展示的职务，否则以position来展示。长度12个汉字内 |
| external_profile.external_corp_name | string | 否 | 企业简称 |
| external_profile.wechat_channels.nickname | string | 否 | 视频号名称 |
| external_profile.external_attr[].type | int32 | 否 | 扩展属性类型。0表示文本，1表示网页，2表示小程序 |
| external_profile.external_attr[].name | string | 否 | 属性名称 |
| external_profile.external_attr[].text.value | string | 否 | 文本值 |
| external_profile.external_attr[].web.url | string | 否 | 网页链接 |
| external_profile.external_attr[].web.title | string | 否 | 网页标题 |
| external_profile.external_attr[].miniprogram.appid | string | 否 | 小程序的appid |
| external_profile.external_attr[].miniprogram.pagepath | string | 否 | 小程序的页面路径 |
| external_profile.external_attr[].miniprogram.title | string | 否 | 小程序的标题 |
| address | string | 否 | 地址。长度最大128个字符 |
| main_department | int32 | 否 | 主部门 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| created_department_list.department_info[].name | string | 新增的部门名称 |
| created_department_list.department_info[].id | int32 | 新增的部门id |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "created",
	"created_department_list": {
		"department_info": [{
			"name": "xxxx",
			"id": 123
		}]
	}
}
```
