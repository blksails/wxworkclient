# 获取代开发模版授权二维码

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97201](https://developer.work.weixin.qq.com/document/path/97201)
- **文档 ID**: `97201`
- **API 名称**: `get_qrcode`
- **请求方法**: `GET`
- **接口地址**: `https://wework.qpic.cn/wwpic/287320_9v_kOiPSQIWHYqN_1626354359/0`
- **分组信息**: 第 3 个接口，共 3 个

## 接口描述

获取代开发模版对应的授权二维码。

## 请求信息

### 请求示例

```text
GET https://wework.qpic.cn/wwpic/287320_9v_kOiPSQIWHYqN_1626354359/0
```

## 其他说明

### 获取代开发模版授权二维码流程

1. 模版上线提交成功后，进入服务商后台-应用管理-代开发应用，进入代开发模版详情页
2. 可查看对应的授权二维码
3. 若需要将多个代开发应用模版组合成一个授权二维码，可在代开发应用首页，点击右上角“批量授权”按钮
