# 门禁通行原始数据回调

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97215](https://developer.work.weixin.qq.com/document/path/97215)
- **文档 ID**: `97215`
- **请求方法**: `POST`
- **接口地址**: `https://127.0.0.1/suite/receive`

## 接口描述

当用户授权设备上报门禁通行原始数据时，并且通行用户在应用可见范围内时，企业微信回调该指令回调到应用配置的数据回调url中。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| msg_signature | string | 是 | 消息签名 |
| timestamp | uint32 | 是 | 时间戳 |
| nonce | uint32 | 是 | 随机数 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName | string | 是 | 企业微信CorpID |
| FromUserName | string | 是 | 固定为sys |
| MsgType | string | 是 | 消息类型，固定为event |
| Event | string | 是 | 事件类型，固定为device_accesscontrol_data_report |
| TimeStamp | uint32 | 是 | 消息创建时间，UNIX时间戳 |
| DeviceSn | string | 是 | 门禁通行设备SN |
| AccessDataList | object[] | 是 | 门禁通行原始数据列表 |
| AccessDataList[].Item.OpenUserId | string | 是 | 门禁通行成员openuserid |
| AccessDataList[].Item.UserType | uint32 | 是 | 门禁通行成员类型：0-企业成员；1-访客；2-学生 |
| AccessDataList[].Item.TimeStamp | uint32 | 是 | 门禁通行时间戳 |
| AccessDataList[].Item.PassType | uint32 | 是 | 门禁通行类型：1-进门；2-出门；3-无法识别方向 |
| AccessDataList[].Item.PassMethod | uint32 | 是 | 门禁通行方法：1-人脸；2-二维码；3-远程开门；4-指纹 |

### 请求示例

```xml
<xml>
    <ToUserName><![CDATA[wpwLgYcwAAayDT6bUVnhZuKebVce4xxx]]></ToUserName>
    <FromUserName><![CDATA[sys]]></FromUserName>
    <MsgType><![CDATA[event]]></MsgType>
    <Event><![CDATA[device_accesscontrol_data_report]]></Event>
    <TimeStamp>1673943183</TimeStamp>
    <AccessDataList>
        <Item>
            <OpenUserId><![CDATA[wowLgYcwAAwh5EDz9jMMeGPGL5uhwxxx]]></OpenUserId>
            <UserType>0</UserType>
            <TimeStamp>1673943181</TimeStamp>
            <PassType>1</PassType>
            <PassMethod>1</PassMethod>
        </Item>
    </AccessDataList>
    <DeviceSn><![CDATA[device_sn]]></DeviceSn>
</xml>
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
