# 服务商上传临时素材

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99310](https://developer.work.weixin.qq.com/document/path/99310)
- **文档 ID**: `99310`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/media/upload?provider_access_token=ACCESS_TOKEN&type=TYPE&attachment_type=3`

## 接口描述

素材上传得到media_id，该media_id仅三天内有效。media_id在同一企业内应用之间可以共享。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| provider_access_token | string | 是 | 服务商provider_access_token，获取方法参见服务商的凭证 |
| type | string | 是 | 媒体文件类型，分别有图片（image）、语音（voice）、视频（video）、普通文件（file） |
| attachment_type | string | 否 | 附件类型，不同的附件类型用于不同的场景。普通场景下不指定该参数，特定场景下必填。目前支持的特定场景有：3 - 收银台 |

### 请求示例

```text
POST https://qyapi.weixin.qq.com/cgi-bin/service/media/upload?provider_access_token=accesstoken001&type=file HTTP/1.1
Content-Type: multipart/form-data; boundary=-------------------------acebdf13572468
Content-Length: 220

---------------------------acebdf13572468
Content-Disposition: form-data; name="media";filename="wework.txt"; filelength=6
Content-Type: application/octet-stream

mytext
---------------------------acebdf13572468--
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| type | string | 媒体文件类型，分别有图片（image）、语音（voice）、视频（video）、普通文件（file） |
| media_id | string | 媒体文件上传后获取的唯一标识，3天内有效 |
| created_at | string | 媒体文件上传时间戳 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "",
   "type": "image",
   "media_id": "1G6nrLmr5EC3MMb_-zK1dDdzmd0p7cNliYu9V5w7o8K0",
   "created_at": "1380000000"
}
```

## 其他说明

### 上传的媒体文件限制

所有文件size必须大于5个字节

- 图片（image）：2MB，支持JPG,PNG格式
- 语音（voice） ：2MB，播放长度不超过60s，仅支持AMR格式
- 视频（video） ：10MB，支持MP4格式
- 普通文件（file）：20MB
