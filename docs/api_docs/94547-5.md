# 创建企业互联/上下游会话

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/94547](https://developer.work.weixin.qq.com/document/path/94547)
- **文档 ID**: `94547`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/ww/createCorpGroupChat`

## 接口描述

创建企业互联/上下游会话。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| groupName | string | 否 | 会话名称。创建单聊时可以忽略。 |
| userIds | string[] | 否 | 参与会话的企业成员列表。仅自建应用使用。 |
| openUserIds | string[] | 否 | 参与会话的企业成员列表。仅第三方应用使用。 |
| externalUserIds | string[] | 否 | 参与会话的外部联系人列表。与发起人需要有好友关系。企业微信 3.1.20。 |
| corpGroupUserIds | Object[] | 否 | 参与会话的互联企业成员列表。企业微信 3.1.20。 |
| corpGroupUserIds[].corpId | string | 是 | 企业Corp ID |
| corpGroupUserIds[].userId | string | 否 | 成员 ID。仅自建应用使用。 |
| corpGroupUserIds[].openUserId | string | 否 | 成员OpenUser ID。仅第三方应用使用。 |

### 请求示例

```text
ww.createCorpGroupChat({
  groupName: '讨论组',
  userIds: ['lisi', 'lisi2'],
  openUserIds: ['wabc3', 'wbcde'],
  externalUserIds: ['exid1', 'exid2'],
  corpGroupUserIds: [
    {
      corpId: 'ww3333',
      userId: 'userid123',
      openUserId: 'wx1111'
    },
    {
      corpId: 'ww4444',
      userId: 'userid123',
      openUserId: 'wx1111'
    }
  ]
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errMsg | string | 通用错误信息 |
| errCode | number | 通用错误码 |
| chatId | string | 创建的群聊 ID |

## 其他说明

### 功能描述

创建企业互联/上下游会话。

### 使用限制

• 本接口必须使用应用身份进行注册
• 企业必须开启互联群功能
• 仅局校互联和上下游企业可调用
• 当前成员必须在应用的可见范围
• 群成员人数不能超过 2000 人
• 如果创建的会话有外部联系人，群成员人数不能超过 40 人
• 当前成员为下游企业成员时，需要打开上下游空间中的“允许外部单位之间互相查看”配置，群成员中才可以包含其他下游企业成员

### 错误信息

| errMsg | 说明 |
| --- | --- |
| createCorpGroupChat:ok | 执行成功 |
| createCorpGroupChat:fail no permission | 应用签名校验失败 |
| createCorpGroupChat:fail exceed user id list size | 超过人数上限 |
| createCorpGroupChat:fail invalid parameter | 参数不合法 |
| createCorpGroupChat:fail need open corp group chat | 企业未开启企业互联群功能 |
| createCorpGroupChat:fail exceed external user id list size | 超过包含外部联系人群人数上限 |
