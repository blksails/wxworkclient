# 专区使用流程

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99864](https://developer.work.weixin.qq.com/document/path/99864)
- **文档 ID**: `99864`
- **API 名称**: `area_process`
- **分组信息**: 第 2 个接口，共 3 个

## 接口描述

企业微信提供了一条专区调用通道，以让应用的后台可以调用部署在专区中的程序，也可以让专区中的程序回调通知应用的后台。
企业微信提供了会话内容展示组件，应用的前端传入消息ID与密钥后，可渲染展示出消息内容，服务商不可直接获得消息明文内容。
企业微信在专区中以SDK的方式提供了一套涉及会话记录、会话搜索、关键词命中记录、模型分析等接口，专区中的程序可以按需调用.
