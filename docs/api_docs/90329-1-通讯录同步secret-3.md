## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90329](https://developer.work.weixin.qq.com/document/path/90329)
- **文档 ID**: `90329`
- **API 名称**: `通讯录同步secret`
- **请求方法**: `GET`
- **接口地址**: `https://developer.work.weixin.qq.com/resource/devtool`
- **分组信息**: 第 1 个接口，共 4 个

## 接口描述

企业如果需要从自有的系统同步通讯录到企业微信，需要使用通讯录同步secret。如果企业委托授权第三方服务商将通讯录从其他系统同步到企业微信，则需要企业授权“通讯录编辑授权”给服务商。
