# 禁用/启用邮箱账号 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95512](https://developer.work.weixin.qq.com/document/path/95512)
- **文档 ID**: `95512`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/exmail/account/act_email?access_token=ACCESS_TOKEN`

## 接口描述

禁用或启用邮箱，可以操作成员邮箱和业务邮箱

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| userid |  | 是 | 成员UserID,userid与publicemail_id至少应该传一项，同时传则只操作userid。不可禁用超管与企业创建人 |
| publicemail_id |  | 是 | 业务邮箱ID,userid与publicemail_id至少应该传一项，同时传则只操作userid |
| type |  | 是 | 1启用，2禁用 |

### 请求示例

```json
{"userid":"member","publicemail_id":1,"type":1}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok"}
```
