# 获取部门家长详情

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92627](https://developer.work.weixin.qq.com/document/path/92627)
- **文档 ID**: `92627`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/user/list_parent`

## 接口描述

获取部门家长详情，包括家长的信息、孩子列表等。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| department_id | string | 是 | 获取的部门id |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| parents | object[] | 家长列表 |
| parents[].parent_userid | string | 家长的userid |
| parents[].mobile | string | 家长手机号，第三方不可获取;代开发应用需要管理员授权该权限才返回 |
| parents[].is_subscribe | int32 | 家长是否关注了“学校通知”，0-未关注，1-已关注 |
| parents[].external_userid | string | 家长的external_userid,仅当家长已关注才返回。对同一个服务商来说，同一个家长微信在不同学校下返回的家长的external_userid是一样的 |
| parents[].children | object[] | 家长孩子列表 |
| parents[].children[].student_userid | string | 学生的userid |
| parents[].children[].relation | string | 家长与学生关系 |
| parents[].children[].name | string | 学生姓名 |

### 权限要求

调用的应用需要满足如下的权限：
- 自建应用：配置到「家校沟通- 读取和编辑家校通讯录的应用」或「家校沟通- 读取家校通讯录的应用」或“家长可使用应用”列表中
- 代开发应用：具有 「家校沟通」权限
- 第三方应用：具有 「家校沟通」使用权限

*注：* 从2023年12月1日0点起，不再支持通过系统应用secret调用接口，存量企业暂不受影响 [查看详情](https://developer.work.weixin.qq.com/document/51165)
