# 批量创建家长 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/100154](https://developer.work.weixin.qq.com/document/path/100154)
- **文档 ID**: `100154`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/school/user/batch_create_parent?access_token=ACCESS_TOKEN`

## 接口描述

请求方式：POST（HTTPS）请求地址：https://qyapi.weixin.qq.com/cgi-bin/school/user/batch_create_parent?access_token=ACCESS_TOKEN

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证。获取方法查看“获取access_token” |
| parents |  | 是 | 家长列表，每次最多100个 |
| 自建应用 | 自建应用 | 否 | 配置到「家校沟通-读取和编辑家校通讯录的应用」中 |
| 代开发应用 | 代开发应用 | 否 | 需拥有「家校沟通」可使用家校通讯录-家校通讯录编辑权限 |
| 第三方应用 | 第三方应用 | 否 | 需拥有「家校沟通」使用和编辑权限 |

### 请求示例

```json
{"parents":[{"parent_userid":"zhangsan_parent_userid","mobile":"18000000000","to_invite":false,"children":[{"student_userid":"zhangsan","relation":"爸爸"},{"student_userid":"lisi","relation":"伯父"}]},{"parent_userid":"lisi_parent_userid","mobile":"18000000001","children":[{"student_userid":"lisi","relation":"爸爸"},{"student_userid":"zhangsan","relation":"伯父"}]}]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| parents.parent_userid |  | 家长UserID。学校内必须唯一。不区分大小写，长度为1~64个字节。只能由数字、字母和“_-@.”四种字符组成，且第一个字符必须是数字或字母。 |
| parents.mobile |  | 家长手机号 |
| parents.to_invite |  | 是否发起邀请，默认为true |
| parents.children |  | 家长的孩子列表，最多10个 |
| parents.children.student_userid |  | 学生的UserID |
| parents.children.relation |  | 家长与学生的关系，最长32字节 |
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| result_list |  | 失败列表 |
| result_list.parent_userid |  | 失败的家长userid |
| result_list.errcode |  | 返回码 |
| result_list.errmsg |  | 对返回码的文本描述内容 |

### 响应示例

```json
{"errcode":1,"errmsg":"invalid parent_userid: lisi_parent_userid","result_list":[{"parent_userid":"lisi_parent_userid","errcode":1,"errmsg":"invalid parent_userid: lisi_parent_userid"}]}
```
