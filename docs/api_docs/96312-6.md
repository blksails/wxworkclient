# 检查用户是否配置了客户联系功能使用权限

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96312](https://developer.work.weixin.qq.com/document/path/96312)
- **文档 ID**: `96312`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/externalcontact/check_follow_user`

## 接口描述

企业和第三方服务商可通过此接口检查部门或成员是否具备客户联系权限。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| userid | array | 否 | 用户的userid |
| partyid | array | 否 | 部门id |

### 请求示例

```json
{
   "userid":["zhangsan","lissi"],
   "partyid":[1]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| authorized | object | 有客户联系使用权限的用户和部门 |
| authorized.userid | array | 有客户联系使用权限的用户id |
| authorized.partyid | array | 有客户联系使用权限的部门id |
| unauthorized | object | 无客户联系使用权限的用户和部门 |
| unauthorized.userid | array | 无客户联系使用权限的用户id |
| unauthorized.partyid | array | 无客户联系使用权限的部门id |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "ok",
   "authorized":
   {
		"userid":["lisi"]
   },
   "unauthorized":
   {
		"userid":["zhangsan"],
		"partyid":[1]
   }
}
```

## 其他说明

### 权限说明

企业需要使用配置到“可调用应用”列表中的自建应用secret所获取的accesstoken来调用。第三方应用需具有“企业客户权限->客户基础信息”权限。
