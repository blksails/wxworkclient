# 移动文件 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97918](https://developer.work.weixin.qq.com/document/path/97918)
- **文档 ID**: `97918`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/wedrive/file_move?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于将文件移动到指定位置。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fatherid | string | 是 | 当前目录的fileid,根目录时为空间spaceid |
| replace | bool | 否 | 如果移动到的目标目录与需要移动的文件重名时，是否覆盖。true:重名文件覆盖 false:重名文件进行冲突重命名处理（移动后文件名格式如xxx(1).txt xxx(1).doc等） |
| fileid | string | 是 | 文件fileid |

### 请求示例

```json
{"fatherid":"FATHERID","replace":true,"fileid":["FILEID1","FILEID2"]}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| file_list | obj[] | 移动文件的信息列表 |
| fileid | string | 文件fileid |
| file_name | string | 文件名字 |
| spaceid | string | 文件所在的空间spaceid |
| fatherid | string | 文件所在的目录fileid, 在根目录时为fileid |
| file_size | uint64 | 文件大小 |
| ctime | uint64 | 文件创建时间 |
| mtime | uint64 | 文件最后修改时间 |
| file_type | uint32 | 文件类型, 1:文件夹 2:文件 3:文档(文档) 4:文档(表格) 5:文档(收集表) 6:文档(幻灯片) |
| file_status | uint32 | 文件状态, 1:正常 2:删除 |
| sha | string | 文件sha |
| md5 | string | 文件md5 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","file_list":{"item":[{"fileid":"FILEID","file_name":"FILE_NAME","spaceid":"SPACEID","fatherid":"FATHERID","file_size":11,"ctime":17777777777,"mtime":2777777777777,"file_type":3,"file_status":2,"sha":"SHA","md5":"MD5"}]}}
```
