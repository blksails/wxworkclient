# NFC接口

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/92412](https://developer.work.weixin.qq.com/document/path/92412)
- **文档 ID**: `92412`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

监听NFC设备的消息回调。当用户离开小程序后，此接口无效。

## 请求信息

### 请求示例

```text
Page({
  onNFCReadMessage: function (e) {
    console.log(JSON.stringify(e))
    if (1 == e.messageType) {
      wx.showModal({
        title: 'NFC-Id',
        content: JSON.stringify(e),
      })
    } else if (2 == e.messageType) {
      console.log('Auto-Stop')
      wx.showModal({
        title: 'NFC-Auto-Stop',
        content: JSON.stringify(e),
      })
    }
  }
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| messageType | uint32 | 消息类型 |
| data.hex | string | 十六进制 id 序列 |
| data.reversed_hex | string | 字节反转十六进制 id 序列 |
| data.dec | uint32 | 十进制 id 数值 |
| data.reversed_dec | uint32 | 字节反转十进制 id 数值 |
