# 更新成员

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90333](https://developer.work.weixin.qq.com/document/path/90333)
- **文档 ID**: `90333`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/user/update`

## 接口描述

更新企业微信通讯录中的成员信息。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| userid | string | 是 | 成员UserID。对应管理端的账号，企业内必须唯一。不区分大小写，长度为1~64个字节 |
| name | string | 否 | 成员名称。长度为1~64个utf8字符 |
| alias | string | 否 | 别名。长度为1-64个utf8字符 |
| mobile | string | 否 | 手机号码。企业内必须唯一。若成员已激活企业微信，则需成员自行修改 |
| department | array | 否 | 成员所属部门id列表，不超过100个 |
| order | array | 否 | 部门内的排序值，默认为0。当有传入department时有效。数量必须和department一致，数值越大排序越前面。有效的值范围是[0, 2^32) |
| position | string | 否 | 职务信息。长度为0~128个utf8字符 |
| gender | string | 否 | 性别。1表示男性，2表示女性 |
| email | string | 否 | 邮箱。长度6~64个字节，且为有效的email格式。企业内必须唯一 |
| biz_mail | string | 否 | 企业邮箱账号。长度6~63个字节，且为有效的企业邮箱格式。企业内必须唯一 |
| biz_mail_alias.item | array | 否 | 企业邮箱别名。长度6~63个字节，且为有效的企业邮箱格式。企业内必须唯一，最多可设置5个别名 |
| is_leader_in_dept | array | 否 | 部门负责人字段，个数必须和department一致，表示在所在的部门内是否为负责人。0-否，1-是 |
| direct_leader | array | 否 | 直属上级，可以设置企业范围内成员为直属上级，最多设置1个 |
| avatar_mediaid | string | 否 | 成员头像的mediaid，通过素材管理接口上传图片获得的mediaid |
| telephone | string | 否 | 座机。由1-32位的纯数字、“-”、“+”或“,”组成 |
| enable | int32 | 否 | 启用/禁用成员。1表示启用成员，0表示禁用成员 |
| extattr.attrs.type | int32 | 否 | 扩展属性类型。0表示文本，1表示网页 |
| extattr.attrs.name | string | 否 | 扩展属性名称 |
| extattr.attrs.text.value | string | 否 | 文本值 |
| extattr.attrs.web.url | string | 否 | 网页链接 |
| extattr.attrs.web.title | string | 否 | 网页标题 |
| external_position | string | 否 | 对外职务，如果设置了该值，则以此作为对外展示的职务，否则以position来展示。不超过12个汉字 |
| external_profile.external_corp_name | string | 否 | 企业简称 |
| external_profile.wechat_channels.nickname | string | 否 | 视频号名称 |
| external_profile.external_attr.type | int32 | 否 | 对外属性类型。0表示文本，1表示网页，2表示小程序 |
| external_profile.external_attr.name | string | 否 | 对外属性名称 |
| external_profile.external_attr.text.value | string | 否 | 文本值 |
| external_profile.external_attr.web.url | string | 否 | 网页链接 |
| external_profile.external_attr.web.title | string | 否 | 网页标题 |
| external_profile.external_attr.miniprogram.appid | string | 否 | 小程序appid |
| external_profile.external_attr.miniprogram.pagepath | string | 否 | 小程序页面路径 |
| external_profile.external_attr.miniprogram.title | string | 否 | 小程序标题 |
| address | string | 否 | 地址。长度最大128个字符 |
| main_department | int32 | 否 | 主部门 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |

### 响应示例

```json
{
   "errcode": 0,
   "errmsg": "updated"
}
```
