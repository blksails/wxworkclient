# 异步任务完成通知 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95797](https://developer.work.weixin.qq.com/document/path/95797)
- **文档 ID**: `95797`

## 接口描述

本事件是在使用异步导入上下游联系人接口时，用于接收任务执行完毕的结果通知。回调推送地址为调用接口应用配置的回调地址。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| ToUserName |  | 否 | 企业微信CorpID |
| FromUserName |  | 否 | 此时固定为sys |
| CreateTime |  | 否 | 消息创建时间（整型） |
| MsgType |  | 否 | 消息类型，此时固定为：event |
| Event |  | 否 | 事件类型：batch_job_result |
| JobId |  | 否 | 异步任务id，最大长度为64字符 |
| JobType |  | 否 | 操作类型，字符串，固定为：import_chain_contact(导入上下游联系人) |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| ErrCode |  | 返回码 |
| ErrMsg |  | 对返回码的文本描述内容 |
