# 获取企业微信系统信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95047](https://developer.work.weixin.qq.com/document/path/95047)
- **文档 ID**: `95047`
- **请求方法**: `GET`
- **接口地址**: `wx.qy.getSystemInfo`

## 接口描述

小程序可以企业微信中调用此接口，获得企业微信终端的版本号

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| Object object | object | 是 | 参数对象 |
| Object object.success | function | 否 | 接口调用成功的回调函数 |
| Object object.fail | function | 否 | 接口调用失败的回调函数 |
| Object object.complete | function | 否 | 接口调用结束的回调函数（调用成功、失败都会执行） |

### 请求示例

```text
wx.qy.getSystemInfo({
  success (res) {
    console.log(res.version)
  }
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| Object res | object | 返回结果对象 |
| Object res.version | string | 企业微信微信版本号 |
