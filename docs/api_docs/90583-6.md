# 设置授权应用可见范围

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90583](https://developer.work.weixin.qq.com/document/path/90583)
- **文档 ID**: `90583`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/agent/set_scope`

## 接口描述

调用该接口前提是开启通讯录迁移，收到授权成功通知后可调用。企业注册初始化安装应用后，应用默认可见范围为根部门。如需修改应用可见范围，服务商可以调用该接口设置授权应用的可见范围。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 查询注册状态接口返回的access_token（跟注册完成回调事件的AccessToken参数一致，请注意与provider_access_token的区别） |

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| agentid | int32 | 是 | 授权方应用id |
| allow_user | array | 否 | 应用可见范围（成员）若未填该字段，则清空可见范围中成员列表 |
| allow_party | array | 否 | 应用可见范围（部门）若未填该字段，则清空可见范围中部门列表 |
| allow_tag | array | 否 | 应用可见范围（标签）若未填该字段，则清空可见范围中标签列表 |

### 请求示例

```json
{
    "agentid":1 ,
    "allow_user":["zhansan","lisi"],
    "allow_party":[1,2,3],
    "allow_tag":[1,2,3]
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码对应的描述 |
| invaliduser | array | 非法成员列表 |
| invalidparty | array | 非法部门列表 |
| invalidtag | array | 非法标签列表 |

### 响应示例

```json
{
    "errcode":0,
    "errmsg":"ok",
    "invaliduser":["zhangshan","lisi"],
    "invalidparty":[2,3],
    "invalidtag":[2,3]
}
```
