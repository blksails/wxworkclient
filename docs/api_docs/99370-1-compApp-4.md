# 微信客服组件授权流程

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99370](https://developer.work.weixin.qq.com/document/path/99370)
- **文档 ID**: `99370`
- **API 名称**: `compApp`
- **请求方法**: `GET`
- **接口地址**: `https://open.work.weixin.qq.com/compApp/wwxxxxxxxx`
- **分组信息**: 第 1 个接口，共 2 个

## 接口描述

开发者可以在服务商助手的「工具-通用配置-微信客服组件」获取授权链接或二维码，企业可使用该授权链接或二维码接入微信客服组件。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| state | string | 否 | 企业授权接入客服组件时的回调参数 |
| mode | string | 否 | 授权模式，1表示每次仅可选择一个客服账号进行授权 |

### 请求示例

```text
https://open.work.weixin.qq.com/compApp/wwxxxxxxxx?state=xxxxxx
https://open.work.weixin.qq.com/compApp/wwxxxxxxxx?mode=1
```
