# 模糊搜索公共邮箱 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/98003](https://developer.work.weixin.qq.com/document/path/98003)
- **文档 ID**: `98003`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/exmail/publicmail/search?access_token=ACCESS_TOKEN`

## 接口描述

该接口用于模糊搜索公共邮箱，可匹配邮箱名也可匹配邮箱地址。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| fuzzy | uint32 | 是 | 1开启模糊搜索，0获取全部公共邮箱 |
| email | string | 否 | 公共邮箱名称或邮箱地址 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误码说明 |
| id | uint32 | 公共邮箱ID |
| email | string | 公共邮箱地址 |
| name | string | 公共邮箱名称 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","list":[{"id":11,"email":"apitest3@gzdev.com","name":"apitest3"},{"id":12,"email":"apitest4@gzdev.com","name":"apitest4"}]}
```
