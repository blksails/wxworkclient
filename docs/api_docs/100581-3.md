## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
|  | number | 是 | 纬度范围为 90 ~ -90 |
|  | number | 是 | 经度范围为 180 ~ -180 |
|  | number | 是 | 速度单位为米/每秒 |
|  | number | 是 | 位置精度 |
| auto:location:report:ok |  | 否 | 执行成功 |

### 请求示例

#### 示例 1: 功能描述

```json
ww.onLocationChange(function(event){console.log(event)})
```

#### 示例 2: 示例代码

```json
wx.onLocationChange(function(res){if(res.errMsg=="auto:location:report:ok"){varlatitude=res.latitude;// 纬度，浮点数，范围为90 ~ -90varlongitude=res.longitude;// 经度，浮点数，范围为180 ~ -180。varspeed=res.speed;// 速度，以米/每秒计varaccuracy=res.accuracy;// 位置精度varlbsIndex=0;}else{//错误处理}});
```

## 其他说明

### ww.onLocationChange(callback)

兼容性：企业微信 2.4.20 开始支持
功能描述监听地理位置的变化。
ww.onLocationChange(function(event){console.log(event)})
使用限制需要提前调用 startAutoLBS需要用户停留在当前页面
参数说明callback: Function参数event: Object属性类型必填说明latitudenumber是纬度范围为 90 ~ -90longitudenumber是经度范围为 180 ~ -180speednumber是速度单位为米/每秒accuracynumber是位置精度
错误信息errMsg说明兼容性auto:location:report:ok执行成功auto:location:report:fail, gps closed.用户关闭了 GPS企业微信 3.0.26

### 功能描述

监听地理位置的变化。

### 使用限制

需要提前调用 startAutoLBS需要用户停留在当前页面

### callback: Function

参数
event: Object
属性类型必填说明latitudenumber是纬度范围为 90 ~ -90longitudenumber是经度范围为 180 ~ -180speednumber是速度单位为米/每秒accuracynumber是位置精度

### 错误信息

errMsg说明兼容性auto:location:report:ok执行成功auto:location:report:fail, gps closed.用户关闭了 GPS企业微信 3.0.26
