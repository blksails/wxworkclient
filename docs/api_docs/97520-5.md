# 保持屏幕常亮

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97520](https://developer.work.weixin.qq.com/document/path/97520)
- **文档 ID**: `97520`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

保持屏幕常亮。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| params.keepScreenOn | boolean | 是 | 是否保持常亮 |
| params.success | Function | 否 | 成功回调 |
| params.fail | Function | 否 | 失败回调 |
| params.cancel | Function | 否 | 取消回调 |
| params.complete | Function | 否 | 完成回调 |

### 请求示例

```text
ww.setKeepScreenOn({
 keepScreenOn: true,
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errMsg | string | 通用错误信息 |
| errCode | number | 通用错误码 |

## 其他说明

### 功能描述

保持屏幕常亮。
在企业微信内打开 H5 页面时，调用该接口让屏幕保持常亮。

### 使用限制

* 本接口必须使用应用身份进行注册
* 成员必须在应用可见范围内

### 参数说明

params: Object

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | keepScreenOn | boolean | 是 | 是否保持常亮 |
|  | success | Function | 否 | 成功回调 |
|  | fail | Function | 否 | 失败回调 |
|  | cancel | Function | 否 | 取消回调 |
|  | complete | Function | 否 | 完成回调 |

### 返回说明

**Promise<Object>**

|  | 属性 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- | --- |
|  | errMsg | string | 是 | 通用错误信息 |
|  | errCode | number | 是 | 通用错误码 |

### 使用说明

仅在当前页面生效，离开页面后设置失效。

### 旧版jweixin调用

### 示例代码

 wx.invoke('setKeepScreenOn',
 	{
		"keepScreenOn": true,
	},
 	function(res) {
 });
