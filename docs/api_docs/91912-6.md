# 获取企业授权信息

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/91912](https://developer.work.weixin.qq.com/document/path/91912)
- **文档 ID**: `91912`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/service/get_auth_info?suite_access_token=SUITE_ACCESS_TOKEN`

## 接口描述

通过永久授权码换取企业微信的授权信息。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| auth_corpid | string | 是 | 授权方corpid |
| permanent_code | string | 是 | 永久授权码，通过get_permanent_code获取 |

### 请求示例

```json
{
	"auth_corpid": "auth_corpid_value",
	"permanent_code": "code_value"
}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| edition_info | object | 企业当前生效的版本信息 |
| edition_info.agent | object[] | 授权的应用信息 |
| edition_info.agent[].agentid | int32 | 应用ID |
| edition_info.agent[].edition_id | string | 版本ID，可在服务商平台查看已配置的付费版本信息 |
| edition_info.agent[].edition_name | string | 版本名称 |
| edition_info.agent[].app_status | int32 | 付费状态 |
| edition_info.agent[].user_limit | int32 | 用户上限 |
| edition_info.agent[].expired_time | int32 | 版本到期时间 |
| edition_info.agent[].is_virtual_version | bool | 是否虚拟版本 |
| edition_info.agent[].is_shared_from_other_corp | bool | 是否由互联企业分享安装 |

### 响应示例

```json
{
	"errcode": 0,
	"errmsg": "ok",
	"edition_info": {
		"agent": [
			{
				"agentid": 1,
				"edition_id": "RLS65535",
				"edition_name": "协同版",
				"app_status": 3,
				"user_limit": 200,
				"expired_time": 1541990791,
				"is_virtual_version": false,
				"is_shared_from_other_corp": true
			}
		]
	}
}
```
