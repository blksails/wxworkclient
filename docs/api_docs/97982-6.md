# 获取邮件内容 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/97982](https://developer.work.weixin.qq.com/document/path/97982)
- **文档 ID**: `97982`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/exmail/app/read_mail?access_token=ACCESS_TOKEN`

## 接口描述

支持应用获取邮件内容。指定单个邮件id，获取邮件eml数据。

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| mail_id | string | 是 | 邮件id |

### 请求示例

```json
{"mail_id":"MAIL_ID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 返回码 |
| errmsg | string | 对返回码的文本描述内容 |
| mail_data | string | 邮件eml内容 |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","mail_data":"MAIL_DATA"}
```
