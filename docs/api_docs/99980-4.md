目录

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 自建应用 | 自建应用 | 否 | 需具备「数据与智能专区权限」 |
| 代开发应用 | 代开发应用 | 否 | 需具备「数据与智能专区权限」 |
| 第三方应用 | 第三方应用 | 否 | 需具备「数据与智能专区权限」 |
| msg_list |  | 是 | 消息列表，每次最多1000个。总和不超过1000个。 |
| jobid |  | 是 | 任务id |

### 请求示例

#### 示例 1: 请求包体：

```json
{"msg_list":[{"msgid":"MSGID1","encrypt_info":{"secret_key":"SECRETKEY1"}},{"msgid":"MSGID2","encrypt_info":{"secret_key":"SECRETKEY2"}}]}
```

#### 示例 2: 请求包体：

```json
{"jobid":"JOBID"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| msg_list.msgid |  | 每条消息对应的msgid。多次出现同一个msgid，以首次出现的为准。目前支持文本、语音、音频存档消息 |
| msg_list.encrypt_info.secret_key |  | 该消息的密钥，将encrypted_secret_key用RSA私钥解密后得到 |
| errcode |  | 错误码 |
| errmsg |  | 错误码说明 |
| jobid |  | 任务id。首次提交时返回 |
| fail_list |  | 提交出错的消息列表，只有msgid重复项返回至该列表 |
| fail_list.errcode |  | 错误码 |
| fail_list.errmsg |  | 错误码说明 |
| fail_list.msgid |  | 每条消息对应的msgid，与入参对应 |
| fail_list.msgid.encrypt_info |  | 每条消息对应的加密信息，与入参对应 |
| errcode |  | 错误码 |
| errmsg |  | 错误码说明 |
| status |  | 任务执行状态0: 执行未完成1: 执行完成2: 执行失败 |
| fail_list |  | 错误的消息列表，如msgid不存在、非文本等，详见“创建标签匹配任务”的FailMsg说明 |
| response_data |  | 摘要提取的结果 |

### 响应示例

#### 示例 1: 返回结果：

```json
{"errcode":0,"errmsg":"ok","jobid":"JOBID","fail_list":[{"errcode":710601,"errmsg":"xxx","msgid":"MSGID2","encrypt_info":{"secret_key":"SECRETKEY2"}}]}
```

#### 示例 2: 返回结果：

```json
{"errcode":0,"errmsg":"ok","status":1,"fail_list":[{"errcode":710601,"errmsg":"xxx","msgid":"MSGID2"}],"response_data":"xxx"}
```

## 其他说明

### 创建摘要提取任务

请求方法通过SDK调用，具体方式参考专区程序使用指引。
请求接口名：create_summary_task
请求包体：
{"msg_list":[{"msgid":"MSGID1","encrypt_info":{"secret_key":"SECRETKEY1"}},{"msgid":"MSGID2","encrypt_info":{"secret_key":"SECRETKEY2"}}]}
参数说明：
参数是否必须说明msg_list是消息列表，每次最多1000个。总和不超过1000个。msg_list.msgid是每条消息对应的msgid。多次出现同一个msgid，以首次出现的为准。目前支持文本、语音、音频存档消息msg_list.encrypt_info.secret_key是该消息的密钥，将encrypted_secret_key用RSA私钥解密后得到
返回结果：
{"errcode":0,"errmsg":"ok","jobid":"JOBID","fail_list":[{"errcode":710601,"errmsg":"xxx","msgid":"MSGID2","encrypt_info":{"secret_key":"SECRETKEY2"}}]}
参数说明：
参数说明errcode错误码errmsg错误码说明jobid任务id。首次提交时返回fail_list提交出错的消息列表，只有msgid重复项返回至该列表fail_list.errcode错误码fail_list.errmsg错误码说明fail_list.msgid每条消息对应的msgid，与入参对应fail_list.msgid.encrypt_info每条消息对应的加密信息，与入参对应

### 获取摘要提取结果

请求方法通过SDK调用，具体方式参考专区程序使用指引。
请求接口名：get_summary_result
请求包体：
{"jobid":"JOBID"}
参数说明：
参数是否必须说明jobid是任务id
返回结果：
{"errcode":0,"errmsg":"ok","status":1,"fail_list":[{"errcode":710601,"errmsg":"xxx","msgid":"MSGID2"}],"response_data":"xxx"}
参数说明：
参数说明errcode错误码errmsg错误码说明status任务执行状态0: 执行未完成1: 执行完成2: 执行失败fail_list错误的消息列表，如msgid不存在、非文本等，详见“创建标签匹配任务”的FailMsg说明response_data摘要提取的结果

### 频率限制说明

每企业每天会话摘要的次数不超过1万次（按调用次数限制）若资源紧张，任务重试超过24小时仍未有足够资源执行，任务将失败
