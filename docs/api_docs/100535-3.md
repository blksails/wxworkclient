## 其他说明

### ww.uploadVoice(params)

功能描述上传语音。
ww.uploadVoice({localId:localId,isShowProgressTips:true})
参数说明params: Object属性类型默认值必填说明localIdstring是需要上传的音频的本地 ID通过 stopRecord 接口获得isShowProgressTipsbooleantrue否是否显示进度提示successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码serverIdstring是音频的服务器端 ID
使用说明上传语音有效期 3 天，可以通过素材管理接口下载语音到自己的服务器，接口返回的的serverId即media_id。

### 功能描述

上传语音。

### params: Object

属性类型默认值必填说明localIdstring是需要上传的音频的本地 ID通过 stopRecord 接口获得isShowProgressTipsbooleantrue否是否显示进度提示successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码serverIdstring是音频的服务器端 ID

### 使用说明

上传语音有效期 3 天，可以通过素材管理接口下载语音到自己的服务器，接口返回的的serverId即media_id。

### 示例代码
