# 变更群成员

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/93232](https://developer.work.weixin.qq.com/document/path/93232)
- **文档 ID**: `93232`
- **请求方法**: `N/A`
- **接口地址**: `N/A`

## 接口描述

兼容性：企业微信 iOS、Android、PC 3.0.36 开始支持。变更群成员。

## 请求信息

### Body 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| params.chatId | string | 是 | 通过企业微信创建群聊接口返回的 chatId |
| params.userIdsToAdd | string[] | 是 | 参与会话的企业成员列表 |
| params.success | Function | 否 | 成功回调 |
| params.fail | Function | 否 | 失败回调 |
| params.cancel | Function | 否 | 取消回调 |
| params.complete | Function | 否 | 完成回调 |

### 请求示例

```text
ww.updateEnterpriseChat({
  chatId: 'CHATID',
  userIdsToAdd: [
    'zhangsan',
    'lisi'
  ]
})
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errMsg | string | 通用错误信息 |
| errCode | number | 通用错误码 |

## 其他说明

### 使用限制

本接口必顿使用应用身份进行注册。目前仅支持添加企业内部成员。仅支持客户群调用。
