## 其他说明

### ww.checkJsApi(params)

功能描述判断当前客户端版本是否支持指定 JS 接口。
ww.checkJsApi({jsApiList:['chooseImage']})
参数说明params: Object属性类型必填说明jsApiListstring[]是JS 接口列表successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调
返回说明Promise<Object>属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码checkResultObject是检查结果以键值对的形式返回，接口可用时为 true，不可用为 false

### 功能描述

判断当前客户端版本是否支持指定 JS 接口。

### params: Object

属性类型必填说明jsApiListstring[]是JS 接口列表successFunction否成功回调failFunction否失败回调cancelFunction否取消回调completeFunction否完成回调

### 返回说明

Promise<Object>
属性类型必填说明errMsgstring是通用错误信息errCodenumber是通用错误码checkResultObject是检查结果以键值对的形式返回，接口可用时为 true，不可用为 false
