## 接口描述

企业微信JS-SDK提供了 npm 和 cdn 两种引入途径。

## 请求信息

### 请求示例

#### 示例 1: 通过 npm 引入

```text
npm install @wecom/jssdk

import * as ww from '@wecom/jssdk'
```

#### 示例 2: 通过 script 标签引入

```text
<script src="https://wwcdn.weixin.qq.com/node/open/js/wecom-jssdk-2.3.3.js"></script>
```
