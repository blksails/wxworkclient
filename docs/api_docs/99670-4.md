# 获取会话记录

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/99670](https://developer.work.weixin.qq.com/document/path/99670)
- **文档 ID**: `99670`
- **请求方法**: `GET`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/chat/get_chat_record`

## 接口描述

获取企业会话内容存档的记录。

## 请求信息

### Query 参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token | string | 是 | 调用接口凭证 |
| chatid | string | 是 | 会话id |
| seq | uint32 | 否 | 起始消息id。可以传入msgid获取大于该msgid的消息 |
| limit | uint32 | 否 | 返回消息条数，最多10000条 |

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode | int32 | 错误码 |
| errmsg | string | 错误信息 |
| msgid | uint32 | 消息id |
| msgtime | uint32 | 消息发送时间 |
| msgtype | uint32 | 消息类型 |
| content | object | 消息内容 |
