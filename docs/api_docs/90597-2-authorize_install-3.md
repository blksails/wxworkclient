# 从企业微信应用市场发起

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/90597](https://developer.work.weixin.qq.com/document/path/90597)
- **文档 ID**: `90597`
- **API 名称**: `authorize_install`
- **请求方法**: `GET`
- **接口地址**: `https://open.work.weixin.qq.com/3rdapp/authorize_install`
- **分组信息**: 第 2 个接口，共 2 个

## 接口描述

企业可以在应用市场找到想要使用的应用并授权安装。

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| auth_code | string | 临时授权码 |
