# 接口调整说明 2019/05/23

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/96307](https://developer.work.weixin.qq.com/document/path/96307)
- **文档 ID**: `96307`
- **分组信息**: 第 8 个接口，共 9 个

## 接口描述

为使API的请求URL表述与企业微信逐步丰富的外部联系人管理能力相符合，现对部分API的URL命名进行调整，请开发者使用新的URL进行调用，原URL仍可正常使用。
相关变更如下:
- 获取客户列表: /cgi-bin/externalcontact/list
- 获取配置了客户联系功能的成员列表: /cgi-bin/externalcontact/get_follow_user_list
- 获取客户详情: /cgi-bin/externalcontact/get
- 配置客户联系「联系我」方式: /cgi-bin/externalcontact/*_contact_way
- 添加企业群发消息模板: /cgi-bin/externalcontact/add_msg_template
- 获取企业群发消息发送结果: /cgi-bin/externalcontact/get_group_msg_result
- 获取员工行为数据: /cgi-bin/externalcontact/get_user_behavior_data
- 获取离职成员的客户列表: /cgi-bin/externalcontact/get_unassigned_list
- 离职成员的外部联系人再分配: /cgi-bin/externalcontact/transfer
