# 获取对接规则id列表 - 文档 - 企业微信开发者中心

## 基本信息

- **文档地址**: [https://developer.work.weixin.qq.com/document/path/95631](https://developer.work.weixin.qq.com/document/path/95631)
- **文档 ID**: `95631`
- **请求方法**: `POST`
- **接口地址**: `https://qyapi.weixin.qq.com/cgi-bin/corpgroup/rule/list_ids?access_token=ACCESS_TOKEN`

## 接口描述

上下游系统应用可通过该接口获取企业上下游规则id列表

## 请求信息

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| access_token |  | 是 | 调用接口凭证 |
| chain_id |  | 是 | 上下游id |
| 自建应用 | 自建应用 | 否 | 配置到「上下游- 可调用接口的应用」中 |

### 请求示例

```json
{"chain_id":"Chxxxxxx"}
```

## 响应信息

### 响应参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| errcode |  | 返回码 |
| errmsg |  | 对返回码的文本描述内容 |
| rule_ids |  | 上下游关系规则的id |

### 响应示例

```json
{"errcode":0,"errmsg":"ok","rule_ids":[1,2]}
```
